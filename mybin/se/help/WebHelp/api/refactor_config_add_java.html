int refactor_config_add_java(_str configName, _str sourcePath, _str jarPath)

Add a new Java compiler configuration to the master refactoring configuration file.

Parameters:

configName - name for this configuration

sourcePath - path to root of the JDK

jarPath - PATHSEP deliminated system JAR list

Returns: int

0 on success, <0 on error

Refactoring Functions
