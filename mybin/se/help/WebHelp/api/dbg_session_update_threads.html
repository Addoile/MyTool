int dbg_session_update_threads(int session_id)

Update the list of threads running in this debugging session.

A thread has a thread name as well as a thread state.

Fundamentally, the responsibility of this method is to merely identify the active threads and their system thread IDs. Optionally, it may gather the thread names and state information. If it does not, that work is deferred until updateThreadNames() or updateThreadStates() is called.

Parameters:

session_id - debugger session ID (1 .. max_session)

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

   int status = dbg_session_update_threads(session_id);
   if (status) {
      return status;
   }

Category:

Debugger Functions
