int tag_clear_context(_str file_name=null, boolean preseveTokenList=false)

Clears all context information.

For synchronization, macros should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

file_name - Name of file to initialize context for

preserveTokens - Preserve the token list for this file

Returns: int

0 on success, <0 or error.

Category:

Tagging Functions
