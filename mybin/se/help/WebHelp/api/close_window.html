_command int close_window(_str doDeleteBufferIf_1FPW='', boolean saveBufferPos=true, ...)

Closes the current window and optionally deletes the buffer. Prompts the user to save changes if the buffer is deleted.

Parameters:

doDeleteBufferIf_1FPW - Determines wheter buffer is deleted.
IF =='' AND one file per window, the buffer is deleted.
IF =='W' the buffer is deleted.
Otherwise the buffer is not deleted.

saveBufferPos - If true, the buffer position information is saved.

Returns: int

Returns 0 if successful.

Category:

Window Functions
