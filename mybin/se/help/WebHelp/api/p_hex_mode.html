boolean p_hex_mode

Determines whether text is displayed in hex or ASCII lines. You need to use the hex function to switch into or out of hex display mode because this function does more than just set this property. This property is available only when the control is displayed. Possible values:

  • HM_HEX_OFF - hex mode off
  • HM_HEX_ON - hex mode on
  • HM_HEX_LINE - hex mode on using line hex

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
