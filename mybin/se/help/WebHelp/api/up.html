int up(int Noflines=1, boolean doScreenLines=false)

Moves cursor number of lines up. If number is not given a value of '1' is used. If line is not in view, line is center scrolled or smooth scrolled into view.

Parameters:

Noflines - Number of lines to move up

doScreenLines - When true, cursor up to previous soft wrap line or previous line.

Returns: int

Returns 0 if successful. Otherwise TOP_OF_FILE_RC is returned.

Example:

    status=up();
    if (status==TOP_OF_FILE_RC) {
         message(get_message(status));
    }

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods
