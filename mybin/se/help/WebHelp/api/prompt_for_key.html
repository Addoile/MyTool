_str prompt_for_key(_str prompt, var keytab_used, var k, var keyname, _str optionch='', _str noPrompt="", _str maxcount="", _str mustbe_mouse_event="")

Prompts the user on the command line for a key sequence. Returns non-zero number if the user presses a cancel key. keytab_used is set to the index of the last key table referenced. k is set to the last key of the sequence. keyname is set to the displayable sequence of keys pressed. The 'R' and 'M' options force the root or mode key table to be used.

Returns: _str

Returns 0 if successful. Otherwise COMMAND_CANCELLED_RC is returned.

Category:

Keyboard Functions
