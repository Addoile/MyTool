int VSAPI vsPropGetZ(int wid, int prop_id, char *pszValue, int ValueLen, int *pValueLen=a)

Copies string value of property into pszValue. The returned string is always null terminated. No more than BufLen characters are copied.

Parameters:

wid - Window id of object or index to resource returned by vsFindIndex, vsNameMatch, VSP_CHILD, or VSP_NEXT. 0 specifies the current object.

prop_id - One of the VSP_??? constants in "vs.h".

pszValue - Ouput buffer for null terminate string. This can be 0.

BufLen - Number of characters allocated to pszValue.

pBufLen - If this is not 0, this is set to the number of characters you need to allocate to pBufLen.

Returns: int

Returns number of characters copied

Example:

     int len;
     char bufname[100];

     len=vsPropGetZ(0,VSP_BUFNAME,bufname,100);

Applies To:

All Window Objects

Category:

Miscellaneous Functions

See Also:

vsPropGetB, vsPropGetZ, vsPropSetZ, vsPropSetB, vsPropGetI, vsPropGetI64, vsPropSetI, vsPropSetI64, vsPropGetL, vsPropSetL
