_command int surround_with(_str prefix='', _str markid='', boolean prompt=true)

Surrounds the selected block of text with a control structure or tag.

sur_text is a Slick-C® function that can only be used inside of a surround-with template. sur_text is used to indicate where the selected text should be placed and can be used multiple times within a single surround-with template. sur_text can take several options. Please see sur_text for more information.

Parameters:

prefix - Specifies the name of the surround-with template to use. If no argument is given or if multiple possible templates are found, the user is prompted to select the template they wish to use.

markid - Specifies the selection block to use.

prompt - if false and multiple matches are found, it is handled as if no matches were found

Returns: int

zero if a surround with template was expanded

Category:

Editor Control Methods
