26 VSCALLBACK_APP_TAG_FILE_MODIFIED

This callback is called after a tag file is modified. This typically occurs when the symbols for a particular file change. Note that this callback does not get called when the vsTagRetagFile function is called.

Parameters:

pszTagFilename - Name of tag file that was modified. If this is "", you should ignore this call back. Currently, this happens when a reference file was modified. We expect this case to be removed in the future since tags references will be in the same tag file.

Category:

Editor Control Callback Functions
