_str _TreeGetCaption(int ItemIndex, int ColIndex=-1)

Returns the caption for the specified tree item.

Parameters:

ItemIndex - Specifies a tree item.

ColIndex - Column index to get. First column is 0. The default is to get all columns, separated by tabs.

Applies To:

Tree View

Category:

Tree View Methods

See Also:

_TreeSetCaption, _TreeSetInfo, _TreeGetInfo, _TreeSetUserInfo, _TreeGetUserInfo
