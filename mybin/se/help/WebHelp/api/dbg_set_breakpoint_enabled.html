int dbg_set_breakpoint_enabled(int breakpoint_id, int enabled)

Enable or disable the given breakpoint. Note that this function should be called only when outside of debugging mode. Normally, you would use the packge-specific callback to enable a breakpoint.

Parameters:

breakpoint_id - Breakpoint to enable

enabled - True of false

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
