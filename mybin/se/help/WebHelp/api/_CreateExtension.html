int _CreateExtension(_str extension, _str langId, _str encoding=null, _str openApplication=null, boolean useAssociation=false)

Creates file extension specific setup data which is used by the File Extension Manager options dialog. This procedure is typically used when associating a physical file extension with a specific language, or when setting up encoding and application preferences for a physical file extension.

Parameters:

extension - File extension to be configured.

langId - Language ID (see p_LangId

encoding - (optional) default file encoding

openApplication - (optional) path for external application used to open this type of file when double-clicked on in project file browser.

useAssociation - (optional, Windows only) use Windows file association to select application for opening this type of file when double-clicked on in the project file browser.

Returns: int

Returns the name index of the setup information created.

Example:

   _CreateExtension('ada', 'ada');
   _CreateExtension('ads', 'ada');

Category:

Miscellaneous Functions
