void _BookmarkRemove(int i, int free_markid=1)

Deletes a bookmark.

Parameters:

i - Index of bookmark where 0<=i<_BookmarkQCount().

free_markid - When non-zero, _free_selection is called for the markid belonging to this bookmark.
