int VSAPI vsLineMarkerAddB(const char *pszBufName, seSeekPos RealLineNum, int isRealLineNum, seSeekPos NofLines, int BMIndex, int type, const char *pszMessage=a, void *pUserData=a, const char *pszLineData=a, void *reserved=a)

Adds a new line marker.

Parameters:

pszBufName - Name of buffer.

RealLineNum - Line number or real line number (not counting no save lines).

isRealLineNum - Indicates whether RealLineNum is a real line number or not.

NofLines - Number of lines in this item. Specify zero if the type you are using does not have the VSMARKERTYPEFLAG_DRAW_BOX flag set.

BMIndex - Bitmap to display at the line specified. Specify 0 for no bitmap. The following are names of bitmaps used for debugging:

_breakpt.ico
Indicates a line with an enabled break point.
_breakpn.ico
Indicates a line with a disabled break point.
_execpt.ico
Indicates current execution line.
_stackex.ico
Indicates a line on the execution call stack.

type - Type allocated by vsMarkerTypeAlloc().

pszMessage - HTML message displayed when mouse is over bitmap. Only a subset of HTML supported by our mini-HTML control is supported.

pUserData - Pointer to user data.

pszLineData - This is usually 0 but may contain some of the contents of the line.

Returns: int

Returns index of new line marker.

Example:

// Add debug bitmap to give the visual effect of a
// break point.
int type=vsMarkerTypeAlloc();
int LineMarkerIndex=vsLineMarkerAddB("c:\\projects\\java\\main.java",
                       10,
                       1,4,
                       vsFindIndex("_breakpt.ico",VSTYPE_PICTURE),
                       type);

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
