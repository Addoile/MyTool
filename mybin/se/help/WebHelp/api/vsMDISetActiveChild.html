void VSAPI vsMDISetActiveChild(int wid)

This function sets the active MDI child edit window. This function must be called by OEM's who implement our MDI API so Visual SlickEdit knows at all times which edit window is active. This function is called during the Windows WM_ACTIVATE message or during the MFC CView::OnActivateView handler.

See our MDI sample program for an example.

Parameters:

wid - Window id of object. 0 specifies the current object.

Category:

Miscellaneous Functions
