boolean _IsKeyPending(boolean mouse=true, boolean macro=false)

Test if a keyboard or mouse event is available.

This function does not check for MOUSE-MOVE events. If you want to check for mouse-move events, use _IsEventPending.

Parameters:

mouse - Set to false if you do not want to check for mouse events. Defaults to true.

macro - Set to true if you want to check for events from recorded macro playback. Defaults to false.

Returns: boolean

true if event is available.

Category:

Keyboard Functions
