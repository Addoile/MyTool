int _BookmarkGetInfo(int i, _str &BookmarkName=0, int &markid=0, int &vsbmflags=0, int &buf_id=0, int determineLineNumber=1, int &RealLineNumber=0, int &Col=0, long &BeginLineROffset=0, _str &LineData="", _str &Filename="", _str &DocumentName="")

Retrieves information for a bookmark.

Parameters:

i - Bookmark index where 0<=i<_BookmarkQCount().

BookmarkName - Set to name of bookmark.

markid - Set to selection id previous given to _BookmarkAdd. Specify NULL if you don't need this value.

vsbmflags - Set to combination of VSBMFLAGS_??? ORed together. Specify NULL if you don't need this value.

buf_id - Set to buffer id of bookmark. Specify NULL if you don't need this value.

determineLineNumber -

RealLineNumber - Set to real line number of bookmark. Specify NULL if you don't need this value.

Col -

BeginLineROffset - Set to real offset to beginning of line. Imaginary line data not counted. Specify NULL if you don't need this value.

LineData - Set to text on the line of the bookmark.

Filename - Set to filename the bookmark is in.

DocumentName - Set to document name the bookmark is in.

Returns: int

Returns 0 or TEXT_NOT_SELECTED_RC if successful. TEXT_NOT_SELECTED_RC indicates that the bookmark is not active. This occurs when a buffer with bookmarks is closed.

Category:

Bookmark Functions

See Also:

_BookmarkAdd, _BookmarkRemove, _BookmarkRestore, _BookmarkGetInfo, _BookmarkFind, _BookmarkQCount
