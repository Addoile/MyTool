int dbg_get_threadgroup(int threadgroup_id, _str &name, _str &group_name)

Get the details about the specified thread group.

Parameters:

threadgroup_id - index of thread group in table

name - (reference) name of thread group

group_name - (reference) name of parent thread group

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
