int tag_remove_from_file(_str file_name, boolean remove_file=true)

Remove all references from the given references (browse database or object) file from the database. This is an effective, but costly way to perform an incremental update of the data imported from a references file. First remove all items associated with that file, then insert them again.

Parameters:

file_name - full path of file the reference info came from

remove_file - if non-zero, the file is removed from the database

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
