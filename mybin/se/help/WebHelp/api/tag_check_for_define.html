int tag_check_for_define(_str tag_name, int line_no, _str(&tag_files)[], _str &id_defined_to, ...)

Look up tag_name and see if it is a macro preprocessing symbol (#define). If so, return the value of symbol in alt_tag_name. Note that this function does not normally support complex macros, that is, macros having arguments or defined to something more complicated than a simple identifier or empty string. However, if the 'arglist' variable is passed in, then it will return more complex macro definitions if one is found.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Parameters:

tag_name - Tag name to check if it is a macro define.

line_no - If found in current context, define must be before 'line'

tag_files - (onput) Tag files to search

id_defined_to - (output) Returns value of #define macro.

arglist - (output) Returns the argument list of the #define macro

Returns: int

Returns the number of matches was found, 0 of none found.

Category:

Tagging Functions
