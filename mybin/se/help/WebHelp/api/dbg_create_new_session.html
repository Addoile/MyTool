int dbg_create_new_session(_str package_name, _str session_name, boolean copy_setup)

Create a new debugger session object handle. If successful, the new session will be made the active session.

Parameters:

package_name - name of debugger integration package (gdb, jdwp, dotnot, ...)

session_name - name of debugger session

copy_setup - copy existing setup data for this session name?

Returns: int

session ID of the new debugger session

Since:

10.0

Example:

To create a GDB debugger session 

   int session_id = dbg_create_new_session("gdb", "myUnixApp");
   if (session_id < 0) {
      printf("could not create session object\n");
   } else {
      printf("new session ID = %d\n",session_id);
   }

Category:

Debugger Functions
