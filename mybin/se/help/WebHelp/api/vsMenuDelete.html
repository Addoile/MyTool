int VSAPI vsMenuDelete(int menu_handle, int position)

Deletes a menu or menu item in a loaded menu or a menu resource.

Parameters:

menu_handle - Handle of loaded menu returned by VSP_MENUHANDLE, vsMenuLoad, vsMenuFind, or vsMenuGetState. This handle may also be a handle to menu resource returned by vsFindIndex or vsNameMatch.

menu_pos - Position within menu menu_handle. The first menu item position is 0.

Returns: int

Returns 0 if successful.

Example:

void VSAPI dllproc ()
{
     int status;
     int mh,mpos;

     status=vsMenuFind (
               vsPropGetI(VSWID_MDI,VSP_MENU_HANDLE),
               "help -using",&mh,&mpos,'M');
     if (!status) vsMenuDelete (mh,mpos);
}
