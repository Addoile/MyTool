void VSAPI vsQTMDIRegisterStatusWindow(int mdi_wid, QWidget *pwidget)

This function registers the MDI status line window. SlickEdit will set focus to the hwnd window when there are no MDI child windows. The callback VSINIT.pfnSetFocus is called with VSWID_STATUS to set focus to this window. If this callback is not defined, the focus is not changed.

See our MDI sample program for an example.

Parameters:

pwidget - Qt widget

Category:

Miscellaneous Functions
