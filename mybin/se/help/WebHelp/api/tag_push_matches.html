int tag_push_matches()

Push the current match set onto the stack of match sets and initialize the new stack top to an empty match set

For synchronization, macros should perform a tag_lock_matches(true) prior to invoking this function.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
