int tag_find_in_class(_str class_name)

Find the first tag belonging to the given class. Use tag_get_info to extract the details about the tag.

Parameters:

class_name - name of class, containing tags

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Category:

Tagging Functions

See Also:

tag_get_class, tag_get_info, tag_next_in_class, tag_next_class, tag_reset_find_in_class
