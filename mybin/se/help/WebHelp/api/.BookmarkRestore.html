void _BookmarkRestore(int vsbmflags=-1)

Restores inactive bookmarks for the file specified. This function is typically called when a new file is loaded, so that inactive bookmarks may be restored.

Parameters:

vsbmflags - One or more of these flags must be in the bookmark for the bookmark to be restored.

Category:

Bookmark Functions
