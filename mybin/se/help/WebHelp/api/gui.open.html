_command typeless gui_open(typeless flags="")

Opens one or more files for editing. The Open dialog box is used to prompt for files and open options. If a file is already open, that file is activated.

Parameters:

ofn_flags - defaults to (OFN_READONLY|OFN_ALLOWMULTISELECT) and may be a combination of the following flag constants defined in "slick.sh":

OFN_ALLOWMULTISELECT
Allow multiple file selection. When set, caller must process a return value which may have space delimited filenames, some of which are double quoted.
OFN_FILEMUSTEXIST
File(s) selected must exist.
OFN_CHANGEDIR
Don't restore original directory on exit
OFN_NOOVERWRITEPROMPT
Don't prompt user with overwrite existing dialog.
OFN_SAVEAS
File list box does not select files and user is prompted whether to overwrite an existing file.
OFN_READONLY
Show read only button. See OFN_PREFIXFLAGS flag.
OFN_KEEPOLDFILE
Show keep old name button. See OFN_PREFIXFLAGS flag.
OFN_PREFIXFLAGS
Prefix return value with -r if OFN_READONLY flag given and -n if
OFN_KEEPOLDFILE flag given.

Returns: typeless

Returns 0 if successful.

Applies To:

Edit Window

Category:

File Functions
