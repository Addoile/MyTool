_str _file_date(_str filename, _str option)

Parameters:

filename - file to retrieve date of

option - Currently you must specify the 'B' option which returns the date in binary string comparison form. '' is returned if filename is not found. The time returned is in milliseconds. The actual accuracy may depend on the Operating System.

Returns: _str

Returns date of filename.

Category:

File Functions

See Also:
