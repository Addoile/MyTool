void VSAPI vsMDIRegisterStatusWindow(int mdi_wid, VSSYSHWND hwndframe, VSSYSHWND hwnd)

This function registers some MDI status line window handles. SlickEdit will set focus to the hwnd window when there are no MDI child windows. The callback VSINIT.pfnSetFocus is called with VSWID_STATUS to set focus to this window. If this callback is not defined, the focus is not changed.

See our MDI sample program for an example.

Parameters:

hwndframe - System window handle which contains the client window. For Windows, this is always the same as hwnd. For UNIX, specify the same value as hwnd if the client window is not contained by a window with decorations.

hwnd - System window handle client window.
