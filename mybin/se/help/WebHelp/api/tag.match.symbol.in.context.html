int tag_match_symbol_in_context(_str prefix, _str search_class, int tree_wid, int tree_index, _str(&tag_files)[], int &num_matches, int max_matches, int filter_flags, int context_flags, boolean exact_match, boolean case_sensitive, boolean strict, boolean find_parents, boolean find_all, ...)

Deprecated. 

Match the given symbol based on the current context information. Order of searching is:

  1. local variables in current function
  2. members of current class, including inherited members
  3. globals found in the current file
  4. globals imported explicitely from current file
  5. globals, (not static variables), found in other files
  6. any symbol found in this file
  7. any symbol found in any tag file
Failing that, it repeats steps (1-6) with filter_flags set to -1, thus disabling any filtering, unless 'strict' is true. The current object must be an editor control or current buffer.

For synchronization, macros should perform a tag_lock_matches(true) prior to invoking this function.

Parameters:

prefix - symbol prefix to match

search_class - name of class to search for matches

tree_wid - window id of tree control to insert into, 0 indicates to insert into a match set

tree_index - tree index to insert items under, ignored if (tree_wid == 0)

tag_files - (reference to _str[]) tag files to search

num_matches - (Input, Output) number of matches

max_matches - maximum number of matches allowed

filter_flags - Tag filter flags, only insert tags passing this filter. See tag_filter_type for more details.

context_flags - VS_TAGCONTEXT_*, tag context filter flags

exact_match - exact match or prefix match (0)

case_sensitive - case sensitive (1) or case insensitive (0)

strict - strict match, or allow any match?

find_parents - find parents of the given class?

find_all - find all instances, for each level of scope

search_file - (optional) file to search for matches in (for imports)

Returns: int

0 on success, <0 on error.
