int def_cvs_flags

CVS options. Bitset of CVS_

  1. CVS_ALWAYS_USE_DASH_D -- always use the "-d" option with cvs update
  2. CVS_USE_TOOLBAR_BITMAPS -- show CVS file status in project tool window
  3. CVS_FIND_NEXT_AFTER_DIFF -- advance to next item after diff
  4. CVS_RESTORE_COMMENT -- restore last checkin comment
  5. CVS_RESTORE_TAGS -- restore last checkin tag(s)
  6. CVS_SHOW_LABELS_IN_TREE -- show labels in version history tree

Default:

CVS_RESTORE_COMMENT;

Category:

Configuration Variables
