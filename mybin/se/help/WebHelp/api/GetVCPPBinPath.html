int GetVCPPBinPath(_str &Path, int UserSpecifiedVersion, int AppendExename=0)

Gets the path of the binaries for Visual C++/Visual Studio

Parameters:

Path - Path of binaries

UserSpecifiedVersion - Version of Visual C++/Visual Studio user has. If not certain, use -1

AppendExename - (optional) If true, append the name of the executable

Returns: int

0 if succesful

Category:

Miscellaneous Functions
