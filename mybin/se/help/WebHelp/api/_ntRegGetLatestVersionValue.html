_str _ntRegGetLatestVersionValue(int root, _str prefixPath, _str suffixPath, _str valueName)

Retrieves a data value from the latest version subkey underneath the given subkey.

Parameters:

root - one of the HKEY_* constants

prefixPath - path of subkey which contains version subkeys

suffixPath - additional path to be added to the version subkeys

valueName - name of data value to fetch

Returns: _str

_str retrieved data value, "" if none is found

Category:

Miscellaneous Functions
