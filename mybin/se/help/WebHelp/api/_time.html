_str _time(_str option='T')

Parameters:

retval_option - may be one of the following:

'T'
Return time in the format hh:mmcc where hh is an hour between 1 and 12, mm is the minutes between 0 and 59, and cc is the "am" or "pm".
'B'
Return binary time in milliseconds. This options is used for comparing dates with the :< and :> operators.
'M'
24-hour (Military) time in the format hh:mm:ss where hh is an hour between 0 and 23, mm is the minutes between 0 and 59, and ss is the seconds between 0 and 59.
'L'
Returns time in the current local format. Under Windows, this uses the regional settings "Time style." For other platforms, this has not yet been defined.
'F'
Return time in the format YYYYMMDDhhmmssfff where YYYY is the year, MM is the month, DD is the day, hh is an hour between 0 and 23, mm is the minutes between 0 and 59, ss is the seconds between 0 and 59, and fff is the fractional second, in milliseconds.
'G'
Returns seconds elapsed since UNIX Epoch (Midnight of January 1, 1970). This value is always in UTC/GMT.

Returns: _str

Returns system time in the format specified.
