_str _lbmulti_select_result(_str not_supported="", _str cwd="", typeless isfileman_list="")

Returns: _str

Returns '' if no items are selected in the list box. Items which contain space characters are placed in double quotes. Items which contain double quotes are not supported. buffer_name is a buffer where each line contains a filename. If the cwd parameter is given and is not '', the list box text is assumed to be file names and will be converted to absolute form relative to the directory, cwd.

Applies To:

List Box

Category:

List Box Methods

_lbdeselect_all, _lbselect_all, _lbselect_line, _lbdeselect_line, _lbinvert, _lbisline_selected, _lbfind_selected
