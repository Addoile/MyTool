int VSAPI vsTagFindRegex(const char *pszTagRegex, const char *pszSearchOptions=a, int find_first=a)

Starting with the currently selected tag, compare the tag name with the given regular expression.

Parameters:

pszTagRegex - regular expression to search for tags matching

pszSearchOptions - search options, passed to vsStrPos()

find_first - find first match or next match

Returns: int

0 if a match is found, <0 on error or not found.

Category:

Tagging Functions
