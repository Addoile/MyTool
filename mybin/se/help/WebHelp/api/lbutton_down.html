typeless lbutton_down()

Occurs when the left mouse button is pressed. The lbutton_up event occurs when the left button is released. This event already has default processing which is not typically overridden. For an edit window you can bind this event to one of the commands mou_click, mou_click_line, mou_click_block, mou_click_menu, mou_click_menu_line, or mou_click_menu_block.
