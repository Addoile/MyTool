void VSAPI vsDeleteSelection(int wid, int markid=-, int Reserved=-)

Deletes the marked text specified. No clipboard is created. This function performs a "binary" delete when in hex mode (p_hex_mode==true). A binary delete allows bisecting of end of line pairs like CR, LF.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

mark_id - Selection id to duplicated. If -1 is specified the active selection.

Returns: void

Returns 0 if successful. Otherwise TEXT_NOT_SELECTED_RC is returned. On error, message is displayed.

Category:

Selection Functions
