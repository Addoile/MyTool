_command typeless list_usersys_objects()

Opens a new buffer called "vusrsNNN.e" and inserts Slick-C® source code for system dialog box templates (forms) that have been modified since SlickEdit was installed. No user created dialog box templates are inserted.

Returns: typeless

Returns 0 if successful. Returns 1 if no templates are defined. Other errors are non-zero.

Applies To:

Edit Window

Category:

Miscellaneous Functions

See Also:

list_sys_objects, list_objects
