int _grid_width(int new_width=-1)

Returns: int

Returns and optionally sets the grid width used by the dialog editor. This affects the distance displayed between the dots displayed on a form that is being edited. If the new_width parameter is given, the grid width is set to this value. The new_width parameter is in twips (1440 twips are one inch on the display).
