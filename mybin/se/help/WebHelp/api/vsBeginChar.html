int VSAPI vsBeginChar(int wid, int doBeginChar=a, int *pCharLen=a, int beginComposite=a, unsigned *pUTF32Array=a, int *pUTF32ArrayLen=a)

Retrieves the current character and/or places the cursor at the beginning of the current character.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

doBeginChar - When non-zero, cursor position is adjusted if necessary.

pCharLen - Set to number of bytes in current UTF-8 character.

beginCompsite - When non-zero, operation supports composite characters.

pUTF32Array - (Output) If the buffer is UTF-8, this is set to the UTF-32 characters in the current character. If the buffer is not UTF-8, the current DBCS character is placed in the first entry and the array length is set to 1. The DBCS lead byte is in the first 8 bits (array[0]&0xff).

pUTF32ArrayLen - Number of UTF-32 characters placed in pUTF32Array.

Returns: int

Returns 0 if the cursor is already at the beginning of the current character. Otherwise, a non-zero value is returned.

Category:

Unicode Functions
