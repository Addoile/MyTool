int def_actapp

Application activation options.

  • ACTAPP_AUTORELOADON -- Automatically reload files that have changed on disk?
  • ACTAPP_SAVEALLONLOSTFOCUS -- Save all files when SlickEdit loses focus?
  • ACTAPP_SUPPRESSPROMPTUNLESSMODIFIED -- Automatically reload with no prompting unless the buffer is modified?
  • ACTAPP_WARNONLYIFBUFFERMODIFIED -- Only warn the user when the file is modified outside SlickEdit?
  • ACTAPP_AUTOREADONLY -- Automatically update read-only status of files on disk?

Default:

ACTAPP_AUTORELOADON|ACTAPP_AUTOREADONLY

Category:

Configuration Variables
