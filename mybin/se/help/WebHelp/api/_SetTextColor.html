int _SetTextColor(int ColorIndex, int Nofbytes, boolean replacePreviousColor=true)

Sets the text color for the number of bytes specified starting from the cursor.

Parameters:

ColorIndex - index of color in color table

Nofbytes - number of bytes to color code

replacePreviousColor - When true, previous color is replaced with new color. When adding lots of color, this slows down adding color.

Returns: int

Applies To:

Edit Window, Editor Control

Example:

int gColorIndex;
definit()
{
     if (arg(1)=='L' && gColorIndex) return;
     gColorIndex=_AlocColor();
     _default_color(gColorIndex,0xffffff,0xFF0000);
}
defmain()
{
     // Change the color of 5 bytes of text starting from cursor
     _SetTextColor(gColorIndex,5);
}

Categories:

Edit Window Methods, Editor Control Methods
