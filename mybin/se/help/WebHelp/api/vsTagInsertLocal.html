int VSAPI vsTagInsertLocal(const char *tag_name, const char *tag_type, const char *file_name, int start_linenum, int start_seekpos, int scope_linenum, int scope_seekpos, int end_linenum, int end_seekpos, const char *class_name=a, int tag_flags=a, const char *signature=a, const char *class_parents=a, const char *template_sig=a, int name_line_no=a, int name_seekpos=a)

Add a local variable tag and its information to the locals list. The context for the a local tag includes all tag information, as well as the ending line number and begin/scope/end seek positions in the file. If unknown, the end line number/seek position may be deferred, see tag_end_local().

For synchronization, macros should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

tag_name - tag string

tag_type - string specifying tag_type

file_name - full path of file the tag is located in

start_linenum - start line number of tag within file

start_seekpos - start seek position of tag within file

scope_linenum - start line number of start of tag inner scope

scope_seekpos - start seek position of tag inner scope

end_linenum - (optional) ending line number of tag within file

end_seekpos - (optional) end seek position of tag within file

class_name - (optional) name of class that tag is present in, use concatenation (as defined by language rules) to specify names of inner classes.

tag_flags - (optional) see VS_TAGFLAG_* above.

signature - (optional) tag signature (return type, arguments, etc)

class_parents - (optional) classes that this class inherits from

template_sig - (optional) template signature

name_line_no - (optional) line number that symbol name is located on

name_seekpos - (optional) Seek position of the first character of the symbol name

Returns: int

sequence number (local_id) of local variable on success, or <0 on error.

Category:

Tagging Functions
