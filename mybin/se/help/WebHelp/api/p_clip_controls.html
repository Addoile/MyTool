boolean p_clip_controls

Determines whether the child controls are clipped when the control is painted. When this property is off, dialogs sometimes have problems repainting. Painting a dialog box is slightly faster when p_clip_controls is off. However, the machines and video performance get faster, this property will become useless. This property is not yet supported under UNIX.

Applies To:

Form, Frame

Categories:

Form Properties, Frame Properties
