int getAutoCompleteMinimumLength(_str langID, int defaultValue=null)

Gets the minimum length required for autocomplete to be triggered for this language. This value is available on the AutoComplete form for each applicable language.

Parameters:

langID - language ID (see p_LangId)

defaultValue - if the value is not available, then we just return this value. If this parameter is not specified, then the default value for the overall def-var is used.

Returns: int

minimum autocomplete length for this language

Category:

LanguageSettings API
