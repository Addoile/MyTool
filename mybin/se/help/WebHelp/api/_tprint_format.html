void _tprint_format(int output_view_id, _str filename="", _str options="", _str print_cheader="", _str print_cfooter="", _str print_lheader="", _str print_lfooter="", _str print_rheader="", _str print_rfooter="")

This function is only supported by the UNIX version. Prints the contents of the current buffer according to the options given.

It is much easier to create input parameters to this function by using macro recording. Use the Text Mode Print dialog box to set your options. Then print a document while macro recording is on.

Parameters:

output_window_id - specifies the window/buffer to get the formatted text. Usually the _create_temp_view function is used to create an empty window/buffer.

filename - is inserted into the header and footer parameters wherever a %f occurs. Usually this is the buffer name.

cheader, cfooter, lheader, lfooter, rheader, and rfooter are strings which may have escape sequences with the following meaning embedded into them:

%f
Buffer name
%p
Page number
%d
Date

The letters 'c', 'l', and 'r' correspond to the header or footer at the center, left, and right of the page.

The header is printer at the top of each page. The footer is printed at the bottom of each page.

options - is a string in the format:
flags, bl_after_header, bl_before_footer, lines_per_page, cols_per_line, linenums_every,

flags - are reserved and should be 0.

bl_after_header - indicates the number of blank lines between the header line and the text on the page. This value is ignored if all header parameters are blank.

bl_before_footer - indicates the number of blank lines between the last line of text printed on the page and the footer line.

lines_per_page - indicates the maximum number of lines that can be printed. This includes header, footer, and blank lines.

cols_per_line - indicates the maximum number of columns that can be printed on a line.

linenums_every - when non-zero, lines at intervals of this value are printed with line numbers.

Returns: void

Returns 0 if successful.

Applies To:

Edit Window

Categories:

Buffer Functions, Edit Window Methods
