int tag_remove_from_class(_str class_name, boolean remove_class=false)

Remove all tags associated with the given class from the database.

Parameters:

class_name - Full name of the class the tag is associated with if NULL, all non-class tags are removed.

remove_class - if non-zero, the class is removed from the database, in addition to the tags associated with the class.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
