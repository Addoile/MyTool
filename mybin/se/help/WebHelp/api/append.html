_command typeless append(_str dest_filename="")

Appends selected text to filename specified. This command has no effect on files already in memory. Select text first with one of the commands select_char, select_line, or select_block.

Returns: typeless

Returns 0 if successful. Common return codes are TEXT_NOT_SELECTED_RC, BLOCK_SELECTION_REQUIRED_RC, and TOO_MANY_SELECTIONS_RC. On error, message is displayed.

Categories:

File Functions, Selection Functions
