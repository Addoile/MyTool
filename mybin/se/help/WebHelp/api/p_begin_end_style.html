int p_begin_end_style

Determines the brace style of the current buffer. This property is available only when the control is displayed.

Example:

   // begin/end style 1
   p_begin_end_style = 0;
   if (x) {
      x = 5;
   }

   // begin/end style 2
   p_begin_end_style = VS_C_OPTIONS_STYLE1_FLAG;
   if (x)
   {
      x = 5;
   }

   // begin/end style 3
   p_begin_end_style = VS_C_OPTIONS_STYLE2_FLAG;
   if (x)
      {
      x = 5;
      }

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
