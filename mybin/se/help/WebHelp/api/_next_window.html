void _next_window(_str option="")

Switches to the next window. By default, the next window is the window that was created after the current window.

Parameters:

option - String of zero or more of the following options:

H
Specifies that hidden windows should not be skipped. Used for debugging.
R
Specifies not to set the screen refresh flags and not to set the focus.
F
Specifies to not to set the focus.
M
Specifies to skip to the next MDI child window which may or may not have an editor control.

Category:

Window Functions
