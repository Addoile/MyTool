void VSAPI vsSearchInitSkipped(int reserved=a)

When the VSP_TRUNCATELENGTH is non-zero, replace operations are skipped if the resulting line becomes longer than the truncation length. This function is used in conjunction with vsSearchQNofSkipped and vsSearchQSkipped to return information about what was skipped. Call this function before performing a search and replace operation to reset NofSkipped and Skipped lines data returned by the vsSearchQNofSkipped and vsSearchQSkipped functions.
