_command void wfont_zoom(_str size="+1")

Modify the font size for the current editor window.

Parameters:

size - font size change amount

  • +n -- increase font size by 'n' pixels
  • -n -- decrease font size by 'n' pixels
  • n -- set font size to 'n'

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
