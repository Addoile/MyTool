int tag_list_in_file(int tree_wid, int tree_index, _str prefix, _str(&tag_files)[], _str file_name, int filter_flags, int context_flags, var vnum_matches, int max_matches, boolean exact_match, boolean case_sensitive)

List the symbols found in files having the given 'base' filename and passing the given pushtag and context flags.

For synchronization, macros should perform a tag_lock_matches(true) and tag_lock_context(false) prior to invoking this function.

Parameters:

tree_wid - window id of tree control to insert into, 0 indicates to insert into a match set, see tag_insert_match.

tree_index - Parent tree node index to insert item under, ignored if tree_wid is zero.

prefix - Tag name or prefix of tag name to search for.

tag_files - (Input) Tag files to search

file_name - Only list tags coming from file with this "base" name.

filter_flags - Tag filter flags, only insert tags passing this filter. See tag_filter_type for more details.

context_flags - Context flags representing what to allow or restrict to. See tag_check_context_flags for more details.

vnum_matches - (Input, Output) Number of matches

max_matches - Maximum number of matches allowed

exact_match - Specifies exact match or prefix match (0)

case_sensitive - Specifies search case sensitivity.

Returns: int

Returns 0 on success, <0 on error.

Category:

Tagging Functions

See Also:

tag_insert_match, tag_list_any_symbols, tag_tree_insert_tag, tag_tree_prepare_expand
