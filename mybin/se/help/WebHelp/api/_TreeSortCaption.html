void _TreeSortCaption(int ItemIndex, _str PrimarySortOptions="", _str SecondarySortOptions="")

Sort children of node specified where caption is the primary key and optionally the user info is the secondary key.

Parameters:

ParentIndex - Tree item index to parent containing children to be sorted.

PrimarySortOptions - String of one or more of the following options:

'F'
Sort filenames
'N'
Sort numbers
'E'
Exact Case. Sort is case insensitive if this option is not specified
'I'
Sort case insensitive
'B'
First try case insensitive compare, then perform insensitive
'D'
Descending. Sort is ascending if this option is not specified
'2'
Sort based only on filename part of caption
'T'
Traverse children (sort recursively)
'='
Place parents nodes at the top after sort. Here we consider a parent nodes to be those with ShowChildren >= 0.
'U'
Remove duplicates
'H'
Hide duplicates
'C'
Sort current sort column only

SecondarySortOptions - String of sort options for secondary key. See PrimarySortOptions.

Applies To:

Tree View

Category:

Tree View Methods

See Also:

_TreeSortUserInfo
