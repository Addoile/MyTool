int last_index(_str new_value="", _str option="D")

Returns: int

All index values returned by this function are indexes into the names table. SlickEdit stores information for names of procedures, commands, variables, event tables, and more in the names table.

When the second parameter is 'D', this function returns (and sets if new_value not ''), the index of the last command executed by a key or the built-in call_key function.

When the second parameter is 'C', this function returns (and sets if new_value not ''), the index of the last command executed by any mechanism.

When the second parameter is 'P', this function returns the prefix key message displayed on the message line for the last key sequence executed. new_value parameter is ignored.

When the second parameter is 'K', this function returns the index of the last event table used to determine a key binding. new_value parameter is ignored.

When the second parameter is 'W', this function returns (and sets if new_value not ''), the command line/menu wait value. This flag is set to non-zero when a command is invoked from the command line or a menu.
