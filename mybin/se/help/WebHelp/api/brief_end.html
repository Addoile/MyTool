_command typeless brief_end()

For a text box or combo box, the cursor is placed after the last character in the text box.

Edit Window and Editor Control: Pressing the END key once moves the cursor to the end of the current line. Pressing the END key again moves the cursor to the bottom of the window. Pressing the END key again moves the cursor to the bottom of the buffer. This command may only be bound to the END key.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods
