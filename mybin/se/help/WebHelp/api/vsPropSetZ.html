void VSAPI vsPropSetZ(int wid, int prop_id, const char *pszValue)

Sets value of string property to pszValue.

Parameters:

wid - Window id of object. 0 specifies the current object.

prop_id - One of the VSP_??? constants in "vs.h".

pszValue - New value for property.

Example:

     vsPropSetZ(0,VSP_BUFNAME,"c:\\autoexec.bat");

Applies To:

All Window Objects

Category:

Miscellaneous Functions

See Also:

vsPropGetB, vsPropGetZ, vsPropSetZ, vsPropSetB, vsPropGetI, vsPropGetI64, vsPropSetI, vsPropSetI64, vsPropGetL, vsPropSetL
