int _suspend()

Pushes a new level for a signal handler. After a _suspend has been executed, errors which halt the interpreter will set the variable rc to the error code and _resume execution at the first statement after _suspend. The _resume function restores the interpreter stack and code position to the state it was in after the last _suspend was executed. Be sure to execute _resume to restore the previous signal handler. This function sets the rc variable to 0 if successful. Interpreter is stopped if too many suspends are performed.

Example:

_suspend(); // rc is set to 0 by
_suspend in first pass.
if (rc) {
     if (rc==1) { // normal return?
         return(0);
     }
     message('Please specify floating point number');
     return(1);
}
// If arg(1) is not a valid number, run time error will occur which
cause _resume
f=arg(1);
++f;
// Initiate normal return
rc=1;
_resume();
