int recycle_file(_str filePath)

Uses a platform-specific method to send a file to the Trash or Recycle Bin.

If def_delete_uses_recycle_bin is FALSE (0), or a trash command is not available on the current platform, then delete_file is used.

Parameters:

filePath - Full path to file to be deleted

Returns: int

Returns 0 if successful. Error codes similar to those returned by delete_file.

Category:

File Functions
