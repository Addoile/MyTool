int VSAPI vsFontQType(int flags, const char *pszFontName, void *reserved=a)

Parameters:

flags - One or more of the VSFONTLISTFLAG_??? flags ORed together.

VSFONTLISTFLAG_SCREEN_FONTS
Test screen font.
VSFONTLISTFLAG_PRINTER_FONTS
Test printer font.

pszFontName - Name of font to list sizes for. Specify 0 to list available font names.

Returns: int

Returns flags of information about this font.

VSFONTTYPE_RASTER
Windows only. Raster font.
VSFONTTYPE_DEVICE
Windows only. Device font.
VSFONTTYPE_TRUETYPE
Windows only. True Type font.
VSFONTTYPE_FIXED
Fixed pitch font.
VSFONTTYPE_OUTLINE
Outline font.
VSFONTTYPE_KERNING
No longer used.

Category:

Font Functions
