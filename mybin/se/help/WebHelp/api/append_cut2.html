_str append_cut2(_str mark='', boolean doCopy=false, _str name='')

Appends selected text to the clipboard and optionally deletes selected text. Specify 'C' as the second argument if you do not want the selected text deleted. mark_id is a handle to a selection returned by one of the built-ins _alloc_selection or _duplicate_selection. A mark_id of '' or no mark_id parameter identifies the active selection.

Parameters:

mark -

doCopy -

name -

Returns: _str

Returns 0 if successful. Common return codes are TEXT_NOT_SELECTED_RC, SOURCE_DEST_CONFLICT_RC, and INVALID_SELECTION_HANDLE_RC. On error, message is displayed

Categories:

Edit Window Methods, Selection Functions
