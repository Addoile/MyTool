void AutoCompleteSetOptions(_str lang, int flags)

Set the auto complete options wich are a bitset of AUTO_COMPLETE_*.

The options are stored per extension type using def_autocomplete_[ext].

Parameters:

lang - language ID -- see p_LangId

flags - bitset of AUTO_COMPLETE_* options

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Category:

Completion Functions
