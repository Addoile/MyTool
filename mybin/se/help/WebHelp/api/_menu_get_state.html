int _menu_get_state(int menu_handle, _str command_or_category, int &mf_flags, _str option='C', _str &caption="", _str &command="", _str &categories="", _str &help_command="", _str &help_message="")

Gets menu item values for the menu items which contain the command or category specified.

Parameters:

menu_handle - Handle of loaded menu returned by p_menu_handle, _menu_load, _menu_find, or _menu_get_state.

command_or_category - category, or position within menu_handle of menu item. If a command is specified, the command must match exactly. See option parameter.

option - May be 'M', 'C', 'P'. If the 'M' option is given, command_or_category is the command of the menu item to be found. If the 'C' option is given, command_or_category is the category of the menu item to be found. If the 'P' option is given, command_or_category is the position (0.._menu_info(menu_handle)-1) of the menu item with in menu_handle. Defaults to 'C'.

mf_flags - mf_flags may be zero or more of the following flags defined in "slick.sh":

  • MF_CHECKED
  • MF_UNCHECKED
  • MF_GRAYED
  • MF_ENABLED
  • MF_SUBMENU

caption - Menu item title.

command - For sub-menu, command is a menu handle which may be used in calls to _menu_get_state or _menu_info. Otherwise, this is the command to be executed when the menu item is selected.

categories - Menu item categories. May be one or more categories separated with a '|' character. See _menu_set_state function for more information about categories.

help_command - Menu item help command. Executed when F1 pressed on menu item.

help_message - Menu item help message. Displayed when cursor is on menu item.

Returns: int

Returns 0 if successful.

Example:

// This code traverses a menu that has been loaded with _menu_load
#include "slick.sh"
defmain()
{
     traverse_menu(_mdi.p_menu_handle);
}
static void traverse_menu(menu_handle)
{
    Nofitems=_menu_info(menu_handle,'c');
    for (i=0;i<Nofitems;++i) {
         _menu_get_state(menu_handle,i,mf_flags,'p',caption,command,
                      categories,help_command,help_message);
         messageNwait("p_caption="caption)
         if (mf_flags & MF_SUBMENU) {
             traverse_menu(command);
         }
    }
}
