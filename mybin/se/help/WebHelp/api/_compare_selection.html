int _compare_selection(_str mark1_id, _str mark2_id)

Compare size and dimensions of selection handles. Used to compare duplicated selection handles for equality. mark1_id and mark2_id are a handle to a selection returned by one of the built-in _alloc_selection or _duplicate_selection.

Parameters:

mark1_id - selection handle

mark2_id - selection handle

Returns: int

0, if selections are equal

Category:

Selection Functions

See Also:

_alloc_selection, _duplicate_selection, _free_selection, _get_selinfo
