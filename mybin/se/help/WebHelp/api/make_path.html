int make_path(_str path, typeless doShellMessages='')

Creates the necessary multi-level directories in path so that files may be copied to path.

Parameters:

path - Input path or filename. All subdirectories are created. Last character may be a file separator character.

Returns: int

Returns 0 if successful. Common return codes are PATH_NOT_FOUND_RC, INSUFFICIENT_DISK_SPACE_RC, and ACCESS_DENIED_RC.

Category:

File Functions
