void VSAPI vsBufRefresh(int buf_id, int flags=VSREFRESH_BUFTEXT)

This function tells the editor to refresh parts of the buffer specified. Currently only the break point manager uses this function because it modifies bitmaps which are stored outside the editors control.

Parameters:

buf_id - Buffer id returned by vsPropGetI(wid,VSP_BUF_ ID), vsBufEdit, or vsBufMatch.

flags - Specifies buffer properties that need to be refreshed when vsRefresh is called or the editor automatically calls vsRefresh when a macro terminates.

Category:

Buffer Functions
