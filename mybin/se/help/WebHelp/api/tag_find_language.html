int tag_find_language(_str &lang, _str search_for=null)

API function for finding what languages are tagged in this database, typically, there is only one. Note that if files are added and removed from a project, stale language types may be left around. Languages are determined by the language ID for the file, as represented by p_LangId.

Parameters:

lang - (reference) set to first language found

search_for - (optional) language ID to search for

Returns: int

0 on success, <0 on error.

Since:

13.0

Category:

Tagging Functions

See Also:

p_LangId, tag_get_language, tag_next_language, tag_set_language
