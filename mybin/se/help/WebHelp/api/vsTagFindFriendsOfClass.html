int VSAPI vsTagFindFriendsOfClass(const char *szClassName, VSHREFVAR tag_files, char *szFriendList, int maxFriendList)

Find all the friends of the given class in the current context and tag files.

Parameters:

szClassName - class to find friends of

tag_files - list of tag files to search

szFriendList - (reference) set to list of friends, separated by VS_TAGSEPARATOR_parents

maxFriendList - number of bytes allocated to szFriendList

Returns: int

0 on success, <0 on error. Having no friends is considered as a success.

Category:

Tagging Functions

See Also:

vsTagListSymbolsInClass, vsTagCheckFriendRelationship
