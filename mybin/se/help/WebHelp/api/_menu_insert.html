int _menu_insert(int menu_handle, int menu_pos, int mf_flags, _str caption, _str command="", _str categories="", _str help_command="", _str help_message="")

Inserts a menu or menu item into a loaded menu or a menu resource.

Parameters:

menu_handle - Handle of loaded menu returned by p_menu_handle, _menu_load, _menu_find, or _menu_get_state. This handle may also be a handle to menu resource returned by find_index or name_match. menu_pos Position within menu menu_handle. Menu item is inserted before position. The first menu item position is 0. Specify -1 or a position greater than the last menu item to insert after the last menu item.

mf_flags - mf_flags may be zero or more of the following flags defined in "slick.sh":

  • MF_CHECKED
  • MF_UNCHECKED
  • MF_GRAYED
  • MF_ENABLED
  • MF_SUBMENU

caption - Menu item title.

command - The command to be executed when the menu item is selected. This parameter is ignored if mf_flags contain the MF_SUBMENU flag.

categories - Menu item categories. May be one or more categories separated with a '|' character. See _menu_set_state function for more information about categories.

help_command - Menu item help command. Executed when F1 pressed on menu item.

help_message - Menu item help message. Displayed when cursor is on menu item.

Returns: int

Returns 0 if successful.

Example:

#include "slick.sh"
defmain()
{
     // Insert 2 items into a menu resource called menu1
     index=find_index("menu1",oi2type(OI_MENU));
     if (!index) {  // IF menu1 does not already exist?
         // Create new menu resource in names table
         index=insert_name("menu1",oi2type(OI_MENU));
         if (index<=0) {
             message("unable to create menu1");
             return("");
         }
     }
     // Insert File menu
     _menu_insert(index, 0,MF_ENABLED|MF_SUBMENU,"&File",
                               "","","help file menu","");
     // Insert New menu item within File menu
     _menu_insert(index.p_child,0,MF_ENABLED,"&New","New","",
                             "help file menu", "");
     // Load and display the menu as a pop-up menu
     show('menu1');
}
