void tag_get_instance_info(int inst_id, _str &inst_name, _str &inst_type, int &inst_flags, _str &inst_class, _str &inst_args, _str &file_name, int &line_no)

Extract the supplementary information associated with a tag instance.

Parameters:

inst_id - Unique ID of instance to get info about. Use tag_match_instance() to get this ID.

inst_name - (Output) Name of tag instance (case insensitive)

inst_type - (Output) Tag type name. See tag_get_type for a list of all standard tag types.

inst_flags - (Output) Reference attributes (see VS_REFFLAG_*)

inst_class - (Output) class associated with tag (zero for global)

inst_args - (Output) Arguments associated with tag (eg. function args)

file_name - (Output) Name of file where tag instance is located

line_no - (Output) Line which tag instance is on

Category:

Tagging Functions

See Also:

tag_find_refer_by, tag_find_refer_to, tag_insert_instance, tag_match_instance, tag_next_refer_by, tag_next_refer_to
