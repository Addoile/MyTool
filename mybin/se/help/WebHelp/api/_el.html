typeless& _el(typeless index)

Parameters:

index - index / key correspond to item to return.

Returns: typeless&

Returns a reference to an element in a hash table or array.

NOTE: For backwards compatibility, this function will also work for structs and class instances, returning a reference to the given field (specified by field index, see _fieldindex). This use of _el() is discouraged.

Example:

t[0]=1;
t._el(0)=4;       //  Same as t[0]=4;
p= &t._el(0);     // Get a pointer to t[0] element
*p=5;             // Same as t[0]=5;
messageNwait("t[0]="t._el(0));

Category:

Miscellaneous Functions
