int dbg_destroy_session(int session_id)

Destroy the given session and remove it from the session list.

Parameters:

session_id - debugger session ID (1 .. max_session)

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

To destroy the first debugger session. 

   dbg_destroy_session(1);

Category:

Debugger Functions

See Also:

dbg_create_new_session, dbg_get_num_sessions
