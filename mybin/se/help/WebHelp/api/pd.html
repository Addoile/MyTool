_command int pd(_str cmdline="", _str quiet="")

Adds the current working directory to the top of the directory stack and makes the supplied directory (cmdline) the new working directory. With no arguments (cmdline==''), swaps the current working directory with the top of the directory stack.

Parameters:

cmdline - directory to switch to

quiet - suppress messages

Returns: int

0 on success, <0 on error

Category:

File Functions

See Also:

cd, popd
