int VSAPI vsRefactorGetErrorFileName(int handle, int index, char *filename, int maxLength)

Get the name of the file while failed to parse from the refactoring transaction.

Parameters:

handle - Handle to transaction

index - Index of file

filename - Filename of file that was modified

maxLength - Maximum filename length

Returns: int

Length of error string on success, <0 on error

Category:

Refactoring Functions
