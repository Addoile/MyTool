_command typeless dde(_str params='')

Supported under Windows, Windows NT, and Windows 95/98. When another application wants to communicate with SlickEdit using DDE, the other application should use the server name of SlickEdit, the system topic, and call this command so that the command is invoked on the correct object. The DDE command string: 

        'dde -mdi -refresh edit c:\junk'
can be sent to SlickEdit to have SlickEdit open the file "c:\junk". The first instance of SlickEdit has the server AND application window name "SlickEdit". All instances of SlickEdit that follow have the server and application window name "SlickEdit Server n" where n starts at 1. Invoke SlickEdit with the +new option to create another instance. The editor_name function may be used to get the server name.

Parameters:

params - params is a string in the format: [-mdi] [-refresh] [-multi] command

The switches have the following meaning:

-mdi
Invoke command on mdi edit window.
-refresh
Refresh the screen after the command is executed. If you do not specify this command, you will need to send the dderefresh command after sending all dde commands to insure that the screen is correctly updated.
-multi
Indicates that command contains multiple commands separated with semicolons.
-cd directory
Temporarily change the current directory to directory when executing the command.
