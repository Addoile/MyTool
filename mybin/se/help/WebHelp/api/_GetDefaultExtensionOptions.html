int _GetDefaultExtensionOptions(_str extension, VS_LANGUAGE_OPTIONS &langOptions)

Deprecated. Use _GetDefaultLanguageOptions()

Gets language specific options for a specific file extension.

Parameters:

extension - File extension. For list of file extension types, use our File Extension Manager dialog ("Tools", "Options...", "File Extension Manager")

langOptions - Initialized to language specific options for extension.

Returns: int

Returns 0 if successful.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions
