void switch_buffer(_str old_buffer_name, _str option='', _str swold_pos=null, _str swold_buf_id=-1)

This procedure is called just after SlickEdit switches to a new buffer. When not in EMACS emulation, this procedure does nothing. Under EMACS emulation, this procedure keeps track of the last buffer edited in the current window.

Parameters:

flag - flag = 'Q' if file is being closed flag = 'W' if focus is being indicated

Category:

Buffer Functions
