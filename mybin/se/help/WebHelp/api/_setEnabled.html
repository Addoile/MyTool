void _setEnabled(int tabIndex, boolean enabled)

Enables a specific tab. Used this method instead of the p_ActiveEnabled property when setting enabled state of a non-active tab.

The active window MUST be a tab control (OI_SSTAB).

Parameters:

tabIndex - Tab-index to set enabled.

enabled - Set to true to enable the tab, false to disable.

Applies To:

SSTab

Category:

SSTab Methods
