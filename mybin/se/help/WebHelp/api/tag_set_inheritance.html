int tag_set_inheritance(_str class_name, _str parents)

Modify the set of parent classes for a given class. Use the NULL or empty string to indicate that class_name is a base class.

Parameters:

class_name - Qualified name of class or interface to update inheritance information for. See VS_TAGSEPARATOR_class and VS_TAGSEPARATOR_package for details on constructing this string.

parents - Delimited list of immediate class ancestors. It is important to remember that for some languages, all classes have a default ancestor, such as java.lang/Object in Java. Otherwise, use the empty string to indicate that class_name is a base class. See VS_TAGSEPARATOR_parents for more details on constructing this string.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
