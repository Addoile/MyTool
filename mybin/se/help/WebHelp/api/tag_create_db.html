int tag_create_db(_str file_name, int db_type=VS_DBTYPE_tags)

Create a tag database, with standard tables, index, and types.

Parameters:

file_name - file path where to create new database If file_name exists, it will be truncated.

db_type - if not given, creates tag database. if (db_type==VS_DBTYPE_references), then creates a tag references database.

Returns: int

Returns database handle >= 0 on success, <0 on error.

Since:

3.0

Category:

Tagging Functions
