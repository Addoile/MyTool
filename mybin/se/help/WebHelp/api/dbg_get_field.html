int dbg_get_field(int class_id, int field_id, _str &name, _str &signature, _str &value, int &flags)

Get the information about the field at the given index in the field table of the given class.

Parameters:

class_id - index of class in class table

field_id - index of field in field table of class

name - (reference) name of field

signature - (reference) set to type of field

value - (reference) set to value of field

flags - (reference) set to bitset of DEBUG_MODEL_FLAGS_

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
