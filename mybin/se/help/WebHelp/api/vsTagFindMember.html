int VSAPI vsTagFindMember(const char *pszClassName, const char *pszFilename=a, int find_first=a)

Retrieve the first tag included in this tag database which belongs to the given class name, and if specified, comes from the given file. Use vsTagGetInfo and/or vsTagGetDetail to extract the details about the tag.

Parameters:

pszClassName - Class name to search for members of

pszFilename - (optional), tag must come from this file name

find_first - find first match or next match

Returns: int

0 on success, <0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Category:

Tagging Functions
