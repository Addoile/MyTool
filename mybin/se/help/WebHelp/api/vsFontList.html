int VSAPI vsFontList(void(VSAPI*pfnCallback)(const char*psz), int flags=VSFONTLISTFLAG_SCREEN_FONTS, const char *pszFontName=a)

Parameters:

pfnCallback - Function called for each font name or font size found.

flags - One or more of the VSFONTLISTFLAG _??? flags ORed together.

VSFONTLISTFLAG_SCREEN_FONTS
List screen fonts.
VSFONTLISTFLAG_PRINTER_FONTS
List printer fonts
VSFONTLISTFLAG_FIXED_ONLY
List fixed fonts only.
VSFONTLISTFLAG_OUPUT_WIDTHxHEIGHT
Windows only: When listing font sizes, list fonts in format wwwxhhh. where www is the pixel width and hhh is the pixel height.

pszFontName - Name of font to list sizes for. Specify 0 to list available font names.

Returns: int

Returns 0 if successful.

Category:

Font Functions
