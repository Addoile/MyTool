void VSAPI vsSetAllOldLineNumbers(int wid, int Reserved=a)

Sets all old line numbers for the current buffer to the line number. Use the vsGoToOldLineNumber method to go to an old line number. SlickEdit uses old line numbers to better handle going to an error line after lines have been inserted or deleted. Note that the Slick-C® save command takes an option to set the old line numbers on save.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
