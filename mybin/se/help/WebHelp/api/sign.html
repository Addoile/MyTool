int sign(typeless x, boolean zeroIsPositive=false)

Returns the sign (+1, 0, -1) of number.

Parameters:

x - Integer or float.

zeroIsPositive - (optional). Set to true if you want 0 to be considered a positive number (+1). Defaults to false.

Returns: int

+1 for positive signed value, -1 for negative signed value, 0 for zero value.

Category:

Miscellaneous Functions
