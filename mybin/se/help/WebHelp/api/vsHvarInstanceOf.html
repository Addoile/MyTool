int VSAPI vsHvarInstanceOf(VSHVAR hvar, VSPSZ class_name)

Check if the given variable is an instance of the named class or interface.

Parameters:

hvar - handle for interpreter variable

class_name - name of class, struct, or interface to test

Returns: int

'true' if 'hvar' is an instance of 'name' 'false' otherwise. Returns CLASS_NAME_NOT_FOUND_RC if the class is not found. Returns INVALID_HVAR_ARGUMENT_RC if 'hvar' is not a struct or class instance.

Category:

Miscellaneous Functions
