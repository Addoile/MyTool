int VSAPI vsBufMatch(const char *pszBufName, int *pIndex=a, int BufMatchFlags=a, VSBUFMATCHINFO *pBufMatchInfo=a, int Reserved=a)

Find file(s) currently open in the editor matching the given file name prefix.

Parameters:

pszBufName - File name prefix to match, use "" to match any file.

pIndex - (output), set to buffer index of buffer found. If it points to -1, find the first matching buffer.

BufMatchFlags - Buffer matching options, a bitset of:

  • VSBUFMATCH_HIDDEN -- Return buffers with (buf_flags & VSBUFFLAG_HIDDEN) true.
  • VSBUFMATCH_EXACT -- Exact buffer name match instead of prefix matching.
  • VSBUFMATCH_BUFNAMEONLY -- Find buffer names only.
  • VSBUFMATCH_DOCUMENTNAMEONLY -- Find document names only.
  • VSBUFMATCH_BUFFERIDONLY -- Return buffer ID only

pBufMatchInfo - (optional, output) set to detailed buffer info

Reserved - (unused)

Returns: int

0 on success, <0 on error

Attention:

This function is thread-safe. On Unix, this function is not yet thread-safe because it uses a call to zabsolute(), which can do a chdir()
