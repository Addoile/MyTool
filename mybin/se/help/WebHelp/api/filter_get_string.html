_str filter_get_string(_str &string)

Sets variable in_string to next partial line of selection being filtered. A non-zero value is returned if all text in the selection has been filtered. The procedure filter_init must be called before this function.

Returns: _str

Returns 0 if successful. A non-zero value is returned if all text in the selection has been filtered.

Applies To:

Edit Window, Editor Control

Category:

Selection Functions

See Also:

filter_init, filter_put_string, filter_restore_pos, filter_selection
