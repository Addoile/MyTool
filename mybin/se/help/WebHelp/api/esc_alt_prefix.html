_command typeless esc_alt_prefix(_str completion_info='')

Defines an alternative ALT key and defines a completion/enter argument key. For example, after binding this command to the ESC key, pressing ESC 'x' is identical to Alt+X when the cursor is in the text area. When the cursor is on the command line pressing the ESC key will complete argument at the cursor. If a unique argument is found, it is returned as if the ENTER key was pressed.
