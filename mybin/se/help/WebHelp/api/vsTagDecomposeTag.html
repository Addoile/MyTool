int VSAPI vsTagDecomposeTag(const char *pszProcName, char *pszTagName, char *pszClassName, char *pszTypeName, int &tag_flags, char *pszSignature=a, char *pszReturnType=a, char *pszTmplSignature=a)

Decompose the canonical tag display string of the form: 

   tag_name(class_name:type_name)flags(signature)return_type

This is used to speed up find-tag and maketags for languages that do not insert tags from DLLs. All output strings are set to the empty string if they do not match, tag_flags is set to 0 if there is no match.

Parameters:

pszProcName - tag display string

pszTagName - (reference) the name of the tag

pszClassName - (reference) class/container the tag belongs to

pszTypeName - (reference) the tag type, (see VS_TAGTYPE_*)

tag_flags - (reference) integer tag flags (see VS_TAGFLAG_*)

pszSignature - (optional reference) arguments/signature of tag

pszReturnType - (optional reference) return type / value of tag

pszTmplSignature - (optional) template signature

Returns: int

0 on success.

Category:

Tagging Functions
