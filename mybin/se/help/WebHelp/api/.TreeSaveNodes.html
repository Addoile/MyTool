void _TreeSaveNodes(TREENODESTATE &state, int index=TREE_ROOT_INDEX)

Save the specified node and all of it's children. This is used in conjunction with _TreeRestoreNodes to save and restore the contents of a tree control to a Slick-C® data structure.

Parameters:

state - (output) filled with state of specified tree node

index - (input) tree node to start from

Applies To:

Tree View

Category:

Tree View Methods

See Also:

_TreeCopy, _TreeRestoreNodes
