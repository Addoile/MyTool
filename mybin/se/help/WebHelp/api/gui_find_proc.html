_command int gui_find_proc()

Searches for a Slick-C® macro tag you specify. Global functions and event tables are tagged. In addition, you can enter a static function name defined in the current file. A dialog box is displayed which allows you to type the tag name in or select from a list of existing tag names. Completion may be used to assist you in entering the name.

Returns: int

Returns 0 if successful.

Applies To:

Edit Window

Category:

Search Functions

See Also:

find_proc, list_tags
