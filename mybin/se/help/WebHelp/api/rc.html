typeless rc

rc is a global variable that is automatically defined by the Slick-C® translator. We have shifted the Slick-C® language away from needing this variable. However, there are still a few functions such as buf_match, and get_env which requires its use for error checking. Many other functions set the rc variable, but we have not documented them to prevent unnecessary use of the rc variable. See also Error handling and the RC Variable.
