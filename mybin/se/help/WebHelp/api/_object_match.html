_str _object_match(_str name, boolean find_first)

Parameters:

name - object name to match

find_first - 'true' to find first, 'false' to find next

Returns: _str

Returns next Slick-C® object name which is a prefix match of name. find_first must be non-zero to initialize matching. Returns '' when no more matches are found.

Category:

Completion Functions
