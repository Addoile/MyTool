int _TreeGetPrevSiblingIndex(int ItemIndex)

Parameters:

ItemIndex - Specifies a tree item.

Returns: int

Returns the tree item index of the previous sibling of the node specified by ItemIndex. -1 is returned if the node specified has no previous sibling.

Applies To:

Tree View

Category:

Tree View Methods

See Also:

_TreeGetParentIndex, _TreeGetNextSiblingIndex, _TreeGetFirstChildIndex
