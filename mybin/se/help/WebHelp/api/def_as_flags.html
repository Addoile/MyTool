int def_as_flags

Auto-save flags. Composed of a bitset of AS_

  • AS_ON -- Auto-save is on
  • AS_ASDIR -- Auto-save to different directory
  • AS_SAMEFN -- Auto-save to same file
  • AS_DIFFERENT_EXT -- Auto-save to Generated extension
  • AS_SAVE_WINDOWS -- Auto-save window configuration

Default:

AS_ASDIR

Category:

Configuration Variables
