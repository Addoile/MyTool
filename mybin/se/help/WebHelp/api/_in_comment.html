boolean _in_comment(boolean in_ml_comment=false)

Determines if the cursor is in a comment

Parameters:

in_ml_comment - If in_ml_comment is true, true is return only if the cursor is inside a multi-line comment. This is useful determining if after pressing ENTER (splitting the line at the cursor) the cursor will be inside a multi-line comment.

Returns: boolean

Returns true if cursor is within a comment and p_lexer_name is not "".

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
