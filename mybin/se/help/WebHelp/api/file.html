_command int file(_str cmdline='', int flags=-1)

Saves current file under name specified and quits the current file. If filename is not specified, buffer name is used. Default options for the file command may be set with the File Options. See _save_file() function for a list of valid options.

Parameters:

cmdline - is a string in the format: options[filename]

Returns: int

Returns 0 if successful. Common return codes are INSUFFICIENT_DISK_SPACE_RC, PATH_NOT_FOUND_RC, ACCESS_DENIED_RC, ERROR_OPENING_FILE_RC, ERROR_WRITING_FILE_RC, DRIVE_NOT_READY_RC, and INVALID_OPTION_RC. On error, message is displayed.

Applies To:

Edit Window

Categories:

Edit Window Methods, File Functions
