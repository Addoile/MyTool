void VSAPI vsCallKey(int wid, int event, const char *pszShowKeys=a, int Reserved=a, int version=a)

This function is used to redirect a key. For example, in our OEM MDI sample program, we redirect client window key presses to send the key press to VSMDI_WID.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

event - An event returned by vsLastEvent or a event constant from "vsevents.h".

pszShowKeys - Previous prefix key message.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Keyboard Functions
