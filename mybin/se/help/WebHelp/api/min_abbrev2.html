_str min_abbrev2(_str word, typeless &word_list, _str options, _str &aliasfilename, boolean findAliases=true, boolean noPrompt=false)

Performs list completion (like '?' key) on partial argument word with valid argument list word_list and the extension specific aliases. The third word of the options argument must be the minimum abbreviation length for list completion to take place. This procedure is typically used to allow syntax expansion to occur when space bar is pressed after an incomplete keyword. For example, typing "m<Space>" while in C mode inserts a template for main(){}.

Parameters:

word - current word under cursor

word_list - array or hash table of words handled by syntax expansion

options - syntax expansion options = name_info(p_index)

aliasfilename - (reference) set to name of alias file

findAliases - Look for aliases and set aliasfilename

noPrompt - Do not prompt. Return semicolon delimited list of matches

Returns: _str

Returns completed word if successful. If the word is from the extension specific alias file, aliasfilename is set to the full path of the alias file. Otherwise aliasfilename is set to "". This function returns "" if word is not a valid partial match, or if the user hit ESCAPE, otherwise,

Applies To:

Edit Window, Editor Control
