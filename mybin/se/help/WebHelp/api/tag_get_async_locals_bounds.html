int tag_get_async_locals_bounds(int &startSeekpos, int &endSeekpos)

Get the start and end seek position bounds for the last completed tagging job. This is only needed, in general, for local variable tagging.

Parameters:

startSeekpos - (output) start seek position

endSeekpos - (output) end seek position

Returns: int

0 on success, <0 on error

Category:

Tagging Functions
