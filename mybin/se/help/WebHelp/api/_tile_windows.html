void _tile_windows(_str option='V')

Resizes and moves MDI edit windows so that they do not overlap.

The option letter must be one of the following:

'V'
(Default) Attempt to tile windows vertically.
'H'
Attempt to tile windows horizontally.
'U'
Until windows (one group of tabs)

Applies To:

MDI Window

Categories:

MDI Window Methods, Window Functions
