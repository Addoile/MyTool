VSDLLEXPORT int SEIsTaggingSupported(const slickedit::SEString &fileName, const slickedit::SEString &defaultLangId)

Parameters:

fileName - Name of file to test

defaultLangId - Language ID of file to test

Returns: VSDLLEXPORT int

Returns 'true' if the given file has support for a tagging callback function or a load-tags callback function.
