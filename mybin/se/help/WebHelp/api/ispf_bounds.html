_command void ispf_bounds(_str cmdline='')

This command is used to set or reset the left and right edit boundaries in ISPF emulation. The left_col and right_col must be integers, or an asterisk may be used in either one's place to specify the current setting. right_col defaults to left_col if not specified.

Parameters:

cmdline - Synax of cmdline: 

   left_col [right_col]

Category:

ISPF Primary Commands

See Also:

ispf_change, ispf_find, ispf_exclude, ISPF Line Commands
