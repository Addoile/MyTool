int tag_set_local_name_location(int local_id, int name_line_no, int name_seekpos)

Set the name and line number positions for the local symbol name with the given local ID.

For synchronization, macros should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

local_id - Local ID [ from 1 to tag_get_num_of_locals() ]

name_line_no - line number that symbol name is located on

name_seekpos - Seek position of the first character of the symbol name

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions

See Also:

tag_insert_local, tag_end_local, tag_get_detail2
