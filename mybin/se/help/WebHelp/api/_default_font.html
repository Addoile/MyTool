_str _default_font(int fieldnum, _str font="")

Gets and optionally sets an editor default font.

Parameters:

fieldnum - is one of the following constants defined in "slick.sh":

  • CFG_CMDLINE
  • CFG_MESSAGE
  • CFG_STATUS
  • CFG_WINDOW_TEXT

font - is a string in the format: font_name, font_size, font_flags

font_size - is a font point size.

font_flags - is a combination of the following flags defined in "slick.sh":

  • F_BOLD
  • F_ITALIC
  • F_STRIKE_THRU
  • F_UNDERLINE

Returns: _str

Returns the current font string value in the same format as the font argument.

Example:

// Set the default command line.
font="Courier,10,"(F_BOLD|F_ITALIC)
_default_font(CFG_CMDLINE, font);
