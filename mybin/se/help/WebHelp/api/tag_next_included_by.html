int tag_next_included_by(_str file_name, _str &included_by)

Retreive the name of next the source file that included (directly or indirectly), the given file (expected to be an include file).

Parameters:

file_name - Full path of file that was included

included_by - (Output) Full path of source file including file_name.

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Category:

Tagging Functions
