void _GetScreen(int &screen_x, int &screen_y, int &screen_width, int &screen_height)

Returns the x, y, width, and height for the monitor this window is on. The monitor is determined by checking the monitor that the form for this window is primarily on.

Parameters:

screen_x - Pixel X coordinate for this monitor

screen_y - Pixel Y coordinate for this monitor

screen_width - Pixel width for this monitor

screen_height - Pixel height for this monitor

Category:

Display Functions
