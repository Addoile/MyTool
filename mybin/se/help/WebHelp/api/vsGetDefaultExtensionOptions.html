int VSAPI vsGetDefaultExtensionOptions(const char *pszExtension, VS_LANGUAGE_OPTIONS *pLangOptions, int version=VS_LANGUAGE_OPTIONS_VERSION)

Deprecated. Use vsGetDefaultLanguageOptions

Gets language specific options for a specific file extension.

Parameters:

pszExtension - File extension. For list of file extension types, use our File Extension Manager dialog ("Tools", "Options...", "File Extension Manager")

pLangOptions - Initialized to language specific options for pszExtension.

version - normally VS_LANGUAGE_OPTIONS_VERSION

Returns: int

Returns 0 if successful.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions

See Also:
