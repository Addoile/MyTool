_command _str current_func_signature(boolean dosticky_message=true)

Returns the signature of the current function. If display_message is true or not specified, the current function signature is displayed on the message line.

Parameters:

dosticky_message - Display message on message line, remember to pass "false" if you are calling this just to get the current proc signature.

Returns: _str

the signature of the current procedure or "" if it can't be found

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
