int VSAPI vsLoadTemplate(int index_or_wid, int parent, int flags, VSLOADDATA *pLoadData, int x=-, int y=-, int width=-, int height=-, int state='N')

Load a Slick-C resource template by index.

Parameters:

index_or_wid - Resource template index or window id of already-created object. If VSLOAD_WINDOW or VSLOAD_REINIT flags are used, then it is the window id of the already-created object. Otherwise it is the resource template.

parent - Parent window id.

flags - Load flags. See VSLOAD_*.

pLoadData - Load data. This includes ON_CREATE arguments passed to the ON_CREATE events of the resource being loaded. Set to 0 if no load data. See VSLOADDATA type.

x - Initial window geometry that overrides what is stored in the template. Only useful when using VSLOAD_MDI_CHILDFORM because you cannot create an MDI child window hidden, move it, then make it visible.

y -

width -

height -

state - Initial window state. 'N'=Normalized, 'M'=Maximized, 'I'=Iconized/minimized. Ignored if not using VSLOAD_MDI_CHILDFORM.

Returns: int

Window id of loaded resource. Returns <0 on error.

Category:

Window Functions
