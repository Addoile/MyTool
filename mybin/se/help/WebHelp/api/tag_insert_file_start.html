int tag_insert_file_start(_str file_name)

Set up for inserting a series of tags from a single file for update. Doing this allows the tag database engine to detect and handle updates more effeciently, even in the presence of duplicates.

Parameters:

file_name - full path of file the tags are located in

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
