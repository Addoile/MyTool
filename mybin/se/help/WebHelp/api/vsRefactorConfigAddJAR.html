int VSAPI vsRefactorConfigAddJAR(const char *configName, const char *jarPath, int jarIndex=-)

Add a jar to the system jars for a configuration

Parameters:

configName - name of configuration to modify

jarPath - jar to add

jarIndex - where to insert jar, -1 to insert at end of list

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
