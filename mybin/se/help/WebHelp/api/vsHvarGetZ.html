VSPSZ VSAPI vsHvarGetZ(HVAR hvar)

Returns: VSPSZ

Returns pointer to Lstring value of interpreter variable corresponding to hvar. Currently, this data format is the most efficient for retrieving interpreter variable data because no format change is required.

Example:

       VSHVAR hvar;
       VSPLSTR plstr;
       char buffer[100];

       // The rc variable is a global variable which always exists.
       index=vsFindIndex("rc",VSTYPE_VAR);
       // Get handle to interpreter variable
       hvar=vsGetVar(index);
       xprintf("result=<%s>",vsHvarGetZ(hvar));

Category:

Miscellaneous Functions
