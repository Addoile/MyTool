_command void hide_code_block()

Hides lines in current code block. The definition of the current block depends on the language. Invoking this command from a key, menu, or button bar multiple times in succession hides larger code blocks.

Applies To:

Edit Window, Editor Control

Categories:

Search Functions, Selective Display
