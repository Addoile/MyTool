void VSAPI vsShiftSelectionRight(int wid, int markid=-, int Reserved=a, unsigned char Reserved2=' ')

Inserts space character at left edge of marked area specified. mark_id is a handle to a selection returned by vsAllocSelection or vsDuplicateSelection. A mark_id of -1 mark_id parameter identifies the active selection.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
