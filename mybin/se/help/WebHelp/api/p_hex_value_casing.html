int p_hex_value_casing

Determines the preferred style of casing for hex value elements for the current buffer. This property is available only when the control is displayed and is valid only in languages where value elements are used (e.g. HTML, XML).

values and respective results:

  1. WORDCASE_LOWER - lowercase
  2. WORDCASE_UPPER - uppercase

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
