int def_diff_edit_options

Difference editor options. Bitset of DIFFEDIT_*.

  • DIFFEDIT_START_AT_TOP -- start at top of file
  • DIFFEDIT_START_AT_FIRST_DIFF -- start at first diff in file
  • DIFFEDIT_CURFILE_INIT -- initialize Diff dialog with current file
  • DIFFEDIT_AUTO_JUMP -- automatically jump to next diff after merge
  • DIFFEDIT_SHOW_GAUGE -- show gauge during diff
  • DIFFEDIT_NO_AUTO_MAPPING -- disable automatic directory mapping in diff dialog
  • DIFFEDIT_AUTO_CLOSE -- automatically close diff dialog after last difference
  • DIFFEDIT_NO_PROMPT_ON_MFCLOSE -- do not prompt when multi-file diff dialog is closed
  • DIFFEDIT_BUTTONS_AT_TOP -- move diff buttons to the top of dialog
  • DIFFEDIT_SPAWN_MFDIFF -- run multi-file diff in background
