_command void prev_sentence()

Used in EMACS emulation. Places the cursor after the end of the current sentence. End of sentence is defined to be one of the character ., !, or ? followed by any number of the characters ., !, ?, ", ', ), or ] followed by end of line or two spaces. Cursor is placed at the first non-blank character of the sentence.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods
