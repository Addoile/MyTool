int _GetSymbolColoringOptions(_str lang='')

Get the symbol coloring options which are a bitset of SYMBOL_COLOR_*.

The options are stored per extension type. If the options are not yet defined for an extension, then use SYMBOL_COLOR_BOLD_DEFINITIONS as the default.

Parameters:

lang - language ID -- see p_LangId

Returns: int

bitset of SYMBOL_COLOR_* options.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Category:

Completion Functions
