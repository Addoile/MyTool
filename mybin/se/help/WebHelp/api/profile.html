_command void profile(_str option="")

This command is used to control the Slick-C® performance profiler.

To activate the profiler, execute the command "profile on". To stop collecting profiling information, execute the command "profile off". To view collected profiling data, execute the command "profile view". This will also stop profiling data collection. To profile a single Slick-C® command, just pass the command and it's arguments to the profile command. For example "profile list-tags".

The profiler has no effect on performance when it is inactive and only a minimal effect on performance when it is collecting data.

Profiling data can be saved and viewed at a later time by using the "profile save" and "profile load" commands, respectively.

The profiler can also be used to profile executing the command tied to a specific key by using the "profile key" command. It will prompt for a key and then execute the corresponding command and display the profiling results.

Profiling results are displayed in a multi-column non-modal tree dialog. Each line in the tree represents one function, either a Slick-C® or an exported DLL function, which was called while we were collecting profiling data. All times are reported in milliseconds. The table columns represent the following information:

  • Function -- name of function called
  • Module -- the name of the module function comes from
  • Offset -- P-code offset of function within module
  • Calls -- number of calls to function
  • F+D Time -- total time in function and descendants
  • Percent -- percentage of total time spent in function and descendants
  • Avg F+D -- average time in function and descendants
  • Min F+D -- minimum time in function and descendants
  • Max F+D -- maximum time in function and descendants
  • Func Time -- total time in function only
  • Percent -- percentage of total time spent in function
  • Avg Time -- average time in function
  • Min Time -- minimum time in function
  • Max Time -- maximum time in function

Parameters:

option - [ on | off | view | load [file] | save [file] | <command> ]
