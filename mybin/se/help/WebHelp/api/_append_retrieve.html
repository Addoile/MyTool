void _append_retrieve(int ctl_wid, _str value, _str ctl_name='')

This command is typically called to add retrieve information for a combo box control. After adding combo box retrieve information, the _retrieve_list method can be called to fill the combo box list. Dialog box retrieval functions call this function to store retrieve data of radio button, command button, check box, text box, and combo box controls.

Parameters:

ctl_wid - the window id (p_window_id) of the control. The default object_name is form_name.ctl_name.

value - the text of a combo box or text box. For a button control, value is 0 or 1.

ctl_name - an override name for the combo box

Example:

defeventtab form1;
ok.lbutton_up()
{
    // When OK button is pressed, you will want to save combo box
    // retrieve information.
    _append_retrieve(_control combo1,combo1.p_text);
}
ok.on_create()
{
     // Fill in the combo box list
     combo1._retrieve_list();
}
