_str b_match(_str name, boolean find_first)

Parameters:

name - buffer name to match

find_first - 'true' to find first, 'false' to find next

Returns: _str

Returns next buffer name which is a prefix match of name. find_first must be non-zero to initialize matching. Returns '' when no more matches are found.

Categories:

Buffer Functions, Completion Functions

See Also:

f_match, dir_match
