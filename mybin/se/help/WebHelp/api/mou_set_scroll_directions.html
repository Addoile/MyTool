void mou_set_scroll_directions(int &past_bottom, int &past_right, int &past_top, int &past_left, int &new_x, int &new_y, int mx, int my)

Set the scroll directions based on the position of the mouse relative to the current window.

Parameters:

past_bottom - Set to 'true' if the mouse is below the window

past_right - Set to 'true' if the mouse is to the right of the window

past_top - Set to 'true' if the mouse is above the window

past_left - Set to 'true' if the mouse is to the left of the window

new_x - Set to horizontal position bounding the mouse within the window

new_y - Set to vertical position bounding the mouse within the window

mx - Horizontal position of mouse

my - Vertical position of mouse

Category:

Mouse Functions
