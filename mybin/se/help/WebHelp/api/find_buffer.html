_command typeless find_buffer(_str buf_name_arg='')

Activates the buffer specified if it exists. If buf_name_arg is not given, user will be prompted for name of buffer to activate.

Parameters:

buf_name_arg - Name of buffer to activate.

Returns: typeless

typeless Returns 0 if successful. Common return codes include FILE_NOT_FOUND_RC (buffer buf_name_arg does not exist).

Category:

Buffer Functions
