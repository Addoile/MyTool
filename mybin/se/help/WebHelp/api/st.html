_command typeless st(_str module_name="")

Compiles the module specified. If no name is specified and the current buffer has the extension e or cmd, the current module is saved (if necessary) and compiled.

Returns: typeless

Returns 0 if successful. A positive return code indicates a return code from the Slick-C® translator meaning there is an error in the source module. A negative return code indicates an error trying to find or execute the Slick-C® translator.
