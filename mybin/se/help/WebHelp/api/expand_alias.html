_command int expand_alias(_str alias_name='', _str insert_space_arg='', _str aliasfilename='', boolean isDocCommentExpansion=false, boolean isAutoExpansion=false)

Ctrl+Space bar

Expands the alias name before the cursor by deleting the alias name and replacing it with its value.

Parameters:

alias_name - Option name of alias

insert_space_arg - If not '', space character is inserted if alias is not found.

aliasfilename - Single alias file to look in.

isDocCommentExpansion - True if this is a special case of a doc comment expansion. False for a regular alias.

isAutoExpansion - true if we are trying to auto-expand an alias through syntax expansion after a user hit space

Returns: int

Returns 0 if successful. Otherwise a non-zero number is returned.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods

alias, alias_cd
