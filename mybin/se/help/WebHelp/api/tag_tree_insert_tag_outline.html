int tag_tree_insert_tag_outline(int tree_wid, int tree_index, int include_tab, int tree_flags, const char *pszTagName, const char *pszTypeName, const char *pszFileName, int line_no, const char *pszClassName=a, int tag_flags=a, const char *pszSignature=a, const char *pszTemplateArgs=a, const char *pszNodeText=a, VSHVAR hvarUserInfo=a)

Internal function for inserting a tag entry with supporting info into the given tree control using the outline view rendering style.

Parameters:

tree_wid - window ID of the tree control

tree_index - parent index to insert item under

include_tab - append class name after signature if 1, prepend class name with :: if 0

force_leaf - if < 0, force leaf node, otherwise choose by type

tree_flags - flags passed to vsTreeAddItem

pszTagName - name of entry

pszTypeName - type of tag, (see VS_TAGTYPE_*)

pszFileName - path to file that is located in

line_no - line number that tag is positioned on

pszClassName - name of class that tag belongs to

tag_flags - tag attributes (see VS_TAGFLAG_*)

pszSignature - arguments and return type

pszTemplateArgs - template signature

hvarUserInfo - per-node user data for tree control

Returns: int

tree index on success, <0 on error.

Category:

Tagging Functions
