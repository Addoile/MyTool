void _delete_temp_view(_str temp_wid='', boolean dodelete_buffer=true)

Deletes temp window and buffer corresponding to temp_wid.

Parameters:

temp_wid - should have been returned by the _create_temp_view function. If temp_wid is not given, current window and buffer are deleted.

dodelete_buffer - If false, the buffer attached to the window is not deleted.

Category:

Selection Functions
