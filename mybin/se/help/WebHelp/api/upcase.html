_str upcase(_str string, int utf8=-1)

Parameters:

string - Input string.

utf8 - -1 specifies that input and output string are UTF-8 if UTF-8 support is enabled. 0 specifies that input and output string are SBCS/DBCS. 1 specifies that input and output string are UTF-8.

Returns: _str

Returns string converted to upper case.

Example:

   upcase('ABc')  == 'ABC'
   upcase('abc')  == 'ABC'

Category:

String Functions

See Also:

lowcase, upcase_selection, upcase_word
