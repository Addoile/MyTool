int _drive_type(_str drive)

(Supported under Windows, Windows 95/98, and Windows NT only) Returns the drive type of the drive specified. drive is a drive letter followed by a colon character. The drive type may be one of the following constants defined in "slick.sh":

DRIVE_NOROOTDIR
Indicates that the drive is not valid.
DRIVE_REMOVABLE
Drive is a floppy drive.
DRIVE_FIXED
Drive is a non-removable drive.
DRIVE_REMOTE
Drive is a network drive.
DRIVE_CDROM
Drive is a CD-ROM
DRIVE_RAMDISK
Drive is a RAM (in memory) disk.

Parameters:

drive -

Returns: int

Example:

defmain()
{
     message("drive type of A: is "_drive_type("A:"));
}
