_command int debug_stop(boolean quiet=true, boolean stay_in_debug_mode=false)

Terminate the debugger.

Parameters:

quiet - Do not issue error messages, just stop debugging.

stay_in_debug_mode - Do not switch out of debug mode. This is used to avoid toolbar switching when restarting the debugger.

Returns: int

0 on success, <0 on error.

Category:

Debugger Commands
