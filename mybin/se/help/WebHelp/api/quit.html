_command int quit(boolean saveBufferPos=true, boolean allowQuitIfHiddenWindowActive=false)

The quit or q command, closes the current file.

When one file per window is on (default), the buffer is closed if the current window is last window displaying the buffer.

When one file per window is off, the current buffer is always closed and all windows are closed if there are no buffers left to display. Before a buffer is deleted, you will be prompted whether you wish to save changes to the buffer if necessary.

Parameters:

saveBufferPos - save the current position in the file

allowQuitIfHiddenWindowActive -

Returns: int

Returns 0 if successful. Possible return codes are 1 (concurrent process buffer must be exited), and COMMAND_CANCELLED_RC. On error, a message is displayed.

Applies To:

Edit Window

Category:

Buffer Functions
