void _delete_text(int Nofbytes, _str option="")

Deletes number of bytes of text starting from cursor location. Specify -1 to delete to end of line. Specify -2 to delete to end of buffer. option may be "C" to delete columns of text. If the "C" option is given, Nofbytes must not be negative and is the number of columns to delete from the current line.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
