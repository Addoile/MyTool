int _get_sel(int &start_pos=0, int &end_pos=0)

Returns: int

Returns the cursor position (same as end_pos). start_pos, if given, is set to the cursor position from which the selection was started. end_pos, if given, is set to the current cursor position. The lowest cursor position is "1".

Applies To:

Text Box, Combo Box

Categories:

Combo Box Methods, Text Box Methods

See Also:

_set_sel, p_sel_start, p_sel_length
