int VSAPI vsQTextWidth(int wid, const char *pText, int TextLen)

Parameters:

wid - Window id of object. 0 specifies the current object.

pText - Text to return width of.

TextLen - Number of characters in pText. -1 means pText is null terminated and length is determined.

Returns: int

Returns the display width of text string specified based on the current font. Return value is in the parent scale mode (p_xyscale_mode).

Applies To:

All Window Objects
