int _TreeCloseEditor(int index, int col, int commitData)

Close the editor (text/combo box) currently displayed in the tree.

Parameters:

index - index of tree node with editor

col - column of tree node with editor

commitData - if not 0, save the data in the editor to the tree.

Returns: int

int 0 if succesful, otherwise INVALID_ARGUMENT_RC

Applies To:

Tree View

Category:

Tree View Methods
