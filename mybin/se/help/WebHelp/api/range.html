typeless range(int start_val, int end_val, int step_by=1)

Parameters:

start_val - start value (inclusive)

end_val - end_value (inclusive)

step_by - step amount, if step is default of 1 and the (start_value > end_value), automatically change step to -1;

Returns: typeless

Return a sc.lang.Range object for iterating over the given range of integers by the specified step amount.

Category:

Miscellaneous Functions
