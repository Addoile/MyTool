int _clex_skip_blanks(_str options='')

Starts from cursor position and skips over spaces, tabs, and comments.

Parameters:

options - may contain one or more of the following option letters:

-
Search backwards
m
Search within selection
h
Search through hidden lines.
c
Skip spaces within comments
q
Quick mode -- does not check for embedded code

Returns: int

Returns 0 if non-blank character is found, nonzero otherwise. If this functions fails the cursor is moved but its final location may not be the top or bottom of the buffer (we need to change this should be more concrete).
