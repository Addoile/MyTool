int VSAPI vsCallbackAdd(int id, VSCALLBACK_AR vscallback, void *pfn, int reserved1=a, int reserved2=a)

Adds a callback function.

Parameters:

id - Buffer id for VSCALLBACK_BUF_??? callbacks. Window id for VSCALLBACK_WIN_??? callbacks. Ignored for all other callbacks.

vscallback - One of the following constants which * correspond to a callback function:

  • VSCALLBACK_AR_BUF_LINES_INSERTED
  • VSCALLBACK_AR_BUF_LINES_DELETED
  • VSCALLBACK_AR_BUF_DELETE_TEXT
  • VSCALLBACK_AR_BUF_INSERT_TEXT
  • VSCALLBACK_AR_BUF_REPLACE_TEXT
  • VSCALLBACK_AR_BUF_LINES_INSERTED2
  • VSCALLBACK_AR_BUF_LINES_DELETED2
  • VSCALLBACK_AR_BUF_TEXT_CHANGE
  • VSCALLBACK_AR_BUF_DRAW_LINE
  • VSCALLBACK_AR_APP_DELETE_BUFFER
  • VSCALLBACK_AR_APP_SELECT_MODE
  • VSCALLBACK_AR_APP_ADD_BUFFER
  • VSCALLBACK_AR_APP_RENAME_BUFFER
  • VSCALLBACK_AR_APP_RENAME_DOCUMENT

pfn - Pointer to callback function. Make sure you remove your callback function with vsCallbackRemove if you unload a DLL which contains your callback function.

Returns: int

Returns 0 if successful. Otherwise NOT_ENOUGH_MEMORY_RC is returned. The break point manager defines VSCALLBACK_AR_BUF_LINES_INSERTED, VSCALLBACK_AR_BUF_LINES_DELETED, and VSCALLBACK_AR_APP_DELETE_BUFFER.

Applies To:

Edit Window, Editor Control

Categories:

Buffer Functions, Edit Window Methods, Editor Control Methods, Miscellaneous Functions

See Also:

vsCallbackSet, vsCallbackRemove
