int VSAPI vsNameReplace(int index, const char *pszName, const char *pszInfo=a, int reserved=a)

Parameters:

index - Index into names table of item.

pszName - New name.

pszInfo - Optional new name information. If null, current name information is preserved. This function is used to rename a names table item.

Returns: int

If successful, returns index. Otherwise a negative error code is returned.

Attention:

This function is thread-safe.

Category:

Names Table Functions
