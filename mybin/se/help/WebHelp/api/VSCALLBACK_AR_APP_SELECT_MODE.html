4 VSCALLBACK_AR_APP_SELECT_MODE

void (VSAPI *pfnSelectMode)(int buf_id,int wid,const char *pszExtension);

The SELECT_MODE callback is called when Slick-C® function _SetEditorLanguage is called. This occurs when vsBufEdit or vsCommandName is called.

This callback is typically used to set the extension specific VSCALLBACK_BUF_SET_LINE_COLOR callback.

Parameters:

wid - Window id of editor control.

pszExtension - Extension corresponding to file extension setup. Note that this extension DOES NOT have to match the actual extension on the buffer.

Category:

Editor Control Callback Functions
