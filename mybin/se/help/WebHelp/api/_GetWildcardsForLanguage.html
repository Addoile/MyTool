_str _GetWildcardsForLanguage(_str lang)

Find all the file name extensions associated with the given language and create a file name wildcard expression for those file types.

Parameters:

lang - Language ID (see p_LangId

Returns: _str

List of wildcards, separated by semicolons corresponding to the file name extensions associated with the given language mode.

Category:

Miscellaneous Functions
