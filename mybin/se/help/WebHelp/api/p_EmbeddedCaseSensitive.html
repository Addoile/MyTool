_str p_EmbeddedCaseSensitive

Determines embedded language case sensitivity. This returns the case sensitivity property for the embedded language lexer or if the cursor is not in an embedded language context, it will return the same thing as p_LangCaseSensitive.

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties

See Also:

p_embedded, p_EmbeddedLexerName, p_LangCaseSensitive
