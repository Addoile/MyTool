void*VSAPI vsRealloc(void *pBuf, size_t len)

Reallocates memory.

Parameters:

pBuf - Null pointer or pointer allocated by vsAlloc.

len - New allocation size for pBuf.

Returns: void*VSAPI

If successful, resizes the memory allocated by vsAlloc to allow up to len bytes and returns a pointer to the newly allocated memory. pBuf may be 0 to allocate new memory. Otherwise 0 is returned. The new memory is initialized to the previous value if any pointed to by pBuf. Use the vsFree function to free memory allocated by this function.

Category:

Miscellaneous Functions
