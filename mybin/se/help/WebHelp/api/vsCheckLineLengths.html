int VSAPI vsCheckLineLengths(int wid, int AllowedLineLen, char *pszLineNumbers=a, int MaxLineNumbersLen=a, int FromCursor=a, int *pMaxLineLen=a, int CountTrailingBlanks=a, int reserved2=a)

Parameters:

wid - Window id of editor control. 0 specifies the current object.

AllowedLineLen - Lines lengths greater than this length are flagged.

pszLineNumbers - (Ouput only) ASCIIZ string. Set to a space delimited list of line numbers of lines which are too long. Lines with the VSLF_NOSAVE flag set are ignored.

MaxLineNumebersLen - Number of characters allocated to pszLineNumebers.

FromCursor - Set this to zero to check lines at and after the cursor. Otherwise, the entire buffer is checked.

pMaxLineLen - (Output only) Set to the length of the longest line.

Returns: int

Returns 0 if there are no lines longer than the AllowedLineLen specified. At the moment only the physical line length is checked as if tab characters count as 1 character. We may change this in the future.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
