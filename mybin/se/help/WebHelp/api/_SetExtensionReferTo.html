void _SetExtensionReferTo(_str extension, _str lang)

Allows you to specify that a file extension is associated with the specified language. This information is used to determine what language mode to choose when a file with the given extension is opened.

Parameters:

extension - File extension. For list of file extension types, use our Options dialog ("Tools", "Options...", "File Extension Manager").

lang - File language ID (see p_LangId). For list of language types, use our Language Options dialog ("Tools", "Options...", "Language Manager")

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions

See Also:

_SetDefaultLanguageOptions, _DeleteExtensionOptions, _DeleteLanguageOptions
