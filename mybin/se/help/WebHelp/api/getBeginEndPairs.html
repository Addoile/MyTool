_str getBeginEndPairs(_str langID, _str defaultValue=null)

Gets the Begin/End pairs for this language. This value is available on the GUI on the General form for each applicable language.

Parameters:

langID - language ID (see p_LangId)

defaultValue - if the value is not available, then we just return this value. If this parameter is not specified, then the default value for the overall def-var is used.

Returns: _str

begin/end pairs value for this language

Category:

LanguageSettings API
