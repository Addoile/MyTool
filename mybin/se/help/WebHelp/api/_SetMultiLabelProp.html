void _SetMultiLabelProp(int index, _str caption, int styles, int min_width, int max_width, int alignment_flags)

This method allows you to subdived a single label control into multiple labels with borders and various options. index starts at 0 and indicates which label you are setting the width, flags, and state for. You must initialially create multiple labels in order (that is 0,1,2... etc.).

Example:

// Create a form with a label control named label1
void label2.on_create()
{

}
