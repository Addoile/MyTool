boolean _dbcsIsLeadByte(_str ch)

Returns: boolean

Returns true if the 1 byte length string ch is a valid first character of a double byte character (DBCS). Always return false, if the operating system is not in Japanese mode.

Example:

        // IF there are at least two bytes left in the current line and the first byte is
        // a dbcs lead byte.
        if (p_col<_text_colc() && _dbcsIsLeadByte(get_text())) {
             p_col+=2;  // Move cursor over double byte character
        } else {
             p_col+=1;  // Move cursor over single byte character
        }
