typeless& _nextel(typeless &index)

_nextel() is the Slick-C intrinsic function behind the Slick-C "foreach" statement.

Parameters:

index - index / key to find next duplicate of

Returns: typeless&

Returns reference to next element in array or hash table. If index is empty (see _makeempty()), the first element is returned. If there are no more elements, index is set to empty. See example below.

For Slick-C classes that implement IIterable, _nextel() will call the class-specific _nextel() method.

For strings, _nextel() will parse out the next (possibly quoted) filename in the string, stripping whitespace. It will not strip quotes.

Example:

t:["a"]=1;
t:["b"]=2;
t:["c"]=3;
// Traverse the elements in hash table
for (i._makeempty();;) {
    t._nextel(i);
    if (i._isempty()) break;
    messageNwait("index="i" value="t:[i]);
}
// Traverse the elements using 'foreach'
foreach (i => v in t) {
    messageNwait("index="i" value="v);
}
