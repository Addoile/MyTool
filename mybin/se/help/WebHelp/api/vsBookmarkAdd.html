int VSAPI vsBookmarkAdd(const char *pszBookmarkName, int markid, int vsbmflags=VSBMFLAG_SHOWTOOLTIP | VSBMFLAG_STANDARD | VSBMFLAG_SHOWPIC, seSeekPos RealLineNumber=seSeekPos(-1), int col=a, seSeekPos BeginLineROffset=seSeekPos(0), char *pszLineData=a, char *pszFilename=a, char *pszDocumentName=a)

Adds a new bookmark.

Parameters:

pszBookmarkName - Name of bookmark to add.

markid - Selection id returned from vsAllocSelection or vsDuplicateSelection.

vsbmflags - Combination of VSBMFLAGS_??? ORed together.

RealLineNumber - Real line number of bookmark. Specify -1 if markid is an active bookmark.

col - Column of bookmark. Specify 0 if markid is an active bookmark.

BeginLineROffset - Real offset to beginning of line. Imaginary line data not counted. Specify 0 if markid is an active bookmark.

pszLineData - Text on the line of the bookmark. Specify 0 if markid is an active bookmark.

pszFilename - Filename the bookmark is in. Specify 0 if markid is an active bookmark.

pszDocumentName - Document name the bookmark is in. Specify 0 if markid is an active bookmark.

Category:

Bookmark Functions

vsBookmarkAdd, vsBookmarkRemove, vsBookmarkRestore, vsBookmarkGetInfo, vsBookmarkFind, vsBookmarkQCount
