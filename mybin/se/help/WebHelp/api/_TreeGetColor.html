void _TreeGetColor(int nodeIndex, int col, int &FGColor, int &BGColor, int &Flags)

Get color for a "cell", on column in a tree control

Parameters:

int - nodeIndex Index to set the color for

int - col Column (0..N-1) to set the color for

int - Gets the FGColor RGB color for text in this cell

int - Gets the BGColor RGB color for background in this cell

int - Gets the flags for this cell

Applies To:

Tree View

Category:

Tree View Methods
