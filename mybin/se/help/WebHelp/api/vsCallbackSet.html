void VSAPI vsCallbackSet(int id, VSCALLBACK vscallback, void *pfn)

Sets a callback function.

Parameters:

id - Buffer id for VSCALLBACK_BUF_??? callbacks. Window id for VSCALLBACK_WIN_??? callbacks. Ignored for all other callbacks.

vscallback - One of the following constants which correspond to a callback function:

  • VSCALLBACK_WIN_UPDATE_EDITOR_STATUS
  • VSCALLBACK_WIN_GOTFOCUS
  • VSCALLBACK_WIN_LOSTFOCUS
  • VSCALLBACK_WIN_CMDLINE_FUNCTIONS
  • VSCALLBACK_WIN_MESSAGE
  • VSCALLBACK_WIN_GET_MESSAGE
  • VSCALLBACK_WIN_FONT_CHANGE
  • VSCALLBACK_APP_QUERY_END_SESSION
  • VSCALLBACK_APP_MENU_ADD_FILE_HISTORY
  • VSCALLBACK_APP_MENU_ADD_WORKSPACE_HISTORY
  • VSCALLBACK_APP_POPUP_MENU
  • VSCALLBACK_APP_POPUP_MENU2
  • VSCALLBACK_APP_ADD_BUFFER
  • VSCALLBACK_APP_RENAME_BUFFER
  • VSCALLBACK_BUF_QDEBUGBITMAP
  • VSCALLBACK_BUF_SET_LINE_COLOR
  • VSCALLBACK_APP_READ_ONLY_ERROR
  • VSCALLBACK_APP_KEY_OR_MOUSE_EVENT_READ
  • VSCALLBACK_APP_BPMQ_DEFAULT_BITMAP
  • VSCALLBACK_APP_MESSAGEBOX
  • VSCALLBACK_APP_DISPATCH_MESSAGE
  • VSCALLBACK_APP_MODAL_WAIT
  • VSCALLBACK_APP_TAG_FILE_REFRESH
  • VSCALLBACK_APP_TAG_FILE_ADD_REMOVE
  • VSCALLBACK_APP_TAG_FILE_MODIFIED
  • VSCALLBACK_APP_MFFIND_ACTIVATE_TOOLBAR_ AND_SEARCH_TAB
  • VSCALLBACK_APP_MFFIND_IS_TOOLBAR_VISIBLE
  • VSCALLBACK_APP_MFFIND_QEDITOR_CTL_WID
  • VSCALLBACK_APP_MFFIND_CLOSE_TOOLBAR
  • VSCALLBACK_APP_DELETE_WINDOW
  • VSCALLBACK_APP_REFRESH_BREAKPOINTS
  • VSCALLBACK_APP_BREAKPOINT_LINE_DELETED

pfn - Pointer to callback function or 0 to remove callback function. Make sure you remove your callback function with vsCallbackSet(id,vscallback,0) if you unload a DLL which contains your callback function.

Applies To:

Edit Window, Editor Control

Categories:

Buffer Functions, Edit Window Methods, Editor Control Methods, Miscellaneous Functions

See Also:

vsCallbackAdd, vsCallbackRemove
