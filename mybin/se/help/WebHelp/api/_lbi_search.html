_str _lbi_search(_str &last_data, _str text, _str case_sense="", ...)

Searches for an item in the list box which starts with the string, text. Set last_data to '' to initialize or reinitialize searching. If you modify any items in the list box, you need to set the last_data input variable to ''. Otherwise, on subsequent calls to this function, pass in the previous last_data value for fastest search. When the last_data variable is '', searching always begins from the top of the list box which can be slow if there are many items in the list box.

Parameters:

options - is a string of zero or more of the following letters:

E
Match case exactly
I
(Default) Ignore case
R
Interpret text as a SlickEdit regular expression.
U
Interpret text as a UNIX regular expression.
B
Interpret text as a Brief regular expression.

Returns: _str

Returns 0 if match is found

Applies To:

List Box

Category:

List Box Methods
