int _syshelp(_str filename, _str keyword, int help_option=-1)

Displays help. This function is not available in the UNIX version.

Parameters:

option - may be one of the following constants defined in "slick.sh":

HELP_FORCEFILE
Load help_filename specified. If the help file is already loaded, the help window receives the focus. help_item is ignored.
HELP_CONTENTS
Display table of contents help item of help_filename. help_item is ignored.
HELP_KEY Display help on help_item from help file help_filename.
HELP_PARTIALKEY
Displays Search dialog box which lists help items and allows you to enter a help item. This constant is supported by Windows only and is translated to HELP_KEY for X Windows.

Windows, Windows 95/98, and Windows NT help files have the extension ".HLP". help_filename specifies one or more help files separated with plus signs.

Returns: int

Returns 0 if successful.

Example:

// Windows example.  Display help on SetFocus function.
_syshelp("win31wh.hlp+mscxx.hlp", "SetFocus");
