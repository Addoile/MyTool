int VSAPI SETagSetFileDate(int dbHandle, const slickedit::SEString &fileName, const VSINT64 fileDate=0, int fileType=0, const slickedit::SEString &includedBy=(const char*) 0)

Set the date of tagging for the given file. If the date is not specified, use the file modification time found on disk. This function has the side effect of positioning the file iterator on the given file name.

Parameters:

dbHandle - database handle

fileName - File path of source file to setting tagging date for.

fileDate - (20 bytes, optional) Modification date of file when tagged. Format is YYYYMMDDHHMMSSmmm (year, month, day, hour, minute, second, ms)

fileType - Type of file to search for, source vs. reference

includedBy - (optional) File path of source file which includes pszFilename

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if the file is not found.

Category:

Tagging Functions
