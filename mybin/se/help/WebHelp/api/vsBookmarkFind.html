int VSAPI vsBookmarkFind(const char *pszBookmarkName, int vsbmflags=VSBMFLAG_STANDARD)

Finds bookmark index.

Parameters:

pszBookmarkName - Name of bookmark to find. vsbmflags One or more of these flags must be in the bookmark found.

Returns: int

Returns index of boomark 0..vsBookmarkQCount() or -1 to indicate the bookmark was not found.

Category:

Bookmark Functions
