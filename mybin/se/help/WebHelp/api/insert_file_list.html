int insert_file_list(_str cmdline)

Inserts a file list of the files specified with columns for size, date, time, attributes, and name.

Parameters:

cmdline - has the following syntax: 

     {filespec  | ['-' | '+' option_letters] [-exclude exfiles]}

option_letters - may be 'H','S','A','D','P','T','V' with the following meanings:

H
Include hidden files (defaults to OFF)
This option is always turned on under Windows if the "Show all files" explorer option is set. On UNIX, this option is ignored.
S
Include system files (defaults to OFF)
This option is always turned on under Windows if the "Show all files" explorer option is set.
A
Include archive files (defaults to OFF)
D
Include directory files (default to OFF)
P
Append path (defaults to OFF)
T
Tree file list (defaults to OFF)
U
UNIX dot files (defaults to ON)
V
Verbose output with size, date, attributes. (defaults to ON)
G
List registered data sets. This option is ignored for all platforms except OS/390.
N
Stat of data sets. By default, no trailing slash is appended to partitioned data sets (directories). We have this default mainly due to the fact that this feature can cause a hang. Use this option when you have to differentiate files from directories. This option is ignored for all platforms except OS/390.
Z
Return data set member info in the same format as ISPF. This option is ignored for all platforms except OS/390.

filespec - may contain operating system wild cards such as '*' and '?'. If filespec is not specified, current directory is used.

exfiles - may contain operating system wild cards such as '*' and '?'. Specifying a directory will exclude a named directory and contained subdirectories

Returns: int

Returns 0 if successful. Common return codes are:

  • FILE_NOT_FOUND_RC
  • PATH_NOT_FOUND_RC
  • NO_MORE_FILES_RC
  • ACCESS_DENIED_RC

Example:

insert_file_list('\*.*");  // List just root directory with directories.
insert_file_list('+tp \*.*"); // List all files.  Append paths.
insert_file_list('-v *.*');   // List files in current directory.  Just
file names.
insert_file_list('+tp \*.* -exclude *\CVS\'); //List all file, append paths, excluding "CVS" subdirectories
insert_file_list('+tp \*.* -exclude *.txt'); //List all files, append paths, excluding .txt files

Applies To:

Edit Window, Editor Control, List Box

Categories:

Edit Window Methods, Editor Control Methods, File Functions, List Box Methods
