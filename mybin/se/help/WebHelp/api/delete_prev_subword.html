_command void delete_prev_subword()

Same as delete_prev_word, except ignores the def_subword_nav setting and deletes from the cursor to the beginning of the previous subword.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods

See Also:

delete_prev_word, delete_prev_full_word
