int tag_set_context_template_signature(int local_id, _str template_args)

Set the template signature for the given context tag.

For synchronization, macros should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

context_id - Context ID [ from 1 to tag_get_num_of_context() ] of context tag representing class or struct to set inheritance relationships for.

template_signature - Template signature for the tag, in language specific format, normally delimited by commas.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
