int tag_find_friends_of_class(_str class_name, _str(&tag_files)[], _str &friend_list)

Find all the friends of the given class in the current context and tag files.

Parameters:

class_name - class to find friends of

tag_files - List of tag files to search

friend_list - (reference) set to list of friends, separated by VS_TAGSEPARATOR_parents

Returns: int

0 on success, <0 on error. Having no friends is considered as a success.

Category:

Tagging Functions
