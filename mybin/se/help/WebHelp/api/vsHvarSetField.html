int VSAPI vsHvarSetField(VSHVAR hvar, int index, VSHVAR value)

Modify the contents of the given non-static data member of a class or struct.

Parameters:

hvar - handle for interpreter variable

index - index >=0 of non-static data member

value - value to set field to

Returns: int

0 on success. Returns INVALID_ARGUMENT_RC if the field index is out of range. Returns INVALID_HVAR_ARGUMENT_RC if 'hvar' is not a struct or class instance.

Category:

Miscellaneous Functions

See Also:

vsHvarFieldIndex, vsHvarSetFieldByName
