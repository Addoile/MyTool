int VSAPI vsRefactorCStandardMethods(int handle, const char *filename, const char *symbolName, seSeekPos startSeekPos, seSeekPos endSeekPos, char **className, int &existingClassMethods)

Analyzes a symbol to determine it's class and determine which of the standard methods are already implemented for that class.

Parameters:

handle - Handle to transaction

filename - Filename to find symbol in

symbolName - Name selected symbol

startSeekPos - Start seek position of symbol

endSeekPos - End seek position of symbol

className - Returns the name of the class that symbolName refers to

existingClassMethods - Returns flags indicating which methods are already implemented in the class. Uses bitset of VSREFACTOR_METHOD_* (above)

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
