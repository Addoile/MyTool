int tag_list_occurrences(_str file_name, _str tag_name=null, boolean case_sensitive=true, int start_seekpos=0, int stop_seekpos=0)

Default function for matching occurrences and inserting them into the database. Simply searches for words that are not in comments, strings, numbers, keywords, line numbers, or preprocessing, using the color coding engine.

Parameters:

file_name - Name of buffer to search for occurrences in

tag_name - Name of identifier to search for, null of anything

case_sensitive - Case sensitive identifier search?

start_seekpos - Seek position to start searching at, 0 means TOP

stop_seekpos - Seek position to end searching at, 0 means EOF

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
