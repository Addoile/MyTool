int _print(_str sysmessage="", _str reserved="", void(*pfnCallback)(int reason, _str caption="")=0, _str font="", _str lheader="", _str lfooter="", _str options="", _str cheader="", _str cfooter="", _str rheader="", _str rfooter="")

Prints the contents of the current buffer according to the options given. This function is not supported by the UNIX version.

It is much easier to create input parameters to this function by using macro recording. Use the Print Setup dialog box to set your options. Then print a document while macro recording is on.

Parameters:

sysmessage - is a description given to the operating system which identifies this particular print job. Usually this is the buffer name.

reserved - should be '' to avoid compatibility problems.

callback_index - is a pointer to a function or a names table index to a global function which is called with PRINT_ONINIT to initialize, and PRINT_ONEXIT to uninitialize. The callback function can cancel printing by calling the _print_cancel function.

font - is a string in the format: font_name, font_size, font_flags[,]

font_flags - is a combination of the flags below:

  • F_BOLD
  • F_ITALIC
  • F_STRIKE_THRU
  • F_UNDERLINE
  • F_PRINTER

lheader, - lfooter, cheader, cfooter, rheader, and rfooter are strings which may have escape sequences with the following meaning embedded into them:

%f
Buffer name
%p
Page number
%d
Date

The letters 'l', 'c', and 'r' correspond to the header or footer at the left, center, and right of the page.

The header is printed at the top of each page. The footer is printed at the bottom of each page.

options - is a string in the format: left_ma, AfterHeader_ma, right_ma, BeforeFooter_ma, space_between_ma, print_flags, linenums_every ,top_ma, bottom_ma

Windows: The left_ma, top_ma, right_ma, and bottom_ma parameters are margin space from the outside edge of the paper to the printed text. AfterHeader_ma is the space in inches between the header and the first line on the page. BeforeFooter_ma is the space in inches between the last line on a page and the footer.

space_between_ma is the spacing in twips between columns. This parameter is only used if the print_flags have the PRINT_TWO_UP flag.

print_flags is a combination of the flags below:

  • PRINT_LEFT_HEADER
  • PRINT_RIGHT_HEADER
  • PRINT_CENTER_HEADER
  • PRINT_LEFT_FOOTER
  • PRINT_RIGHT_FOOTER
  • PRINT_CENTER_FOOTER
  • PRINT_TWO_UP

The PRINT_LEFT_HEADER, PRINT_RIGHT_HEADER, and PRINT_CENTER_HEADER flags have no effect if the cheader, cfooter, rheader, and rfooter parameters are given.

Returns: int

Returns 0 if successful.

Applies To:

Edit Window
