int VSAPI vsSetDefaultColor(int cfgfield, int FGColor, int SetFGColor, int BGColor=a, int SetBGColor=a, int FontFlag=a, int SetFontFlag=a, int cfgParentColor=a, int SetParentColor=a)

Gets rgb color values (always in Windows format) and font style flag for the cfgfield specified.

Parameters:

cfgfield - One of the VSCFG_??? constants or a color index allocated by vsAllocColor.

FGColor - New rgb foreground color.

SetFGColor - Indicates whether to use FGColor as new foreground color.

BGColor - New rgb background color.

SetBGColor - Indicates whether to use BGColor as new foreground color.

FontFlag - Set to one of the VSFONTFLAG_?? constants or 0.

SetFontFlag - Indicates whether to use FontFlag as new foreground color.

cfgParentColor - One of the VSCFG_??? constants or a color index allocated by vsAllocColor

SetParentColor - Indicates whether to use cfgParentColor as new parent color id.

Returns: int

Returns 0 if successful.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions

See Also:

vsAllocColor, vsFreeColor, vsSetTextColor, vsSetDefaultColor
