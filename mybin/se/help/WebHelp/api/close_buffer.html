_command int close_buffer(boolean saveBufferPos=true, boolean allowQuitIfHiddenWindowActive=false)

Prompts the user to save changes and deletes the current buffer. The previous buffer is displayed in the current window. This function is not affected by the One File per Window configuration option.

Returns: int

Returns 0 if the buffer was deleted. Otherwise, a non-zero number is returned.

Applies To:

Edit Window

Category:

Buffer Functions
