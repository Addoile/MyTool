int dbg_session_update_stack(int session_id, int thread_id)

Update the stack trace for the indicated thread.

Parameters:

session_id - debugger session ID (1 .. max_session)

thread_id - Thread ID to update stack for

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

   int thread_id = dbg_get_cur_thread();
   int status = dbg_session_update_stack(session_id, thread_id);
   if (status) {
      return status;
   }

Category:

Debugger Functions
