int _xmlcfg_copy(int DestHandle, int DestNodeIndex, int SrcHandle, int iSrcNodeIndex, int flags)

Duplicates a tree

Parameters:

DestHandle - Handle of destination tree

DestNodeIndex - Index of destination node

SrcHandle - Handle to source tree

iSrcNodeIndex - Index of source node

flags - Specifies one of the following flags:

VSXMLCFG_COPY_CHILDREN
copy children of source as the last children of the dest
VSXMLCFG_COPY_AS_CHILD
copy the source as the last child of the dest
VSXMLCFG_COPY_BEFORE
copy the source as a sibling before the dest
Specify no flags to copy as a sibling after dest. Note that this function can not copy children as siblings. Use the _xmlcfg_copy_children_as_siblings() function to perform this operation.

Returns: int

Returns first node created or VSRC_XMLCFG_NO_CHILDREN_COPIED if successful. Otherwise, an negative error code other than VSRC_XMLCFG_NO_CHILDREN_COPIED is returned.

Example:

// Add tree as last child
_xmlcfg_copy(DestHandle,DestNodeIndex,
   SrcHandle,SrcNodeIndex,VSXMLCFG_COPY_A_CHILD);

// Add tree before dest node
_xmlcfg_copy(DestHandle,DestNodeIndex,
   SrcHandle,SrcNodeIndex,VSXMLCFG_COPY_BEFORE);

// Add tree after dest node
_xmlcfg_copy(DestHandle,DestNodeIndex,
   SrcHandle,SrcNodeIndex);

Category:
