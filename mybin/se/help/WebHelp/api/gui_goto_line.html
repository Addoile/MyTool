_command int gui_goto_line()

Places the cursor on a line number you specify. A dialog box is displayed which prompts you for a line number.

Returns: int

Returns 0 if successful

Applies To:

Edit Window, Editor Control

Categories:

Editor Control Methods, Edit Window Methods, CursorMovement Functions, Search Functions
