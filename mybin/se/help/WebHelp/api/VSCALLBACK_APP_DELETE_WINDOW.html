32 VSCALLBACK_APP_DELETE_WINDOW

void (VSAPI *pfnDeleteWindow)(int wid,int modal)
This callback gets called before a dialog window gets deleted. It may also get called for a control window that gets deleted.

Parameters:

wid - Window id of the window being deleted.

modal - Non-zero the the window id is for a form about to exit a modal wait state.

Category:

Editor Control Callback Functions
