int VSAPI vsRefactorConfigGetType(int configIndex, char *configType, int maxLength)

Retrieve the type of the given configuration. Currently will return "c" for C/C++ and "java" for Java.

Parameters:

configIndex - index of configuration to query name of [0..n-1]

configName - [reference] set to name of configuration

maxLength - length of the configName buffer

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions

See Also:

vsRefactorConfigAdd, vsRefactorConfigCount
