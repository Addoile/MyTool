int dbg_get_breakpoint(int breakpoint_id, int &count, _str &condition, _str &thread_name, _str &class_name, _str &method_name, _str &file_name, int &line_number, boolean &enabled, _str &address)

Get the details about a breakpoint.

Parameters:

breakpoint_id - index of variable in list of breakpoints

count - (reference) number of times to skip breakpoint -1 indicates that it is a one-time breakpoint

condition - (reference) expression for conditional breakpoint

thread_name - (reference) name of thread breakpoint is restricted to

class_name - (reference) name of class breakpoint is restricted to

method_name - (reference) name of method breakpoint is set in

file_name - (reference) name of file breakpoint is in

line_number - (reference) line number breakpoint occurs at

enabled - (reference) is the breakpoint enabled or disabled

address - (reference) hex address to search from breakpoint matching

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
