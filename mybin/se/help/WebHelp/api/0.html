_command void 0(_str LineNumber='')

The 0 command is called if the first character of a command is a digit. The first parameter to this special function is the entire command line including the first digit. This command moves the cursor to the specified line number. The leading digit 0 zero is not required. Any line number may be typed on the command line and the (_command) for '0' will be called.

Parameters:

LineNumber -

Applies To:

Edit Window, Editor Control

Example:

Command Line Example

100
Moves cursor to line 100.

Category:

CursorMovement Functions
