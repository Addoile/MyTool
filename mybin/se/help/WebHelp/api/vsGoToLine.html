int VSAPI vsGoToLine(int wid, seSeekPos LineNumber, int OnlyCountRealLines=a, int doCenterScroll=a)

Searches for line specified and places the cursor on the line.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

LineNumber - Indicates the old line number to find.

OnlyCountRealLines - When non-zero, only real lines are counted. The "real line number" does not count non-savable lines (that is lines with the VSLF_ NOSAVE flag).

doCenter - When non-zero, if scrolling is necessary to make the destination line visible, the line is scrolled to the center of the screen.

Returns: int

Returns 0 if successful. If the line specified is past the end of the file, BOTTOM_OF_FILE_RC is returned and the cursor is placed at the end of the file. Other possible error codes are INVALID_OBJECT_HANDLE_RC and PROPERTY_OR_METHOD_NOT_ALLOWED_RC.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
