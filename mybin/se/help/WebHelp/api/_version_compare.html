int _version_compare(_str ver1, _str ver2, int resolution=-1)

Compares two version numbers in the form of x.x.x.x(...) to see which is greater.

Parameters:

ver1 -

ver2 -

resolution - how far in the comparison to go, use -1 to compare the whole numbers. See example.

Returns: int

int 0 indicates versions are equal, int > 0 means ver1 is greater, int < 0 means ver2 is greater

Example:

_version_compare('15.0.0.5', '15.0.0.6');          // returns -1
_version_compare('15.0.0.5', '15.0.0.5');          // returns 0
_version_compare('15.0.0.7', '15.0.0.6');          // returns 1
_version_compare('15.0.0.7', '15.0.0.6', 3);       // returns 0, as the first three sections of each are equal

Category:
