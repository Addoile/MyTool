_command typeless dos_copy(_str cmdline='')

Passes text to the operating system copy (UNIX: cp) command. See your operating system manual for argument syntax.

In ISPF emulation, this command is not called when invoked from the command line. Instead ispf_copy is called. Use dos_copy to explicitly invoke the copy command.

Returns: typeless

Return code depends on operating system. Use the copy_file built-in for predictable and portable results.

Category:

File Functions
