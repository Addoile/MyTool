int tag_check_for_enum(_str symbol, _str class_name, _str(&tag_files)[], boolean case_sensitive)

Lookup 'symbol' and see if it could be an enumerated type. If so, return 1, otherwise return 0.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Parameters:

symbol - symbol to investigate

class_name - namespace or class to search

tag_files - (reference to _str[]) tag files to search

case_sensitive - use case-sensitive comparisons?

Returns: int

1 if 'symbol' could be an enum, 0 otherwise, <0 on error.

Category:

Tagging Functions
