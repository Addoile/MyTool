boolean _LanguageReferencedIn(_str ref_lang, _str lang=null)

Does the current source language have the given language in it's list of languages which symbols can be referenced in?

If 'lang' is not specified, the current object must be an editor control.

Parameters:

ref_lang - language ID to check reference in

lang - current language ID (default=p_LangId)

Returns: boolean

'true' if the language matches, 'false' otherwise.

Since:

17.0

Applies To:

Edit Window, Editor Control

Category:

Tagging Functions
