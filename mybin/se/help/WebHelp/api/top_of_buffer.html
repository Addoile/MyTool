_command void top_of_buffer()

Places cursor at first line and first column of buffer. If the "Preserve Column on Top/Bottom" option is on, the cursor is placed at the top of the buffer and the column positon is unchanged.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods
