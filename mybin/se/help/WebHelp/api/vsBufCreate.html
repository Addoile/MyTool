int VSAPI vsBufCreate(const char *pszTempOption="+t", const char *pszLoadOptions=a, int reserved1=a, long reserved2=a)

Parameters:

pszTempOption - May be "+t", "+tu", "+td", or "+tm" which corresponds to current OS, UNIX, DOS, or Macintosh file format.

pszLoadOptions - Specifies options that occur before pszTempOption. See vsLoadFiles function for a list of options. It is not likely you will need to use any options except possibly in a strange macro.

Returns: int

Creates an empty buffer and returns the buffer id if successful (0 is a valid buffer id). Otherwise a negative error code is returned.
