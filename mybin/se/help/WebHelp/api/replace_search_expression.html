_command void replace_search_expression(_str name='', boolean go=false)

Replace text in buffer using search string, replace string, and search options stored from an entry in the saved search expressions list.

Parameters:

name - Saved search expression name. If no name is passed in, a selection list is displayed to pick from the list of saved expressions.

go - Replace all without prompt.

Applies To:

Edit Window, Editor Control @categories Search_Functions
