char*VSAPI vsUTF8CharNext(const char *pBuf, const char *pEndBuf)

Parameters:

pCurrent - Buffer contain UTF-8 text.

pEndBuf - Pointer to end of pBuf text where pEndBuf-pCurrent is the buffer length.

Returns: char*VSAPI

Returns pointer to the beginning of the next composite character or pEndBuf if pCurrent==pEndBuf.

Category:

Unicode Functions
