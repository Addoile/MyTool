_str p_rawpos

Returns 'A' if the current buffer buffer contains SBCS/DBCS data and that SlickEdit is running in UTF-8 mode. Otherwise, '' is returned. This property is read-only. This property is intended for use with the pos(), lastpos(), and parse functions in order to simplify supporting Unicode and SBCS/DBCS mode buffers.

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
