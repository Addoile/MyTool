int refactor_c_push_down(int handle, _str filename, _str className, int beginSeekPos, int endSeekPos, _str derivedClass, _str membersToMove, _str &memberWorkingOn, _str class_def_info_list, _str origClassDefFiles)

Push Down class member(s) to a derived class

Parameters:

handle - Handle to transaction

className - Name of class

filename - Name of file containing class

beginSeekPos - Seek position of beginning of class reference

endSeekPos - Seek position of end of class reference

derivedClass - Name of superclass to move symbol to

membersToMove - Class members to move to super class

memberWorkingOn - (out)This is used for errors. This is the member being worked when error occurs.

classDefInfoList - Classes and Files to move members to. Format: NumOfClasses @ className0 @ classFile0 @ classNameN @ classFileN @

origClassDefFiles - Files that contain the original class's function definitions and static initializers Format: NumOfFiles @ defFile0 @ defFileN @ membersToMove is in the form: number of members [@] member [@] member [@] ...

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
