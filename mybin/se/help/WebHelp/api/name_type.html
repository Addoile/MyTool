int name_type(int index)

Returns: int

Returns an integer name type corresponding to name table entry index. If index is not an index to a valid name, 0 is returned. All valid types are defined in "slick.sh"

Example:

         index= find_index("find_next");
         message('find_next name type is 'name_type(index));

Category:

Names Table Functions
