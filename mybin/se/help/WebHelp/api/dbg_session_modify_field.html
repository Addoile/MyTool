int dbg_session_modify_field(int session_id, _str field_path, _str new_value)

Modify the value of the specified field of the specified class.

Parameters:

session_id - debugger session ID (1 .. max_session)

field_path - path to field to modify

new_value - value to set variable to

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

To change the value of the the field with ID 8 under the class with ID 22 to the integer value 48: 

  int status = dbg_session_modify_field(session_id, "8 22", "48");
  if (status) {
     return status;
  }

Category:

Debugger Functions
