int dbg_find_breakpoint(_str file_name, int line_number, boolean &enabled, int start_id=0, _str address=null)

Find the breakpoint at the given file and line number.

Parameters:

file_name - name of file breakpoint is in

line_number - line number breakpoint occurs at

enabled - (reference) is the breakpoint enabled or disabled

start_id - (optional) breakpoint ID to start searching at, default 0

address - (optional) hex address to search for breakpoint matching

Returns: int

breakpoint ID > 0 on success, <0 on error.

Category:

Debugger Functions
