int getHexMode(_str langID, int defaultValue=null)

Retrieves the hex mode for the given language. Possible values:

  • HM_HEX_OFF - hex mode off
  • HM_HEX_ON - hex mode on
  • HM_HEX_LINE - hex mode on using line hex

Parameters:

langID - language ID (see p_LangId)

defaultValue - (optional) the value to return if no value is saved for this setting in the given language. If this is not specified, then the application's default value for this language setting is returned

Returns: int

hex mode for language
