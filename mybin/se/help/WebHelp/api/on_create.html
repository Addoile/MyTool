event on_create()

All objects are sent an on_create event when a dialog box template is loaded with the show or _load_template function. The show and _load_template function can be given a variable number of arguments to pass to the on_create event. The on_create event can not be used to change the focus. Use the on_load function to set the initial focus of a dialog box.

Example:

#include 'slick.sh'
_command mycommand()
{
    show('form1','first argument', 'second argument');
}

defeventtab form1;
command1.on_create()
{
   arg1=arg(1);
   arg2=arg(2);
   message('arg1='arg1' arg2='arg2);
}

Applies To:

Text Box, Combo Box, List Box, Directory List Box, File List Box, Drive List, Command Button, Check Box, Radio Button, Frame, Hscroll Bar, Vscroll Bar, Spin, Gauge, Image, Form, Tree View
