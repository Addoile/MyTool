int _reflow_selection(_str markid="")

Reformats the selection specified according to the current margin settings. If the buffer containing the specified selection is active when this function is invoked, the resulting lines are inserted after the end of the selection. Otherwise the resulting lines are inserted after the cursor. Character mark is not supported. mark_id is a handle to a selection returned by one of the built-ins _alloc_selection or _duplicate_selection. A mark_id of '' or no mark_id parameter identifies the active selection.

Returns: int

Returns 0 if successful. Common return codes are TEXT_NOT_SELECTED_RC, and LINE_OR_BLOCK_SELECTION_REQUIRED_RC. On error, message is displayed.

Applies To:

Edit Window, Editor Control

Category:

Selection Functions
