VSSTATIC unsigned short*VSAPI vsUTF16CharRead2(const unsigned short *pwBuf, const unsigned short *pwEndBuf, unsigned &uch)

Reads a UTF-16 or UTF-32 (surrogate) character. This function does not read composite characters. This function is faster than the vsUTF16CharRead() function.

Parameters:

pwBuf - Buffer contain UTF-16 text.

pwEndBuf - Pointer to end of buffer.

uch - Recieves UTF-32 character.

Returns: VSSTATIC unsigned short*VSAPI

Returns pointer to the next character to read.

Category:

Unicode Functions
