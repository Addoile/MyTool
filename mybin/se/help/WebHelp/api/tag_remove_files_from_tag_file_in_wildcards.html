int tag_remove_files_from_tag_file_in_wildcards(_str pszTagDatabase, int rebuildFlags, _str pszDirectoryPath, _str pszWildcardOpts)

Remove all the files matching the given wildcards from the tag file.

Parameters:

pszTagDatabase - name of tag file to rebuild

rebuildFlags - bitset of VS_TAG_REBUILD_

  • VS_TAG_REBUILD_SYNCHRONOUS -- Remove the files in the foreground instead of finishing the work using a background thread.

pszDirectoryPath - base path to search for wildcards in

pszWildcardOpts - command line options for wildcards

  • filespec -- wildcard path/file specification
  • -T -- find files recursively in subdirectories
  • -X filespec -- exclude files matching this spec

Returns: int

0 on success, <0 on error.

Since:

16.0

Category:

Tagging Functions
