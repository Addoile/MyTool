int p_encoding_set_by_user

Set to -1 if the buffer was loaded using with automatic encoding. Set to the encoding chosen if the user overrode automatic encoding processing. load_files(),_save_file(), and auto-reload set this property. This buffer property is needed so that auto-reloaded knows how to reload the buffer when it is modified by another application.

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
