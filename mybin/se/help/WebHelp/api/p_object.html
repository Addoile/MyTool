int p_object

Returns the object type of the current window. The p_object property can be one of the following constants defined in "slick.sh":

OI_MDI_FORM
OI_FORM
OI_TEXT_BOX
OI_CHECK_BOX
OI_COMMAND_BUTTON

OI_VSCROLL_BAR

OI_PICTURE_BOX
OI_IMAGE
OI_GAUGE
OI_SPIN
OI_MENU
OI_MENU_ITEM
OI_SSTAB
OI_DESKTOP
OI_SSTAB_CONTAINER
OI_EDITOR
OI_MINIHTML
OI_SWITCH
OI_TEXTBROWSER
When menus are displayed, there is no instance handle. Menu and menu item properties can only be accessed in menu resources. The _menu_get_state function can be used to get the menu flags. These flags indicate whether the menu item is a submenu which contains other menu items.

Applies To:
