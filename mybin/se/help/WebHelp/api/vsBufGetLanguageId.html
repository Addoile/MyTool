VSPSZ VSAPI vsBufGetLanguageId(int buf_id, char *pszLanguageId, int MaxLangId, int *pMaxLangIdLen=a)

Parameters:

buf_id - Buffer id returned by vsPropGetI(wid,VSP_BUF_ID), vsBufEdit, or vsBufMatch.

pszLanguageId - Name of language setting corresponding to buf_id specified. This can be NULL, in which case, only the length of the language ID will be set.

MaxLangId - Number of characters allocated to pszLanguageId. We recommend at least 32.

pMaxLangIdLen - If this is not 0, this is set to the number of characters needed to be allocated.

Returns: VSPSZ

Returns language ID for the language mode corresponding to buffer id specified. Returns pszLanguageId.

Attention:

This function is thread-safe.

Category:

Buffer Functions
