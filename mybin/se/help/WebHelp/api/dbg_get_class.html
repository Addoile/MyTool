int dbg_get_class(_str class_path, _str &name, _str &outer_class, _str &signature, _str &file_name, int &flags, _str &loader)

Get the basic information about the indicated class

Parameters:

class_path - index path of class in class table

name - (reference) name of class

outer_class - (reference) outer class name

signature - (reference) signature of class (type)

file_name - (reference) source file that class is in

flags - (reference) bitset of DBG_FLAGS_

class_loader - (reference) class loader

Returns: int

0 on success, <0 on error

Category:

Debugger Functions
