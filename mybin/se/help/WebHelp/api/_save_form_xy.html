void _save_form_xy(_str form_name=null)

Saves x, y window position in the ".command" buffer. This buffer is saved in the auto restore file ("vrestore.slk" by default) when auto restore is on. The _restore_form_xy method may be used to restore the previous window position saved by this function.

Parameters:

form_name - (optional) form name to use to store position data

Applies To:

Form

Category:

Form Methods
