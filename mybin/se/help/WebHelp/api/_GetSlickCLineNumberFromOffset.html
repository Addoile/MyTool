int _GetSlickCLineNumberFromOffset(_str module_name, _str proc_name, int offset)

Determine the line number of a function found at the given offset within a compiled and loaded Slick-C module.

Parameters:

module_name - name of module function comes from

proc_name - name of function offset is within

offset - code offset within function.

Returns: int

Return the line number of the given offset in the source file on success, Return error code <0 on error.

Category:

Macro Programming Functions
