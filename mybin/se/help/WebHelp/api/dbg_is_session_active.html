int dbg_is_session_active(int session_id)

Return true if the given session is active.

Parameters:

session_id - debugger session ID (1 .. max_session)

Returns: int

true (1) if the session is active, 0 otherwise.

Since:

10.0

Category:

Debugger Functions

See Also:

dbg_find_session, dbg_get_session_name
