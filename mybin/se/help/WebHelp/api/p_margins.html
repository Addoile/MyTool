_str p_margins

Determines the margins for the current buffer. This property is available only when the control is displayed. p_margins is a string in the format: left_ma right_ma [new_paragraph_ma ]. If new_paragraph_ma is not given, it defaults to left_ma.

Example:

   parse p_margins with left_ma right_ma new_para_ma;
   ++left_ma;++right_ma;++new_para_ma;
   p_margins=left_ma' 'right_ma' 'new_para_ma;

Applies To:

Edit Window
