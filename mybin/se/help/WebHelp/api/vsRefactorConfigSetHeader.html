int VSAPI vsRefactorConfigSetHeader(const char *configName, const char *configHeader)

Save the full path to the configuration header for the given configuration.

Parameters:

configName - name of configuration to query

configHeader - new path to configuration header

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
