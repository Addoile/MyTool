_command typeless put(_str dest_filename="", _str save_flags="")

Writes selection to dest_filename specified. The -p option selects to write the selection to the print using the default print options. The default printer options may be set with the Print Setup dialog box ("File", "Print...", press "Setup..." button) and the Print dialog box ("File", "Print...").

Returns: typeless

Returns 0 if successful. Common return codes are TEXT_NOT_SELECTED_RC, and TOO_MANY_SELECTIONS_RC. On error, message is displayed.

Applies To:

Edit Window

Category:

Selection Functions

See Also:

gui_write_selection, get, gui_append_selection, append
