VSArgumentCompletionFlags

Control the default behavior of text box argument and command line auto completion. This value is a bitset of the flags defined by VSARGUMENT_COMPLETION_*. The default is for everything to be enabled, including the use of TAB and S_TAB to cycle up and down through the list.

  • VSARGUMENT_COMPLETION_ENABLE
  • VSARGUMENT_COMPLETION_NO_TAB_NEXT
  • SARGUMENT_COMPLETION_COMMANDLINE
  • VSARGUMENT_COMPLETION_UNC_PATHS

Default:

VSARGUMENT_COMPLETION_ALL_OPTIONS

Category:

Configuration Variables
