void VSAPI vsTagComposeTag(char *proc_name, int proc_name_max, const char *tag_name, const char *class_name, const char *type_name, int tag_flags=a, const char *signature=a, const char *return_type=a, const char *template_signature=a)

Create the canonical tag display string of the form: 

   tag_name(class_name:type_name)flags(signature)return_type

This is used to speed up find-tag and maketags for languages that do not insert tags from DLLs.

Parameters:

proc_name - output string for "composed" tag

proc_name_max - number of bytes allocated to proc_name

tag_name - the name of the tag

class_name - class/container the tag belongs to

type_name - the tag type, (see VS_TAGTYPE_*)

tag_flags - (optional) integer tag flags (see VS_TAGFLAG_*)

signature - (optional) function signature

return_type - (optional) function return type

template_signature - (optional) template signature

Returns: void

nothing.

Category:

Tagging Functions
