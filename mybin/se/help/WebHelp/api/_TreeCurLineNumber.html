int _TreeCurLineNumber(int lineNumber=-1)

Parameters:

lineNumber - If lineNumber>=0, the current node is set to the node corresponding to the line number specified.

Returns: int

Returns the line number of the current node in the tree as if the tree we are list box. If lineNumber>=0, the current node is set to the node corresponding to the line number specified.

Applies To:

Tree View

Category:

Tree View Methods
