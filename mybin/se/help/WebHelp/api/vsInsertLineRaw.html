void VSAPI vsInsertLineRaw(int wid, const char *pBuf, int BufLen=-)

This function is identical to the vsInsertLine function, except that the input string is in the same format as the internal buffer data which can be SBCS/DBCS or UTF-8. See "Unicode and SBCS/DBCS C API Programming".

Applies To:

Edit Window, Editor Control

Edit Window Methods, Editor Control Methods
