_str nls(_str msg, ...)

Returns: _str

Returns string with '%s' values replaced with arg1, arg2, etc. The first parameter must be a string constant.

Note that the Slick-C Translator optimizes the case where the nls procedure is called with a simple string constant. In this case, the code generation is equivalent to just specifying the string constant (i.e. the nls procedure is not called.).
