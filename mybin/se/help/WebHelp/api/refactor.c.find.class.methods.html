int refactor_c_find_class_methods(int handle, _str filename, _str symbolName, int beginSeekPos, int endSeekPos)

Find methods that are members of this symbol's class The methods are stored internally and can be retrieved using refactor_c_get_num_class_methods() and refactor_c_get_class_method()

Parameters:

handle - Handle to transaction

filename - Filename to do refactoring within

symbolName - Name of reference to class

beginSeekPos - Begin Seek position of reference

endSeekPos - End Seek position of reference

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
