int dbg_find_disassembly_line(_str source_file, _str address, int &line_number, int &inst_offset)

Find the line and offset of the instruction corresponding to the given program counter address.

Parameters:

source_file - source file

address - program counter address

line_number - (reference) set to line number

inst_offset - (reference) set to line offset

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
