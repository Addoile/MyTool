int VSAPI vsRefactorCMoveMethod(int nHandle, VSPSZ szSrcMethodName, VSPSZ szSrcClassName, VSPSZ szDstMethodName, VSPSZ szDstClassName, int nSrcIdx, int nDstIdx, char **files, int nFiles, int nReceiver, int nFlags)

Move a method. Move a method from one class to another and fix all references.

Parameters:

nHandle - Handle to transaction

szSrcMethodName - Name of method to move

szSrcClassName - Name of class to move method from.

szDstMethodName - Name of destination method

nReceiver - The index of the delegate chosen.

nFlags - Flags for access modifiers

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
