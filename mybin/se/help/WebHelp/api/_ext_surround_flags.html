int _ext_surround_flags(_str lang='')

Deprecated. Use _GetSurroundModeFlags()

Get the extension specific surround options.

The options are stored per extension type. If the options are not yet defined for an extension, then use def_surround_mode_options as the default.

Parameters:

lang - language ID (p_LangId)

Returns: int

bitset of VS_SURROUND_MODE_* options.

Applies To:

Edit Window, Editor Control

Category:

Tagging Functions
