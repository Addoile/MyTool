int tag_filter_type(int type_id, int filter_flags, _str type_name=null, int tag_flags=0)

Filter the given tag type based on the given filter flags

Parameters:

type_id - tag type ID

filter_flags - Union of the following filtering flags:

VS_TAGFILTER_PROC
proc, func, constr, destr, trigger
VS_TAGFILTER_PROTO
proto, procproto
VS_TAGFILTER_DEFINE
define
VS_TAGFILTER_ENUM
enum, enumc
VS_TAGFILTER_GVAR
gvar
VS_TAGFILTER_TYPEDEF
typedef
VS_TAGFILTER_STRUCT
struct, class
VS_TAGFILTER_UNION
union
VS_TAGFILTER_LABEL
label
VS_TAGFILTER_INTERFACE
interface
VS_TAGFILTER_PACKAGE
package, program, library
VS_TAGFILTER_VAR
var
VS_TAGFILTER_CONSTANT
const
VS_TAGFILTER_PROPERTY
prop
VS_TAGFILTER_LVAR
lvar, param
VS_TAGFILTER_MISCELLANEOUS
Misc. + user defined tag types
VS_TAGFILTER_DATABASE
database, table, column, index
VS_TAGFILTER_GUI
form, eventtab, menu, control
VS_TAGFILTER_INCLUDE
include
VS_TAGFILTER_SUBPROC
nested procedure or function

Common combinations of tag filters.

VS_TAGFILTER_ANYTHING
Any tag type
VS_TAGFILTER_ANYPROC
PROC,PROTO, SUBPROC
VS_TAGFILTER_ANYDATA
VAR, LVAR, GVAR, PROPERTY
VS_TAGFILTER_ANYSTRUCT
STRUCT, UNION, INTERFACE

type_name - Look up type ID using this name

tag_flags - Check tag flags for VS_TAGFLAG_maybe_var

Returns: int

Returns 1 if the type is allowed according to the flags, 0 if not.

Category:

Tagging Functions

See Also:

tag_get_type, tag_find_type, tag_next_type
