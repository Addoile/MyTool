5 VSCALLBACK_WIN_GET_MESSAGE

void (VSAPI *pfnGetMessage)(int wid,char *pszMsg,int MaxStringLen);
The GET_MESSAGE callback function allows the user to save the message line so that it can be restored later. Currently the editor AutoSave features uses this to preserve the message.

Parameters:

wid - Window id of editor control.

pszMsg - String to receive message being displayed on message line. This may or may not be the message displayed by the CallbackMessage function. This can be 0.

MaxMsgLen - Number of characters allocated to pszMsg.

pMaxMsgLen - If this is not 0, this is set to the number of characters you need to allocate to pszMsg.

Category:

Miscellaneous Defines
