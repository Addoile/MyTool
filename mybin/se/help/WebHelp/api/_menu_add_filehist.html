void _menu_add_filehist(_str filename)

Adds file history to end of File menu on the SlickEdit MDI menu bar. The filename argument MUST be specified in absolute. Use the absolute function to do this. This requirement is imposed so that unnecessary disk reads (during auto restore) do not occur. Calling the absolute function for floppy drives is slow.
