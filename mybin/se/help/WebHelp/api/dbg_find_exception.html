int dbg_find_exception(_str class_name, int &enabled, int start_id)

Find the exception with the given class name

Parameters:

class_name - name class corresponding to exception

enabled - (reference) is the exception enabled or disabled

start_id - (optional) exception ID to start searching at, default 0

Returns: int

exception ID > 0 on success, <0 on error.

Category:

Debugger Functions
