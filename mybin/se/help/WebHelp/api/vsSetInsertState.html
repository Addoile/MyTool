void VSAPI vsSetInsertState(int wid, int state, int Reserved=a)

Sets the insert state on (non-zero) or off (0). Each text box has its own insert state. However, all edit windows and editor controls share the same insert state.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

State - New value for insert state.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods
