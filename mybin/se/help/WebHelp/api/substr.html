_str substr(_str string, int start, int length=-1, _str pad=" ")

Parameters:

string - input string

start - start position

length - number of characters to get, default is to get to end of string

pad - padding character if 'start+length' goes beyond end of string

Returns: _str

Returns length characters of string beginning at start. By default, length is the rest of string. If length is greater than length of string, the return string is padded with blanks or pad character if specified. 'start' must be a positive whole number.

Example:

   substr('test',2)     :== 'est'
   substr('test',1,5)   :== 'test '
   substr('',1,5,'*')   :== '*****'
   substr('test',2,4)   :== 'est '
