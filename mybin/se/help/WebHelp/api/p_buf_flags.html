int p_buf_flags

Determines buffer flags. This property is only available when the control is displayed. The buffer flags may be zero or more of the following constant flags defined in "slick.sh":

VSBUFFLAG_HIDDEN
Affects Edit Window only. next_buffer and prev_buffer commands won't switch to this buffer.
VSBUFFLAG_THROW_AWAY_CHANGES
Affects Edit Window only. Allow quit without prompting for save when this buffer is modified.
VSBUFFLAG_KEEP_ON_QUIT
Affects Edit Window only. Don't delete buffer on quit.
VSBUFFLAG_REVERT_ON_THROW_AWAY
Affects Edit Window only. Replace buffer with copy on disk when the buffer is deleted with the quit command.
VSBUFFLAG_PROMPT_REPLACE
Indicates whether to prompt when replacing existing file on save.
VSBUFFLAG_DELETE_BUFFER_ON_CLOSE
Indicates whether the buffer should be deleted when a list box or editor control is deleted.

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
