boolean p_indent_case_from_switch

Determines whether to indent a case statement from its parent switch statement. This property is available only when the control is displayed.

Example:

   // p_indent_case_from_switch = true
   switch (num) {
      case 1:
      case 2:
      default:
         break;
   }

   // p_indent_case_from_switch = false
   switch (num) {
   case 1:
   case 2:
   default:
      break;
   }

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
