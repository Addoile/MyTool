void set_command(_str command_text, int start_sel=-1, int end_sel=-1, _str prompt=null)

Be sure to look at the macro function get_string which allow a macro to read command line input from the user.

set_command changes current setting of the command line. Text of command line is set to command_text. The selection columns (1.. MAXINT) are set if start_sel is >=0. If end_sel<0, end_sel is set to start_sel. The prompt parameter sets the command prompt. The prompt may only be modified by the set_command primitive.

Example:

          command_text='dir ';
          set_command(command_text, length(command_text)+1);

Applies To:

Text Box, Combo Box

Categories:

Combo Box Methods, Text Box Methods
