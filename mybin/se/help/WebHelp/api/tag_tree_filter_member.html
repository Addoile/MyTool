int tag_tree_filter_member(int filter_flags, _str type_name, int in_class, int tag_flags, int &i_access, int &i_type)

Deprecated. Use tag_tree_filter_member2

Filter the given item based on the given filtering flags, and determine the access level and member type. Though this function is typically used by the symbol browser for filtering items, it can also be used simply to determine the access level and bitmap type for the tag, just ignore the return value in this case. This function is deprecated, use tag_tree_filter_member2 instead.

Parameters:

filter_flags - Filtering options flags, see tag_tree_filter_member2.

type_name - Tag type, See tag_get_type.

in_class - if 1, treat as a class member, (class_name != '')

tag_flags - Set of tag attribute flags. See tag_insert_simple for more information about the tag flags.

i_access - (Output) Access level (public, package, protected, or private) of tag, see tag_tree_filter_member2.

i_type - (Output) Tag bitmap type to use to represent tag, see tag_tree_filter_member2.

Returns: int

0 if the item is filtered out, 1 if it passes the filters.
