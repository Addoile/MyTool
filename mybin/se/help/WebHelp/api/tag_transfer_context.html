int tag_transfer_context(_str file_name)

Insert all the tags from the current context into the currently open tag file. Assumes that the context is up-to-date at the time that this function is called.

For synchronization, macros should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

file_name - typically p_buf_name, absolute path of buffer

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if the context info is not set or from the wrong file or buffer.

Category:

Tagging Functions
