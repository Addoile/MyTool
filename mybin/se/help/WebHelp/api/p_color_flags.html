int p_color_flags

Determines coloring styles for a buffer. Color flags are defined in "slick.sh" and may be one or more of the following constants:

LANGUAGE_COLOR_FLAG
Color language specific elements.
MODIFY_COLOR_FLAG
Color modified lines.
CLINE_COLOR_FLAG
Color the current line.

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties

See Also:

_clex_find, _clex_load, _clex_skip_blanks
