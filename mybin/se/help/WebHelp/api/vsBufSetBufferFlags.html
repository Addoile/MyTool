void VSAPI vsBufSetBufferFlags(int buf_id, int BufFlags)

Set the buffer flags for the given buffer.

Parameters:

buf_id - Buffer id returned by vsPropGetI(wid,VSP_BUF_ID), vsBufEdit, or vsBufMatch.

BufFlags - Bitset of the following flags

  • VSBUFFLAG_HIDDEN -- hidden buffer, next buffer won't switch to this buffer.
  • VSBUFFLAG_THROW_AWAY_CHANGES -- Allow quit without prompting on modified buffer
  • VSBUFFLAG_KEEP_ON_QUIT -- Don't delete buffer on QUIT.
  • VSBUFFLAG_REVERT_ON_THROW_AWAY -- Revert buffer
  • VSBUFFLAG_PROMPT_REPLACE -- Prompt before replacing buffer contents
  • VSBUFFLAG_DELETE_BUFFER_ON_CLOSE -- Indicates whether a list box
  • VSBUFFLAG_FTP_UPLOAD_IN_PROGRESS -- Specifies that buffer is currently being uploaded via FTP
  • VSBUFFLAG_FTP_BINARY -- Specifies that the FTP buffer should be transferred binary by default

Attention:

This function is thread-safe.
