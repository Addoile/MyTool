int tag_get_inheritance(_str class_name, _str &parents)

Retrieve the set of parent classes for a given class.

This function has the side effect of position the class iterator on the given class. Returns BT_RECORD_NOT_FOUND_RC if class_name is not in the database.

Parameters:

class_name - Qualified name of class or interface to update inheritance information for. See VS_TAGSEPARATOR_class and VS_TAGSEPARATOR_package for details on constructing this string.

parents - Delimited list of immediate class ancestors. See VS_TAGSEPARATOR_parents for more details on decomposing this string.

Returns: int

0 on success, < 0 on error. Returns BT_RECORD_NOT_FOUND_RC if class_name is not in the database.

Category:

Tagging Functions
