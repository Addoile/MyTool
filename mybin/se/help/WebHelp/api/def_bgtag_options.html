_str def_bgtag_options

Options for background file tagging. Consists of three parts (space separated):

  1. activate interval -- seconds in idle time to wait before starting background tagging
  2. maximum files to check -- maximum files per iteration
  3. maximum number to retag -- maximum to retag per iteration
  4. timeout before retagging -- time in seconds before re-activating background tagging

Default:

'30 10 3 600';

Category:

Configuration Variables
