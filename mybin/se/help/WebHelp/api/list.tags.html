_command typeless list_tags(_str options="")

Scans the current buffer for tags and displays the tags in a selection list. Only functions are listed. You may select a tag to go to. Unlike the push_tag and find_tag commands, this function does not require a tags file. Currently the C, C++, Pascal, REXX, AWK, Modula-2, dBASE, Cobol, Fortran, Ada, and Assembly languages are supported. See "tags.e" for information on adding support for other languages.

Parameters:

options - use 'a' to show all tags, not just procs

Returns: typeless

Returns 0 if successful user selected to go to a tag.

Applies To:

Edit Window

Category:

Miscellaneous Functions

See Also:

push_tag, find_tag, find_proc, make_tags, gui_make_tags
