void VSAPI vsSetTextColor(int wid, VSCOLORINDEX ColorIndex, seSeekPos ColorLen, bool replacePreviousColor=true)

Copies color indexes for ColorLen characters of text starting from the cursor. A 0 color index indicates a null color.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

ColorIndex - Valid color indexes are any of the VSCFG_??? constants or a color index allocated by vsAllocColor. 0 indicates a NULL color. Use the NULL color to get our built-in color coding.

ColorLen - Repeat the specified color for this many bytes. abs(ColorLen) times.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

vsAllocColor, vsFreeColor, vsSetDefaultColor, vsGetDefaultColor
