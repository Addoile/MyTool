int _write_state(_str path)

Write out the current state of the Slick-C interpreter to the given file path. The current state includes all global variables, names table entries, loaded modules, and loaded DLLs. It does not include the current function stack, locals, or user data stored in Slick-C dialogs and controls.

Parameters:

path - File to save state file to.

Returns: int

0 on success, <0 on error.

Category:

Names Table Functions
