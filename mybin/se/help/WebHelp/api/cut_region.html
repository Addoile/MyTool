_command typeless cut_region()

Used in EMACS emulation. Deletes the text in the highlighted area and copies it to the clipboard. If no highlighted area exists, an invisible mark is used which contains the last highlighted text not deleted or the last text pasted by the emacs_paste command.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
