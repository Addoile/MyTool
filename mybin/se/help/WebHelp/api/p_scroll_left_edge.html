int p_scroll_left_edge

Determines the left edge scroll position in pixels when the window is scrolled. A scroll position of zero means the first character of every line can be seen. A negative scroll position means the window is not scrolled and the p_left_edge property determines the left edge scroll position. This property is available only when the control is displayed.

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
