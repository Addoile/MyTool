_command void argument(...)

Prompts you for a repeat count and executes the next key you press the number of times specified. The following keys have special meaning while being prompted for the repeat count:

0..9
Appends one of the digits 0..9 to the counter.
Alt+0..Alt+9
EMACS emulation only. Appends one of the digits 0..9 to the counter.
-, Alt+Minus
Toggles the repeat count from positive to negative.
Pressing the key bound to the argument command while being prompted for the repeat count multiplies the repeat count by 4.

The global variable "_argument" is set to the repeat count given before the command is executed. The command invoked may test the value of this variable and set it to '' when finished processing the argument to do some special processing when executed with a repeat count.

Applies To:

Edit Window

Category:

Keyboard Functions
