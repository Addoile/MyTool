_str last_event(_str newEvent=null, boolean keysOnly=false)

Parameters:

newEvent - set the last event

keysOnly - report only the last keyboard event, no MOUSE_MOVE, etc.

Returns: _str

Returns a binary string representation of the last event returned from get_event or test_event. If the event parameter is specified, the last event is set to event. For ASCII keys, a string of length 1 is returned. For extended keys like Alt+L, and ESC, a string of length 2 or 3 is returned. Use the mou_mode function if you want button-up events returned by get_event.

Example:

         event=get_event();
         message('The key pressed was 'event2name(last_event()));

Category:

Keyboard Functions
