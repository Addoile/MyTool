void VSAPI vsDeleteLanguageOptions(const char *pszLangId)

Removes the language setup options for the specific language, as specified by the given language ID. Note that 'lang' must be a real language ID, not a referred file extension.

Parameters:

pszLangId - Language ID (see p_LangId For list of languages, use our Language Options dialog ("Tools", "Options","Language Setup...").

Since:

13.0

Category:

Configuration Functions

See Also:

vsSetDefaultLanguageOptions, vsGetDefaultLanguageOptions, vsSetDefaultExtensionReferTo, vsGetDefaultExtensionReferTo, vsDeleteExtensionOptions
