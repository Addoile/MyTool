void _kbd_add_callback(int callback_index, _str(&keyList)[], _str options=null)

Add an extensions specific keyboard callback

Parameters:

callback_index - Index into names table of command or function to call. Signature is boolean mycallback(). Return true if no further processing of this key is necessary.

keyList - Array of keys that may be handled by this callback.

options - options may contain one or more of the following option letters: 'K' - All keys

Applies To:

Edit Window, Editor Control

Example:

boolean _on_key_my_callback() {
    say('key='last_event());
    return(false);
}
// Note that these callbacks typically use the naming convention _on_key_<p_LangId>
_str keyList[];
keyList[0]='(';keyList[1]=')';
_kbd_add_callback(find_index("_on_key_my_callback",PROC_TYPE),keyList);

Categories:

Edit Window Methods, Editor Control Methods

See Also:

_kbd_remove_callback, _kbd_try_callbacks
