_command void process_enter()

New binding of ENTER key when in process mode. When invoked while the cursor is on the last line of the build window, a blank line is inserted after the current line and the contents of the current line after the "read point" is inserted in the ".process- command" retrieve buffer for latter use by the process_up or process_down command. Otherwise the fundamental mode ENTER key binding is executed. Process output is inserted at the character before the "read point".
