int _spell_check(_str &word, _str &replace_word, int &col, _str string="")

Starts spell checking from the cursor position in the current buffer. The output variable, col is set to the starting column position (p_col) of the word found and the cursor is placed at the this column. If the string argument is given, the string argument is spell checked instead of the current buffer. In this case, col is set to the character position within the string of the word found.

Parameters:

word -

replace_word -

col -

string -

Returns: int

Return values are as follows:

SPELL_NO_MORE_WORDS_RC
End of buffer reached.
SPELL_REPEATED_WORD_RC
Found two of the same words found in a row.
SPELL_WORD_NOT_FOUND_RC
Possible misspelled word. word variable set to the misspelled word.
SPELL_REPLACE_WORD_RC
History word found. Replace word at cursor with replace_word.
SPELL_ERROR_READING_MAIN_DICT_RC

Applies To:

Editor Control, Edit Window

Categories:

Edit Window Methods, Editor Control Methods
