int dbg_add_exception(int stop_when, int count, _str condition, _str class_name, _str thread_name)

Add a exception. The exception is initially disabled at the time that it is inserted. You must call the package-specific callback to enable the exception.

Parameters:

stop_when - THROWN, CAUGHT, UNCAUGHT

count - number of times to skip breakpoint -1 indicates that it is a one-time breakpoint

condition - expression for conditional exception

class_name - name of exception to catch

thread_name - name of thread breakpoint is restricted to

Returns: int

exception index >0 on success, <0 on error.

Category:

Debugger Functions
