int def_background_tagging_maximum_jobs

Specifies the maximum number of active background tagging jobs that can be in any stage of the background tagging queue. This limit is in place to guard against background tagging using excessive amounts of memory while processing files.

This option has no effect if def_background_tagging_threads == 0. This setting requires you to restart the editor for a change to take effect.

Default:

500

Category:

Configuration Variables
