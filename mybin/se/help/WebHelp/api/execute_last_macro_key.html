_command void execute_last_macro_key()

Ends macro recording if currently recording and saves it as the last recorded macro for this key. This macro may be bound to an array of keys, such as Ctrl+0 through Ctrl+9, in order to have unique last recorded macros for each key.

Displays message if a macro was defined.
