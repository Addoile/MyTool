int VSAPI vsCallbackRemove(int id, VSCALLBACK_AR vscallback, void *pfn)

Removes a callback function. Make sure you call this function if you unload a DLL which has a callback function.

Parameters:

id - Buffer id for VSCALLBACK_AR_BUF_??? callbacks. Window id for VSCALLBACK_AR_WIN_??? callbacks. Ignored for all other callbacks.

vscallback - One of the following constants which correspond to a callback function:

  • VSCALLBACK_AR_BUF_LINES_INSERTED
  • VSCALLBACK_AR_BUF_LINES_DELETED
  • VSCALLBACK_AR_BUF_TEXT_CHANGE
  • VSCALLBACK_AR_APP_DELETE_BUFFER

pfn - Pointer to callback function to remove

Applies To:

Edit Window, Editor Control

Categories:

Buffer Functions, Edit Window Methods, Editor Control Methods, Miscellaneous Functions

See Also:

vsCallbackSet, vsCallbackAdd
