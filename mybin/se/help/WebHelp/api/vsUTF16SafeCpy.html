void VSAPI vsUTF16SafeCpy(unsigned short *pszDest, const unsigned short *pszSrc, size_t DestLen)

Copies a null terminated UTF-16 string. Resulting string is always null terminated even if the entire source string cannot be copied.

Parameters:

pszDest - Destination buffer.

pszSrc - Null terminated UTF-16 string.

DestLen> - Number of characters allocated to pszDest. Must be at least 1.

Category:

Unicode Functions
