void save_pos(typeless &p, typeless useRealLineNumbers="")

Used for saving and restoring location within a buffer. DO NOT use this function if you plan to modify the buffer before calling the restore_pos function. Use the _save_pos2 function instead. Saves buffer location information into p. Use the restore_pos procedure to restore the location within a buffer.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods

See Also:

restore_pos, _save_pos2, _restore_pos2
