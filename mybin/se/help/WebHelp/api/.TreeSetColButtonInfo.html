void _TreeSetColButtonInfo(int ButtonIndex, int ButtonWidth, int ButtonFlags=-1, int State=-1, _str Caption='')

Set up the column button info for a multi-column tree control. This is used to add headers and set sorting properties for the colunms.

Parameters:

ButtonIndex - column index, 0 is the first column

ButtonWidth - column width, in twips

ButtonFlags - button flags, bitset of TREE_BUTTON_

State - initial state of button, depressed or normal

Caption - Button caption

Applies To:

Tree View

Category:

Tree View Methods
