int VSAPI vsRefactorGetRText(int wid, char *text, int textLen)

Get the entire source file text from the given editor control.

Parameters:

wid - Window ID

text - text to insert

textLen - number of bytes allocated to 'text'

Returns: int

number of bytes copied on success, if text is NULL, return the number of bytes required. If text is not NULL, but textLen is less than the buffer size, return INSUFFICIENT_MEMORY_RC.

Category:

Refactoring Functions
