_str reindent_line(_str s, int syntax_indent, boolean forceIndentEmptyLine=false)

Returns: _str

Returns line with leading spaces and/or tabs reformatted to adhere precisely to the users indent_with_tabs style (See indent_with_tabs command). If the line is completely blank (no tabs or spaces), uses the forceIndentEmptyLine and LangaugeSettings.getInsertRealIndent value to figure out what to do.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
