char*VSAPI vsGetMessage(int rc, char *pszMessage, int MaxMsgLen, int *pMaxMsgLen=a)

Copies corrensponding message into pszMessage. Set to "Message not available" for a non-existant error code.

Parameters:

rc - Error code. One of ???_RC constants defined in "rc.h".

pszMessage - Output buffer for message. This can be 0. We recommend VSMAXMESSAGE.

MaxMessageLen - Number of characters allocated to pszMessage. We recommend VSMAXMESSAGE.

pMaxMessageLen - If this is not 0, this is set to the number of characters you need to allocate to pszMessage.

Returns: char*VSAPI

Returns pszMessage.

Category:

Miscellaneous Functions

See Also:

vsMessage, vsGetMessageLine
