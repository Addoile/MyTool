void tag_tree_decompose_tag(_str proc_name, _str &tag_name, _str &class_name, _str &type_name, int &tag_flags, _str &arguments=null, _str &return_type=null, _str &template_args=null)

Decode the given proc_name into the components of the tag. See tag_tree_compose_tag for details about how tags are encoded as strings.

Parameters:

proc_name - Encoded tag from tag_tree_compose_tag on an extension specific tag search function.

tag_name - (Output) the name of the tag

class_name - (Output) class/container the tag belongs to

type_name - (Output) Tag type name. See tag_get_type for a list of all standard tag types.

tag_flags - (Output) Tag flags indicating symbol attributes, see (Output) tag_insert_simple for details.

arguments - Arguments, such as function or template class parameters.

return_type - (Output) Return type of symbol or value of constant or both, separated by VS_TAGSEPARATOR_equals.

Category:

Tagging Functions
