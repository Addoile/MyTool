_str fnq_match(_str name, boolean find_first)

Parameters:

name - file name to search for

find_first - 'true' to find first, 'false' to find next

Returns: _str

Returns next file name which is a prefix match of name. find_first must be non-zero to initialize matching. Returns '' when no more matches are found.

"fnq" means "file no quote". This function does not quote the files returned.

Completion Functions, File Functions
