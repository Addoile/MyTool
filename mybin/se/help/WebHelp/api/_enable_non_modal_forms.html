_str _enable_non_modal_forms(boolean value, int skip_wid, _str wid_list='')

Enables or disables forms specified. For simple, modal dialog boxes, you should not use this function. If you want to write a loop which does some process and have a dialog box show the status of the process and allow canceling, you should use this function.

Parameters:

enable - If non-zero, forms are enabled.

skip_wid - Form with window id equal to this value will not be effected.

wid_list - Specific space delimited window ids of the forms to be enabled or disabled. Usually, this variable is the list of window ids to be reenabled. The list of windows disabled by this function is returned by this function.

Returns: _str

'' is returned if the enable argument was non-zero. Otherwise, a space delimited list of the window ids that were disabled is returned.

Example:

#include "slick.sh"
static  typeless gcancel
_command test()
{
    // Show the form modeless so there is no modal wait
    form1_wid=show('form1');
    disabled_wid_list=_enable_non_modal_forms(0,form1_wid);
    gcancel=0;
    for (;;) {
          // Read mouse, key, and all other events until none are left
          // or until the variable gcancel  becomes true.
          process_events(gcancel);
          if (gcancel) break;
    }
    _enable_non_modal_forms(1,0,disabled_wid_list);
    form1_wid._delete_window();
}
defeventtab form1;
cancel.lbutton_up()
{
     gcancel=1;
}
