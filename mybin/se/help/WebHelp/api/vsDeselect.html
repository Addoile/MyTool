int VSAPI vsDeselect(int wid, int markid=-)

Clears selection specified. mark_id is a handle to a selection returned by vsAllocSelection or vsDuplicateSelection. A mark_id of -1 identifies the active selection.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

mark_id - Selection id to duplicated. If -1 is specified the active selection.

Returns: int

Returns 0 if mark_id is a valid selection handle. Otherwise INVALID_SELECTION_HANDLE_RC is returned.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
