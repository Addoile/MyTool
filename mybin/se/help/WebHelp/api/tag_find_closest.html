int tag_find_closest(_str tag_name, _str file_name, int line_no, boolean case_sensitive=false)

Retrieve tag in the given file closest to the given line number. Use tag_get_info to extract the details about the tag.

Parameters:

tag_name - Tag name to search for

file_name - Full path to file containing tag

line_no - Line that tag is expected to be present on

Returns: int

0 on success, <0 on error, or BT_RECORD_NOT_FOUND_RC if no such tag.

Category:

Tagging Functions

tag_find_equal, tag_next_equal, tag_reset_find_tag
