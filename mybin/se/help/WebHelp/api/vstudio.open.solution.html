int vstudio_open_solution(_str solutionFilePath, _str VStudioVersion)

Opens the Visual Studio Solution (.sln) file in the specified version of Visual Studio .NET.

Parameters:

solutionFilePath - Full path to the .sln file

VStudioVersion - Version of Visual Studio to open (7.1, 8.0, and up)

Returns: int

1 indicates success -1 indicates Visual Studio could not be started. 0 indicates Visual Studio is started, but the solution could not be opened

Category:

Miscellaneous Functions
