void VSAPI vsBPMSetPicture(int flags, int index)

Deprecated. Use vsLineMarker* functions Determines the debug bitmap which gets displayed for the specified debug flags.

Parameters:

flags - Specifies the one or more of the VSBPFLAG_??? flags. You can add your own.

index - An index into the names table of the bitmap to display. Use vsUpdatePicture to register a bitmap.
