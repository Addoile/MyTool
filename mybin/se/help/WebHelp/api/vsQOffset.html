seSeekPosRet VSAPI vsQOffset(int wid)

Parameters:

wid - Window id of editor control. 0 specifies the current object.

Returns: seSeekPosRet

Returns character offset of cursor from beginning of buffer.

IMPORTANT: This function includes lines with VSLF_NOSAVE set which means that these file offsets will not match what is on disk if the file has non-savable lines. Use vsGoToROffset, vsQROffset, p_RLine, and p_RNoflines for dealing with disk seek positions and line numbers.
