long VSAPI vsExecute(int wid, const char *pszCommand, const char *pszOptions="SMDA")

Runs the command specified. pszCommand can be any command that can be invoked from the SlickEdit command line. Command may be an internal command (_command), an external operating system command, or an external Slick-C® batch macro (with defmain entry point). If no arguments are given, the command line is erased and the command is inserted into the command retrieve buffer. wid identifies the object to be operated on. 0 specifies the current object.

Parameters:

pszOptions - is a string of one or more of the following letters:

W
If command is an external program, run slkwait program to wait for program to complete. This option allows you to view the results after running a DOS text mode application which displays results to the screen. Defaults to OFF.
A
If command is an external program, run program asynchronously (no wait). Defaults to OFF.
R
Insert command into command retrieve buffer (.command) and clear command line. Defaults to OFF unless pszCommand is NULL.
Z
Return pointer to ASCIIZ instead of long. BEWARE: This is a pointer to the "rc" global variable which will change if vsExecute is called again or if you or another macro sets the "rc" variable. Make a copy of the string if necessary.
L
Return pointer to VSLSTR instead of long. BEWARE: This is a pointer to the "rc" global variable which will change if vsExecute is called again or if you or another macro sets the "rc" variable. Make a copy of the string if necessary.
S
Start new undo step.
M
Record source if macro recording.
D
Do refresh. Refresh the screen after executing the command.

Returns: long

Return value of command executed is returned.

Applies To:

All Window Objects

Category:

Miscellaneous Functions
