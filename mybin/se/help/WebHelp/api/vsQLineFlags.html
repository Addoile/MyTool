int VSAPI vsQLineFlags(int wid)

Parameters:

wid - Window id of editor control. 0 specifies the current object.

Returns: int

Gets line status flags for the current line. wid identifies the object to be operated on. 0 specifies the current object. See vsSetLineFlags function for information on meaning of flags returned.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

vsQLineFlags, vsSetLineFlags, vsResetModifiedLineFlags
