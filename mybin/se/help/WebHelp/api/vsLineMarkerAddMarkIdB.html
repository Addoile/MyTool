int VSAPI vsLineMarkerAddMarkIdB(const char *pszBufName, int markid, int BMIndex, int type, const char *pszMessage=a, void *pUserData=a, const char *pszLineData=a, seSeekPos RealLineNum=seSeekPos(0), seSeekPos BeginLineROffset=seSeekPos(0), int col=a, void *reserved=a)

Adds a new line marker. Note that using a mark id instead of a line number has the advantage that the number of lines in the file do not need to be calculated. This is only useful for data files and not source files. However, the draw box feature is not supported.

Parameters:

pszBufName - Name of buffer.

MarkId - Mark id allocated by vsAllocSelection().

BMIndex - Bitmap to display at the line specified. Specify 0 for no bitmap.

type - Type allocated by vsMarkerTypeAlloc().

pszMessage - HTML message displayed when mouse is over bitmap. Only a subset of HTML supported by our mini-HTML control is supported.

pUserData - Pointer to user data.

pszLineData - This is usually 0 but may contain some of the contents of the line.

RealLineNum -

Returns: int

Returns index of new line marker.

Category:

Marker Functions
