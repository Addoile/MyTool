int VSAPI vsScrollMarkupSetTypeColor(int iType, int iColorInfo, int iIsRGB=a)

Set the color for a type of scroll bar markers

Parameters:

wid - Window ID of editor control

iType - Type returned by vsScrollMarkupAllocType

iColorInfo - a SlickEdit color index, or an RGB color. If this is an RGB color - be sure isRGB is 1.

iIsRGB - If true, iColorInfo is an RGB color.

Returns: int

int 0 if successful

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
