int _TreeSearch(int ItemIndex, _str searchCaption, _str options="", _str searchUserInfo=null, int ColIndex=-1)

Searches a tree control.

Parameters:

ItemIndex - By default, this is the parent index of the children to be searched. If the 'S' option is specified, this is the first sibling searched.

searchCaption - Item caption to search for.

options - A string of zero or more of the following:

'I'
Case insensitive search.
'P'
Prefix match.
'S'
Search siblings. When this is specified, itemIndex is the first sibling searched. By default, itemIndex is the parent of the items to be searched.
'T'
Search recursively through the tree. By default, only one level is searched.
'H'
Search hidden nodes.

searchUserInfo - Option item user info to search for.

ColIndex - Column to search. First column is 0. The default is to search all columns.

Returns: int

Returns the index of the tree item or -1 if the search fails.

Applies To:

Tree View

Category:

Tree View Methods

See Also:

_TreeGetCaption, _TreeSetInfo, _TreeGetInfo, _TreeSetUserInfo, _TreeGetUserInfo
