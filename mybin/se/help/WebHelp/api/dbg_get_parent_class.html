int dbg_get_parent_class(int class_id, int parent_id, _str &class_name)

Get the name of the specified parent class

Parameters:

class_id - index of class in class table

parent_id - index of parent class (usually '1')

class_name - (referene) name of parent class

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
