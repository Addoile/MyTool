void _clex_xmlSetConfig(boolean forceUpdate, _str pszTagFilename)

Sets the tag filename for the current buffer used for XML Context Tagging® and may create or update an XML color coding lexer associated with this tag file.

Parameters:

ForceUpdate - Indicates whether to always update an existing XML color coding lexer for this TagFilename. This option is most useful for allowing DTD changes to be reapplied. It can also be useful if you have muliple buffers using the same tag file (same DTD).

TagFilename - Name of tag file.

Applies To:

Edit Window, Editor Control

Categories:

Editor Control Methods, Miscellaneous Functions
