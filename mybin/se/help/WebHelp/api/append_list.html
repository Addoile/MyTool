_command typeless append_list(_str path='', _str arg2='', _str arg3='', _str arg4='')

Appends a tree directory list of the files specified to the current buffer. If filespec is not specified, the current directory list is used. This command is useful when in fileman mode. An '@' sign prefix to filespec indicates that filespec is a file or buffer which contains a list of file names to be used as arguments to this command. Use the dir or list command to change the mode to fileman mode.

Parameters:

cmdline - a string in the format: [@]filespec [[@]filespec ... ]

Returns: typeless

Returns 0 if successful. Common return codes are TOO_MANY_SELECTIONS_RC, TOO_MANY_FILES_RC, TOO_MANY_OPEN_FILES_RC (OS limit). On error, message is displayed.

Category:

File Functions

See Also:

dir, list, append_dir, append_list, fileman
