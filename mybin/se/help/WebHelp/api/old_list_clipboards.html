_command typeless old_list_clipboards()

Displays List Clipboards dialog box. Allows you to select a clipboard from a list of your most recently used (15 is the default maximum) clipboards to insert at the cursor. Subsequent calls to the paste command will insert the same text as your last list_clipboards. LINE type clipboards are inserted before or after the current line depending upon the Line insert style. By default, they are inserted after the current line.

Returns: typeless

Returns 0 if successful. Common return codes are 1 (clipboard empty, nothing selected, or clipboard text too long), TOO_MANY_SELECTIONS_RC, TOO_MANY_WINDOWS_RC, and TOO_MANY_FILES_RC. On error message is displayed.

Applies To:

Edit Window, Editor Control

Category:

Clipboard Functions
