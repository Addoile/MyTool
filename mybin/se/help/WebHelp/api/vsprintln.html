void VSAPI vsprintln(const char *pBuf, int BufLen=-)

Outputs debug data to a scrolling window. The next debug output message will be printed on the next line. This function is not yet supported under UNIX. This function is used to assist debugging.

Use vsprintln to implement your own xprintf

Parameters:

pBuf - Data to be printed.

BufLen - Number of characters in pBuf. -1 specifies that pBuf is null terminated and its length can be determined using this.

Category:

Miscellaneous Functions
