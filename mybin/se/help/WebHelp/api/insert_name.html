int insert_name(_str name, int type, _str info="", int copy_index=0)

Inserts name into names table with type and optional info. Valid types are listed in "slick.sh" and have the suffix "_TYPE". Internal name types may not be inserted except for PICTURE_TYPE and EVENTTAB_TYPE. If the type given is PICTURE_TYPE, name must be a filename.

Returns: int

If successful, the index of the new name table entry is returned. Otherwise 0 is returned and rc is set to INTERPRETER_OUT_OF_MEMORY_RC. On error, message displayed.

Category:

Names Table Functions
