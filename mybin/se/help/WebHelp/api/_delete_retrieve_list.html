void _delete_retrieve_list(_str object_name='')

Delete text box history into list box. When no argument is given, the history is retrieved based on the active form name and active control name (p_name). If the cmdline argument does not contain a '.', the history is based on the active form name and cmdline specifies an alternate control name.

Parameters:

cmdline - is a string in the format: form_name[.ctl_name]

Applies To:

Combo Box, List Box

Categories:

Combo Box Methods, List Box Methods

See Also:
