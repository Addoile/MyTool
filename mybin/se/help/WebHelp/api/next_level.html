_command typeless next_level()

Used in EMACS emulation. Searches for an open parenthesis character '(', '[', or '{' and places the cursor after the matching close parenthesis. Cursor position is unchanged if matching close parenthesis is not found. Comment or string literals may cause this command to match parenthesis incorrectly.

Returns: typeless

Returns 0 if successful. Common return value is TOO_MANY_MARKS_RC. On error, message displayed.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods

See Also:

prev_level, cut_level, cut_prev_level
