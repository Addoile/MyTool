int VSAPI SETagInsertOccurrence(int dbHandle, const slickedit::SEString &occur_name, const slickedit::SEString &file_name=(const char*) 0, slickedit::SEHashTable*occurHashTable=NULL)

Insert a new occurrence into the word index.

Parameters:

dbHandle - database handle

occur_name - Word to be indexed

file_name - Path of file occurrence is located in

occurHashTable - hash table mapping file names to a hash set of symbol names this is used to cache results and avoid inserting the same symbol twice.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
