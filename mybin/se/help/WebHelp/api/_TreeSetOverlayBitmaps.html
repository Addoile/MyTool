void _TreeSetOverlayBitmaps(int nodeIndex, int bitmapArrayIndex[])

Specify bitmaps to overlay on top of the note bitmap

Parameters:

nodeIndex - Index of existing tree node

bitmapArrayIndex - Array of bitmap indexes, [0] is drawn first, then [1], etc. All have to be the same size as the bitmap specified when the node was added (or scale well to that size)

Applies To:

Tree View

Category:

Tree View Methods
