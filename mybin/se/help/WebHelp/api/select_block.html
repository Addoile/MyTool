_command void select_block()

Starts, extends, or locks a block selection. Used for processing columns of text. The first select_block becomes the pivot point. Most select styles allow the selection to be extended as the cursor moves. For these styles, you can invoke this command again to lock the selection so that is does not extend as the cursor moves. In SlickEdit emulation, the selection does not extend as the cursor moves. For this selection style, subsequent calls to this command will select the text between the pivot point and the cursor.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
