int VSAPI vsHvarCallMethod(int wid, VSHVAR hvar, int index, int Nofargs, VSARGTYPE *pArgList)

Call a class method with the given argument list.

Parameters:

wid - window ID

hvar - handle for interpreter variable

index - names table index of class function

Nofargs - number of arguments

pArgList - array of arguments

Returns: int

0 on success. Returns INVALID_ARGUMENT_RC if the method index is invalid. Returns INVALID_HVAR_ARGUMENT_RC if 'hvar' is not a struct or class instance.

Category:

Miscellaneous Functions

See Also:

vsHvarFindMethod, vsHvarCallMethodByName
