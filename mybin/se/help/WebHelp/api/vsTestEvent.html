int VSAPI vsTestEvent(int wid=a, char *pszOptions=a)

Deprecated. Use vsIsEventPending

Parameters:

wid - Window id of editor control. 0 specifies the current object.

pszOptions - A string of zero or more of the following options:

'R'
Specifies no screen refresh.
'K'
Return keys from physical keyboard and not keyboard macro that is being played back.
'P'
UNIX only: Process non-key/mouse events first before testing for a key or mouse event. This option is ignored for non-UNIX platforms.
'M'
When given, only key events are tested.

Returns: int

This function returns non-zero pending key or mouse event. If there is no pending event, 0 is returned.. See our OEM MDI sample program for sample code which translates windows messages into our event constants.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Keyboard Functions
