void _SccDisplayOutput(_str str, boolean clear_buffer=false, boolean strIsViewId=false, boolean doActivateOutput=true)

Append a line of information to the Output tool window.

Parameters:

str - message to append

clear_buffer - (default false) clear window before adding this message?

strIsViewId - (default false) 'str' is actually the view ID of a buffer to switch to

doActivateOutput - (default true) activate the Output tool window if it is not already up.

Category:

Miscellaneous Functions
