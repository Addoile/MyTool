int p_ShowSpecialChars

Determines what special characters are displayed for the current buffer. This property is only available when the object is displayed. The p_ShowSpecialChars property is one or more of the following flags defined in "slick.sh":

  • SHOWSPECIALCHARS_NLCHARS
  • SHOWSPECIALCHARS_TABS
  • SHOWSPECIALCHARS_SPACES
  • SHOWSPECIALCHARS_EOF
  • SHOWSPECIALCHARS_FORMFEED
  • SHOWSPECIALCHARS_ALL

Applies To:

Editor Control, Edit Window

Categories:

Edit Window Properties, Editor Control Properties
