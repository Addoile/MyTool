_str alias_match(_str name, int find_first)

Returns the name of the alias matching the prefix name_prefix. If a match is not found, '' is returned. Specify find_first == 1, to start from the first alias and 0 to find the next match. You must call this function with find_first==2, to terminate matching after no match is found.

Returns: _str

The name of the alias matching the prefix specified by name.

Category:

Completion Functions
