int p_eventtab

Determines the event table used for user level 1 inheritance. This property is automatically set when a Slick-C® module is loaded which defines an event table that identifies the object.

Example:

   index=find_index('eventtab_name',EVENTTAB_TYPE);
   if (index) {
      p_eventtab=index;
   }

Applies To:

All Window Objects

Category:

All Windows Properties
