void VSAPI vsBufName2Caption(int wid, char *pszCaption, int MaxCaptionLen, char state, int *pMaxCaptionLen=a, bool showModified=true)

This function is used to convert the buffer name for an MDI child edit window into a window title. Currently buffers with no name have a caption in the form "Untitled" where nnn is the buffer id of the buffer.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

pszCaption - Output string for window caption.

MaxCaptionLen - Number of characters allocated in pszCaption. We recommend VSMAXFILENAME.

state - Indicates the state of the MDI child edit window. One of the following:

'I'
Caption is for an iconized window. No path is placed in the caption.
'N'
Caption is for a normalized window.
'M'
Caption is for a maximized window.

pMaxCaptionLen -

showModified - If set to false, the modified indicator (*) will not be appended to caption when buffer is modified. Defaults to true.

Applies To:

Edit Window, Editor Control

Categories:

Buffer Functions, Edit Window Methods, Editor Control Methods
