int VSAPI vsHvarCopy(HVAR hvar, HVAR src)

Sets value of interpreter variable specified to be a copy of the given interpreter variable.

Parameters:

hvar - interpreter variable (destination)

src - interpreter variable (source)

Returns: int

Returns <0 on error.

Category:

Miscellaneous Functions
