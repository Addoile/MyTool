_command void ispf_reset(_str cmdline="")

Resets the line prefix area for specific types of lines or all lines in the buffer.

Syntax:

   RESET [ ALL | CHANGE | COMMAND | ERROR | EXCLUDED |
           FIND | LABEL | SPECIAL ] [ line_range ]

ALL
Reset all line prefixes. This is the default.
CHANGE
Search for lines with a change flag (==CHG>).
COMMAND
Search for lines containing an edit line command.
ERROR
Search for lines with an error flag (==ERR>).
EXCLUDED
Search only for excluded lines.
LABEL
Search for lines with a label.
SPECIAL
Search for lines with special non-data lines, (e.g. =COLS>)
line_range
Two numbers or labels, specifying the start and end

Category:

ISPF Primary Commands

See Also:

ispf_do_lc, ispf_locate, ISPF Line Commands
