boolean tag_check_context_flags(int context_flags, int tag_flags, _str type_name)

Returns true if the tag having the given tag_flags and type matches the requirements set by the given context flags.

Parameters:

context_flags - VS_TAGCONTEXT_

tag_flags - Tag flags indicating symbol attributes, see tag_insert_simple for details.

type_name - tag type (from tag details)

Returns: boolean

Returns true if the tag with the given type and flags passes the context flags, otherwise, returns false.

Category:

Tagging Functions
