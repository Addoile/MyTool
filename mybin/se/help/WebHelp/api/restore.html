_command typeless restore(_str options="", int alternate_view_id=0, _str relativeToDir=null, boolean restoreFiles=true, boolean restoringFromProject=false)

Resumes the last edit session which was terminated by the safe_exit command. The safe_exit command will save the window/buffer configuration only if auto restore is set to on. When auto restore is on and the editor is invoked with no file or command parameters, the restore command is automatically invoked. SlickEdit stores auto restore information in the file "vrestore.slk". You may define an environment variable called VSLICKRESTORE and assign it a path where you want SlickEdit to look for this file.
