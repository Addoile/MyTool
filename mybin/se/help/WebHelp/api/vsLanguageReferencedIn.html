int VSAPI vsLanguageReferencedIn(const char *refLangId, const char *pszLangId=a)

Does the current source language have the given language in it's list of languages which symbols can be referenced in?

If 'lang' is not specified, the current object must be an editor control.

Parameters:

refLangId - language ID to check reference in

pszLangId - current language ID (default=p_LangId)

Returns: int

'true' if the language matches, 'false' otherwise.

Since:

17.0

Attention:

This function is thread-safe provided pszLangId is not NULL or the empty string.

Applies To:

Edit Window, Editor Control

Category:

Tagging Functions
