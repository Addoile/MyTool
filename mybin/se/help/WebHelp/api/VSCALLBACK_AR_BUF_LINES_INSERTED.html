0 VSCALLBACK_AR_BUF_LINES_INSERTED

void (VSAPI *pfnInsertLine)(int buf_id,seSeekPos AfterLineNum,seSeekPos Noflines)

The LINES_INSERTED callback is called when lines are inserted into a buffer.

WARNING: It is not safe to access editor buffer data during this callback.

Parameters:

buf_id - Buffer id.

AfterLineNum - Line number (VSP_LINE and not VSP_RLINE) Lines we inserted after this line.

Noflines - Number of lines inserted.

Category:

Editor Control Callback Functions
