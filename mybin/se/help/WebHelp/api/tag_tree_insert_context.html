int tag_tree_insert_context(int tree_wid, int treeIndex, int filter_flags, int include_tab, int force_leaf, int tree_flags, int show_statements)

Insert all the context tags passing through the tag filter flags into the given tree. Use tag_tree_prepare_expand prior to calling this function in order to configure the browser bitmap set.

For synchronization, perform a tag_lock_context(false) prior to invoking this function.

Parameters:

tree_wid - Window ID of the tree control to insert into.

treeIndex - Parent tree node index to insert item under.

filter_flags - Tag filter flags, only insert tags passing this filter. See tag_filter_type for more details.

include_tab - append class name after signature if 1, prepend class name with :: if 0. See tag_tree_make_caption for further explanation.

force_leaf - If < 0, force leaf node, if equal to 0, choose by tag type. Normally "container" tag types, such as classes or structs are automatically inserted as non-leaf nodes. If > 0, insert the items from the current context as a tree, displaying the heirarchy of inner/outer nesting of tags, going by the positions calculated by tag_sort_context().

tree_flags - tree flags to set for this item

show_statements - show statements in "Defs" tab

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
