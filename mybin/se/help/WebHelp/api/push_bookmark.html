_command typeless push_bookmark(_str markid='')

Creates a book mark at the cursor and places it on a stack. After calling this command you may move the cursor within the same file or another file and restore your original cursor position by invoking the pop_bookmark command (Ctrl+, or "Search", "Pop Bookmark"). The name of the new bookmark created is displayed.

Parameters:

markid - named selection to create bookmark on

Returns: typeless

Returns 0 if successful.

Applies To:

Edit Window

Categories:

Bookmark Functions, Edit Window Methods, Editor Control Methods

See Also:

pop_bookmark, set_bookmark, toggle_bookmark, goto_bookmark, bookmark_stack
