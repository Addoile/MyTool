int VSAPI vsFontQDefaultCharSet(int flags, const char *pszFontName, const char *pszFontSize)

Parameters:

flags - One or more of the VSFONTLISTFLAG_??? flags ORed together.

VSFONTLISTFLAG_SCREEN_FONTS
Test screen font.
VSFONTLISTFLAG_PRINTER_FONTS
Test printer font.

pszFontName - Name of font to list sizes for. Specify 0 to list available font names.

pszFontSize - Font size.

Returns: int

Windows only: Returns Windows character set.

Category:

Font Functions
