_str def_file_types

File filters used by File Open and File Save As dialogs. Separate each file filter with a comma. Place file patterns in parentheses. Separate file patterns with a semicolon. The first file filter is used to initialize the file list.

Example:

   Basic Files (*.bas), All Files (*.*)
   C Files (*.cpp;*.cxx;*.c;*.h), All Files (*.*)

Category:

Configuration Variables
