typeless lbutton_up()

Command Button - Occurs when the left mouse button is pressed and released and also when the selection character is pressed.

Check Box - Occurs when the left mouse button is pressed and released and also when the selection character is pressed.

Radio Button -Occurs when the left mouse button is pressed and released and when the p_value property is changed to a non-zero value.

Picture Box, Image -Depends on p_style property as follows:

PSPIC_DEFAULT
Occurs when the left mouse button is pressed and released.
PSPIC_PARTIAL_BUTTON
Occurs when the left mouse button is pressed and released while inside the window.
PSPIC_AUTO_BUTTON
Occurs when the left mouse button is pressed and released while inside the window.
PSPIC_AUTO_CHECK
Occurs when the left mouse button is pressed.
PSPIC_BUTTON
(Image control only) Occurs when the left mouse button is pressed and released while inside the window.
PSPIC_FLAT_BUTTON
(Image control only) Occurs when the left mouse button is pressed and released while inside the window.
PSPIC_HIGHLIGHTED_BUTTON
(Image control only) Occurs when the left mouse button is pressed and released while inside the window.

Other - Occurs when the left mouse button is released.
