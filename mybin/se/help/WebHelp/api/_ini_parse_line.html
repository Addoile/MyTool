int _ini_parse_line(int ini_view, _str &field_name, _str &info, int first=0)

After a call to _ini_find_section, this function is called to parse each field_name=value line in the section.

Returns: int

A non-zero value is returned when the end of the section is reached. 

   ".ini" File Syntax:
         [section1]
         field_name1=value1
         field_name2=value2
         ...
         [section2]
         field_name1=value1
         field_name2=value2
         ...
