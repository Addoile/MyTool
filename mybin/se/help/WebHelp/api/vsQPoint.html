void VSAPI vsQPoint(int wid, seSeekPos *pPoint, seSeekPos *pDownCount, char Option='P')

Parameters:

wid - Window id of editor control. 0 specifies the current object.

pPoint - If option=='P', ouput for seek position to beginning of line. Otherwise, output for current line number. -1 indicates current line number is unknown.

pDownCount - Ouput for number of lines to cursor down to get to the current line. This is non-zero in rare cases such as when editing record files (p_buf_width!=0) which have no end of line characters and blank lines are inserted.

option - 'P' or 'L'.

'P'
Sets pPoint to the seek position to the beginning of the current line. Sets pDownCount to the number of lines to cursor down to get to the current line when starting from the seek position to the beginning of line. It is unusual for pDownCount to be non-zero. This can only occur for record files (p_buf_width!=0) when a line has 0 bytes INCLUDING the end of line characters. Lines of record files have no end of line characters.
'L'
Sets pPoint to current line number. If the current line number is not known, pPoint is set to -1.
