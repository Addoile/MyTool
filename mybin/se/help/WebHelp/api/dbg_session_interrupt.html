int dbg_session_interrupt(int session_id, int thread_id)

Interrupt the execution of a specific thread.

Parameters:

session_id - debugger session ID (1 .. max_session)

thread_id - ID of specific thread to interrupt

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

To interpret the thread id 7: 

   int status = dbg_session_interrupt(session_id, 7);
   if (status) {
      return status;
   }

Category:

Debugger Functions
