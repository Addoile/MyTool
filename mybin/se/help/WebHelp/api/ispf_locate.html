_command void ispf_locate(_str cmdline="")

The locate command is used to find lines with specific attributes in the prefix line. It can be used to find a line with a specific label, line number, command, changed line, error line, or no-save lines.

Syntax:

   LOCATE line_number
   LOCATE label
   LOCATE [NEXT | PREV | FIRST | LAST ]
          [LABEL COMMAND CMD ERROR CHANGE CHG C SPECIAL S EXCLUDED X]
          [range]

Arguments:

line_number
Line number to go directly to.
label
ISPF label to go directly to.
range
Two numbers or labels, specifying the starting and ending lines to search within, inclusive.
NEXT
Find the next occurrence starting from the cursor.
ALL
Find all matches in the specified range.
FIRST
Find only the first occurrence in the specified range.
LAST
Find only the last occurrence in the specified range.
PREV
Find the previous occurrence starting from the cursor.
CHANGE
Search for lines with a change flag (==CHG>).
COMMAND
Search for lines containing an edit line command.
ERROR
Search for lines with an error flag (==ERR>).
EXCLUDED
Search only for excluded lines.
LABEL
Search for lines with a label.
SPECIAL
Search for lines with special non-data lines, (e.g. =COLS>)
QUIET
Do not display messages.

Category:

ISPF Primary Commands
