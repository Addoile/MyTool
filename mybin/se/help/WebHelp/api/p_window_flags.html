int p_window_flags

Returns: int

Returns the window flags. The only window flag is HIDE_WINDOW_OVERLAP which indicates the window is a hidden window which should not get focus. The hidden window stores many views of system buffers such as ".command", ".clipboards", and ".process-command", Currently there can only be one hidden window. This property is available only when the object is displayed.

Applies To:

Edit Window

Categories:

Edit Window Properties, Editor Control Properties, List Box Properties
