int VSAPI vsRefactorConfigGetInclude(const char *configName, int includeIndex, char *includePath, int maxLength)

Get the full path to the specified include path

Parameters:

configName - name of configuration to query

includeIndex - index within include path to query

includePath - set to include path

maxLength - length of includePath buffer

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions

See Also:

vsRefactorConfigAdd, vsRefactorConfigAddInclude, vsRefactorConfigDeleteInclude, vsRefactorConfigDeleteAllIncludes
