void _ok.on_create()

Displays Print dialog box. This form is not supported by the UNIX version.

Returns: void

Returns '' if the dialog box is cancelled. Returns 0 if the user has specified to print the selection and sets the global variables _param1, _param2, _param3, and _param4. The _param1-_param4 variables may be passed as arguments 1-4 to the print_selection function to perform the printing. 1 is returned if the user has selected to print the current buffer and the global variables _param1-_param4 are set. The _param1-_param4 variables may be passes as arguments 1-4 to the _print function to perform the printing

Category:

Buffer Functions
