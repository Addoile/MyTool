_command int copy_to_clipboard(_str name='')

Copies selection to the clipboard. If no text is selected the current line is copied to the clipboard. The optional cbname string specifies the name of the clipboard. This name can be used with the paste command to specify a specific clipboard. Copies of all clipboards are stored in the ".clipboards" buffer. The clipboard may be inserted by the paste command (Ctrl+V). Previous clipboards may be inserted with the list_clipboards command (Ctrl+Shift+V).

IMPORTANT: cbname needs to start with an alphabetic character (a-z). numeric values for cbname are reserved for VI emulation.

NOTE: If there is no selection, the current line is copied.

NOTE: If the copy is done using Ctrl+C, the current window is the Build window, and there is no selection, then execute stop-build. This can be disabled by setting the configuration variable def_stop_process_noselection to false.

Returns: int

Returns 0 if successful. Common return codes are 1 (no text marked in current buffer) and TOO_MANY_SELECTIONS_RC. On error, message is displayed.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Clipboard Functions, Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods
