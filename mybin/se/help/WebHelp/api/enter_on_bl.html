typeless enter_on_bl()

Typically used by a command which is bound to the ENTER key. Called when ENTER key is pressed on a blank line.

Returns: typeless

Returns the column position where the cursor should be placed on a new line that is inserted by an ENTER key command. You may want to write a replacement for this procedure.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
