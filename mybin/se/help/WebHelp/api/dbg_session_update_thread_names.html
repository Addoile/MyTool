int dbg_session_update_thread_names(int session_id)

Update the names of threads and thread groups in this debugging session.

NOTE: You must update the threads and thread groups first.

Parameters:

session_id - debugger session ID (1 .. max_session)

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

   int status = dbg_session_update_threads(session_id);
   if (status) {
      return status;
   }
   status = dbg_session_update_thread_groups(session_id);
   if (status) {
      return status;
   }
   status = dbg_session_update_thread_names(session_id);
   if (status) {
      return status;
   }

Category:

Debugger Functions
