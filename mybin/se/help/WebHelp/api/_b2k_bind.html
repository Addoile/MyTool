int _b2k_bind(_str command, _str key_sequence, _str effect_all_modes, _str options='')

We recommend you use the set_eventtab_index function to bind keys instead of this function. This function is likely to be changed in the future. This function binds command to the binary key_sequence specified. If effect_all_modes is false, the command is bound to the mode event table (p_mode_eventtab). Specify an unbind_option of "U" to unbind the key. You may prefix the unbind_option string with a "C" to effect command mode for VI emulation. The command parameter is ignored when unbinding a key.

Parameters:

command -

key_sequence -

effect_all_modes -

Returns: int

Returns 0 if successful. Can fail if incorrect parameters given.

Applies To:

Edit Window, Editor Control

Category:

Keyboard Functions
