typeless for_each_mdi_child(_str command, ...)

Executes command for each MDI edit window. command must be a Slick-C® function if _external_cmd is '' or not specified. Otherwise command may be any external macro or program. Windows with (p_window_flags & HIDE_WINDOW_OVERLAP) true, are skipped.

Returns: typeless

Returns 0 if successful. Common return codes are STRING_NOT_FOUND_RC, TOO_MANY_SELECTIONS_RC, and FILE_NOT_FOUND_RC.

Category:

Window Functions
