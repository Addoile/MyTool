_command int copy_to_cursor(...)

Copies selection to the cursor. Character or block selections are inserted before the character at the cursor. For line selection, lines are inserted before or after current line depending on the Line insert style. By default line marks are inserted after the current line. Resulting selection is always on destination text.

Returns: int

Returns 0 if successful. Common return codes are TEXT_NOT_SELECTED_RC and SOURCE_DEST_CONFLICT_RC. On error, message is displayed.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
