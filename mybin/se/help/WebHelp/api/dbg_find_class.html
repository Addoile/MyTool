int dbg_find_class(_str class_name, int name_only, int start_id=0)

Find the class with the given class name.

Parameters:

class_name - name of class to look for

name_only - find first match using class name only

start_id - class ID to start searching at

Returns: int

class ID > 0 on success, <0 on error.

Category:

Debugger Functions
