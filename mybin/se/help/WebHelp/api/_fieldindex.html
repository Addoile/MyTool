int _fieldindex(_str fieldName)

Parameters:

fieldName - Name of field to look up.

Returns: int

Returns the index of the given given class field. Indexes start at zero for the first field of the base (least derived) class.

Example:

class AAAA { int x; int y; };
class BBBB : AAAA { int z; };
void testFieldIndex() {
   BBBB b;
   say("x is at position "b._fieldindex("x"));  // 0
   say("y is at position "b._fieldindex("y"));  // 1
   say("z is at position "b._fieldindex("z"));  // 2
}
