_command typeless load(_str arg1='')

Compiles (if necessary) and loads the modules specified. If the module is already loaded it is replaced. If no module is specified and the current buffer has the extension ".e", the current buffer is saved (if necessary) and loaded.

Prior to loading the module, it will invoke the _on_unload_XXX callbacks with a single argument indicating the Slick-C® module to be loaded.

Returns: typeless

Returns 0 if successful. Since this command does not actually load the modules until the macro terminates, it is possible that the load can still fail.

Applies To:

Edit Window

Category:

Macro Programming Functions

See Also:

gui_load, gui_unload, unload
