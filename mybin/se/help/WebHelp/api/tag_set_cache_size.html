int tag_set_cache_size(int cache_size, int cache_max)

Specify the amount of memory to use for the database cache.

Parameters:

cache_size - Amount of memory in bytes.

cache_max - maximum amount of memory to allow cache to use dynamically set depending on the machine's available memory.

Returns: int

0 on success, <0 on error. The minimum size cache allowed is 512k.

Since:

3.0

Category:

Tagging Functions
