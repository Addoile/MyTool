int tag_tree_add_members_in_section(_str prefix, int type_id, int mask, int nzero, _str category_name, int &in_count)

Add globals with the given type the given section of a category, as set up using tag_tree_prepare_expand. Refer to tag_find_global for details about the parameters type_id, mask, and non_zero. Categories are divided into sections when they have so many items that it would be impractical to attempt to view all the items in the category under one tree node. Currently sections are not subdivided into subsections, even if they contain thousands of items.

Parameters:

prefix - Tag name prefix to search for. If given the specific string "[Misc]", it will insert all tags starting with characters that are not identifier start characters.

type_id - Type id (see tag_get_type) If (type_id<0), inserts global tags with any user defined tag types.

mask - Tag flags indicating symbol attributes, see tag_insert_simple for details.

nzero - add member if (mask & tag_flags) is zero or nonzero?

category_name - tag category to add members from

in_count - (Input, Output), number of items inserted. See tag_tree_add_members_of for more information about how this relates to refresh operations and prompting.

Returns: int

0 on error, <0 on error.

Category:

Tagging Functions
