void _map_xy(int from_wid, int to_wid, int &x, int &y=0, int scale_mode=SM_PIXEL)

Converts (maps) the x and y coordinates from the coordinate space relative to the from_wid to the coordinate space relative to the to_wid. If scale_mode specifies the input and output scale mode for the x and y coordinates.

Parameters:

from_wid - Window to map coordinates from.

to_wid - Window to map coordinates to. 0 defaults to the desktop.

x - x-coordinate relative to from_wid.

y - y-coordinate relative to from_wid.

scale_mode - (optional). Scale mode of the from/to windows. May be SM_PIXEL or SM_TWIP. Defaults to SM_PIXEL.

Category:

Miscellaneous Functions
