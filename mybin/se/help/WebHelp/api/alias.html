_command int alias(_str params='')

The alias command displays, creates, or modifies aliases. To expand an alias, execute the expand_alias command (Ctrl+Space bar).

If no arguments are given or cmdline is "", you will prompted to pick an alias file and the Alias Editor dialog box will be displayed. We recommend you use the alias editor for creating, deleting, viewing, or modifying aliases because it is easier to use than this command.

Use the "-e" argument to edit aliases for the current file. Use the "-surround" option to edit surround with aliases for the current file.

Aliases are stored by default in a file called "alias.als.xml". Use the VSLICKALIAS environment variable to define one or more alias files. Separate each alias filename with a ';' (UNIX: ':'). New aliases created with the alias command are placed in the first file defined by the VSLICKALIAS environment variable.

The value of the alias may have any of the following escape sequences embedded in it. 

%<EnvVar>%  Insert the value of the environment variable specified
%\d   Insert the date
%\t   Insert the time
%\i   Indent
%\b   Unindent
%\c   Place cursor
%\n   Current function name
%(ParamName)   Argument replacement.

Prefix the name of the alias with "-d" to delete an alias.

See alias_cd command for information on using an alias to change directory.

Returns: int
