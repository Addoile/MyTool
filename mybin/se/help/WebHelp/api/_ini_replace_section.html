int _ini_replace_section(_str filename, _str section_name, int temp_view_id)

For each field_name=value line in buffer corresponding in temp_view_id, the field_name value is replaced or inserted into section. In addition, the temp_view_id is deleted. Unlike the _ini_put_section function, fields which are not defined are NOT deleted. This allows more than one user to replace multiple field values in one call without deleting field names defined by someone else. This function is slower than the _ini_put_section function which replaces the entire section with the buffer corresponding to temp_view_id.

Returns: int

0 if successful 

     ".ini" File Syntax:
         [section1]
         field_name1=value1
         field_name2=value2
         ...
         [section2]
         field_name1=value1
         field_name2=value2
         ...
