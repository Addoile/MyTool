_str column_process_list(int first_col, int col_width, _str search_string, _str action_fun, _str search_opts='')

Generic column list processing function. For each line in the current buffer containing an occurrence of string search_string within the columns specified by first_col and col_width, action_fun is executed with that line as the current line. The cursor is placed on the last line with a match.

Parameters:

first_col - Start column of area to be searched within.

col_width - Width of area to be searched.

search_string - String to search for in area specified.

action_fun - Name of procedure which operates on lines found by search. Use a pre-existing action function such as zap_line or _fselect_line if possible. The zap_line procedure deletes the current line. The _fselect_line procedure replaces the first character of the current line with '>'.

search_options - Optionally specifies that search_string is a regular expression. Specify 'r' for regular expression. See Regular Expressions for more information.

Returns: _str

Returns 0 if successful. Otherwise TOO_MANY_SELECTIONS_RC is returned.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Search Functions
