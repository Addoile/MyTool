int VSAPI vsTreeGetUserInfo(int wid, int iHandle, VSHVAR *hvar)

Gets the user info value for the specified tree item.

Parameters:

wid - Window id of tree control. 0 specifies the current object.

iHandle - Tree item index.

pszInfo - Buffer to put user info in

iInfoSize - Size of buffer pszInfo

Returns: int

0 if successful

Applies To:

Tree View

Category:

Tree View Methods

See Also:

_TreeSetCaption, _TreeGetCaption, _TreeSetInfo, _TreeGetInfo, _TreeGetUserInfo
