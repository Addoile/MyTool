int _move_to_cursor(_str markid="")

Moves the selection to the cursor. For block and character selections, the text is inserted at the cursor position. In the case of a line selection, the lines are inserted before or after the current line depending upon the Line insert style. Lines are inserted after the current line by default. Resulting selection is placed on the inserted text. Text may be marked with one of the commands select_char (F8), select_line (Ctrl+L), or select_block (Ctrl+B).

Parameters:

mark_id - is a handle to a selection returned by one of the built-ins _alloc_selection or _duplicate_selection. A mark_id of '' or no mark_id parameter identifies the active selection.

Returns: int

Returns 0 if successful. Common return values are TEXT_NOT_SELECTED_RC, SOURCE_DEST_CONFLICT_RC, and INVALID_SELECTION_HANDLE_RC. On error, message is displayed.

Applies To:

Edit Window, Editor Control
