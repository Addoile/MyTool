boolean def_expansion_overrides_locals

If set to true, syntax expansion can still expand a keyword prefix, even if there is a local variable with the same name as the prefix. For example: 

   void foobar(int i) {
      i<space>
   }
will expand to: 
   void foobar(int i) {
      if (<cursor>) {
      }
   }

Default:

false

Category:

Configuration Variables
