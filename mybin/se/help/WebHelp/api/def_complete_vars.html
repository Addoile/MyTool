_str def_complete_vars

If we're in a comment or string, check everywhere. If not, just look for variables (not color coded).

First value is on/off, second value is allow any match (color coded or not) within [value] lines from the cursor position (of the prefix). The first value is redundant with the COMPLETE_LOOK_FOR_ANYTHING flag. This could be cleaned up.

Default:

"1 2"

Category:

Configuration Variables

See Also:

complete_next, complete_prev, complete_more
