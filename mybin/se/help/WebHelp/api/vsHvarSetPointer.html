int VSAPI vsHvarSetPointer(HVAR hvar, HVAR src)

Sets value of interpreter variable specified to be a pointer to the given interpreter variable.

NOTE: As always with pointer types, use with care, because interpreter variables are not reference counted. If the src variable goes out of scope, it will be destructed, leaving this pointer dangling.

Parameters:

hvar - interpreter variable (destination)

src - interpreter variable (source)

Category:

Miscellaneous Functions

See Also:

vsHvarFree, vsHvarAlloc, vsHvarMakeEmpty, vsHvarArrayEl, vsHvarSetI, vsHvarSetI64, vsHvarSetZ
