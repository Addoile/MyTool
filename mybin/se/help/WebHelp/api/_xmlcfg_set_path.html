int _xmlcfg_set_path(int iHandle, _str QueryStr, _str AttrName=null, _str AttrValue=null, int iFlags=0)

Sets or adds an attribute value for the element node specified.

Parameters:

iHandle - Handle to an XMLCFG tree returned by _xmlcfg_open() ,_xmlcfg_create(), or _xmlcfg_open_from_buffer().

QueryStr - This is a small subset of an XPath expression. See _xmlcfg_find_simple() for information on this parameter.

AttrName - Name of attribute.

AttrValue - New value for attribute.

iFlags - -1 indicates that an error should be returned if the attribute does not already exists. Otherwise, the _xmlcfg_add_attribute() function is called to add the attribute.

A combination of the following flags:

VSXMLCFG_ADD_ATTR_AT_BEGINNING
Insert attribute at the beginning of the list of attributes at this node. By default, the attribute is added at the end.

Returns: int

Returns NodeIndex (>=0) for QueryStr (not attribute) if successful. Otherwise a negative return code defined in "rc.sh" is returned.

Example:

// Find the first file element node under /project with name equal to
// "main.c" and set the "options" attribute value.
_xmlcfg_set_path(handle,
    "/project/file[file-eq(@name,'main.c')]",
    "options","-Zi");

Category:

XMLCFG Functions
