int _strBeginChar(_str string, int col, int &charLen=0, boolean beginComposite=true)

Parameters:

string - Input string to perform operation on. When SlickEdit is running in UTF-8 mode, this string is UTF-8. Otherwise, this string is SBCS/DBCS for the active code page.

col - Column position within string. 1..length(string)

CharLen - Set to length of entire character at position col.

BeginComposite - When true, column position returned is the beginning of a composite character (one or more UTF-8 sequences) sequence. Otherwise, of a UTF-8 character sequences is returned. This parameter is ignored when SlickEdit is not running in UTF-8 mode.

Returns: int

Returns the column position of the beginning of the character which contains the byte at column col.

Category:

String Functions
