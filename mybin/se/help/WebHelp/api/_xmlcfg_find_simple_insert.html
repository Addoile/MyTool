int _xmlcfg_find_simple_insert(int iHandle, _str QueryStr, int NodeIndex=TREE_ROOT_INDEX)

Searches for nodes based on an subset XPath expression and inserts the results into the current buffer.

Parameters:

iHandle - Handle to an XMLCFG tree returned by _xmlcfg_open() ,_xmlcfg_create(), or _xmlcfg_open_from_buffer().

QueryStr - This is a small subset of an XPath expression. See _xmlcfg_find_simple() for information on this parameter.

NodeIndex - Tree node index to start search from.

Returns: int

Returns 0 if successful. Otherwise a negative error code is returned.

Example:

// Find the value of all "name" attributes under
// "/project/file" element nodes.
_xmlcfg_find_simple_array(handle,"/project/file/@name");

Category:

XMLCFG Functions
