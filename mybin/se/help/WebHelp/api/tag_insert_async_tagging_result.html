int tag_insert_async_tagging_result(_str fileName, int taggingFlags, int bufferId)

Insert the tags which were gathered by the asynchronous parsing for this tagging job. This applies to updating tag databse, the current context, or for updating local variables. Note that becuase of embedded code sections, there may be more work still required after this method is complete if there are embedded code blocks which also require tagging work.

Parameters:

fileName - Name of file to update tagging results for

taggingFlags - tagging options flags (bitset of VSLTF_*)

bufferId - ID of editor buffer to update tagging for

Returns: int

0 on success, <0 on error

Category:

Tagging Functions
