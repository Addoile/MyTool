int _xmlcfg_get_modify(int handle)

Fetches the XML document modify state.

Parameters:

handle - Handle to an XMLCFG tree returned by _xmlcfg_open(), _xmlcfg_create(), or _xmlcfg_open_from_buffer().

Returns: int

Returns a non-zero value if the XML document has been modified.

Category:

XMLCFG Functions
