_str pgetkey(...)

This function performs a normal get_event and translates the alt_prefix, ctrl_prefix, and case_indirect key bindings into the appropriate key. The translated key is returned and the global variable _key is set to the returned key value.

Parameters:

get_flags - Used as first parameter passed to get_event built-in. See get_event built-in for description of first parameter. Defaults to ''.

root_keys - Name table index of root event table. Defaults to index of "root_keys" event table.

mode_keys - index of mode specific event table.

Category:

Keyboard Functions
