_command typeless dunload(_str filename='')

This function is not yet available in the UNIX version. Unloads a DLL. All functions registered to this DLL with the vsDllExport function are deleted. Do not specify a path in DllFilename. If DllFilename is not given, you a prompted for one.

Returns: typeless

Returns 0 if successful.

Applies To:

Edit Window

Category:

Macro Programming Functions

See Also:

dload, _dllload
