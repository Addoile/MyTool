int _xmlcfg_set_value(int iHandle, int NodeIndex, _str Value)

Sets the value for the node specified. The value returned is null except for comment, attribute, CDATA, and PCDATA nodes.

Parameters:

iHandle - Handle to an XMLCFG tree returned by _xmlcfg_open() ,_xmlcfg_create(), or _xmlcfg_open_from_buffer().

NodeIndex - Tree node index.

Value - New value for node.

Returns: int

Returns 0 if successful. Otherwise a negative return code is returned.

Category:

XMLCFG Functions
