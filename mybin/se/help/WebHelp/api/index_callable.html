int index_callable(int index)

Returns: int

Returns index into the name table of the module to which the procedure or command, corresponding to index, is linked. If index is invalid or does not correspond to a procedure or command, 0 is returned.

Example:

         index=find_index('upcase_filter');
         if (!index_callable(index) ) {
              message('upcase_filter name in name table but not linked');
         } else {
              message('upcase_filter is defined in module
'name_name(index));
         }
