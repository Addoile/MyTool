void tag_get_info(_str &tag_name, _str &type_name, _str &file_name, int &line_no, _str &class_name, int &tag_flags)

Retrieve general information about the current tag (as defined by calls to tag_find_equal, tag_find_prefix, tag_find_in_file, tag_find_in_class, etc.). If the current tag is not defined, such as immediately after opening a database or a failed search), all strings will be set to '', and line_no and tag_flags will be set to 0.

Parameters:

tag_name - (Output) Tag string (native case)

type_name - (Output) Tag type name, see tag_get_type.

file_name - (Output) Full path of file the tag is located in

line_no - (Output) line number of tag within file set to 0 if not defined.

class_name - (Output) name of class that tag is present in, uses concatenation (as defined by language rules) to specify names of inner classes (see insert, above). set to empty string if not defined.

tag_flags - (Output) Tag flags indicating symbol attributes, see tag_insert_simple for details.

Category:

Tagging Functions

See Also:

tag_get_detail, tag_find_equal, tag_find_in_class, tag_find_in_file, tag_find_prefix, tag_next_equal, tag_next_in_class, tag_next_in_file, tag_next_prefix
