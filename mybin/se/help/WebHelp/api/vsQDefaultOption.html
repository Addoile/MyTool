int VSAPI vsQDefaultOption(int option)

Parameters:

option - One of the following options:

VSOPTION_WARNING_ARRAY_SIZE
A warning message is displayed if a Slick-C® array of this size or larger is created.
VSOPTION_WARNING_STRING_LENGTH
A warning message is displayed if a Slick-C® string of this size or larger is created.
VSOPTION_VERTICAL_LINE_COL
Column at which editor control displays a vertical line. 0 indicates no vertical line column.
VSOPTION_WEAK_ERRORS
Indicates whether a less server Slick-C® error should set the find_error commands error location.
VSOPTION_MAXIMIZE_FIRST_MDICHILD
Indicates whether the first MDI Child should be created maximized.
VSOPTION_MAXTABCOL
Specifies the maximum column to expand tabs. Tab characters after this column are not expanded.
VSOPTION_CURSOR_BLINK
Indicates whether the cursor should blink.
VSOPTION_DISPLAY_TEMP_CURSOR
Indicates whether the editor control should display a temporary cursor for an active MDI child which does not have focus.
VSOPTION_LEFT_MARGIN
Left margin in twips between the editor control text and the left edge of the window. This is ignored, when bitmaps are displayed.
VSOPTION_DISPLAY_TOP_OF_FILE
Indicates whether the editor control should display a "Top of File" line.
VSOPTION_HORIZONTAL_SCROLL_BAR
Indicates whether MDI Children should have scroll bars.
VSOPTION_VERTICAL_SCROLL_BAR
Indicates whether MDI Children should have scroll bars.
VSOPTION_HIDE_MOUSE
Indicates whether the mouse should be hidden when you start typing in an editor control.
VSOPTION_ALT_ACTIVATES_MENU
Indicates whether a lone alt key should activate the menu bar.
VSOPTION_DRAW_BOX_AROUND_CURRENT_LINE
Indicates whether the editor control draws a dotted box or ruler around the current line.
  • VSCURRENT_LINE_BOXFOCUS_NONE -- Do not draw box
  • VSCURRENT_LINE_BOXFOCUS_ONLY -- Just draw box, no ruler
  • VSCURRENT_LINE_BOXFOCUS_TABS -- Draw tabs ruler
  • VSCURRENT_LINE_BOXFOCUS_INDENT -- Draw syntax indent ruler
  • VSCURRENT_LINE_BOXFOCUS_DECIMAL -- Darw decimal ruler
VSOPTION_MAX_MENU_FILENAME_LEN
Indicates the maximum length of a filename displayed in a menu.
VSOPTION_PROTECT_READONLY_MODE
Indicates whether a read only mode editor control allows editing. This option has no effect for a an editor control if VSP_PROTECTREADONLYMODE!=0.
VSOPTION_PROCESS_BUFFER_CR_ERASE_LINE
Indicates whether a carriage return not followed by a line feed erase the line in the current process buffer.
VSOPTION_ENABLE_FONT_FLAGS
No longer supported.
VSOPTION_APIFLAGS
VSINIT.APIFlags You can not set this option.
VSOPTION_HAVECMDLINE
Indicates whether a command line is currently available. This gets set when an event is dispatched or vsExecute is called. You can not set this option.
VSOPTION_QUIET
Indicates whether the -Q invocation option was specified.
VSOPTION_SHOWTOOLTIPS
Indicates whether tool tip message should be displayed.
VSOPTION_TOOLTIPDELAY
Determines the tool tip message delay in 10ths of a second.
VSOPTION_HAVEMESSAGELINE
Indicates whether a message line is currently available. This gets set when an event is dispatched or vsExecute is called. You can not set this option.
VSOPTION_HAVEGETMESSAGELINE
Indicates whether get message line is currently available. This gets set when an event is dispatched or vsExecute is called. You can not set this option.
VSOPTION_MACRO_SOURCE_LEVEL
Indicates the source code macro level stored in "vsapi.dll".
VSOPTION_APPLY_LOCAL_STATE_FILE_CHANGES
Indicates if local state file changes need to be applied to the global state file.
VSOPTION_DISPLAYVERSIONMESSAGE
Indicates whether the version of the editor message should be displayed when the editor starts.
VSOPTION_CXDRAGMIN
(Windows only) Indicates the minumum number of pixels in the X directory which indicate a drag instead of just a click operation.
VSOPTION_CYDRAGMIN
(Windows only) Indicates the minumum number of pixels in the Y directory which indicate a drag instead of just a click operation.
VSOPTION_DRAGDELAY
(Windows only) Indicates the minumum amount of time in milliseconds which indicate a drag instead of just a click operation.
VSOPTION_NEXTWINDOWSTYLE
This can be 0 or 1. 1 specifies "Smart next window" mode.
VSOPTION_CODEHELP_FLAGS
Zero or more of the following flags:
VSCODEHELPFLAG_AUTO_FUNCTION_HELP
Indicates whether auto Parameter Info is on/off.
VSCODEHELPFLAG_AUTO_LIST_MEMBERS 0x2
Indicates whether auto list members is on/off.
VSCODEHELPFLAG_SPACE_INSERTS_SPACE
When on, pressing space bar during list members always inserts a space.
VSCODEHELPFLAG_INSERT_OPEN_PAREN
When on, selecting an item during list members which requires an open paren,'<', or additional characters, automatically inserts the additinal characters.
VSCODEHELPFLAG_SPACE_COMPLETION
When on, pressing space during list members completes the word.
VSCODEHELPFLAG_DISPLAY_MEMBER_COMMENTS
When on, source comments are displayed while in list help
VSCODEHELPFLAG_DISPLAY_FUNCTION_COMMENTS
When on, source comments are displayed while in Parameter Info
VSCODEHELPFLAG_AUTO_SYNTAX_HELP
Determines whether auto syntax help is on or off. Only some languages which have Context Tagging® such as COBOL have auto syntax help.
VSCODEHELPFLAG_REPLACE_IDENTIFIER
When on, the current identifier is replaced when an item is selected from list members.
VSOPTION_LINE_NUMBERS_LEN
Indicates the number of digits to use when displaying line numbers. In addition, in ISPF emulation this indicates the number of '=' signs to display.
VSOPTION_LCREADWRITE
A non-zero value indicates that a prefix area should be displayed for read/write files.
VSOPTION_LCREADONLY
A non-zero value indicates that a prefix area should be displayed for read/only files.
VSOPTION_LCMAXNOFLINECOMMANDS
Indicates the maximumum number of line command data allowed durring a search and replace. This is so that the error information does not get extremely large.
VSOPTION_DOUBLE_CLICK_TIME
The maximum amount of time between two clicks to be considered a double click.
VSOPTION_LCNOCOLON
A non-zero value indicates that no colon character should be displayed after line numbers. This option is ignored in hex mode.

Returns: int

Returns current setting of specified option.

Category:

Miscellaneous Functions
