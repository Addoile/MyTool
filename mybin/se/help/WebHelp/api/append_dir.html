_command typeless append_dir(_str path='')

Appends a directory list of the files specified to the current buffer. If filespec is not specified, files in the current directory are listed. This command is useful when in fileman mode. An '@' sign prefix to filespec indicates that filespec is a file or buffer which contains a list of file names to be used as arguments to this command. Use the dir or list command to change the mode to fileman mode.

cmdline is a string in the format: [@]filespec [ [@]filespec ... ]

Returns: typeless

Category:

File Functions

See Also:

dir, list, append_list
