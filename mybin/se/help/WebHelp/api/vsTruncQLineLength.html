int VSAPI vsTruncQLineLength(int wid, int *pEntireLineLen=a, int reserved=a)

Parameters:

wid - Window id of editor control. 0 specifies the current object.

IncludeNLChars - Specify a non-zero value for IncludeNLChars if you want to include the end of line characters in the returned length. This argument is ignored if the VSP_TRUNCATELENGTH property is non-zero.

Returns: int

If VSP_TRUNCATELENGTH is 0, the number of characters in current line not including end of line characters is returned. Specify a non-zero value for IncludeNLChars if you want to include the end of line characters in the returned length.

If VSP_TRUNCATELENGTH is non-zero, the length of the current line as if the text to the right of the truncation length does not exist is returned. In addition, trailing blanks (tab characters count) are not included in the length.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
