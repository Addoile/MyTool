int VSAPI vsTagFindClass(char *pszClassName, const char *pszSearch=a, int normalize=a, int case_sensitive=a, int find_first=a, const char *cur_class_name=a)

Return the first class name in the database, matching 'search_class', normalize the class name if necessary (find the class/package scope that this class belongs to). If (normalize==1) attempt to normalize the class name, finding a package scope that this class could belong to.

Parameters:

pszClassName - (output) output buffer for full class name (1024 bytes) Allocate VS_TAG_MAX_CLASSNAME characters

pszSearch - (optional) name of class to look for, can be NULL

normalize - attempt to normalize class name found?

case_sensitive - case sensitive search?

find_first - find first match or next match

cur_class_name - name of current class context

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
