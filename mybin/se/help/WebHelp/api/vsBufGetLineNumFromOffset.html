seSeekPosRet VSAPI vsBufGetLineNumFromOffset(int buf_id, seSeekPos Offset, seSeekPos *pRLineNum=a)

Converts the offset (not ROffset) into a line number and optionally a real line number.

Parameters:

buf_id - Buffer id returned by vsPropGetI(wid,VSP_BUF_ID), vsBufEdit, or vsBufMatch.

Offset - Byte offset into buffer including no save buffer data (lines with the VSLF_NOSAVE flag set).

pRLineNum - If specified, set to real line number (excludes lines with the VSLF_NOSAVE flag set).

Returns: seSeekPosRet

Returns the line number which contains the offset specified

Categories:

Buffer Functions, Edit Window Methods, Editor Control Methods
