_command void unload(_str filename='')

Unloads module specified. If the module is currently running it may not be unloaded. Do not specify a path for module_name.

Prior to unloading the module, it will invoke the _on_unload_module_XXX callbacks with a single argument indicating the Slick-C® module to be unloaded.

Category:

Macro Programming Functions
