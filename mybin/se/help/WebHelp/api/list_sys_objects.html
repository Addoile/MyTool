_command typeless list_sys_objects()

Opens a new buffer called "sysobjs.e" and inserts Slick-C® source code for all system dialog box (forms) and menu templates that existed when Visual SlickEdit was installed. No user created dialog box templates are inserted. This command is typically used by the developers of SlickEdit.

Returns: typeless

Returns 0 if successful. Returns 1 if no templates are defined (this should never happen). Other errors are non-zero.

Applies To:

Edit Window

Category:

Miscellaneous Functions

See Also:

list_usersys_objects, list_objects
