_command int debug_show_next_statement(boolean quiet=false, int breakpoint_id=0, int exception_id=0, _str event_kind='', _str event_text='', int thread_id=0, int frame_id=1)

Display the next statement to be executed for the current stack frame.

Parameters:

quiet - Do not complain about errors (default false)

breakpoint_id - breakpoint ID from debugger event

execption_id - exception ID from debugger event

event_kind - type of debugger event causing this update

event_text - description of debugger event

thread_id - thread ID to make active, if == 0, use the current thread ID

frame_id - frame ID to make active, if == 0, use the current frame ID if == 1, use the topmost frame ID

Returns: int

0 on success, <0 on error.

Category:

Debugger Commands
