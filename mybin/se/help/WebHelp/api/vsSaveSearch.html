int VSAPI vsSaveSearch(void *pvoid=a)

Saves search information set by one of the functions vsSearch or vsRepeatSearch. Use vsRestoreSearch to restore the search information.

Returns: int

Returns number of bytes needed to be allocated to pvoid.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions, Search Functions

See Also:

vsMatchLength, vsMatchStart, vsMatchCursorLength, vsSelectMatch, vsSearch, vsRepeatSearch, vsSearchReplace, vsRestoreSearch
