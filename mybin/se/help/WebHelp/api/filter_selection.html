_str filter_selection(typeless filter, _str optionL='', boolean doRaw=false)

Filter selected text through an external command. The external command must be capable of redirecting stdin from a file and output to stdout. Takes an optional argument that is the command to run. If "", then user * will be prompted for command to run.

Returns: _str

Returns 0 if successful.

Applies To:

Edit Window, Editor Control

Category:

Selection Functions

See Also:

filter_init, filter_get_string, filter_put_string, filter_restore_pos
