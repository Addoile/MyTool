_str _fm_copy(_str cmdline, _str doPause=false)

Copies filename specified to directory specified. directory is created if it does not exists. Used by fileman mode. For your own macros, you will probably want to use the copy_file function to copy files.

Parameters:

cmdline - is a string in the format: filename directory

Returns: _str

Returns 0 if successful. Common return codes are: ACCESS_DENIED_RC, ERROR_OPENING_FILE_RC, FILE_NOT_FOUND_RC, INSUFFICIENT_DISK_SPACE_RC, ERROR_CREATING_DIRECTORY_RC, ERROR_READING_FILE_RC, ERROR_WRITING_FILE_RC, DRIVE_NOT_READY_RC, and PATH_NOT_FOUND_RC. When invoked from the command line, error messages are displayed.

Category:

File Functions
