int tag_next_prefix(_str tag_prefix, boolean case_sensitive=false, _str class_name=null, boolean skip_duplicates=false)

Retrieve the next tag with the given prefix (case-insensitive). Should be called only after calling vsTagGetTagPrefix(). Should be called only after calling tag_find_prefix. Use tag_get_info to extract the details about the tag.

Parameters:

tag_prefix - Prefix of tag name to search for.

case_sensitive - Specifies search case sensitivity.

class_name - Name of class that tag belongs to.

skip_duplicates - Skip tags with same name, type and class.

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Category:

Tagging Functions
