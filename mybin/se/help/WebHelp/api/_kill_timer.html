int _kill_timer(int timer_handle=-1)

Frees the timer corresponding to timer_handle. timer_handle must have been allocated by the _set_timer function. When timer_handle is -1, the get_event timer is killed (see _set_timer) Beware, if a timer callback function dies, the timer handle is automatically killed to prevent infinite error message loops.
