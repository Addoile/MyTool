_command void enumerate(_str cmdline='')

This commands adds incrementing numbers to a selection.

Parameters:

cmdline - is a string in the format: 

   [-x | -xf] [-l leading] [-t trailing]
         [start [ increment [ DigitWidth ]]]
-x
Specifies hexadecimal output. If -l and -t options are not specified, the buffers extension specific syntax for hex numbers is used. If not hex syntax has been defined in the buffers color coding, "0x" is used for leading text.
-xf
Specifies hexadecimal output. If -l and -t options are not specified, the buffers extension specific syntax for hex numbers is used. If not hex syntax has been defined in the buffers color coding, "0x" is used for leading text. NOTE: the meaning of start and increment change when this option is specified (see below).
-l leading
Specifies text to prefix each number with.
-t trailing
Specifies text to append to each number.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
