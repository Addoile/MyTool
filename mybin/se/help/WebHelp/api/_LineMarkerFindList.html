void _LineMarkerFindList(int(&list)[], int wid, int LineNum, long LineOffset, boolean CheckNofLines)

Finds list of line markers which match.

Parameters:

list - Output. List of line marker indexes.

wid - Window id of editor control. 0 specifies the current object.

LineNum - Line number returned by vsPropGetI(wid,VSP_LINE)

LineOffset - Optional specifies offset to search for line markers which use MarkIds.

CheckNofLines - When on, only line markers which NofLines which include this line are returned.

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
