_command int move_to_cursor(...)

Moves selection specified to the cursor. For block and character selections, the text is inserted at the cursor position. In the case of a line selection, the lines are inserted after the current line. Resulting selection is placed on the inserted text.

Returns: int

Returns 0 if successful. Common return codes are TEXT_NOT_SELECTED_RC, and SOURCE_DEST_CONFLICT_RC. On error, message is displayed.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
