boolean p_readonly_set_by_user

Determines whether the p_readonly_mode property was set by the user or automatically by the editor. This property is available only when the control is displayed.

The editor has an option to automatically set and reset read only mode based on the read only attribute (UNIX: permissions) of the file on disk. We added this property so we could tell when the user has overridden the automatic processing. The read_only_mode and read_only_mode_toggle commands turn this property on.

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
