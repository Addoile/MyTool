int _IsKeyDown(_str ShiftKeyEvent)

Returns: int

Returns a non-zero value if the key specified is currently down. One of the following keys may be specified:

  • CTRL
  • SHIFT

This function is typically used when get_event() returns the on_keystatechange event.

Example:

defmain()
{
     mou_mode(1);
OuterLoop:
     for (;;) {
         event=get_event();
         switch( event ) {
         case ESC:
              break OuterLoop;
          case ON_KEYSTATECHANGE:
              if (_IsKeyDown(CTRL) ){
                   message("CTRL is down");
               } else {
                    message("CTRL is not down");
               }
           }
      }
      mou_mode(0);
           }
