_command int save_all(int sv_flags=-1, boolean skip_unnamed_files=false, boolean only_save_workspace_files=false)

Saves all modified buffers.

Returns: int

Returns 0 if successful. Common return codes are INVALID_OPTION_RC, ACCESS_DENIED_RC, ERROR_OPENING_FILE_RC, ERROR_WRITING_FILE_RC, INSUFFICIENT_DISK_SPACE_RC, DRIVE_NOT_READY_RC, and PATH_NOT_FOUND_RC. On error, message box is displayed.

Category:

File Functions
