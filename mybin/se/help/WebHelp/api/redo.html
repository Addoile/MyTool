_command typeless redo()

If the current buffer allows more than 0 undoable steps, the last undo operation is redone. Use the undo command to undo mistakes made while using redo.

If the current buffer has 0 undoable steps, the current line is restored to its original value before the cursor moved onto it.

Returns: typeless

On successful completion, a descriptive message of what was redone is displayed and a number greater than or equal to zero is returned. Common negative error codes are NOTHING_TO_UNDO_RC, and NOTHING_TO_REDO_RC. On error, message is displayed.
