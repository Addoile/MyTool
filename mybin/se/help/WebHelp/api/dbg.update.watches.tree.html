int dbg_update_watches_tree(int tree_wid, int tree_index, int thread_id, int frame_id, int tab_number, _str watch_path=null)

Update the display of manually-watched variables. Each variable watch has two columns:

  • Name
  • Value

Parameters:

tree_wid - window ID of tree control

tree_index - Index to insert variables at (TREE_ROOT_INDEX)

thread_id - index of thread

frame_id - index of stack frame

tab_number - which tree tab the watch belongs to

watch_path - (optional) position to start updating tree at

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
