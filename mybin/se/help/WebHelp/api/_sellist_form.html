typeless _sellist.on_create(_str title="", typeless flags="", typeless list_data="", _str buttons="", _str help_item="", _str font="", typeless callback="", _str separator="", _str retrieve="", _str completion="", typeless min_list="", typeless initial_value="", typeless SkipPrefixChars="")

Displays a selection list.

Parameters:

title - Title of dialog box.

flags - Combination of the following flags defined in "slick.sh":

SL_ALLOWMULTISELECT
Allow multiple items to be selected. Multiple items are separated with space characters. If an item contains spaces, double quotes are placed around the item.Currently this function does not support items which contain double quotes. If the return string is longer than 200characters, a "@buffer_name" string is returned. The buffer buffer_name contains the items selected.
SL_NOTOP
Don't position cursor at top of list.
SL_VIEWID
Indicates that input_data is a window id.
SL_FILENAME
Indicates that input_data is a filename that should be loaded and placed in the list box. File must contain blanks in column one.
SL_BUFID
Indicates that input_data is a buffer id. Buffer is deleted unless the SL_NODELETELIST flag given.
SL_NOISEARCH
Indicates whether typing characters while list box is active performs incremental searching in the list.
SL_NODELETELIST
No longer has any effect
SL_SELECTCLINE
Select the current line in the list box when initializing.
SL_MATCHCASE
Case sensitive incremental searching in list box. Has no effect if SL_NOISEARCH flag is given.
SL_INVERT
Display Invert button
SL_SELECTALL
Display Select All button. Must be set if SL_BUTTON_SELECTALL is included in the buttons list
SL_HELPCALLBACK
Call the callback routine when the help button is invoked.
SL_DEFAULTCALLBACK
Call the callback routine when default button invoked.
SL_COMBO
Display combo box above list box
SL_MUSTEXIST
Selected item must exist. Has no effect if SL_ALLOWMULTISELECT is given.
SL_DESELECTALL
Deselect all items in list box when initializing.
SL_SIZABLE
Dialog can be resized.

list_data -

buttons - Comma delimited string of button captions (p_caption). The buttons appear to the right of the list box. Cancel, and Help button are inserted after the first button in the list if there positions are explicitly specified. The first button is the default button. If buttons is '', a default button of OK is used. A button may be given a control name (p_name) by appending colon and the control name. This allows for easier access to the control in the call back function. The control name for the default button is always "_sellistok" and may not be changed. This parameter may be '' to specify OK, Cancel, and Help buttons. help_item Specifies help displayed when F1 is pressed or the help button is pressed. If the help_item starts with a '?' character, the characters that follow are displayed in a message box. The help string may also specify a unique keyword in the "vslick.hlp" (UNIX: "uvslick.hlp") help file. The unique keywords for "vslick.hlp" (UNIX: "uvslick.hlp") are contained in the file "vslick.lst" (UNIX: "uvslick.lst"). In addition, you may specify a unique keyword for any windows help file by specifying a string in the format:
keyword:help_filename.

help_item -

font - Font string of the form "font_name,font_size, font_flags". The font flags are defined in "slick.sh" and have the prefix "F_". font_size is in points. May be '' to specify default list box font.

min_list -

initial_value - Initial value displayed in combo box. Useful only if combo box is displayed.

SkipPrefixChars -

Returns: typeless

Returns '' if the dialog box is cancelled. Otherwise, the selected item is returned or the value specified by the call back function is returned.

_str show('_sellist_form', title, flags, input_data [, buttons [,help_item [,font [,callback_name [, item_separator [,retrieve_name [, combo_completion [,min_list_width [,initial_value ]]]]]]]]] )
