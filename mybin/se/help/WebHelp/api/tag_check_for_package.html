int tag_check_for_package(_str tag_name, _str(&tag_files)[], boolean exact_match, boolean case_sensitive, ...)

Look up tag_name and see if it is a package, namespace, module or unit.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Parameters:

tag_name - Tag name to check if it is a package or package prefix.

tag_files - (reference to _str[]) list of tag files to search

exact_match - Look for exact match rather than prefix match

case_sensitive - Specifies case sensitive comparison

Returns: int

Returns non-zero value if the given tag is a package name or the prefix of a package name. Otherwise returns 0.

Category:

Tagging Functions
