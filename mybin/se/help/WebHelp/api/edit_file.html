_str edit_file(_str filename)

Loads filename. Filename may contain options allowed by load_files built-in.

Parameters:

filename - may contain options. Only one filename may be given

Returns: _str

Returns 0 if successful. Otherwise an error code is returned in which negative numbers are valid input codes to the get_message built-in. On error, message is displayed.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, File Functions
