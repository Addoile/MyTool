int copy_file(_str source_filename, _str dest_filename)

Copies the file source_name to dest_name.

Returns: int

Returns 0 if successful. Common return codes are: ACCESS_DENIED_RC, ERROR_OPENING_FILE_RC, FILE_NOT_FOUND_RC, INSUFFICIENT_DISK_SPACE_RC, ERROR_CREATING_DIRECTORY_RC, ERROR_READING_FILE_RC, ERROR_WRITING_FILE_RC, DRIVE_NOT_READY_RC, and PATH_NOT_FOUND_RC.

Category:

File Functions
