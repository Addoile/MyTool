typeless _getfield(_str indexOrName)

Parameters:

indexOrName - Name or position of field

Returns: typeless

Return a reference to the given class field. Fields may be specified either by name or by index (see _fieldindex). This function pays no attention to field visibility (private or protected).

Example:

class AAAA { int x; int y; };
class BBBB : AAAA { int z; };
void testGetField() {
   BBBB b;
   b.x = 0;
   b.y = 1;
   b.z = 2;
   int zero = b._getfield("x");
   int one  = b._getfield(1);
   int two  = b._getfield("z");
   b._getfield(0) = 10;
   int ten  = b._getfield("x");
}

Category:

Miscellaneous Functions
