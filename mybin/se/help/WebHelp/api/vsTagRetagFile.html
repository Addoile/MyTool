int VSAPI vsTagRetagFile(const char *pszFilename, const char *pszExt=a, int reserved=a)

Retag the given file, using the given extension-specific parser.

Parameters:

pszFilename - Name/Path to source file to tag

pszExt - File extension to use for tagging this source code

reserved - future expansion

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
