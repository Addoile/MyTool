seSeekPosRet VSAPI vsQROffset(int wid)

Parameters:

wid - Window id of editor control. 0 specifies the current object.

Returns: seSeekPosRet

Returns the number of bytes before the cursor. Non-savable lines (lines with the VSLF_NOSAVE flag set) are not included. This methed is intended for dealing with disk seek position. However, if you have changed the load options to translate files when they are openned, these offsets will not match what is on disk.

Applies To:

Edit Window, Editor Control

Categories:

Editor Control Methods, Search Functions

See Also:

vsGoToPoint, vsQPoint, vsGoToROffset
