_command int debug_configurations()

Display the GDB configuration management dialog in order to manage different GDB executable configurations.

Format of debugger configuration file 

<Debugger>
   <Package Name="GDB">
      <Configuration
         Name="PalmOS"
         Path='C:\palmsdk\bin\gdb_palm.exe'
         Platform="VSWINDOWS"
         Arguments=""
      />
   </Package>
</Debugger>
