int tag_close_db(_str file_name=null, boolean leave_open=false)

Close the current or specified tag database.

Parameters:

file_name - Explicite filename of database to close otherwise the current open database is closed.

leave_open - Leave the tag file open read-only

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
