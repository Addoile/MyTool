void _retrieve_prev()

Moves the line position of the retrieve buffer ".command", one line up and places the contents of the line on the command line. If the current line is the first line of the buffer, the cursor is placed on the last line and the contents of the last line are placed on the command line. When the execute function is issued, the current line of the retrieve buffer becomes the last line. However, the first execution of retrieve_prev after execute is issued will place the contents of the last line on the command line without moving the line position.

Applies To:

Command Line
