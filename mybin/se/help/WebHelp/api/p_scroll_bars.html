int p_scroll_bars

Determines the scroll bars displayed for the current window. The p_scroll_bar property may be one of the following constants defined in "slick.sh":

SB_NONE
No scroll bars.
SB_HORIZONTAL
Display horizontal scroll bar.
SB_VERTICAL
Display vertical scroll bar. May only appear when there are enough items in the list.
SB_BOTH
Display horizontal and vertical scroll bars.

Applies To:

Editor Control, List Box, File List Box, Directory List Box, Tree View

Categories:

Editor Control Properties, List Box Properties, Directory List Box Properties, Tree View Properties
