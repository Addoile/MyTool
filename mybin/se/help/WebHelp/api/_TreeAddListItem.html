int _TreeAddListItem(_str ItemText, int Flags=0, int Index=TREE_ROOT_INDEX, int ShowChildren=TREE_NODE_LEAF, typeless userInfo=null)

Used to a list style treeview(treeview control with column buttons turned on). Easier to use because index defaults to TREE_ROOT_INDEX, and TREE_ADD_AS_CHILD is always or'd into the flags.

Parameters:

ItemText - Text for this line. Use a tab character to start a new column.

Flags - Flags to add this item. TREE_ADD_AS_CHILD will always be or'd in.

Index - Index to add item under. Defaults to TREE_ROOT_NODE.

ShowChildren - This is the same as the ShowChildren state on _TreeAddItem. Set to 1 or 0 if you wish to have children show. It defaults to TREE_NODE_LEAF because you don't normally want children in this style of tree.

userInfo - Allows you to specify per-node user data. See _TreeSetUserInfo.

Returns: int

Returns the index of the new tree item.

Applies To:

Tree View

Category:

Tree View Methods
