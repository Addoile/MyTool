long _negative2dec(long result, int base, _str &msg)

Returns: long

Returns non-decimal negative_number converted to a 16, 32, or 64 bit number. This function is useful for displaying negative hexadecimal typed numbers (long, int, short). The msg variable is set to the number of bits required to represent the negative number.

Example:

defmain()
{
   result=_negative2dec('-f',16,msg);
   result=dec2hex(result,16);
   message(msg' result='result);
}
