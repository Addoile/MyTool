void tag_type_set_filter(int type_id, int filter_flags)

Set the filter flags for given type ID.

Parameters:

type_id - Tag type ID, in range VS_TAGTYPE_OEM <= type_id <= VS_TAGTYPE_MAXIMUM

filter_flags - New tag filter flags, only insert tags passing this filter. See tag_filter_type for more details.

Category:

Tagging Functions
