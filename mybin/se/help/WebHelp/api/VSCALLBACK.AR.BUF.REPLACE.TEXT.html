7 VSCALLBACK_AR_BUF_REPLACE_TEXT

void (VSAPI *pfnReplaceText)(int buf_id,seSeekPos StartOffset,seSeekPos InsertNofBytes,seSeekPos DeleteNofBytes)

The REPLACE_TEXT callback is called when any text is replaced into a buffer.

WARNING: It is not safe to access editor buffer data during this callback.

Parameters:

buf_id - Buffer id.

StartOffset - Byte offset (including lines with NOSAVE data) of first byte being replaced.

InsertNofBytes - Number of bytes being inserted.

DeleteNofBytes - Number of bytes being deleted.

Category:

Editor Control Callback Functions
