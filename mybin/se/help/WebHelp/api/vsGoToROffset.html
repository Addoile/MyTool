int VSAPI vsGoToROffset(int wid, seSeekPos offset)

Places the cursor at the character offset specified. Non-savable lines (lines with the VSLF_NOSAVE flag set) are not included. This methed is intended for dealing with disk seek position. However, if you have changed the load options to translate files when they are openned, these offsets will not match what is on disk.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

offset - Real character offset to go to.

Returns: int

Returns 0 if offset is a valid offset. Otherwise an negative error code is returned.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods, Search Functions
