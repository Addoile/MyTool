int VSAPI vsDeleteLine(int wid)

Deletes the current line.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

Returns: int

Returns 0 unless last line of buffer is deleted.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

vsInsertLine, vsReplaceLine, vsGetLine, vsGetText
