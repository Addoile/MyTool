int VSAPI vsTagSetDate(const char *pszFilename, const char *pszDate=a, int file_type=a, const char *pszIncludedBy=a)

Set the date of tagging for the given file. If the date is not specified, use the file modification time found on disk. This function has the side effect of positioning the file iterator on the given file name.

Parameters:

pszFilename - File path of source file to setting tagging date for.

pszDate - (20 bytes, optional) Modification date of file when tagged. Format is YYYYMMDDHHMMSSmmm (year, month, day, hour, minute, second, ms)

file_type - Type of file to search for, source vs. reference

pszIncludedBy - File path of source file which includes pszFilename

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if the file is not found.

Category:

Tagging Functions
