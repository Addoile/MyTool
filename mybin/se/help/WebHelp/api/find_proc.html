_command typeless find_proc(_str name='', boolean force_case_sensitivity=false)

Finds Slick-C® source code or help for the Slick-C® identifier tag_name specified. This function supports tagging global functions, static functions if in current file, event tables, and dialog boxes (form names). Unlike the find_tag command, the find_proc command does not use a tag file.

Returns: typeless

Returns 0 if successful. Otherwise a non-zero value is returned and a message box is displayed.

Applies To:

Edit Window

Category:

Search Functions

See Also:

f, find_tag, push_tag, gui_find_proc
