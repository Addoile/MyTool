_command int next_char()

Move the cursor right one character. If the cursor is at the end of the line, move down to the beginning of the next line.

Returns: int

0 on success, <0 if at then end of the file.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods

See Also:

right, down, prev_char, traverse_char
