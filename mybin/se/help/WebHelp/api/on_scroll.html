event on_scroll()

The on_scroll event is sent when you click and drag the thumb box of a scroll bar. The on_change event is sent after dragging the thumb box is completed.

Example:

#include 'slick.sh'

defeventtab form1;
hscroll1.on_scroll()
{
   message('on_scroll p_value='p_value);
}
hscroll1.on_change()
{
   message('on_change p_value='p_value);
}

Applies To:

Hscroll Bar, Vscroll Bar

Categories:

Hscroll Bar Events, Vscroll Bar Events
