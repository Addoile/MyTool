int VSAPI vsRefactorAddFile(int handle, const char *filename, const char *userIncludePath, const char *systemIncludePath, const char *defineOptions, const char *compilerConfig)

Add a file to the refactoring transaction.

Parameters:

handle - Handle to transaction

filename - Filename to add

userIncludePath - User include path for this file

systemIncludePath - System include path for this file

defineOptions - #defines and #undefs for this file

compilerConfig - Compiler configuration name

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
