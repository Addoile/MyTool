int _sort(_str options="", start=0, int Nofitems=-1)

Sorts an array.

Parameters:

options - options may contain one or more of the following option letters:

D
Descending. When not specified sort is ascending.
I
Ignore case. Default is case sensitive.
N
Sort numbers. Supports floating point.
F
Sort filenames.

start - array index to start sorting at

Nofitems - number of items to sort

Returns: int

int

Example:

a[0]="def";
a[1]="abc";
// Sort array of strings in ascending order.
a._sort();

Miscellaneous Functions
