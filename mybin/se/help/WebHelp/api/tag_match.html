VSPSZ VSAPI tag_match(VSPSZ name, int find_first)

Parameters:

name - tag name to search for

find_first - 'true' to find first, 'false' to find next

Returns: VSPSZ

Returns next tag name which is a prefix match of name. find_first must be non-zero to initialize matching. Returns '' when no more matches are found.

Categories:

Completion Functions, File Functions
