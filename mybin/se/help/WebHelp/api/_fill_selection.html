int _fill_selection(_str ch, _str mark_id="")

Fills the selection specified with character. mark_id is a handle to a selection returned by one of the built-ins _alloc_selection or _duplicate_selection. A mark_id of '' or no mark_id parameter identifies the active selection.

Returns: int

Returns 0 if successful. Otherwise, TEXT_NOT_SELECTED_RC is returned.

Applies To:

Edit Window, Editor Control

Category:

Selection Functions
