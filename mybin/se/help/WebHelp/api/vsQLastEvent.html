int VSAPI vsQLastEvent(int NewLastEvent=VSEV_NULL, int version=a)

Parameters:

NewLastEvent - If this is not VSEV_NULL, the last event is set to this value.

Returns: int

Returns a last event. Last event is set when a key is pressed. This function is typically used by a command which wants to know what key it was invoked by.

Category:

Keyboard Functions
