int VSAPI vsSelectChar(int wid, int markid=-, int SelectFlags=a)

Starts or extends the character selection specified. Used for processing sentences of text which do not start and end on line boundaries. The first vsSelectChar becomes the pivot point. Subsequent vsSelectChar calls will extend the selection between the pivot point and the cursor.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

markid - mark_id is a selection handle allocated by vsAllocSelection or vsDuplicateSelection. A mark_id of -1 specifies the active selection or selection showing and is always allocated.

SelectFlags - Zero or more of the following flags Ored together:

VSSELECT_BEGINEND
Specifies that selecting an area of text requires _select_char be executed to select the end of the text area as well as the beginning. If VSSELECT_CURSOREXTENDS is not specified, this select style is used.
VSSELECT_CURSOREXTENDS
Specifies that the selection extend as the cursor moves.
VSSELECT_INCLUSIVE
Specifies an inclusive selection. Currently only character selections are affected by this option.
VSSELECT_NONINCLUSIVE
Specifies a non-inclusive selection. Currently only character selections are affected by this option. If the 'I' letter is not specified, the character selection will be non-inclusive selection.
VSSELECT_PERSISTENT
A value of 'P' specifies a persistent select style and may be specified in addition to the other options above. Macros use this to help determine if a selection should be unhighlighted when the cursor moves.

Returns: int

Returns 0 if successful. Possible returns are TEXT_NOT_SELECT_RC. and TEXT_ALREADY_SELECTED_RC.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions

See Also:

vsSelectBlock, vsSelectLine
