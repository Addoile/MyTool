int VSAPI vsRefactorConfigGetJAR(const char *configName, int jarIndex, char *jarPath, int maxLength)

Get the full path to the specified system jar

Parameters:

configName - name of configuration to query

jarIndex - index within include path to query

jarPath - set to jar path

maxLength - length of includePath buffer

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions

See Also:

vsRefactorConfigAdd, vsRefactorConfigAddJAR, vsRefactorConfigDeleteJAR, vsRefactorConfigDeleteAllJARs
