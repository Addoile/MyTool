_str _macro_path_search(_str name, _str options='')

Searches the macros directory for a file.

Parameters:

name - name of file to look for

options - a string of zero or more of the following:

'P'
Program search. Does not effect UNIX.
'M'
Program search including search for .e and .ex Slick- C batch macros.
'S'
Don't search in current directory. Does not effect UNIX.

Returns: _str

_str full path to file, relative to macros directory. Empty string if file is not found

Category:

File Functions
