int VSAPI vsGetRText(int wid, int Nofbytes, seSeekPos point, char *pszBuf, int *pNofbytesRead=a)

Places Len bytes of data starting from the byte offset specified by RealSeekPos into pszBuf. The resulting string is always null terminated. The buffer pszBuf must be large enough to contain Len+1 bytes of data. Lines with the VSLF_ NOSAVE flag set are not copied into the output buffer. Use this function to read all data but no save lines.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

Len - Number of characters to copy into pszBuf.

RealSeekPos - Specifies the real seek position from which to start copying. Specify VSNULLSEEK to start copying for the cursor position.

pszBuf - Output buffer for text. The buffer pszBuf must be large enough to contain Len+1 bytes of data.

Returns: int

Returns number of bytes copied not including null character.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
