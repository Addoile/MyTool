int VSAPI vsReplaceLine(int wid, const char *pBuf, int BufLen=-)

Sets the current line of the current buffer to pBuf.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

pBuf - New text for current line. This should not contain line separator characters.

BufLen - Number of characters in pBuf. If BufLen is -1, pBuf must be null terminated.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

vsGetLine, vsInsertLine, vsDeleteLine
