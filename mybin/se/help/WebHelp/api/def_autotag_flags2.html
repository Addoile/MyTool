int def_autotag_flags2

Control dynamic tagging options. Consists of a bitset of AUTOTAG_* flags.

  • AUTOTAG_ON_SAVE -- Tag file on save
  • AUTOTAG_BUFFERS -- Background tag buffers
  • AUTOTAG_PROJECT_ONLY -- Background tag project buffers only (obsolete?)
  • AUTOTAG_FILES -- Background tag all files
  • AUTOTAG_SYMBOLS -- Refresh tag window (symbols tab)
  • AUTOTAG_FILES_PROJECT_ONLY -- Background tag project files only
  • AUTOTAG_CURRENT_CONTEXT -- Background update current context
  • AUTOTAG_UPDATE_CALLSREFS -- Update call tree and references on change event for symbols browser and proctree

Default:

AUTOTAG_ON_SAVE | AUTOTAG_BUFFERS | AUTOTAG_SYMBOLS | AUTOTAG_FILES_PROJECT_ONLY | AUTOTAG_CURRENT_CONTEXT
