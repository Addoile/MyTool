int VSAPI vsTagCloseDB(const char *pszFilename=a, int leave_open=a, int dbHandle=-, bool releaseExtraWriteLock=false)

Close the current database.

Parameters:

pszFilename - (optional) File path of database to close.

leave_open - (optional) If true, do not really close the file, instead flush the buffer if open read-write, then leave the file open read-only.

dbHandle - (optional) Explicit database handle to close. If this is given, pszFilename is ignored.

releaseExtraWriteLock - (optional) If this database was opened using vsTagOpenWriterDB(), and then locked for writing in advance using vsTagWriteLockDB(), release the extra write lock now. This is done to prevent a race condition that could lead to deadlock.

Returns: int

database handle >= 0 on success, < 0 on error.

Category:

Tagging Functions
