void VSAPI vsBPMSetAll(int flags, int mask)

Deprecated. Use vsLineMarker* functions Adds or removes the debug flags specified for all lines which have debug flags.

Parameters:

flags - Zero or more of the the VSBPFLAG_??? flags.

mask - One or more of the the VSBPFLAG_??? flags. Only these debug flags are effected.

Returns: void

Returns 0 if successful. It is possible but very unlikely that you could run out of memory.
