int dbg_session_expand_class(int session_id, _str class_path)

Update the items under a class recursively. This includes methods, static members, fields, and parent classes.

Parameters:

session_id - debugger session ID (1 .. max_session)

class_path - List of class ID's and field ID's to expand

Returns: int

0 on success, <0 on failure

Since:

10.0

Example:

To expand the class with class ID 34: 

   int status = dbg_session_expand_class(session_id, "34");
   if (status) {
      return status;
   }

To expand a structured static class member with ID 39 under class ID 34: 

   int status = dbg_session_expand_class(session_id, "34 39");
   if (status) {
      return status;
   }

Category:

Debugger Functions
