void VSAPI vsHvarHashtabInsertEl(HVAR hvarArray, HVAR hvarValue, VSPSZ pBuf, int BufLen=-)

Inserts an element into a hash table. Nofitems defaults to 1.

Parameters:

hvarHashtab - hash table to insert item into

hvarValue - value to insert into hash table

pBuf - index or key corresponding to item to look for

BufLen - length of hash key

Example:

   vsHvarMakeEmpty(VSHVAR_RC);
   hvar = vsHvarAlloc();
   vsHvarSetZ(hvar, "world");
   vsHvarHashtabInsertEl(VSHVAR_RC, hvar, "hello");
   vsHvarFree(hvar);

Category:

Miscellaneous Functions
