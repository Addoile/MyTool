_str prompt(_str arg1='', _str msg='', ...)

immed_return_value
Defaults to ''
msg
Defaults to the name of the last command (_command) called
default_value
Defaults to ''
cursor_placement
Defaults to ''

If the value of the macro variable "def_prompt" is not 0, this function displays msg and prompts for a single argument to a command. default_value may be specified as the user's default input response. The users response is returned. If the user has selected to abort, execution is stopped unless a menu is currently displayed on screen. If immed_return_value is given and not '', its value is immediately returned.

If the value of the macro variable "def_prompt" is 0, this function places msg followed by default_value on the command line and stops. If immed_return_value is given and not '', its value is immediately returned.
