int create_ext(int &kt_index, _str extension, _str ref_language, _str mode_name='', _str setup_info='', _str compile_info='', _str syntax_info='', _str be_info='', _str include_info='', _str word_chars='', _str lexer_name='', _str color_flags='')

Deprecated. Use _CreateLanguage() and _CreateExtension().

Creates file extension specific setup data which is used by the Language Options dialog box. This procedure is typically used when adding support for languages not already provided by SlickEdit's default configuration.

See defload procedure in the macro file "modula.e" for an example.

Parameters:

kt_index - (output) Names table index to mode event table for this extension. The key table name is derived from the input variable mode_name with the string "-keys" appended. Not set if ref_extension is not ''.

extension - File extension to be setup.

ref_language - If not '', this extension will refer to the same setup data as files with of the language represented by 'ref_language' All other parameters are ignored if 'ref_language' is specified.

mode_name - Name given to the extension's mode.

setup_info - A string which specifies the mode_name, tabs, margins, key table name, word wrap setting, indent with tabs setting, and show tabs style. See defload procedure in the macro file "modula.e" for an example.

compile_info - Compile command.

syntax_info - A string of numbers separated by spaces.
first number specifies "Syntax Indent". Second number specifies syntax expansion on or off.
Numbers that follow specify extension specific options.

be_info - String of "Begin/end pairs".

include_info - Default include path or copy book search path

word_chars - set of characters used in identifiers/words

lexer_name - name of lexer for Color Coding

color_flags - Color coding flags

Returns: int

Returns the name index of the setup information created.
