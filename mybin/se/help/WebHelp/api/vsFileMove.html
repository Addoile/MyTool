int VSAPI vsFileMove(const char *pszDestFilename, const char *pszSrcFilename)

Renames the file specified. This function will fail if the destination file exists or the destionation path is on a different drive or file system.

Parameters:

pszDestFilename - New name for file.

pszSrcFilename - Existing name of file.

Returns: int

Returns 0 if successful.

Category:

File Functions

vsFileOpen, vsFileClose, vsFileRead, vsFileWrite, vsFileSeek, vsFileMove
