int dbg_update_exceptions_tree(int tree_wid, int tree_index, int pic_enabled, int pic_disabled)

Update the display of currently set exception breakpoints. Each exception breakpoint has two columns:

  • Class name
  • Thrown, Caught, or Uncaught

Parameters:

tree_wid - window ID of tree control

tree_index - Index to insert variables at (TREE_ROOT_INDEX)

pic_enabled - bitmap to display for an enabled breakpoint

pic_disabled - bitmap to display for a disabled breakpoint

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
