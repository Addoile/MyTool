int dbg_session_terminate(int session_id, int exit_code)

Stop the debugger and terminate execution.

Parameters:

session_id - debugger session ID (1 .. max_session)

exit_code - exit code for halted executable

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

   int status = dbg_session_terminate(session_id, 0);
   if (status) {
      return status;
   }

Category:

Debugger Functions
