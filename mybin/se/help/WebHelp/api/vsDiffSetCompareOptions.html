int VSAPI vsDiffSetCompareOptions(int iFlags)

Use this function to set the compare options for DIFFzilla®. Most options apply to the interface itself, and can be passed in at the time you are comparing the files. This function is used with the VSDIFF_COMP_OPT_* flags to actually control things like wheter or not to compare spaces in when comparing files.

Parameters:

iFlags - A combination of VSDIFF_COMP_OPT_* flags

Returns: int

0 if successful

Category:

File Functions
