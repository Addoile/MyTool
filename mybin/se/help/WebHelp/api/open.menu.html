_command void open_menu(_str result="")

If no parameters are specified, the Open Menu dialog box is displayed. Existing menus may be edited and new menus may be created. Use the -show option followed by the menu name to run a menu as a pop-up menu. The -new option allows you to create a new menu and optional specify the name of the new menu.

Parameters:

cmdline - is a string in the format: [-show menu_name| -new [menu_name]]

Category:

Menu Functions
