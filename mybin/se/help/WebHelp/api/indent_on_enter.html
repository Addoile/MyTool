void indent_on_enter(int syntax_indent, ...)

Determines action taken when syntax indenting is on and the ENTER key is pressed and the current line starts with a language key word such as IF, WHILE, FOR etc. If a new line is inserted, it is indented by the indent_amount given. The optional column parameter may be given to specify an exact column for the cursor to be placed.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
