int dbg_update_breakpoints_tree(int tree_wid, int tree_index, _str rel_dir, int pic_enabled, int pic_disabled)

Update the display of currently set breakpoints. Each breakpoint has four columns:

  • Class
  • Method
  • File
  • Line

Parameters:

tree_wid - window ID of tree control

tree_index - Index to insert variables at (TREE_ROOT_INDEX)

rel_dir - Show breakpoint file names relative to this directory

pic_enabled - bitmap to display for an enabled breakpoint

pic_disabled - bitmap to display for a disabled breakpoint

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
