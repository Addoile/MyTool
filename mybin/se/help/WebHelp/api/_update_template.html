int _update_template()

Updates a dialog box template in the names table. If the p_template property is 0, a new dialog box template is inserted. Otherwise, the existing dialog box template is replaced. Specify the name of the form (p_name) to the show command to display the dialog box.

Returns: int

Returns the index into the names table of the dialog box template. The p_template property of the form is set to the index into the names table of the dialog box template. On error, a negative error code is returned which can be used by get_message to retrieve the error message.

Applies To:

Form

Categories:

Form Methods, Names Table Functions
