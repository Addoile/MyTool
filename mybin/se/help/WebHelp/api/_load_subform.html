int _load_subform(_str form_name, int parent_wid, _str options='')

Loads a form as a child into another form, without the form frame.

Parameters:

form_name - Form name to be loaded.

parent_wid - Parent window to insert form.

options - Load options.

''
Default.
'H'
Hidden.

Returns: int

window id of created subform

Applies To:

Form

Category:

Form Methods
