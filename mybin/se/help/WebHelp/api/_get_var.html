typeless& _get_var(int index)

Parameters:

index - name table index of variable to get contents of.

Returns: typeless&

Returns the contents of the variable corresponding to the name table index given.

Example:

// VAR_TYPE is defined in "slick.sh"
index=find_index('rc',VAR_TYPE);
message('rc='_get_var(index));

Category:

Miscellaneous Functions

See Also:

_set_var, find_index, name_match
