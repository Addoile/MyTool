_command void grid(_str cmdline='')

Displays or sets the grid width and height used by the dialog editor. This effects the distance displayed between the dots displayed on a form that is being edited. If no parameters are given, the current grid width and height are displayed on the command line. The width and height parameters are in twips (1440 twips are one inch on the display).

Parameters:

cmdline - is a string in the format: width height

Category:

Miscellaneous Functions

See Also:

gui_grid, _grid_width, _grid_height
