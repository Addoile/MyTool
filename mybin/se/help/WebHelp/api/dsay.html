void dsay(_str string, _str log=null, int depth=0)

Write a string to the SlickEdit log file. The log file is located in the user's configuration directory, under the "logs" subdirectory.

Parameters:

string - string to display

log - can be a single word, and word.log will be stored in the standard logs subdirectory. Leave this blank to use the default (vs.log). Can also specify a full path to the log file.

depth - controls space before this line. There will be depth * 3 spaces before the text specified in the string param.

Category:

Miscellaneous Functions
