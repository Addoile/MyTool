event on_close()

Occurs when dialog box is closed via the system menu of the form or when the user cancels the dialog box (ESC or Alt+F4). When this event falls through to the dialog manager, the dialog manager looks to see if the form has this event. If the form has a handler for this event, the form on_close is called. If the form does not have an on_close event, the dialog manager executes the lbutton_up event handler for the command button with the p_cancel property set to true. If no such button exists, the dialog managers closes the dialog box.

Applies To:

Form

Category:

Form Events
