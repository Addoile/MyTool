int VSAPI vsHvarSetWID(VSHVAR hVar, int wid)

Sets value of interpreter variable to window id specified. The window id is represented as an integer.

Example:

       // Get handle to interpreter variable
       hvar=vsGetVar(index);
      // Specify integer for variable value
         vsHvarSetWID(hvar,wid);

Category:

Miscellaneous Functions
