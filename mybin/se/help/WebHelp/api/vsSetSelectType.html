int VSAPI vsSetSelectType(int wid, int markid, int type, char option='L')

This function changes the selection type for a specified selection handle.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

markid - mark_id is a selection handle allocated by the vsAllocSelection or vsDuplicateSelection function. A mark_id of -1 specifies the active selection or selection showing and is always allocated.

type - One of the following:

If option=='L' or option=='T'

  • VSSELECT_LINE
  • VSSELECT_CHAR
  • VSSELECT_BLOCK
  • VSSELECT_NONINCLUSIVEBLOCK

If option=='S'

  • VSSELECT_CURSOREXTENDS

If option=='U'

  • VSSELECT_PERSISTENT

option - 'L', 'T', 'S', or 'U'. For 'L' or 'T' the select type is set to the type given. When the 'L' option, type==VSSELECT_LINE,, and the last line of a character selection has 0 bytes selected, the last line is not included in the new line selection.

For 'S', the selection style is set to extend as the cursor moves or not to extend as the cursor moves.

For 'U', the selection style is set to be persistent or not persistent.

Returns: int

Returns 0 if the markid specified is valid. Otherwise a negative error code is returned.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
