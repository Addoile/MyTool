39 VSCALLBACK_WIN_FONT_CHANGE

void (VSAPI * vspfnChangeFont)(int wid, VSPSZ pszFontName, int FontWidth, int FontHeight, int CharSet);

The FONT_CHANGE callback function is used to notify of a font change in an editor control window.

Actual font name and metrics being used by an editor control window. This can be different from what the window properties report (e.g. VSP_FONTNAME, VSP_FONTWIDTH, etc.) because the operating system will often map a font differently when it is used.

Note:
This callback is not called the first time an editor control is created. Use vsGetWindowFontInfo after calling vsCreateEditorCtl to retrieve font name and metrics when an editor control window is created.

Parameters:

wid - Editor control window id.

pszFontName - Actual font name used to draw text.

FontWidth - Font width in pixels.

FontHeight - Font height in pixels. This is significant when a WidthxHeight font is selected (e.g. "Terminal 8x12" on Windows).

pCharSet - Character set. Windows only. Indicates the character set of the font. These character sets map directly onto the Windows character sets. See VSCHARSET_??? constants defined in "vs.h".
