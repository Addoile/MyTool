int _show_selection(_str mark_id="")

Makes the mark corresponding to mark_id visible. Currently only one mark may be showing at a time. mark_id is a handle to a selection returned by one of the built-ins _alloc_selection or _duplicate_selection. A mark_id of '' or no mark_id parameter identifies the active selection.

Returns: int

Returns 0 if successful. Otherwise INVALID_SELECTION_HANDLE_RC is returned. On error, message is displayed.

Category:

Selection Functions
