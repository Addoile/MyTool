VSPSZ VSAPI vsHvarParent(VSHVAR hvar, int index=0)

Parameters:

hvar - handle for interpreter variable

index - 0 for super class, 1 ... n for interfaces

Returns: VSPSZ

Return the name of the given parent or interface which this class derives from or implements, respectively. Returns INVALID_ARGUMENT_RC if the field index is out of range. Returns INVALID_HVAR_ARGUMENT_RC if 'hvar' is not a struct or class instance.

Category:

Miscellaneous Functions

See Also:

vsHvarFieldName, vsHvarClassParent
