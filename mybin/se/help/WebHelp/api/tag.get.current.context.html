int tag_get_current_context(_str &cur_tag_name, int &cur_flags, _str &cur_type_name, int &cur_type_id, _str &cur_context, _str &cur_class, _str &cur_package)

Determine the name of the current class or package context. The current object needs to be an editor control.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Parameters:

cur_tag_name - name of the current tag in context

cur_flags - Tag flags indicating symbol attributes, see tag_insert_simple for details.

cur_type_name - type (VS_TAGTYPE_*) of the current tag

cur_type_id - type ID (VS_TAGTYPE_*) of the current tag

cur_context - class name representing current context

cur_class - This parameter is broken. It only returns the class if the class is in a package. cur_context minus the package name

cur_package - only package name for the current context

Returns: int

0 if no context, context ID >0 on success, <0 on error.

Category:

Tagging Functions
