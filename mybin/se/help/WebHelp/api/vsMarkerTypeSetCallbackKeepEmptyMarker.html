int VSAPI vsMarkerTypeSetCallbackKeepEmptyMarker(int type, int(VSAPI*pfnKeepEmptyMarker)(int MarkerIndex, int BMIndex, int type, void*pUserData))

Sets the KeepEmptyMarker callback for the marker type specified. The purpose of this callback is to give you more control over whether this marker should be deleted. DO NOT free user data in this callback. Instead, define the vsMarkerTypeSetCallbackFreeUserData() callback.

For line markers, this callback is called when the line containing this marker is deleted and NofLines<=0.

For stream markers, this callback is called when all the text in the range is deleted.

Parameters:

type - Type returned from vsMarkerTypeAlloc()

pfnPicKeepEmptyMarker - Callback function. Returns 0 if the marker should be deleted.

Returns: int

Returns 0 if the marker type is valid.

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
