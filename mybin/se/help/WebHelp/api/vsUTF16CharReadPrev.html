unsigned short*VSAPI vsUTF16CharReadPrev(const unsigned short *pLine, const unsigned short *pCurrent, unsigned &uch)

Reads the previous UTF-16 or UTF-32 (surrogate) character. This function does not read composite characters.

Parameters:

pLine - Buffer contain UTF-16 text.

pCurrent - Pointer to current character in pLine.

uch - Recieves UTF-32 character.

Returns: unsigned short*VSAPI

Returns pointer to the previous character.

Category:

Unicode Functions
