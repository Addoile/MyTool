_str _form_match(_str name, boolean find_first)

Parameters:

name - form name to match

find_first - 'true' to find first, 'false' to find next

Returns: _str

Returns the name of the form matching the prefix name_prefix. If a match is not found, '' is returned. Specify find_first == 1, to start from the first form and 0 to find the next match. This function scans the names table for forms and not instances of forms already displayed.

Category:

Form Functions
