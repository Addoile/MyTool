_command typeless pop_bookmark()

Places the cursor at the buffer cursor position stored in the top bookmark of the bookmark stack and pops the bookmark off the bookmark stack. The push_bookmark and push_tag commands add bookmarks to the bookmark stack.

Returns: typeless

Returns 0 if successful.

Applies To:

Edit Window

Category:

Bookmark Functions

See Also:

push_bookmark, set_bookmark, toggle_bookmark, sb, goto_bookmark, gb, push_tag, bookmark_stack
