_command typeless safe_exit(_str doNotExit='', _str endingSession=false)

Exits editor. If any files need to be saved, the number of buffers which need to be saved is displayed and you are asked whether you want to exit any way or write them all.

Parameters:

doExit - force exit when this macro is finished

endingSession - (Windows only) Is the user logging out rather than just closing SlickEdit?

Returns: typeless

Exits editor if successful. Common return codes are COMMAND_CANCELLED_RC, ACCESS_DENIED_RC, and INSUFFICIENT_DISK_SPACE_RC. On error, message is displayed.

Category:

Miscellaneous Functions
