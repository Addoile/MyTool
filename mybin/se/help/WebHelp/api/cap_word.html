_command typeless cap_word()

Capitalizes the first character of the current word and places the cursor after the current word.

Returns: typeless

0 if successful. Returns 1 if no word exists at cursor. On error, message is displayed.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods

See Also:

lowcase_word, upcase_word
