int VSAPI vsTagContextMatchTags(VSHREFVAR errorArgs, char *tag_name, int max_tag_name, int exact_match, int case_sensitive, int find_parents, int &num_matches, int max_matches=VS_TAG_MAX_MATCHES, int filter_flags=VS_TAGFILTER_ANYTHING, int context_flags=VS_TAGCONTEXT_ALLOW_locals, VSHREFVAR visited=a, int depth=a)

The current object must be an editor control positioned on the tag that you wish to find matches for, with the edit mode selected.

Parameters:

errorArgs - Array of strings for return code

tag_name - Name of occurrence to match against

max_tag_name - Amount of space allocated to tag_name

exact_match - Exact match or prefix match?

case_sensitive - Case sensitive tag search?

find_parents - Find instances of the tag in parent classes

num_matches - (output) number of matches found

max_matches - maximum number of matches to find

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
