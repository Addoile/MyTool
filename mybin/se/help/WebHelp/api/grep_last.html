_command int grep_last()

Displays selection list of files/buffers and occurrences found by the last multi-file find performed (See Find dialog box). When a file or buffer is chosen, the file is opened and the cursor is placed on the first occurrence of the search string. Use the find_next command (Ctrl+G) to find the next occurrence of the search string.

Returns: int

Returns 0 if a file from the list is selected for editing.

Applies To:

Edit Window

Category:

Search Functions
