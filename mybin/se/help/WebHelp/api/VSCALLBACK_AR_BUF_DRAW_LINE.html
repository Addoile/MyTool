11 VSCALLBACK_AR_BUF_DRAW_LINE

void (VSAPI *pfnDrawLine)(int buf_id, int wid, int y, int height, seSeekPos LineNum,seSeekPos RealLineNum, int PastBottomOfFile, int SoftWrap)

The DRAW_LINE callback is called when lines are drawn for a buffer.

WARNING: It is not safe to access editor buffer data during this callback.

Parameters:

buf_id - Buffer id.

wid - Window id of editor control. 0 specifies the current object.

y - y coordinate origin of drawn line.

height - Height of the drawn line (not the same as the font height).

LineNum - Line number of drawn line.

RealLineNum - Real line number of drawn line. This will differ from LineNum when you have NOSAVE lines in your buffer. See vsSetLineFlags.

PastBottomOfFile - (boolean). Set to true if drawn line is past the bottom of the file (i.e. between the last line of file and the bottom of the editor control window). 1

SoftWrap - (boolean). Set to true if drawn line is soft-wrapped. This will be false for the first "line", and true for all other "lines" that are part of the soft-wrapped line.

Category:

Editor Control Callback Functions
