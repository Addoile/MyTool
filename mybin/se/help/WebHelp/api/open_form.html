_command typeless open_form(_str cmdline="")

Opens one or more dialog boxes (forms) specified for editing with the dialog editor. If a form is already opened for editing, it is brought to the foreground and given focus. The -new option specifies that a new form with the given name be created if it does not already exist. If no arguments are given or cmdline is "", a selection list of existing forms is displayed.

Parameters:

cmdline - is a string in the following format: [-new] form1 [form2 [form3 ...]]

Returns: typeless

Returns 0 if there are no errors.

Category:

Form Functions
