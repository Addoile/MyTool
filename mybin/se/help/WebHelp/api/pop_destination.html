boolean pop_destination(boolean force=false, boolean closeBuffer=false)

Decrement the reference count for a file that was jumped to by a tagging or search operation.

Parameters:

force - force destination to be popped, independent of reference count.

closeBuffer - close the current buffer?

Returns: boolean

'true' if the reference count hit zero, 'false' if not in table or still referenced

Applies To:

Edit Window

Category:

Bookmark Functions

See Also:

def_autoclose_flags, push_destination, push_bookmark
