typeless _GetDialogInfoHt(_str key, int wid=0, boolean usePUser2=false)

Retrieve data from a dialog indexed by key. Use this instead of using multiple p_user variables for several controls.

Use _SetDialogInfoHt to store data to a dialog by hash table key.

This function differs from _GetDialogInfo because it retrieves data from a hash table that is indexed by string key, rather than an array. Use _SetDialogInfo, _GetDialogInfo to store/retrieve data to/from a dialog by integer index.

Parameters:

key - A unique hash table key that represents a given item.

wid - (optional). If specified, use this control's p_user instead of p_active_form. 0 specifies the active form. Defaults to 0.

usePUser2 - If true, get the items from the p_user2 property

Returns: typeless

Data found with hash table key.

Category:

Form Functions

See Also:

_SetDialogInfoHt, _SetDialogInfo, _GetDialogInfo
