_command int ispf_change(_str arglist="")

The change command is used to replace one string with another. It accepts the same search options as find, locate, and exclude. See ispf_find for more details about the search arguments. 

   CHANGE string1 string2
          [FIRST| LAST | NEXT | PREV ]
          [WORD| PREFIX | SUFFIX ]
          [X| EX| NX ]
          [ ALL ]
          [StartCol-EndCol]
          [StartLabel-EndLabel]

Returns: int

Returns 0 if successful.

Category:

ISPF Primary Commands

See Also:

ispf_find, ispf_exclude, ispf_locate
