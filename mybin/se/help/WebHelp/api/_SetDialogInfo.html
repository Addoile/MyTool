void _SetDialogInfo(int index, typeless value, int wid=0, boolean usePUser2=false)

Store indexed data to a dialog. Use this instead of using multiple p_user variables for several controls.

Use _GetDialogInfo to retrieve data from a dialog by hash table key.

This function differs from _SetDialogInfoHt because it stores data in an array that is indexed by integer, rather than a hash table key. Use _SetDialogInfoHt, _GetDialogInfoHt to store/retrieve data to/from a dialog by hash table key.

Parameters:

index - A unique int that represents a given item. There should be a constant for this.

value -

wid - (optional). If specified, use this control's p_user instead of p_active_form. 0 specifies the active form. Defaults to 0.

usePUser2 - If true, store the data on the p_user2 property

Category:

Form Functions

See Also:

_GetDialogInfo, _SetDialogInfoHt, _GetDialogInfoHt
