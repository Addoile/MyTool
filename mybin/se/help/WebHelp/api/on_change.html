event on_change(int reason, int index=0)

Text Box - Occurs when the p_text property changes.

Combo Box - Occurs when the p_text property changes. The on_change event is called with a reason argument which may be one the following constants defined in "slick.sh":

CHANGE_OTHER
Change probably caused by typing.

CHANGE_CLINE
Change occurred because selected line in list box changed and the list was visible.

CHANGE_CLINE_NOTVIS
Change occurred because a key was pressed which scrolls the list (Up, Down, PgUp, PgDn) while the list was invisible.

CHANGE_CLINE_NOTVIS2
Same as CHANGE_CLINE_NOTVIS. Sent to user level 2 inheritance only. User level 2 inheritance will receive the CHANGE_CLINE_NOTVIS reason as well if the user level 1 inheritance does not catch the on_change event.

List Box - The on_change event is called with a reason argument set to CHANGE_SELECTED. Occurs when items are selected or deselected because of a key press or mouse event. None of the _lb??? functions cause a list box on_change event.

Directory List Box - The on_change event is called with a reason argument which may be one of the following reason constants defined in "slick.sh":

CHANGE_SELECTED
Occurs when items are selected or deselected because of a key press or mouse event. None of the _lb??? functions cause an on_change event. CHANGE_PATH _dlpath function was called which changed the directory displayed.

File List Box - The on_change event is called with a reason argument which may be one of the following reason constants defined in "slick.sh":

CHANGE_SELECTED
Occurs when items are selected or deselected because of a key press or mouse event. None of the _lb??? functions cause an on_change event.

CHANGE_FILENAME
_flfilename function was called which changed the file names listed.

Drive List - The on_change event is called with a reason argument which may be one the following constants defined in "slick.sh":

CHANGE_OTHER
p_text property changed probably because of key press.

CHANGE_CLINE
p_text property changed because selected line in list box changed and the list was visible.

CHANGE_CLINE_NOTVIS
p_text property changed because a key was pressed which scrolls the list (Up, Down, PgUp, PgDn) while the list was invisible.

CHANGE_CLINE_NOTVIS2
Same as CHANGE_CLINE_NOTVIS. Sent to user level 2 inheritance only. Use level 2 inheritance will receive the CHANGE_CLINE_NOTVIS reason as well if the user level 1 inheritance does not catch the on_change event.

CHANGE_DRIVE
The drive changed by selecting a different drive from the combo list box.

Vscroll Bar, Hscroll Bar - Sent after dragging the thumb box is completed. The p_value property has the new scroll position.

Spin - The on_change event is called with a reason set to CHANGE_NEW_FOCUS. This event occurs before an on_spin_up or on_spin_down event to allow you to return the window id of the control you want to get focus after spinning is completed. Return '' if you don't care.

Tree View - The on_change event is called with a reason argument which may be one the following constants defined in "slick.sh":

CHANGE_EXPANDED
Occurs when you expand a level of the tree. During this event you can fill in the children at this level of the tree. arg(2) is the tree item index.

CHANGE_COLLAPSED
Occurs when you collapse a level of the tree. arg(2) is the tree item index.

CHANGE_LEAF_ENTER
User pressed Enter key while on leaf tree item. arg(2) is the tree item index.
Will usually be called with the index of the item changing as the second argument

SSTab - The on_change event is called with a reason argument which may be one the following constants defined in "slick.sh":

CHANGE_TABACTIVATED
Occurs when a different tab is activated. arg(2) is the tab order (p_ActiveTab).

CHANGE_TABDEACTIVATED
Occurs before a different tab is activated. arg(2) is the tab order (p_ActiveTab).

CHANGE_TAB_DROP_DOWN_CLICK
The drop-down list was clicked.

CHANGE_TABMOVED
Tab was moved from from-index to to-index, where arg(2) is from-index, arg(3) is to-index.

Parameters:

reason - change event reason, one of CHANGE_

index - identifies the item being changed

Applies To:

Text Box, Combo Box, List Box, Directory List Box, File List Box, Drive List, Vscroll Bar, Hscroll Bar, Spin, Tree View, SSTab

Categories:

Combo Box Events, Directory List Box Events, Drive List Events, File List Box Events, Hscroll Bar Events, List Box Events, Spin Events, Text Box Events, Tree View Events, Vscroll Bar Events
