int _SlickCDebugHandler(int max_time=5000)

Poll for commands for the Slick-C® JDWP debugger server to handle and respond to. If debugging is not current enabled or connected, this function will do nothing.

Parameters:

max_time - maximum amount of time in ms to spend processing before returning control to SlickEdit.

Returns: int

0 on success, <0 on error.

Macro Programming Functions
