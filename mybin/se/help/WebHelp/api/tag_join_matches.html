void tag_join_matches()

Transfer the contents of the current match set to the previous match set, and pop it off of the stack. This is similar to tag_pop_matches(), except that the contents of the match set are not thrown away.

For synchronization, macros should perform a tag_lock_matches(true) prior to invoking this function.
