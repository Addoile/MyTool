int VSAPI vsFindIndex(const char *pszName, int flags)

Parameters:

pszName - Name to find.

flags - One or more of the VSTYPE_??? flags Ored together.

VSTYPE_PROC
Matches function
VSTYPE_VAR
Matches variable
VSTYPE_EVENTTAB
Matches event table
VSTYPE_COMMAND
Matches command
VSTYPE_CLASS
Matches class name
VSTYPE_INTERFACE
Matches interface name
VSTYPE_STRUCT
Matches struct name
VSTYPE_CONST
Matches const name
VSTYPE_ENUM
Matches enumerated type
VSTYPE_MODULE
Matches module
VSTYPE_PICTURE
Matches picture.
VSTYPE_CLASS
Matches any Slick-C® class type name
VSTYPE_OBJECT
Matches any type of dialog box template. Use vsoi2type(VSOI_???) to find a specific type of object.
VSTYPE_MISC
Matches miscellaneous.

Returns: int

Returns name table index to pszName of one of the types specified by flags. 0 is returned if a match is not found. A name type match is considered if (flags & vsNameType(index)) is true. Search is not case sensitive except for names of type VSTYPE_PICTURE and VSTYPE_MODULE under UNIX. All underscore characters in name are converted to dash characters before searching takes place except for names of type VSTYPE_PICTURE or VSTYPE_MODULE. The name type flags are listed in the file "vs.h".

Attention:

This function is thread-safe.

Example:

     int index=vsFindIndex("form1",vsoi2type(VSOI_FORM));

Names Table Functions
