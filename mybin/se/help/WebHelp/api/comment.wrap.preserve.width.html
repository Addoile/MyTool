_command typeless comment_wrap_preserve_width(_str yesno="")

If yes, the editor uses the preserve width on existing comments option when performing comment wrapping. To set the comment wrap style for a specific file extension, use the Extension Options dialog box (" Tools", " Configuration..", "File Extension Setup...", select the Comment Wrap tab).

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
