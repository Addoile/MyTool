_str _default_option(_str option, _str newValue=null)

Gets and optionally sets an editor default option.

Parameters:

option - may be one of the following letters:

'A'
boolean. Alt menu. When true, won't accidentally get placed on menu bar when press Alt key not followed by another key.
'C'
boolean. Display MDI child cursor position when do not have focus. Default is on.
'E'
boolean. Set Slick-C® error position on weak interpreter error. Effects find_error command.
'F'
boolean. Maximize first MDI child. Default is off.
'H'
boolean. Display horizontal scroll bar on edit window. Default is on.
'I'
boolean. When on, a carriage return not followed by a new line in the build window erases current line. Default is on.
'K'
boolean. Cursor blink. Default is on.
'L' int. Window left margin in twips for edit window and editor control. Ignored when there are selective display
bitmaps or any other line prefix bitmaps. Default is 100.
'N' _str. Past EOF character. Determines the character displayed on lines past the end of the file. Useful for VI emulation.
'P'
boolean. Hide mouse when type character. Default is off.
'Q' _str. This option is ignored.
VSOPTIONZ_SPECIAL_CHAR_XLAT_TAB
_str. Determines the characters display when special characters are display like tabs, new lines, spaces, etc. This strings must be 6 characters in length. The special characters in order are EOLCH1, EOLCH2, TAB(9), SPACE(32), VIRTUAL TAB SPACE, and EOF(26).
'R' Column to display vertical line in edit window. 0 specifies no vertical line. Defaults to 0.
'S' int. Default search flags.
'T'
boolean. Display top of file line for edit window and editor control. Defaults to on.
'U'
boolean. Draw box focus rect around current line for edit window and editor control. Defaults to off.
'V'
boolean. Display vertical scroll bar on edit window. Defaults to on.
'W' int. Maximum length of filename placed in menu in
bytes. Effects X windows only. Defaults to 40.
'Z'
boolean. Protect read only mode. When on, the editor will not allow read only files to be modified. Saving a read only file always displays the Save As dialog box. Defaults to on.
VSOPTION_WARNING_ARRAY_SIZE
int. A message box is displayed when a macro creates an array larger than this value to optional let the use terminate the macro.
VSOPTION_WARNING_STRING_LENGTH
int. A message box is displayed when a macro creates a string larger than this value to optional let the use terminate the macro.

newValue -

Returns: _str

Returns the (possibly newly set) current value for corresponding option.

Category:

Miscellaneous Functions
