int dbg_resolve_path(_str file_name, _str &full_path)

Attempt to quickly resolve the path to the given source file. First it checks the "source file" to "resolved file" cached mapping. If that fails, it tries to find the file in one of the source dirs.

Parameters:

file_name - source file name (with extension)

full_path - (reference) fully resolved, absolute path name

Returns: int

0 on success, <0 on error

Category:

Debugger Functions
