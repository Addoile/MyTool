int fmsay(_str msg)

This command is intended to be executed by the for_select command and not directly from the command line. Displays string in fileman status window and opens shell window if it is not already open. Useful for testing the commands for_all, for_mark, for_select, and for_dir. 

   Command Line Example

         for-all fmsay %L

Returns: int

Returns 0.

Category:

File Functions

See Also:

for_all, for_mark, for_select, for_dir
