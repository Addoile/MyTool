int set_name_info(int index, _str info)

Replace info, corresponding to name table index. If index is not a valid index, the interpreter is halted. Info may be changed for any name type.

Returns: int

Returns 0 if successful. Otherwise, INTERPRETER_OUT_OF_MEMORY_RC is returned. Pending message set.

Category:

Names Table Functions
