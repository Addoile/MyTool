int tag_next_global(int type_id, int mask, int non_zero)

Retrieve the next tag included in this tag database with global scope that is one of the given type (type_id) and that matches the given tag flag mask ((mask & tag_flags) != 0) == non_zero. Should be called only after calling tag_find_global. Use tag_get_info to extract the details about the tag.

Parameters:

type_id - First tag type ID. See tag_get_type for a list of all standard tag types. if (type_id<0), returns tags with ID>VS_TAGTYPE_LASTID

mask - Tag flags indicating symbol attributes, see tag_insert_simple for details.

non_zero - if 1, succeed if mask & tag.flags != 0 if 0, succeed if mask & tag.flags == 0

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Category:

Tagging Functions
