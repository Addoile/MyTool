_command typeless edit_with(_str command='')

Loads the filename specified and executes the command specified. If command modifies the file, a prompt appears asking whether to save the file. The file is quit if it has been saved or is not modified. This command is useful in fileman mode for executing the change command on the selected files.

Parameters:

cmdline - a string in the format: filename command

Returns: typeless

Returns 0 if command executes successfully. If command returns a non-zero value, the edit_with function returns the commands return code which could be anything.

Example:

   for-select edit-with %f  c/string1/string2
   for-select edit-with %f  c/string1/string2/**   No prompting

Category:

Miscellaneous Functions
