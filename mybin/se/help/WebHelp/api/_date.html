_str _date(_str option='U')

Parameters:

option - return value format option, may be one of the following:

  • 'U' US format date mm/dd/yyyy.
  • 'L' Locale dependent date string. Under Windows, this uses the regional settings "Short date style". For other platforms, this option is still under development.
  • 'I' ISO 8601 format date yyyy-mm-dd.

Returns: _str

Returns the system date.
