int VSAPI vsTagInsertContext(int outer_context, const char *tag_name, const char *tag_type, const char *file_name, int start_line_no, int start_seekpos, int scope_line_no, int scope_seekpos, int end_line_no, int end_seekpos, const char *class_name=a, int tag_flags=a, const char *signature=a, const char *class_parents=a, const char *template_sig=a, int name_line_no=a, int name_seekpos=a)

Add a tag and its context information to the context list. The context for the current tag includes all tag information, as well as the ending line number and begin/scope/end seek positions in the file. If unknown, the end line number/seek position may be deferred, see tag_end_context().

For synchronization, threads should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

outer_context - context ID for the outer context (eg. class/struct)

tag_name - tag string

tag_type - string specifying tag_type

file_name - full path of file the tag is located in

start_line_no - start line number of tag within file

start_seekpos - start seek position of tag within file

scope_line_no - start line number of start of tag inner scope

scope_seekpos - start seek position of tag inner scope

end_line_no - (optional) ending line number of tag within file

end_seekpos - (optional) end seek position of tag within file

class_name - (optional) name of class that tag is present in, use concatenation (as defined by language rules) to specify names of inner classes.

tag_flags - (optional) see VS_TAGFLAG_* above.

signature - (optional) tag signature (return type, arguments, etc)

class_parents - (optional) classes that this class inherits from

template_sig - (optional) template signature

name_line_no - (optional) line number that symbol name is located on

name_seekpos - (optional) Seek position of the first character of the symbol name

Returns: int

sequence number (context_id) of tag context on success, or <0 on error.

Category:

Tagging Functions
