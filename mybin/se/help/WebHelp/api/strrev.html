_str strrev(_str string, int utf8=-1)

Parameters:

string - Input string.

utf8 - -1 specifies that input and output string are UTF-8 if UTF-8 support is enabled. 0 specifies that input and output string are SBCS/DBCS. 1 specifies that input and output string are UTF-8.

Returns: _str

Returns string with characters reversed.

Example:

   strrev('ABc')  == 'cBA'
   strrev('abc123')  == '321cba'

String Functions
