_command void ispf_delete(_str cmdline="")

Deletes lines in the given line range, or the entire buffer.

Sytnax:

   DELETE [ ALL| X| NX ] [ line_range ]

line_range
Two numbers or labels, specifying the starting and ending lines to delete.
ALL
Delete all lines in the range, excluded or not excluded. If no line range is given, all implies the entire buffer.
X
Only delete excluded lines.
NX
Only delete displayed lines.

Category:

ISPF Primary Commands
