int tag_occurrences_end(_str file_name)

Clean up after inserting a series of occurrences from a single file for update. Doing this allows the tag database engine to remove any occurrences from the database that are no longer valid.

Parameters:

file_name - full path of file the tags are located in

Returns: int

0 on success, <0 on error.int VSAPI

Category:

Tagging Functions
