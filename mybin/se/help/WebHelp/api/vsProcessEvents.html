void VSAPI vsProcessEvents(int *pcancel=0, int ProcessTimerEvents=0, int AllowUserInput=0)

Reads and dispatches all system messages until there are no more messages or the variable *pcancel becomes non- zero.

Parameters:

pcancel - Optionally points to a user variable. When the user variable is non-zero, this function returns.

ProcessTimerEvents - Determines whether timer events should be processed.

Category:

Keyboard Functions
