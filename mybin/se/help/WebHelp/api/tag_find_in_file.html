int tag_find_in_file(_str file_name)

Find the first tag in the given file. Tags are returned unsorted. Use tag_get_info to extract the details about the tag.

Parameters:

file_name - full path of file containing tags

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
