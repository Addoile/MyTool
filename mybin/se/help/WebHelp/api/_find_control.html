int _find_control(_str ctlName)

Parameters:

ctlName - name of control to search for

Returns: int

Returns instance handle (window id) of control, control_name, in the current form. If the current form does not have a control with name (p_name), ctlName, 0 is returned.

Example:

   defeventtab form1;
   command1.lbutton_up()
   {
       wid=_find_control("command1");  // Look for self
       if (p_window_id==wid) {
            message("Found myself");
       } else {
            message("Not possible");
       }
   }

Applies To:

All Window Objects

Category:

Form Methods
