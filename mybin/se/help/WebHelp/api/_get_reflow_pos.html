int _get_reflow_pos(int &Noflines_down, int &col)

Returns: int

Returns relative position where cursor should be placed in resulting reflowed text output from the _reflow_selection function. If the cursor does not need to be moved, Noflines_down will be a negative number, otherwise Noflines_down indicates the number of lines to move down in the reflowed text, and col specifies the column to move to.

Applies To:

Edit Window, Editor Control

Category:

Selection Functions
