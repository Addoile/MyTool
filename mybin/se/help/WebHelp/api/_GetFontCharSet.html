int _GetFontCharSet(_str FontName, _str FontSize, _str options="")

Parameters:

options - is a string of zero or more of the following option letters:

F
Display fixed fonts only.
P
Display printer fonts.
S
(Default) Display screen fonts.

The S and P options may not both be specified.

Returns: int

Returns SHIFTJIS_CHARSET if the font specified contains Japanese characters.

Applies To:

Label, Text Box, Edit Window, Editor Control, Frame, Command Button, Radio Button, Check Box, Combo Box, List Box, Drive List, File List Box, Directory List Box
