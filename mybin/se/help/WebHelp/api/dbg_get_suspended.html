int dbg_get_suspended(int thread_id, _str &name, _str &group_name, int &flags)

Get the details about a suspended thread. Note that the index of a suspended thread is not technically the same as the index of that same thread on the complete threads table. It is the n'th thread on the threads table which has the 'suspended' flag set.

Parameters:

thread_id - index of suspend thread in thread table

name - (reference) thread name

group_name - (reference) thread group name

flags - (reference) bitset of DEBUG_MODEL_THREAD_FLAGS_

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
