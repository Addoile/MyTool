bool VSAPI vsTagCheckFriendRelationship(const char *szClassName, const char *szFriendList)

Parameters:

szClassName - class which tag is coming from

szFriendList - list of classes that are friendly to our context

Returns: bool

Returns true if the given given class is among the friends on the given friend list. Otherwise it returns false. Since this language feature is specific to C++, the check is always case sensitive.

Category:

Tagging Functions
