unsigned short*VSAPI vsUTF8ToUTF16(const char *pBuf, int BufLen=-1, int *pDestLen=0, int i=0)

Converts a UTF-8 text to UTF-16. To free the memory allocated by this function, specify 0 for pBuf, -1 for BufLen, anything for DestLen, and the correct destination temp buffer. There is typically no need to free the memory allocated by this function unless you converted a very large amount of data.

Parameters:

pBuf - Input UTF-8 buffer.

BufLen - Length of buffer. Specify -1 to compute length from a null terminated buffer.

DestLen - Length of resulting UTF-16 string.

i - Destination temp buffer index where 0<=i<=9. This allows you to call this function multiple times without the previous result being overwritten by the new result.

Returns: unsigned short*VSAPI

Returns pointer to null terminated UTF-16 string.
