_str _typename()

Returns: _str

For a given Slick-C variable, return the name of it's type, as currently seen by the interpreter.

This function works with Slick-C classes, arrays, hash tables, and strings. It is unable to distinguish between integers, booleans, and enumerated types.

Example:

class AAAA { int x; int y; };
void sayAAAA() {
   AAAA a;
   say(a._typename());
}

Category:

Miscellaneous Functions
