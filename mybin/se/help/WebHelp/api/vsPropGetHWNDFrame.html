void*VSAPI vsPropGetHWNDFrame(int wid, bool alwaysReturnChildHWND=false)

Parameters:

wid - Window id of object or index to resource returned by vsFindIndex, vsNameMatch, VSP_CHILD, or VSP_NEXT. 0 specifies the current object.

alwaysReturnChildHWND - When false, shell HWND is returned. With Qt, this is typically what you want because it is unsafe to retrieve a child widget HWND because of drawing issues that will occur afterwards.

Returns: void*VSAPI

Returns the window handle for the given window ID.

Example:

     HWND hwnd = (HWND)vsPropGetHWND(0);

Applies To:

All Window Objects

Category:

Miscellaneous Functions

See Also:

vsPropGetB, vsPropGetZ, vsPropSetZ, vsPropSetB, vsPropGetI, vsPropGetI64, vsPropSetI, vsPropSetI64, vsPropGetL, vsPropSetL, vsPropGetHWND, vsPropGetHWNDFrame
