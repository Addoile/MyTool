int tag_get_num_async_tagging_jobs(_str jobKind="A")

Parameters:

jobKind - The kind of job to return the number of. Currently, only "A" (All) is supported.

  • "A" for all jobs
  • "U" for unfinished jobs
  • "Q" for just queued jobs
  • "F" for finished jobs
  • "P" for finished jobs which were postponed to avoid delays
  • "R" for running jobs
  • "L" for jobs waiting for file to be read from disk
  • "T" for the number of threads running

Returns: int

Return the number of background tagging jobs queued, finished, and running.

Tagging Functions
