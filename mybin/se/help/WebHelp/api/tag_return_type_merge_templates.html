void tag_return_type_merge_templates(struct VS_TAG_RETURN_TYPE &rt, struct VS_TAG_RETURN_TYPE &outer_rt)

Merge the template arguments for the return type represented by 'outer_rt' with the template arguments for the return type represented by 'rt'. This is used to handle nested templates.

Parameters:

rt - return type to merge template arguments to

outer_rt - return type to mrege template arguments from

Category:

Tagging Functions
