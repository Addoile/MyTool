void VSAPI vsSetPrevIndex(char option, int index)

Parameters:

option - Indicates last index information to return.

'C'
Set the index of the previous command or function executed by a key, vsCallKey, or vsExecute.
'L'
Set the index of the previous command or function executed by a key or the vsCallKey function.

index - New value for index.

Returns: void

Returns a last index corresponding to option given.

Category:

Keyboard Functions
