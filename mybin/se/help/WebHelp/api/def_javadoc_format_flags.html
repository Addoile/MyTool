int def_javadoc_format_flags

These setting is used by the JavaDoc editor when reformatting JavaDoc comments. This is a bitset of VSJAVADOCFLAG_*.

  • VSJAVADOCFLAG_BEAUTIFY -- Beautify JavaDoc comments when exiting the JavaDoc editor
  • VSJAVADOCFLAG_ALIGN_PARAMETERS -- Align function parameters
  • VSJAVADOCFLAG_ALIGN_EXCEPTIONS -- Align exception names
  • VSJAVADOCFLAG_BLANK_LINE_AFTER_PARAMETERS -- Add blank line after parameter list
  • VSJAVADOCFLAG_BLANK_LINE_AFTER_RETURN -- Add blank line after @return statement
  • VSJAVADOCFLAG_BLANK_LINE_AFTER_DESCRIPTION -- Add blank line after symbol's description
  • VSJAVADOCFLAG_ALIGN_RETURN -- Align comments for @return statement
  • VSJAVADOCFLAG_ALIGN_DEPRECATED -- Align comments for @deprecated
  • VSJAVADOCFLAG_BLANK_LINE_AFTER_EXAMPLE -- Add balnk line after example
  • VSJAVADOCFLAG_BLANK_LINE_AFTER_LAST_PARAM -- Add blank line after last parameter comment
