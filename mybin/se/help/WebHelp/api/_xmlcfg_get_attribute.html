typeless _xmlcfg_get_attribute(int iHandle, int NodeIndex, _str AttrName, _str DefaultValue='')

Gets the value of an attribute

Parameters:

iHandle - Handle to an XML tree

NodeIndex - Index to get attribute from

AttrName - Name of attribute to retrieve the value for

DefaultValue - Value to return if attribute does not exist or an error occurs.

Returns: typeless

Returns value of attribute if successful. Otherwise DefaultValue is returned.

Category:

XMLCFG Functions
