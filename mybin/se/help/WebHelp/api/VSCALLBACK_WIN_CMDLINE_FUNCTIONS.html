3 VSCALLBACK_WIN_CMDLINE_FUNCTIONS

pfn is &VSCMDLINE_FUNCTIONS

For this callback, pfn is a pointer to the VSCMDLINE_FUNCTIONS structure. The VSCMDLINE_FUNCTIONS structure defines all the function required to implement a command line. See VSCMDLINE_FUNCTIONS structure in "vs.h" for more information. This allows each editor control to have its own command line.
