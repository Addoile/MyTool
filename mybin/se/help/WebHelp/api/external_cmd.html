int external_cmd(_str name, _str params, _str shell_options, _str exact_options)

Utility command for invoking an external macro or program from a menu. Normally, when an external macro or program is executed from a menu, there is no message displayed if the macro or program is not found (if it can't be executed for some strange reason). Prefix your external macro or program you want to execute with extern to get better error handling.

Returns: int

Returns 0 or return code of program or macro if successful. Common negative return codes are FILE_NOT_FOUND_RC, and ACCESS_DENIED_RC. Return value of external macro can be anything.

Category:
