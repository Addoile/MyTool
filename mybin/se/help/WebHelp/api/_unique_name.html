void _unique_name(_str prefix="")

Gives the object a unique name (p_name) different from other controls on the form. If p_name is not '' and is already unique (no other controls have the same name), it is not changed.

Parameters:

prefix - prefix to assign to name

Applies To:

All Window Objects
