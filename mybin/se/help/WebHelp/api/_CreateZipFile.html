int _CreateZipFile(_str filename, _str(&files)[], int(&zipStatus)[], _str(&archiveFilenames)[]=null)

Creates a zip package with the given file name, containing the given array of files. All file names (in the zip file name and in the array) must include full paths to the file.

Parameters:

filename - name of zip file

files - files to be zipped up and included

Returns: int

0 for success, error code otherwise

Category:

File Functions
