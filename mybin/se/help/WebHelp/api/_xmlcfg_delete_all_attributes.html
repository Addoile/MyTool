int _xmlcfg_delete_all_attributes(int handle, int elementNode)

Deletes all the attributes of a given node.

Parameters:

handle - Handle to an XMLCFG tree returned by _xmlcfg_open(), _xmlcfg_create(), or _xmlcfg_open_from_buffer().

elementNode - Node index of element to delete attributes of

Returns: int

0 on success, <0 on error.

Category:

XMLCFG Functions
