int VSAPI vsTagGetOccurrence(char *occur_name, char *file_name)

Retrieve information about the current occurrence, as defined by tag_find_occurrence/tag_next_occurrence.

Parameters:

occur_name - (output, VS_TAG_MAX_TAGNAME) Word to be indexed

file_name - (output, VS_TAG_MAX_FILENAME) Path of file occurrence is located in

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
