int _xyHitTest(int x, int y)

Return the tab-index of the tab that contains specified global (x,y) coordinates.

The active window MUST be a tab control (OI_SSTAB).

Parameters:

x - Global x-coordinate.

y - Global y-coordinate.

Returns: int

Tab-index of tab containing (x,y), or -1.

Applies To:

SSTab

Category:

SSTab Methods
