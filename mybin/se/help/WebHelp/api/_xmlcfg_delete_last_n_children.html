int _xmlcfg_delete_last_n_children(int handle, int parent, int numToDelete)

Deletes the last n children from a node.

Parameters:

handle - handle to xml tree

parent - parent node of children to delete

numToDelete - number of children to delete. if this number is greater than the number of children that the node has, then all children are deleted

Returns: int

int 0 on success, < 0 on error

Category:

XMLCFG Functions
