_command typeless lowcase_word()

Translates the current word to lower case and places the cursor after the current word.

Returns: typeless

Returns 0 if successful. Returns 1 if no word exists at cursor. On error, message is displayed.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods

See Also:

upcase_word, cap_word
