int tag_next_type(_str &type_name)

Retreive the name of the next type included in this tag database.

Parameters:

type_name - (Output) Tag type name. See tag_get_type for a list of all standard tag types.

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if no more.

Category:

Tagging Functions

See Also:

tag_get_type, tag_find_type, tag_reset_find_type
