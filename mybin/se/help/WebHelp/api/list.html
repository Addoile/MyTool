_command typeless list(_str path='')

Creates a buffer and inserts a tree directory list of the files specified with columns for size, date, time, attributes, and name. The mode is changed to fileman mode. When in fileman mode F1 becomes help on file management keys and the right mouse button brings up a menu of file management commands.

Parameters:

cmdline - has the following syntax:

     {[ [- | +] option_letters] [ [@]filespec ]}

option_letters - may be 'H','S','D','P','T' with the following meaning:

H
Include hidden files. Defaults to off. Ignored by UNIX version. This option is always turned on under Windows if the "Show all files" explorer option is set.
S
Include system files. Defaults to off. Ignored by UNIX version. This option is always turned on under Windows if the "Show all files" explorer option is set.
D
Include directory files. Defaults to off.
P
Append path. Defaults to on.
T
Tree file list. Defaults to on.

filespec - may contain operating system wild cards such as '*' and '?'. If filespec is not specified, the current directory is used. '@' sign prefix to filespec indicates that filespec is a file or buffer which contains a list of file names to be used as arguments to this command.

Command line examples:

list c:\
List all files on drive C.
list @backup.lst
List files specified by each line of the file "backup.lst".
list
List all files starting from the current directory.
list +HS c:\
List all files on drive C including hidden and system files.

Returns: typeless

Returns 0 if successful. Common return codes are TOO_MANY_SELECTIONS_RC, FILE_NOT_FOUND_RC, PATH_NOT_FOUND_RC, and TOO_MANY_SELECTIONS_RC. On error, message is displayed.

Applies To:

Edit Window
