int _restore_form_xy(boolean do_wh_only=false, _str form_name=null, boolean do_span=false)

Restores x, y window position saved by the _save_form_xy function.

Parameters:

do_wh_only - (optional) only do width and height, do not adjust the x and y position of form.

form_name - (optional) form name to use to store position data

do_span - (optional) allow the form to span in width to an adjacent monitor.

Returns: int

Returns 0 if the old (x, y) position information is found.

Applies To:

Form

Category:

Form Methods
