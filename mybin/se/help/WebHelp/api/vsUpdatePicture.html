int VSAPI vsUpdatePicture(const char *pszFilename, int reserved=a)

Loads a .bmp, .ico, .png, or .xpm file. We recommend that the filename start with an "_" (ex. "c:\slickedit\_stackex.ico"). This tells SlickEdit that the bitmap is global (not attatched to a Slick-C® dialog box).

Parameters:

pszFilename - Filename of picture to load. The filename stored in the names table has no path. So if you use vsFindIndex to lookup your bitmap, make sure you strip the path.

Returns: int

If succcessful, returns names table index of picture. Otherwise, a negative error code is returned.

Category:

File Functions
