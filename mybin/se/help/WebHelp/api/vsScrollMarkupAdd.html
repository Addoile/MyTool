int VSAPI vsScrollMarkupAdd(int wid, seSeekPos iLineNum, int iType, seSeekPos iLength=seSeekPos(1))

Add a scroll bar mark

Parameters:

wid - Window ID of editor control

iLineNum - Line number to put mark on

iType - Type returned by vsScrollMarkupAllocType

iLength - Number of lines in this mark, defaults to 1

Returns: int

int 0 if successful

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
