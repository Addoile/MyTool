void _change_all_wfonts(_str font_name, _str font_size, int font_flags, int charset=VSCHARSET_DEFAULT, int cfg=CFG_SBCS_DBCS_SOURCE_WINDOW)

Changes the font of all MDI edit windows. font_size is a point size. font_flags is a combination of the following flag constants defined in "slick.sh": 

F_BOLD
F_ITALIC
F_STRIKE_THRU
F_UNDERLINE

font_name -
