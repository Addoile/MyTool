_str _escape_re_chars(_str filename, _str regex_type="")

Parameters:

filename - string to escape

regex_type - regular expression syntax type

Returns: _str

Returns string with regular expression special characters prefixed with \. Specify the 'U' option to escape UNIX regular expression special characters. Specify the 'B' option to escape Brief regular expression special characters. Specify the '&' option to escape a simple Wildcard expression.

Example:

_escape_re_chars("abc") == "abc"
_escape_re_chars("*.c") =="\*.c"
_escape_re_chars("*.c",'U')   =="\*\.c"
_escape_re_chars("*.c",'B')   =="\*\.c"
_escape_re_chars("+++") =="\+\+\+"
