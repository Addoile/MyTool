int VSAPI vsSetTimeout(int ms)

Set the timeout amount for performance critical functions. The timeout is not a strict timeout, it's a software timeout. Use _CheckTimeout() to test if the timeout is expired.

It is good practice to clear the timeout after you are done with it by calling _SetTimeout(0).

Parameters:

ms - number of milliseconds to time out after. use 0 to clear an existing timeout.

Returns: int

Normally will return 'ms', but if there is an existing earlier timeout, the earlier timeout will be returned.

Category:

Miscellaneous Functions
