int tag_insert_match_fast(int match_type, int local_or_context_id)

Add a tag to the match set. The tag database allows you to maintain a list of tags very efficiently through the match set. This is useful when searching for specific tags. Many of the Context Tagging® functions are designed to insert directly into the match set and use this function to do so for efficiency.

For synchronization, macros should perform a tag_lock_matches(true) and tag_lock_context(false) prior to invoking this function.

Parameters:

match_type - Specifies where the tag to insert comes from:

VS_TAGMATCH_tag
current tag from database
VS_TAGMATCH_context
item in current context (buffer)
VS_TAGMATCH_local
item in local variable set (function)
VS_TAGMATCH_match
item in match set

local_or_context_id - 0 if inserting current tag from database, otherwise, specifies the unique integer ID of the item from the context, locals, or match set, as specified by match_type.

Returns: int

sequence number (match_id) of matching tag on success, or <0 on error.
