_str _menu_match(_str name, boolean find_first)

Parameters:

name - menu name to match

find_first - 'true' to find first, 'false' to find next

Returns: _str

Returns next menu name in names table which is a prefix match of name. find_first must be non-zero to initialize matching. Returns '' when no more matches are found.
