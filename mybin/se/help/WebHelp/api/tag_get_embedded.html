int tag_get_embedded(int i, int &start_seekpos, int &end_seekpos)

Retrieve the embedded context information for the given position 'i' (first is 0).

For synchronization, threads should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

i - index of embedded context to retrieve

start_seekpos - (Output) set to start seek position for embedded context

end_seekpos - (Output) set to end seekposition for embedded context

Returns: int

0 on success, <0 on error

Category:

Tagging Functions
