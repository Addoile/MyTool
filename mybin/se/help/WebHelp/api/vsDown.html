int VSAPI vsDown(int wid, seSeekPos Noflines)

Moves cursor down the number of lines specified by Noflines. If the destination line is not in view, it is center scrolled or smooth scrolled into view.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

Noflines - Number of lines to move the cursor down.

Returns: int

Returns 0 if successful. Otherwise BOTTOM_OF_FILE_RC is returned.

Example:

        status=vsDown(0,1);
        if (status) {
            // Hit bottom of file
        }

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods
