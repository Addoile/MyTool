int dbg_get_member(int thread_id, int frame_id, _str member_path, _str &name, _str &class_name, _str &signature, int &flags, _str &value, _str &raw_value)

Get the details about a member of 'this' object.

Parameters:

thread_id - index of thread

frame_id - index of stack frame

member_path - path to member in list of member variables

name - (reference) name of member

class_name - (reference) name of class the member is in

signature - (reference) type of variable

value - (reference) value of variable

flags - (reference) set to bitset of DEBUG_MODEL_FLAGS_

is_in_scope - (reference) is the variable in scope?

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
