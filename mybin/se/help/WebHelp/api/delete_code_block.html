_command void delete_code_block()

Attempts to identify the block of code starting at the current line, and prompts to delete the entire block of code.

Returns: void

Returns 0 if successful. Common return codes are 1 (no block at cursor) and TOO_MANY_SELECTIONS_RC. On error, message is displayed.
