_command typeless for_all(_str commandArg='', _str shell_wait='', _str echo_command='', typeless command_view_id='')

Executes command specified on all the lines of the current buffer. Parts of the line being executed are inserted into the command specified according to the following specifications:

%E
File extension with dot (Directory buffers only)
%F
Filename (Directory buffers only)
%N
Name without extension (Directory buffers only)
%L
Entire line
%R
Current project
%V
Drive with : (Directory buffers only). This is not very useful under UNIX.
%nnn
Nth word is inserted
%(envvar)
Value of environment variable envvar
%%
percent character

Example:

for-all fmsay %L
Displays all lines of the current buffer

Category:

Miscellaneous Functions
