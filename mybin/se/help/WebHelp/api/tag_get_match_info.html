int tag_get_match_info(int match_id, VS_TAG_BROWSE_INFO &cm)

Populate the given symbol struct with all the known information corresponding to the given match ID in the current match set.

For synchronization, macros should perform a tag_lock_matches(true) prior to invoking this function.

Parameters:

match_id - match ID

cm - (output) symbol information

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
