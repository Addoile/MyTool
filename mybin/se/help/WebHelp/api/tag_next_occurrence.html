int tag_next_occurrence(_str tag_name, boolean exact_match=true, boolean case_sensitive=false)

Find the next occurrence with the given tag name or tag prefix. Use tag_get_occurrence (below) to get details about the occurrence.

Parameters:

tag_name - Tag name or prefix to search for

exact_match - Exact (word) match or prefix match (0)

case_sensitive - Case sensitive search?

Returns: int

0 on success, BT_RECORD_NOT_FOUND_RC if not found, <0 on error.

Category:

Tagging Functions

See Also:

tag_find_occurrence, tag_get_occurrence, tag_reset_find_occurrence
