int dbg_update_thread_tree(int tree_wid, int tree_index)

Update the list of threads in the system. The thread tree has four columns:

  • Thread name
  • Group name
  • Thread status
  • Thread State

Parameters:

tree_wid - window ID of the tree control

tree_index - tree index to insert under (TREE_ROOT_INDEX)

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
