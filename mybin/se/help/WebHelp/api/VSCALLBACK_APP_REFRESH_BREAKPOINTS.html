33 VSCALLBACK_APP_REFRESH_BREAKPOINTS

void (VSAPI * vspfnRefreshBreakPoints)(int BPMRefreshFlags);
This callback gets called when a break point is deleted because the line it was on was deleted, and when the line number for a break point changes.

Parameters:

BPMRefreshFlags - Flags for this break point. One or more of the following flags:

VSBPMREFRESH_LINENUMBERS_CHANGED
Breakpoint line number changed because lines were inserted or deleted
VSBPMREFRESH_BREAKPOINTS_AUTO_DELETED
Breakpoint was deleted because lines were deleted

Category:

Editor Control Callback Functions
