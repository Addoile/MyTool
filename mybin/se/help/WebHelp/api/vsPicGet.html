int VSAPI vsPicGet(int PicIndex, seSeekPos *pRealLineNum=a, seSeekPos *pLineNum=a, seSeekPos *pNofLines=a, void **ppUserData=a, int *pBufID=a, char *pszBufName=a, int *pBMIndex=a, int *ptype=a, char **pszMessage=a, char *pszLineData=a, int *pmarkid=a, seSeekPos *pBeginLineROffset=a, int *pcol=a, void *preserved2=a)

Deprecated. . Use vsLineMarkerGet() instead.

Parameters:

PicIndex - Index of pic returned by a vsPicAdd function.

pRealLineNum - Set to real line number.

pLineNum - Set to line number.

pNofLines - Set to number of lines. Used to determine how lines to draw a box around.

ppuserdata - Set to userdata.

pBufID - Set to buffer id or negative number if file is not currently loaded.

pszBufName - Set to buffer name.

pBMIndex - Set to index into names table of bitmap.

ptype - Set to type.

pszMessage - Set to message.

pszLineData - Set to line data. This parameter is typically only used by pics with a valid mark id.

pmarkid - Set to mark id.

pBeginLineROffset - Set to real seek position to the beginning of the line.

pcol - Set to column.

Returns: int

Returns 0 if successful.

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
