int refactor_config_add_jar(_str configName, _str jarPath, int jarIndex=-1)

Add a JAR to the system JAR list for the configuration

Parameters:

configName - name of configuration to modify

jarPath - path to JAR to add

jarIndex - where to insert JAR in list, -1 to insert at end

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
