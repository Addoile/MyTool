int _retrieve_next_form(_str direction="", _str doSetSelect="")

Retrieves the next response to check boxes, radio buttons, text boxes, and combo boxes, for the active form. This function is used to perform dialog box retrieval and is called by the dialog manager when you press F8.

Applies To:

Form

Categories:

Form Methods, Retrieve Functions

See Also:

_retrieve_prev_form, _save_form_response
