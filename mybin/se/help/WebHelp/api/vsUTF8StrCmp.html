int VSAPI vsUTF8StrCmp(const char *p1, const char *p2, int len1=SESIZE_MAX, int len2=SESIZE_MAX)

Compares two UTF-8 strings. This function is slow because it applies many of the Unicode rules. Under Windows, both buffers are converted to UTF-16 and then the WIN32 CompareStringW() function is called. Use the vsUTF8StrCmpQuick() function for a fast string compare that does not apply Unicode rules.

Parameters:

p1 - UTF-8 string 1.

p2 - UTF-8 string 2.

len1 - Number of bytes in string 1.

len2 - Number of bytes in string 2.

Returns: int

Returns 1 if string 1 is greater than string 2. Returns 0 if string 1 equals string 2. Returns -1 if string 1 is less than string 2.

Category:

Unicode Functions
