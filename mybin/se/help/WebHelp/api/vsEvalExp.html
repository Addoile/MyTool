const char*VSAPI vsEvalExp(char *pszResult, const char *pszSource, int base=a, void *preserved=a)

Evaluates a C syntax mathematical expression.

Parameters:

pszResult - (Ouput only). Result from evaluating the expression specified in pszSource.

pszSource - (Input only). C syntax mathematical expression. See help on math for more information.

base - Specifies the output base for the result. Currently only 2, 8, 10, and 16 are supported.

Returns: const char*VSAPI

Returns 0 if successful.

Category:

Keyboard Functions
