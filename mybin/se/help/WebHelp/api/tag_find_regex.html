int tag_find_regex(_str tag_regex, _str options='')

Retrieve the next tag with the a tag name matching the given regular expression with the given matching options. Use tag_get_info to extract the details about the tag.

Parameters:

tag_regex - tag name regular expression to search for

search_options - search options, passed on to pos()

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Category:

Tagging Functions
