_command int backward_up_sexp()

Moves backwards up into the parent S-expression starting from the cursor position. If no parent S-expression is found, it will move the cursor to the beginning of the file.

Returns: int

Returns 0 if successful, <0 on error.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods

See Also:

next_sexp, prev_sexp, forward_up_sexp, backward_down_sexp, forward_down_sexp
