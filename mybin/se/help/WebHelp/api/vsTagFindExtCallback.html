int VSAPI vsTagFindExtCallback(const char *callback_name, const char *language_id=a)

Deprecated. use vsFindLanguageCallbackIndex()

This function is used to look up a language-specific callback function.

Return the names table index for the callback function for the current language, or an inherited language. The current object should be an editor control.

Parameters:

callback_name - name of callback to look up, with a '%s' marker in place where the language ID would be normally located.

ext - current language ID (default=p_LangId)

Returns: int

Names table index for the callback. 0 if the callback is not found or not callable.

Category:

Tagging Functions
