int _ini_get_value(_str filename, _str section_name, _str field_name, _str &result, _str default_value='', _str case_option=null)

Places value for field_name defined within section of file specified in result variable. The start of a section is identified by a section name in square brackets. All lines until the next section or end of file are part of the section. If section or field_name do not exist, default_value is placed in result.

Parameters:

default_value - has no effect on ini file. It only affects return value when section name or field name does not exist. 

   ".ini" File Syntax:
         [section1]
         field_name1=value1
         field_name2=value2
         ...
         [section2]
         field_name1=value1
         field_name2=value2
         ...

Returns: int

Returns 0 if successful.
