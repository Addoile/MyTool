int VSAPI vsTagOpenDB(const char *pszFilename, int read_only=a)

Open an existing database for read-only or read-write access. The database type (tags, references) is automatically detected. BSC files can not be opened read-write, use tag_read_db instead.

Parameters:

pszFilename - File path of database to open.

read_only - (default true) open the database for read access

Returns: int

database handle >= 0 on success, < 0 on error.

Category:

Tagging Functions
