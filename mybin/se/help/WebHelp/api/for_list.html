_str for_list(_str word, _str command, _str echo_command)

This function is intended to be called to process "@buffer_name" string returned by the _sellist_form function. See for_all command for information on parsing parts of each line into command.

Activates buffer specified by the expression substr(atsign_bufname,2) and applies command to each line in the buffer. Parts of the line being processed may be inserted into the command. On completion, the buffer is deleted.

Parameters:

word -

command -

echo_command -

Returns: _str

Returns 0 if successful. Otherwise a non-zero value is returned. This function may return the return value of the for_all function which can return anything the command returns.

Example:

        status=edit("list");
        if (status) {
             return(status);
        }
        for_list("@"absolute("list"), "e %L",0);
