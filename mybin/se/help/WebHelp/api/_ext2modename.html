_str _ext2modename(_str ext, int &setup_index)

Deprecated. Use _LangId2Modename()

Converts a file extension to the mode name corresponding to the language language referred to by the given file extension.

Parameters:

ext - File extension.

setup_index - Set to names table index for def-language-lang (canonical extension)

Returns: _str

The mode name for the language.
