int _EmbeddedStart(typeless &orig_values, _str key="")

If the cursor is in an embedded language section switch to the language mode of the embedded code.

Parameters:

orig_values - (ref) stores the original language-specific settings for the current file. This will be used to restore the language state when _EmbeddedEnd is called.

key - name for new key table (no longer used)

Returns: int

  • Returns 0 if there is no embedded language
  • Return 1, indicates that the mode has been switched to the embedded language. Caller must call _EmbeddedEnd(orig_values)
  • Returns 2 to indicate that there is embedded language code, but in comment/in string like default processing should be performed.
  • Returns 3 if mode name for the embedded code is the same.
