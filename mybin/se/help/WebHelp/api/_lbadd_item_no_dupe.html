int _lbadd_item_no_dupe(_str item, _str options='', int addOption=0, boolean selectItem=false)

Determines if an item already exists in a combo box. If not, the item is added.

Parameters:

item - item to search for.

options - i' for case-insensitive searching, 'e' for exact atch, '' to use the existing p_case_sensitive value

addOption - where to add the new item (LBADD_TOP to add it to the beginning of the list, LBADD_BOTTOM to add at the end of the list, LBADD_SORT to sort the list after adding)

Returns: int

int

Applies To:

List Box, Combo Box

Categories:

List Box Methods, Combo Box Methods
