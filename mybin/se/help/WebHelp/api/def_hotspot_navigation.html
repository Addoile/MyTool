boolean def_hotspot_navigation

If enabled, alias and syntax expansion can define hotspots in order to have multiple cursor stops.

For example, if you have an alias such as: 

    if ( %\c) {
        %\c
    }

Initially the cursor will be positioned at the first cursor stops (%\c), when you hit Ctrl+[ (next_hotspot), the cursor will jump to the next hotspot within the brace block.

Default:

true

Category:

Configuration Variables

See Also:

expand_alias, c_space, next_hotspot, prev_hotspot
