_command int prev_ref(boolean preview_only=true, boolean quiet=false)

Repositions the cursor on the previous item in the references tool window. If window is not active, it will do a next-ref, followed by a prev-ref to attempt to reposition on the current match.

Parameters:

preview_only - If true, show the reference only in the preview pane of the references tool window.

quiet - If true, do not display message box when there are no more references.

Returns: int

Returns 0 if successful. Otherwise NO_MORE_FILES_RC is returned. On error, message is displayed.

Applies To:

Edit Window,Editor Control

Categories:

Editor Control Methods, Edit Window Methods, Search Functions

See Also:

next_ref, find, replace, gui_replace, gui_find, find_current, find_prev, find_next
