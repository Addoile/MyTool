int VSAPI vsRefactorSetModifiedFileContents(int wid, int handle, const char *filename)

Set the contents of the modified file from the given editor control and store it back with the refactoring transaction.

Parameters:

wid - Editor control window ID

handle - Handle to transaction

filename - Filename of file that was modified

Returns: int

Length of error string on success, <0 on error

Category:

Refactoring Functions
