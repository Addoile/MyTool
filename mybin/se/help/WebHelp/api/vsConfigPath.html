VSPSZ VSAPI vsConfigPath(int NoLocalConfigOption=a)

Parameters:

NoLocalConfigOption - When NoLocalConfigOption==0, the user configuration directory is returned.

When NoLocalConfigOption==1, the user configuration directory is returned if it exists. Otherwise "" is returned.

When NoLocalConfigOption==2, the user configuration directory is returned if it exists. Otherwise the "bin" directory is returned.

Returns: VSPSZ

Return absolute path of users local configuration directory with trailing file separator.
