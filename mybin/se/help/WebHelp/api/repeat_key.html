_command void repeat_key(_str command_string='')

Repeat a key event the number of times specified.

To use, after calling this command, type the number of times you want to repeat a key, followed by the key itself. For instance, the following key sequence (without the dashes): 

repeat_key - 68 - *
inserts 68 instances of '*' to the current cursor location as if the character was manually typed 68 times.

The key to be repeated is not limited to a single character, but can be a command bound to a key stroke (such as Ctrl+V).

This command can also be called from the command line, in which case it can take arguments in the following format: 

repeat_key -[repeat num] [command name] [command arguments]
