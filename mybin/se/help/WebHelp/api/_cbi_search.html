int _cbi_search(_str case_sense='', _str search_string='')

Searches for combo box text (p_text) in the combo box list starting from the beginning of the list. This function is typically used to check if the users input is valid or adjust the current line in the combo box list after more items have been added. Search is case insensitive by default. Set the p_case_sensitive property to true if you want a case sensitive search. Normally the search is considered successful if a line that begins with p_text is found. To search for an exact match, specify '$' for the second argument.

Parameters:

case_sense - 'e' for a case-sensitive search, 'i' for case-insensitive

search_string - pass '$' for an exact match

Returns: int

Returns 0 if text in combo box is found.
