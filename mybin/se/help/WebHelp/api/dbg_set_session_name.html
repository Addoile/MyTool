int dbg_set_session_name(int session_id, _str session_name)

Modify the session name for the given session ID

Parameters:

session_id - debugger session ID (1 .. max_session)

session_name - new name for session

Returns: int

0 on success, <0 on failure

Since:

10.0

Category:

Debugger Functions

See Also:

dbg_find_session, dbg_get_session_name
