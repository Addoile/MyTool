int VSAPI vsQDefaultEventTab()

Returns: int

Returns names table index of default event table. This is typically used to get and set key bindings global to all modes for the current emulation.

Example:

   int index;

index=vsFindIndex("gui_open",VSTYPE_PROC|VSTYPE_COM
MAND);
   if (index) {
      // Bind this function to a F12
       vsSetEventTabIndex(vsQDefaultEventTab(),
                          vsEvent2Index(VS_OFFSET_FKEYS+11),
                          index);

   }

Keyboard Functions
