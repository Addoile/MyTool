int dbg_session_test_version(int session_id, _str debugger_path, _str &version, int timeout)

Test the given debugger executable for version compability. This command is used outside of a debugging session to query if a particular debugger executable is compatible.

Parameters:

session_id - debugger session ID (1 .. max_session)

debugger_path - Path to debugger executable to test

version - On success, set to the version number

timeout - Connection timeout in milliseconds

Returns: int

1 on success, 0 on successful failure, <0 on serious failure (not executable, file not found)

Since:

10.0

Example:

To test the GDB supplied with cygwin, for example 

   _str version;
   int status = dbg_session_test_version(session_id,
                                         'C:\cygwin\bin\gdb.exe',
                                         version,10000);
   if (status==0) {
      _message_box("Inadaquate GDB version");
   } else if (status<0) {
      _message_box("GDB executable failure: ":+get_message(status));
   } else {
      _message_box("GDB version is good");
   }
