int def_diff_options

File differencing options. Bitset of DIFF_

  • DIFF_EXPAND_TABS -- expand tabs to spaces
  • DIFF_IGNORE_LSPACES -- ignore leading white space
  • DIFF_IGNORE_TSPACES -- ignore trailing white space
  • DIFF_IGNORE_SPACES -- ignore all white space in line
  • DIFF_IGNORE_CASE -- ignore text case
  • DIFF_OUTPUT_INTERLEAVED -- display interleaved output instead of side-by-side difference
  • DIFF_DONT_COMPARE_EOL_CHARS -- do not compare end of line characters
  • DIFF_OUTPUT_BOOLEAN -- output boolean
  • DIFF_LEADING_SKIP_COMMENTS -- skip leading comments in each file
  • DIFF_NO_BUFFER_SETUP -- no buffer setup

Default:

0

Category:

Configuration Variables
