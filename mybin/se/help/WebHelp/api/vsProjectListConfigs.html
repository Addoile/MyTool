char**VSAPI vsProjectListConfigs(int *pNofConfigs, int *pActiveConfig=a, int reserved=a, int *preserved=a)

Parameters:

pNofConfigs - Set to the number of configurations in the current project. Set to zero of there is not a project currently open.

pActiveConfig - Set to the index (0..NofConfigs-1) of the active configuration. Set to -1 if no configuration is active.

Returns: char**VSAPI

Returns an array of ASCIIZ strings which are the configrations names in the current project. 0 is returned if an error occurs or there is no project currently open.

Category:

Project Functions
