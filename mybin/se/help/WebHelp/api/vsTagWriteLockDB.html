int VSAPI vsTagWriteLockDB(int dbHandle, int ms=0)

Lock the given database to prevent other threads from accessing it. This will obtain a write lock on the database, blocking all other reader and writer threads until it is released().

Parameters:

dbHandle - database handle returned from vsTagOpenDB()

ms - millisecond timeout to wait before giving up on obtaining the write lock.

Returns: int

0 on success, <0 on error

Category:

Tagging Functions

See Also:

vsTagOpenDB(), vsTagCloseDB(), vsTagWriteLockDB(), vsTagReadLockDB(), vsTagUnlockDB()
