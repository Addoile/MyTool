void insert_line(_str line)

Inserts line into editor buffer after the current line.

Parameters:

line - String to insert. line should not contain newline characters since the appropriate newline characters are appended.

Example:

insert_line("add line after current line");

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

get_line, replace_line, delete_line, get_text
