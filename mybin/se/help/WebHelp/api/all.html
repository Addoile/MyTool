_command void all(...)

Searches entire buffer for lines containing the string specified. Lines not containing the search string are hidden. If a tilde (~) is given in the command line syntax, then lines containing the search string are hidden like the allnot command. See find command for information on options argument.

Alternate command line syntax: all [~]/string[/options]

Applies To:

Edit Window, Editor Control

Categories:

Search Functions, Selective Display

See Also:
