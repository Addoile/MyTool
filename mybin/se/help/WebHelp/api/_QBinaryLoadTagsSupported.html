boolean _QBinaryLoadTagsSupported(_str filename=null)

Parameters:

filename - name of file to test, defaults to current file

Returns: boolean

Returns 'true' if the given file has support for a load-tags callback function, such as for loading symbols from a ZIP file or a DLL with metadata or a Java class file or JAR file.
