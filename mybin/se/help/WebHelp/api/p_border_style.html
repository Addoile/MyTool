int p_border_style

Determines the border of a control.

Form - Border style may be one of the following constants defined in "slick.sh":

  • BDS_NONE - No border.
  • BDS_FIXED_SINGLE - Thin non-sizable border.
  • BDS_SIZABLE - Thick sizable border.
  • BDS_DIALOG_BOX - Standard non-sizable dialog box border.

Text Box, Editor Control, List Box, File List Box, Directory List Box, Mini HTML, Tree View - Border style may be one of the following constants defined in "slick.sh":

  • BDS_NONE - No border.
  • BDS_FIXED_SINGLE - Thin non-sizable border.

Picture Box, Image - Border style may be one of the following constants defined in "slick.sh":

  • BDS_NONE - No border.
  • BDS_FIXED_SINGLE - Thin rectangular border.
  • BDS_SUNKEN - Sunken border. The exact look of this border style OS dependent.
  • BDS_SUNKEN_LESS - Less sunken border. The exact look of this border style OS dependent.
  • BDS_ROUNDED - Thin border with rounded corners.

Label - Border style may be one of the following constants defined in "slick.sh":

  • BDS_NONE - No border.
  • BDS_FIXED_SINGLE - Thin non-sizable border.
  • BDS_SUNKEN - Sunken border. The exact look of this border style OS dependent.

Applies To:

Form, Label, Text Box, Editor Control, List Box, File List Box, Directory List Box, Picture Box, Image, Mini HTML, Tree View

Categories:

Form Properties, Text Box Properties, Editor Control Properties, File List Box Properties, Directory List Box Properties, Picture Box Properties, Image Properties, Mini HTML Properties, Tree View Properties
