int dbg_update_class_tree(int tree_wid, int tree_index, _str class_path, boolean hide_system=false)

Update the debugger class browser tree with the information retrieved by the debugger system. The list has three columns:

  • Class name
  • Outer class name (or package name)
  • Class status (loaded, prepred, unloaded)

Parameters:

tree_wid - window ID of tree control

tree_index - Index of tree control to update under

class_path - Path to members under class to expand

hide_system - Hide system classes

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
