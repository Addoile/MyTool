void VSAPI vsTop(int wid)

Places cursor at first line and first column of buffer. wid identifies the object to be operated on. 0 specifies the current object.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods
