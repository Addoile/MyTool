7 VSCALLBACK_APP_MENU_ADD_FILE_HISTORY

void (VSAPI *pfnMenuAddFileHistory)(const char *pszFilename);
The MENU_ADD_FILE_HISTORY callback gets called when the "edit" command or "gui_open" command are called. Note that MFC MDI apps don't always uses these commands to open a file. vsBufEdit does not call this callback.

Parameters:

pszFilename - Absolute name of file

Category:

Editor Control Callback Functions
