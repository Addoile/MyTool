_str file_list_field(_str filename, int col, int width)

Returns: _str

Returns the size, date, time or attributes/permissions for the filename specified. col is one of the constants DIR_SIZE_COL, DIR_DATE_COL, DIR_TIME_COL, or DIR_ATTR_COL from "slick.sh". width is one of the constants DIR_DATE_WIDTH, DIR_TIME_WIDTH, DIR_ATTR_WIDTH, or DIR_SIZE_WIDTH from "slick.sh". If file is not found or error occurs, '' is returned.

Category:

File Functions
