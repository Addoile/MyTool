typeless _ext_commentwrap_flags(int commentWrapOption, _str lang='')

Deprecated. Use _GetCommentWrapFlags().

Get the comment wrap flags.

The options are stored per extension type. If the options are not yet defined for an extension, then use CFcommentWrapDefaults as the default.

lang - language ID

Returns: typeless
