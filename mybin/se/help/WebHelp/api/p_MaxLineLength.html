int p_MaxLineLength

Determines the maximum line length. At the moment this option is only used by the _save_file built-in when the +CL option is specified and the destination file does not already have a strict record length. This property was intended for use with mainframe source files which have line length restrictions. At the moment only the physical line length is checked as if tab characters count as 1 character. We may change this in the future.

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
