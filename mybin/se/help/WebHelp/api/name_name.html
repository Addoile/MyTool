_str name_name(int index)

Returns: _str

Returns name corresponding to name table entry index. If index is not an index to a valid name, '' is returned. Names are returned in lower case.

Example:

         index= name_match("repeat_se")
         message 'possible match of repeat_se is 'name_name(index)

Category:

Names Table Functions
