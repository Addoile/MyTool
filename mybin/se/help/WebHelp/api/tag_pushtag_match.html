int tag_pushtag_match(int tree_wid, int tree_index, int progress_wid, _str member_filter, _str class_filter, _str(&tag_files)[], int filter_flags, int &num_matches, int max_matches, boolean regex_match, boolean exact_match, boolean case_sensitive)

Match symbols in all tag files against the given member and class filters.

For synchronization, macros should perform a tag_lock_matches(true) prior to invoking this function.

Parameters:

tree_wid - Window ID of the tree control

tree_index - Tree index to insert items under

progress_wid - Window id of progress label

member_filter - Prefix or regular expression for symbols

class_filter - Regex or prefix for class names

tag_files - list of tag files to search

filter_flags - Tag filter flags, only insert tags passing this filter. See tag_filter_type for more details.

num_matches - (Input, Output) number of matches found

max_matches - maximum number of matches to find

regex_match - perform regular expression match?

exact_match - exact match or prefix match?

case_sensitive - Case sensitive match or case-insensitive?

Returns: int

0 on success, 1 if list was truncated, <0 on error.

Since:

5.0a

Category:

Tagging Functions
