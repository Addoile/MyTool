int VSAPI vsGetSelLines(int wid, int markid, seSeekPos *pfirstline, seSeekPos *plastline=0, bool mark_to_cursor=true)

Returns line number information about the selection specified.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

markid - Selection handle returned by vsAllocSelection or vsDuplicateSelection. -1 specifies current selection.

pfirstline - Output for the first line of the selection.

plastcol - Output for the last line of the selection.

Returns: int

Returns 0 if wid and markid are valid.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
