int tag_find_local_iterator(_str tag_prefix, boolean exact, boolean case_sensitive, boolean pass_through_anonymous=false, _str class_name=null)

Find the first local tag with the given tag prefix, or if 'exact', with the exact tag name. Use case-sensitive match if case_sensitive != 0. Use tag_get_local2 to extract details about the tag.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Parameters:

tag_prefix - tag name or prefix of tag name

exact - search for exact match or prefix match

case_sensitive - case sensitive string comparison?

pass_through_anonymous - Pass through anonymous classes

class_name - Class to find item in

Returns: int

Local ID of tag if found, <0 on error or not found.

Category:

Tagging Functions

See Also:

tag_get_local, tag_get_local2, tag_next_local
