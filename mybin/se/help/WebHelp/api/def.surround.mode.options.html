int def_surround_mode_options

This option controls the behavior of the active code block surround mode. It consists of a set of bit flags of VS_SURROUND_MODE_*.

  • VS_SURROUND_MODE_ENABLED -- Enable dynamic surround
  • VS_SURROUND_MODE_DRAW_BOX -- Draw box around surround block
  • VS_SURROUND_MODE_JUMP_FAST -- Jump over block statements
  • VS_SURROUND_MODE_EDITABLE -- Allow primitive editing

Default:

0xFFFF

Category:

Configuration Variables
