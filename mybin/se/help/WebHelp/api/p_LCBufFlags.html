int p_LCBufFlags

Contains flags to control the line number display in the current buffer. The following flags, defined in slick.sh, are used with this property:

VSLCBUFFLAG_READWRITE
prefix area on/off
VSLCBUFFLAG_LINENUMBERS
Line numbers on/off
VSLCBUFFLAG_LINENUMBERS_AUTO
Line numbers automatic

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
