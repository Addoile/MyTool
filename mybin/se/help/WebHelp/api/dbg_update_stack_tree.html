int dbg_update_stack_tree(int tree_wid, int tree_index, int thread_id)

Update the list of functions on the current stack frame. Each entry has two columns:

  • The method name with parameter list
  • The class name

Parameters:

tree_wid - window ID of tree control

tree_index - tree index to insert under (TREE_ROO_INDEX)

thread_id - thread ID to get stack dump for

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
