int _GetCodehelpFlags(_str lang='')

Get the codehelp options for the specified language. The options are a bitset of VSCODEHELPFLAG_*.

The options are stored per language. If the options are not yet defined for a language, then use def_codehelp_flags as the default.

Parameters:

lang - language ID, see p_LangId

Returns: int

bitset of VSCODEHELPFLAG_* options.

Applies To:

Edit Window, Editor Control

Category:

Tagging Functions
