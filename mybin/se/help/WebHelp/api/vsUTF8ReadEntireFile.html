int VSAPI vsUTF8ReadEntireFile(int filehandle, int encoding, char *&pszBuf, int &BufLen, int reuseCurrentPointer=0, int *pXlatError=0)

Converts entire file from encoding specified to UTF-8.

Parameters:

filehandle - Source file handle returned by vsFileOpen().

encoding - One of the following encodings:

  • VSCP_ACTIVE_CODEPAGE
  • VSCP_CYRILLIC_KOI8_R
  • VSCP_ISO_8859_1
  • VSCP_ISO_8859_2
  • VSCP_ISO_8859_3
  • VSCP_ISO_8859_4
  • VSCP_ISO_8859_5
  • VSCP_ISO_8859_6
  • VSCP_ISO_8859_7
  • VSCP_ISO_8859_8
  • VSCP_ISO_8859_9
  • VSCP_ISO_8859_10
  • Any valid Windows code page
  • VSENCODING_UTF8
  • VSENCODING_UTF8_WITH_SIGNATURE
  • VSENCODING_UTF16LE
  • VSENCODING_UTF16LE_WITH_SIGNATURE
  • VSENCODING_UTF16BE
  • VSENCODING_UTF16BE_WITH_SIGNATURE
  • VSENCODING_UTF32LE
  • VSENCODING_UTF32LE_WITH_SIGNATURE
  • VSENCODING_UTF32BE
  • VSENCODING_UTF32BE_WITH_SIGNATURE

pszBuf - Output buffer for UTF-8 data. This buffer is null terminated. If reuseCurrentPoint is non- zero, this must be a pointer to memory allocated by vsAlloc().

BufLen - Number of bytes read into pszBuf. This length does not including the null terminating character.

reuseCurrentPointer - When non-zero, pszBuf must already point to memory allocated by vsAlloc. If pszBuf is not big enough, its size will be increased.

pXlatError - Set to 1 if a translation error occurs.

Returns: int

Returns 0 if successful. Otherwise, a negative return code.
