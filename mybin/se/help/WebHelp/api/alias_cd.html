_command int alias_cd(_str msg='')

Changes the current directory to the alias or directory given. If an alias is given, it must evaluate to a directory. If no parameter is given, you will be prompted for a parameter. T he current directory in the build window is set as well if the build window is started.

Returns: int

0 if successful

Categories:

File Functions, Miscellaneous Functions

See Also:

alias, expand_alias, gui_cd, cd, cdd
