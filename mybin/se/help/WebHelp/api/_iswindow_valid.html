boolean _iswindow_valid(int wid)

Returns: boolean

Returns true if window_id is a valid window handle. This function is used to loop through all windows and to check if a window was deleted.

Example:

_find_formobj(form_name /*,'E' - Find edited forms only | 'N'-Find non-
edited forms only*/)
{
   last=_last_window_id();
   option=upcase(arg(2));
   for (i=1;i<=last;++i) {
      if (_iswindow_valid(i) && i.p_object==OI_FORM &&
           (option=='' || (option=='E' && i.p_edit) ||
           (option=='N' && !i.p_edit)
           )
         ) {
         if (name_eq(i.p_name,form_name)) {
            return(i);
         }
      }
   }
   return(0);
}
