int refactor_c_get_super_classes(int handle, _str symbolName, _str filename, int beginSeekPos, int endSeekPos, _str &superClasses)

Get a list of the super classes that are associated with a given symbol's class

Parameters:

handle - Handle to transaction

symbolName - Name of function

filename - Name of file containing function

beginSeekPos - Seek position of beginning of function reference

endSeekPos - Seek position of end of function reference

superClasses - String containing all super classes deliminated with @ and starts with number of classes. superClasses is in the form: number of classes [@] super_class_name [@] parent_class_name [@] super_class_name [@] parent_class_name [@] ...

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
