_str min_abbrev(_str word, _str word_list, typeless options)

Deprecated. 

This function is present for backward compatibility. New macros should use the min_abbrev2 function. Performs list completion (like '?' key) on partial argument word with valid sorted argument list word_list. The third word of the options argument must be the minimum abbreviation length for list completion to take place. This procedure is typically used to allow syntax expansion to occur when space bar is pressed after an incomplete keyword. For example, typing "m<Space>" while in C mode inserts a template for main(){}.

Parameters:

word - current word under cursor

word_list - space delimited list of words handled by syntax expansion

options - syntax expansion options = name_info(p_index)

Returns: _str

Returns completed word if successful. Otherwise '' is returned.
