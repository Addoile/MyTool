void _set_var(int index, typeless value)

Sets the contents of the variable corresponding to the name table index to value.

Parameters:

index - index into the names table returned by one of the functions find_index or name_match.

value - value to set variable to.

Example:

// VAR_TYPE is defined in "slick.sh"
index=find_index("rc",VAR_TYPE)
_set_var(index,"anything");
message('rc='_get_var(index));

Categories:

Macro Programming Functions, Miscellaneous Functions
