_command int show_matching_paren(_str select_block='', boolean search_backward=true)

Searches for the closest open parenthesis [, {, or ( that is at or before the cursor location and then temporarily places the cursor on the matching end parenthesis for a short period of time.

Applies To:

Edit Window, Editor Control

Category:

Miscellaneous Functions

See Also:

find_matching_paren, keyin_match_paren
