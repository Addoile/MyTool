int GetSystemMetrics(int sm)

Returns value for system-metric sm.

Parameters:

sm - may be one of VSM_* constants defined in "slick.sh":

The following constants are supported by all platforms 

   VSM_TOOLBAR_HANDLE_EXTENT

All other constants are only supported by Windows

Returns: int

System-metric value, 0 if system-metric unknown.

Category:

Miscellaneous Functions
