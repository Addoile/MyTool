void VSAPI vsFillSelection(int wid, int markid=-, unsigned ch=' ')

Fills the selection specified with a character.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

mark_id - Selection id to duplicated. If -1 is specified the active selection.

ch - Character to fill selection with.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
