int _xmlcfg_get_last_child(int iHandle, int NodeIndex, int NodeTypeFlags=~VSXMLCFG_NODE_ATTRIBUTE)

Gets the last child index from the specified index.

Parameters:

iHandle - Handle of tree to get index from

NodeIndex - Index of node to get child node index for

NodeTypeFlags - Indicates node type of child to find. 1 or more VSXMLCFG_NODE_* flags. See _xmlcfg_add() for more information on this parameter.

Returns: int

If successful, returns index of last child. Otherwise a negative error is returned.

Category:

XMLCFG Functions
