void _free_selection(_str markid)

Frees the selection handle or bookmark handle corresponding to mark_id. _free_selection will not free the active selection (the one that is seen on screen). Use _show_selection if necessary to activate another selection before freeing the selection. mark_id is a handle to a selection returned by one of the built-ins _alloc_selection or _duplicate_selection. A mark_id of '' or no mark_id parameter identifies the active selection.

Example:

         // Get handle to active selection
         current_mark_id=_duplicate_selection('');
         // Allocate another mark.
         mark_id=_alloc_selection();
         _select_line(mark_id);
         _show_selection(mark_id);get_event();
         _show_selection(current_mark_id);
         _free_selection(mark_id);
