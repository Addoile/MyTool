int tag_register_type(int type_id, _str pszTypeName, int is_container=0, _str description=null, int filterFlags=VS_TAGFILTER_ANYTHING)

Register a new OEM-defined type.

Parameters:

type_id - Tag type ID, in range VS_TAGTYPE_OEM <= type_id <= VS_TAGTYPE_MAXIMUM

pszTypeName - Tag type name

is_container - 1=tag type is a container (i.e. can have members)

description - (optional) description of the new tag type

filterFlags - (optional) VS_TAGFILTER_

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
