VSPSZ VSAPI vsBufGetModeName(int buf_id, char *pszModeName, int MaxModeName, int *pMaxModeNameLen=a)

Parameters:

buf_id - Buffer id returned by vsPropGetI(wid,VSP_BUF_ID), vsBufEdit, or vsBufMatch.

pszModeName - Name of language mode corresponding to buf_id specified. This can be NULL, in which case, only the length of the language mode name string will be set.

MaxModeName - Number of characters allocated to pszModeName. We recommend at least VSMAXMODENAME

pMaxModeNameLen - If this is not 0, this is set to the number of characters needed to be allocated.

Returns: VSPSZ

Returns language mode name corresponding to buffer id specified. Returns pszModeName.

Attention:

This function is thread-safe.

Category:

Buffer Functions
