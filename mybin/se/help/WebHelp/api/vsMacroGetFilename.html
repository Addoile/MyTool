char*VSAPI vsMacroGetFilename(char *pszFilename, int MaxFilenameLen, int *pMaxFilenameLen=a)

Parameters:

pszFilename - Ouput buffer for filename. This can be 0.

MaxFilenameLen - Number of characters allocated to pszFilename. We recommend VSMAXFILENAME.

pMaxFilenameLen - If this is not 0, this is set to the number of characters you need to allocate.

Returns: char*VSAPI

Returns absolute output filename of the last recorded macro. This is "" if there is no last recorded macro.
