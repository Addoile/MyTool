int _xmlcfg_find_child_with_name(int iHandle, int NodeIndex, _str name, int NodeTypeFlags=VSXMLCFG_NODE_ELEMENT_START | VSXMLCFG_NODE_ELEMENT_START_END)

Finds a child index under NodeIndex with the name name

Parameters:

iHandle - Handle to an XML tree

NodeIndex - Index to a node to search under

name - Name of node to find. This may be null.

NodeTypeFlags - Indicates node type of child to find. 1 or more VSXMLCFG_NODE_* flags. See _xmlcfg_add() for more information on this parameter.

Returns: int

index >=0 if successful. Otherwise, a negative error code.

Category:

XMLCFG Functions
