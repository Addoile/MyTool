_str _flfilename(_str wildcards=null, _str dirpath='', boolean forceRefresh=false, boolean showDotFiles=true)

When arguments are given to this function, the file list of a file list box will display the files from directory path and match any of the wildcards specified. wildcards is a string of semicolon delimited wildcards (Ex. "*.c;*.h"). In addition, this function sends an on_change event to the current control with CHANGE_FILENAME as the first argument if the arguments given are different.

Returns the current wildcards and path used to fill in a file list box. The return value is a string in the following format: 

      path;wildcards

Parameters:

wildcards - wildcards

dirpath - directory path

forceRefresh -

showDotFiles - if true, the file list shows UNIX-style dot files.

Example:

// This examples requires a form with a file list box named list1 and a
button named
// command1
defeventtab form1;
command1.lbutton_up()
{
   // If you use double quotes, you need two \\
   list1._flfilename('*.e;*.h','c:\vslick');
   parse list1._flfilenames with path ';' wildcards
   messageNwait('path='path'  wildcards='wildcards);
}

Applies To:

File List Box

Category:
