const char*VSAPI vsFileAbsoluteTo(char *pszDestFilename, const char *pszSourceFilename, const char *pszToDir=a)

Converts pszSrcFilename to a fully qualified path.

Parameters:

pszDestFilename - Result of convert pszSrcFilename to a fully qualified path.

pszSrcFilename - Filename with path specification which is relative to the directory specified by pszToDir. This may already be fully qualified.

pszToDir - Directory with or without trailing backslash (UNIX: forward slash). If 0 or "" is given, the current directory is used.

Returns: const char*VSAPI

Returns pszDestFilename.

Category:

File Functions

See Also:

vsFileAbsolute, vsFileAbsoluteTo, vsFileRelative
