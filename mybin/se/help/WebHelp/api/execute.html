void execute(_str command=null, _str options="")

Runs the command currently on the command line or the command specified. Command may be an internal command (_command), an external operating system command, or an external Slick-C® batch macro (with defmain entry point). If the command argument is null, the command line is erased and the command is inserted into the command retrieve buffer ".command". See retrieve_prev, retrieve_next.

Parameters:

command -

options - a string of one or more of the following letters:

W
If command is an external program, run slkwait program to wait for program to complete. This option allows you to view the results after running a DOS text mode application which displays results to the screen. Defaults to OFF.
A
If command is an external program, run program asynchronously (no wait). Defaults to OFF.
R
Insert command into command retrieve buffer (.command) and clear command line. Defaults to OFF unless no arguments given.

Category:

Miscellaneous Functions
