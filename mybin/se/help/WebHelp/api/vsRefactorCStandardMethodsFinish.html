int VSAPI vsRefactorCStandardMethodsFinish(int handle, const char *filename, const char *symbolName, seSeekPos startSeekPos, seSeekPos endSeekPos, int methodsToCreate, int formattingFlags, int syntaxIndent)

Finish the create standard methods

Parameters:

handle - Handle to transaction

filename - Filename to do refactoring within

className - Name of class to add standard methods to

startSeekPos - Start seek position of symbol

endSeekPos - End seek position of symbol

methodsToCreate - Which methods to create, bitset of VSREFACTOR_METHOD_* (above)

formattingFlags - Optional flags, bitset of VSREFACTOR_FORMAT_* (above)

syntaxIndent - Syntax indent amount

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
