int def_background_tagging_idle

Specifies the amount of time in milliseconds to wait before starting to gather background tagging results.

This setting will also be used to calculate a minimum amount of idle time to wait before processing background tagging jobs that require foreground tagging work, such as files containing embedded code or files that use a proc-search for finding symbols. The idle time delay in microseconds for slower tagging jobs is calculated by multiplying this setting by 50 and adding 1 second.

Default:

100 ms

Category:

Configuration Variables
