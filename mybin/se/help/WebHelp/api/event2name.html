_str event2name(_str event, _str option="S")

Parameters:

option - defaults to 'S' and may be one of the following:

'S'
Return Slick-C® source code event name.
'L'
Return long menu bar event name.
'C'
Return condensed menu bar event name.

Returns: _str

Returns the event name corresponding to the binary event string. Event may have been returned by get_event(), test_event(), and last_event() into a event name.

Example:

         key=get_event()
          message 'The key pressed was 'event2name(key)

Category:

Keyboard Functions
