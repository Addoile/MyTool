int p_template

Returns: int

Returns the index in the names table which was used to load or update the form. This property is set by the _load_template and _update_template functions. The purpose of this property is to allow you to determine the name of the form (name_name(p_template)) which should be opened for editing when the user selects to edit the current dialog box (Shift+Space). This property is available only when the form is displayed.

Applies To:

Form

Category:

Form Properties
