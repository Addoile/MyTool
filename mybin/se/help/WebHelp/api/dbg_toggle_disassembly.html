int dbg_toggle_disassembly(_str file_name, int on_off)

Toggle whether the disassembly lines for the given file name should be displayed or not.

Parameters:

file_name - source file to remove disassembly for

on_off - new setting for displaying disassembly use -1 to query value

Returns: int

'true' if the disassembly is to be displayed, false if not, <0 on error.

Category:

Debugger Functions
