_command void tabs(_str cmdline="")

Sets up to tab stops in increasing order. The commands ctab and cbacktab may be used to move between tab stops. The '+' symbol is used to set tabs according to an increment value. Tab stops continue past the last tab stop by repeating the difference of the last two tab stop values.

Command line examples:

tabs 1 7 13 16
Creates tab stops 1 7 13 16 19 21 24 ...
tabs +3
Creates tab stops 1 4 7 ...
tabs 1 4
Creates tab stops 1 4 7 ...
tabs 1 49 +3
Creates tab stops 1 4 7 ...
tabs 1 49 +3 53 250 +4
Creates tab stops 1 4 7 ... 49 53 57 61 65 69 ...
tabs 10 +3
Creates tab stops 10 13 16 ...

Parameters:

cmdline - is a string in the format: [+t1]: [+t2]: [+t3] ...

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

ctab, cbacktab, move_text_tab, gui_tabs
