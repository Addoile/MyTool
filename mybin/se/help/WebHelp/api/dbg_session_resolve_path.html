int dbg_session_resolve_path(int session_id, _str file_name, _str &full_path)

Resolve the path to the given source file using debugger-specific lookup methods.

Parameters:

session_id - debugger session ID (1 .. max_session)

file_name - source file name (with extension)

full_path - (reference) fully resolved, absolute path name

Returns: int

0 on success, <0 on error

Since:

10.0

Category:

Debugger Functions
