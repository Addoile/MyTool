void _update_buffers_for_ext(_str ext, _str lang="")

Update the language mode for buffers with the given file extension. This function is used for updating buffers after changes have been made to the file extension mappings in the File Extension Manager.

Parameters:

ext - Physical file extension

lang - Language ID (see p_LangId

Category:

Buffer Functions

See Also:

_CreateExtension, _update_buffers
