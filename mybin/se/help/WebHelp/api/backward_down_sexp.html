_command int backward_down_sexp()

Moves backward and down into the preceeding S-expression starting from the cursor position. If the cursor is not on a close paren, bracket, or brace, it will behave just like prev_sexp.

Returns: int

Returns 0 if successful, <0 on error.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods

See Also:

next_sexp, prev_sexp, backward_up_sexp, forward_up_sexp, forward_down_sexp
