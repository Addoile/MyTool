void _StackDump(int dumpToFile=0, int dumpToScreen=1, int ignoreNStackItems=0)

Dump the current interpreter stack to the debug window, or the log file.

Parameters:

dumpToFile - dump the stack the log file

dumpToScreen - dump the stack to the debug window

ignoreNStackItems - ignore 'n' items on top of stack

Category:

Miscellaneous Functions
