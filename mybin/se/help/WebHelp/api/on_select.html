event on_select()

The on_select event is sent before a key press event when the current buffer has a selection. User level 1 and user level 2 inheritance can not catch this event. Only the default_keys event table can process this event. This event will not occur if the user level 1 or user level 2 inheritance has an event handler for the key.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Events, Editor Control Events
