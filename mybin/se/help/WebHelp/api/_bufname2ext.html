_str _bufname2ext(_str buf_name)

Deprecated. Use _Filename2LangId()

Returns the canonical file extension for the given file.

If the file's actual extension extension matches an _[ext]_Filename2LangId() callback, it will first try the callback to see if the file, based on it's path or name should be referred to an alternate extension.

Otherwise, if the file's actual extension matches a suffix_[ext] callback, it will open the file in a temporary view and call _SetEditorLanguage to determine the file's actual language type.

Parameters:

buf_name - file name and path to test

Returns: _str

Returns referred extension for buffer name, using _Filename2LangId.

Miscellaneous Functions
