typeless _showbuf(int buf_or_view_id, boolean isBuffer=true, _str show_options='-new -modal', _str dialog_caption='', _str save_button='', boolean read_only=false)

Displays the buffer id or the buffer attatched to the window specified modally in a dialog box. This function is intended for debugging macros. The default show options used to display the dialog box are "-new -modal". Specify the show_options arguments to override the default show options. See show function information on show options.

Parameters:

buf_or_window_id - This paramater must be a valid buffer or view id

isBuffer - When true,buf_or_window_id is a buffer id.

show_options - Arguments that are given to the show command before the form name. Default is -new -modal. Specify '' for a modeless dialog

dialog_caption - Caption of hte dialog box

save_button - Specify 'S' to put a save button on this dialog

read_only - if true, buffer is treated as read only, and if there is a save button, it will do a "save as..."

Returns: typeless

the window id of the dialog. Returns 0 if dialog is modal

Category:

Miscellaneous Functions
