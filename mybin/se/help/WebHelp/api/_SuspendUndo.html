int _SuspendUndo(int wid)

Temporarily suspends undo

This function is used in conjuction with vsResumeUndo in order to perform temporary buffer modifications so that undo steps are not recorded for the temporary buffer modifications. As long as the buffer is restored to it's original state before vsSuspendUndo is called, no issues with undo will occur.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

Returns: int

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
