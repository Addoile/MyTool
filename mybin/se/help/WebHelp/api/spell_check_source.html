_command void spell_check_source(...)

Starts spell checking from the cursor position comments and strings in the current buffer. The p_lexer_name property must be set to a section name defined in "vslick.vlx" or user defined ".vlx" file.

Applies To:

Editor Control, Edit Window

Categories:

Edit Window Methods, Editor Control Methods

See Also:

spell_check, spell_check_word, spell_check_files
