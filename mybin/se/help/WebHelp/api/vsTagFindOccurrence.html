int VSAPI vsTagFindOccurrence(const char *tag_name, int exact_match=a, int case_sensitive=a, int find_first=a)

Find the first/next occurrence with the given tag name or tag prefix. Use tag_get_occurrence (below) to get details about the occurrence.

Parameters:

tag_name - Tag name or prefix to search for

exact_match - Exact (word) match or prefix match (0)

case_sensitive - Case sensitive search?

find_first - find first match or next match?

Returns: int

0 on success, BT_RECORD_NOT_FOUND_RC if not found, <0 on error.

Category:

Tagging Functions
