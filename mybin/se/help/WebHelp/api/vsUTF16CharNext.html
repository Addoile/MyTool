unsigned short*VSAPI vsUTF16CharNext(const unsigned short *pCurrent, const unsigned short *pEndBuf)

Parameters:

pCurrent - Buffer contain UTF-16 text.

pEndBuf - Pointer to end of pBuf text where pEndBuf-pCurrent is the buffer length.

Returns: unsigned short*VSAPI

Returns pointer to the beginning of the next composite or surrogate character or pEndBuf if pCurrent==pEndBuf.

Category:

Unicode Functions
