_str _ntGetRegistryValue(int rootKey, _str prefixPath, _str suffixPath, _str valuename, boolean versionSpecific)

Deprecated. Use _ntRegGetLatestVersionValue() for versionSpecific data, otherwise use _ntRegQueryValue()

Retrieves a data value underneath the latest version subkey under the given key.

Parameters:

rootKey - one of the HKEY_* constants

prefixPath - subkey path before the version

suffixPath - subkey path to be appended after the version

valuename - name of data value

versionSpecific - whether to search for the latest version or to just append the suffix directly to the prefix

Returns: _str

_str
