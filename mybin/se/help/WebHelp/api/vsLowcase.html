char*VSAPI vsLowcase(char *pString, int StringLen=-1, int utf8=-1)

Converts string to lower case.

Parameters:

pString - Input string.

StringLen - Number of bytes in pString. Specify -1 to determine length of string if string is null terminated.

utf8 - -1 specifies that input and output string are UTF-8 if UTF-8 support is enabled. 0 specifies that the input and output string are SBCS/DBCS. 1 specifies that the input and output string are UTF-8.

Categories:

String Functions, Unicode Functions
