char*VSAPI vsUTF8CharNext2(const char *pBuf, const char *pEndBuf)

Parameters:

pCurrent - Buffer contain UTF-8 text.

pEndBuf - Pointer to end of pBuf text where pEndBuf-pCurrent is the buffer length.

Returns: char*VSAPI

Returns pointer to the beginning of the next UTF-8 character or pEndBuf if pCurrent==pEndBuf. Use vsUTF8CharNext, to skip composite characters.

Category:

Unicode Functions
