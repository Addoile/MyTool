typeless _retrieve_value(_str cmdline="")

Sets text box p_text property to value saved by the _append_retrieve function. When no argument is given, the history is retrieved based on the active form name and active control name (p_name). If the cmdline argument does not contain a '.', the history is based on the active form name and cmdline specifies an alternate control name.

Parameters:

cmdline - is a string in the format: form_name[.ctl_name]

Returns: typeless

Returns the value placed into the p_text property.

Example:

defeventtab form1;
combo1.on_destroy()
{
    // When OK button is pressed, you will want to save combo box
retrieve information.
    _append_retrieve(_control combo1,combo1.p_text);
}
combo1.on_create()
{
     // Fill in the combo box list
     _retrieve_value();
}
