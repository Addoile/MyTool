int p_style

Determines the style of an object.

Check Box - Style may be one of the following constants defined in "slick.sh":

PSCH_AUTO2STATE
Two state check box.
PSCH_AUTO3STATEA
Three state check box. Gray, check, unchecked.
PSCH_AUTO3STATEB
Three state check box. Gray, unchecked, check.

Combo Box - Style may be one of the following constants defined in "slick.sh":

PSCBO_EDIT
Standard. List drops down.
PSCBO_LIST_ALWAYS
List is always present. Can edit.
PSCBO_NOEDIT
Must select item from list box. Can't edit.

Gauge - Style may be one of the following constants defined in "slick.sh":

PSGA_HORZ_WITH_PERCENT
Fill gauge from left to right and show percentage of completion.
PSGA_VERT_WITH_PERCENT
Fill gauge from bottom to top and show percentage of completion.
PSGA_HORIZONTAL
Fill gauge from left to right.
PSGA_VERTICAL
Fill gauge from bottom to top.

Image - Style may be one of the following constants defined in "slick.sh":

PSPIC_DEFAULT
No default handling for lbutton_down event. Uses p_border_style.
PSPIC_FLAT_BUTTON
Tool-button style. Set the p_picture property to a 1 state picture without a border to get a 3 state toolbar button (0-normal, 1-pushed, 2-mouse over button). This style supports the p_enabled property on all platforms. lbutton_down event calls lbutton_up event or executes command in p_command property. p_value property is restored to its original value.
PSPIC_FLAT_MONO_BUTTON
Tool-button style. Set the p_picture property to a 1 state picture without a border to get a 3 state toolbar button (0-normal, 1-pushed, 2-mouse over button). The picture should be a 1-bit mask and is used, along with p_forecolor, to fill the mask with the foreground color. This style supports the p_enabled property on all platforms. lbutton_down event calls lbutton_up event or executes command in p_command property. p_value property is restored to its original value.
PSPIC_HIGHLIGHTED_BUTTON
Tool-button style. Set the p_picture property to a 1 state picture without a border to get a 3 state toolbar button (0-normal, 1-pushed, 2-mouse over button). This style supports the p_enabled property on all platforms. lbutton_down event calls lbutton_up event or executes command in p_command property. p_value property is restored to its original value.
PSPIC_BUTTON
Tool-button style. Set the p_picture property to a 1 state picture without a border to get a 2 state push button. This style supports the p_enabled property on all platforms. If you set the p_picture or p_caption property, the control will display and act like a tool-button. lbutton_down event calls lbutton_up event or executes command in p_command property. p_value property is restored to its original value.
PSPIC_AUTO_BUTTON
Set the p_picture property to a 1 state picture without a border to get a 2 state push button. lbutton_down event calls lbutton_up event or executes command in p_command property. p_value property is restored to its original value.
PSPIC_AUTO_CHECK
Picture is a 2 or more state picture. lbutton_down event calls lbutton_up after p_value property has be incremented (or set to 0 if past last state).
PSPIC_SIZEVERT
Draws a thick vertical line which is grabbed when sizing a docked toolbar.
PSPIC_SIZEHORZ
Draws a thick horizontal line which is grabbed when sizing a docked toolbar.
PSPIC_GRABBARVERT
Draws two thick vertical lines which are used to grab and dock/undock a toolbar.
PSPIC_GRABBARHORZ
Draws two thick horizontal lines which are used to grab and dock/undock a toolbar.
PSPIC_TOOLBAR_DIVIDER_VERT
Draws a vertical toolbar divider line.
PSPIC_TOOLBAR_DIVIDER_HORZ
Draws a horizontal toolbar divider line.

Picture Box - Style may be one of the following constants defined in "slick.sh":

PSPIC_DEFAULT
No default handling for lbutton_down event.
PSPIC_PUSH_BUTTON
Push-button style. Picture is a 1 state picture which needs button up and down borders drawn around it. The lbutton_down event calls the lbutton_up event or executes the command in the p_command property. The p_value property is restored to its original value.
PSPIC_SPLIT_PUSH_BUTTON
Push-button style with drop-down indicator arrow on right. Picture is a 1 state picture which needs button up and down borders drawn around it. The lbutton_down event calls the lbutton_up event or executes the command in the p_command property. The p_value property is restored to its original value. A reason of CHANGE_SPLIT_BUTTON is passed to lbutton_down/lbutton_up event if user clicks on the drop-down indicator part of the button.
PSPIC_AUTO_BUTTON
Set the p_picture property to a 2 or more state picture and set the p_Nofstates property to indicate the number of picture states. lbutton_down event calls lbutton_up event or executes command in p_command property. p_value property is restored to its original value.
PSPIC_AUTO_CHECK
Picture is a 2 or more state picture. lbutton_down event calls lbutton_up after p_value property has be incremented (or set to 0 if past last state).
PSPIC_FILL_GRADIENT_HORIZONTAL
Gradient fill left to right with p_backcolor to p_forecolor.
PSPIC_FILL_GRADIENT_VERTICAL
Gradient fill top to bottom with p_backcolor to p_forecolor.
PSPIC_FILL_GRADIENT_DIAGONAL
Gradient fill top-left to bottom-right with p_backcolor to p_forecolor.

SSTab - Style may be one of the following constants defined in "slick.sh":

PSSSTAB_DEFAULT
Default style.
PSSSTAB_DOCUMENT_TABS
Mac only. Document tabs style. Used by File Tabs tool-window.

Applies To:

Check Box, Combo Box, Gauge, Picture Box, Image, SSTab
