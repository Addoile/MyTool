int tag_ext_inherits_from(_str parent, _str ext=null)

Deprecated. use _LanguageInheritsFrom()

Does the current extension match or inherit from the given extension?

If 'ext' is not specified, the current object must be an editor control.

Parameters:

parent - extension to compare to

ext - current language (default=p_LangId)

Returns: int

'true' if the extension matches, 'false' otherwise.

Category:

Tagging Functions
