void _next_buffer(_str options="")

Switches to the next buffer.

Parameters:

options - is a string of zero or more of the following:

'H'
Allow switching to the hidden window.
'N'
Don't update the active buffer's non-active cursor position information.
'R'
Don't do any screen updating.

The active cursor position information is saved in the active buffer's non-active cursor position information unless the 'N' option is specified. Each buffer maintains ONE non-active cursor position which is updated when you switch to a buffer within the same window using one of the built-ins _next_buffer, _prev_buffer, _delete_buffer, load_files, _begin_select, and _end_select. When the non-active buffer becomes active, the non-active cursor position information is copied into the windows cursor position. Switching windows DOES NOT update the buffers non-active cursor position information.

The 'H' option determines whether buffers with (p_buf_flags&VSBUFFLAG_HIDDEN) true may be selected.

The 'R' options specifies that the screen should not be refreshed.

Applies To:

Edit Window, Editor Control

Categories:

Buffer Functions, Edit Window Methods, Editor Control Methods

See Also:

_prev_buffer, _delete_buffer
