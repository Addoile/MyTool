_str _Ext2LangId(_str ext)

Find the language type that the given file extension is mapped to. This will first check if there is a language-to-extension mapping for 'ext', then it will check for language setup information 'ext'. Failing that, it will attempt to do a case-insensitive search for a language mapping, if the default operating system file system is case-insensitive.

Parameters:

ext - file name extension

Returns: _str

Returns the language ID for the language mode associated with the given file extension. Returns '' if no match is found.

Category:

Miscellaneous Functions
