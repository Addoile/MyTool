void VSAPI vsConcurProcessInputDirect(VSPSZ pszBuf)

Bypasses the editor's process buffer and inputs directly into running process from pszBuf. This input occurs out-of-band, which means that any input data waiting to be sent from the editor's process buffer will not be sent before the data in pszBuf.

Currently only supported on UNIX.

Parameters:

pszBuf - Char buffer to input direct to process

Category:

Miscellaneous Functions
