VSSTATIC int vsUTF8StrICmpQuick(const char *pBuf1, const char *pBuf2, size_t BufLen1=SESIZE_MAX, size_t BufLen2=SESIZE_MAX)

Case insensitive compare. Characters are compared by character value and not using any Unicode standard.

Parameters:

pBuf1 - Pointer to UTF-8 buffer 1

pBuf2 - Pointer to UTF-8 buffer 2

BufLen1 - Number of bytes in buffer 1. Use -1 if pBuf1 is a NULL terminated string.

BufLen2 - Number of bytes in buffer 2. Use -1 if pBuf2 is a NULL terminated string.

Returns: VSSTATIC int

0     Strings are equal
>0    pBuf1>pBuf2
<0    pBuf1

Category:

Unicode Functions
