int _project_command(_str field_name, _str buf_name, _str word, boolean AccumulateErrors=false, boolean AutoNextError=true, int ActionOverrideFlags=0, boolean buildFirstDone=false, _str debugStepType='go')

Executes project command.

Utility function called by project_compile() and _project_command2()

Parameters:

field_name - Target name (e.g. 'Compile', 'Execute', 'Debug').

buf_name - Parts of this buffer name may be used to fill in the command.

word - Parts of this word may be used to fill in the command.

AccumulateErrors - If redirecting errors, appends to the file rather than overwriting if not redirecting errors, no effect

AutoNextError - If redirecting errors, automatically to to first error if not redirecting errors, no effect

ActionOverrideFlags - One or more of bitwise flags OVERRIDE_*.

buildFirstDone - Set to true when build has already been done (e.g. a Build before a Debug).

debugStepType - Applies to debug commands (e.g. 'debug'). Valid values are 'go'=run to next breakpoint; 'into'=step into debugee; 'reload'=restart debugging (buf_name becomes significant).

Returns: int

0 on success, <0 on error.

Category:

Project Functions
