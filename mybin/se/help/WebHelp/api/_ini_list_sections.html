int _ini_list_sections(_str filename)

Returns: int

Returns 0 if successful. Creates a buffer and view in list box format of all section names in square brackets in filename. 

     ".ini" File Syntax:
         [section1]
         field_name1=value1
         field_name2=value2
         ...
         [section2]
         field_name1=value1
         field_name2=value2
         ...
