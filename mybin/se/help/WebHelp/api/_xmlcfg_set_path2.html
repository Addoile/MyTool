int _xmlcfg_set_path2(int handle, _str QueryStr, _str element, _str AttrName, _str AttrValue)

another element with the attribute specified below QueryStr.

Parameters:

handle - Handle to an XMLCFG tree returned by _xmlcfg_open(), _xmlcfg_create(), or _xmlcfg_open_from_buffer().

QueryStr - This is a small subset of an XPath expression. See _xmlcfg_find_simple() for information on this parameter.

element - Name of tag to add.

AttrName - Name of attribute to set value for ElementName

AttrValue - Value for attribute.

Returns: int

Returns index of element node if successful. Otherwise a negative error code is returned.

Example:

// Add another [Set] element under /Workspace/Environment
// and set the value.
Node=_xmlcfg_set_path2(handle,
    "/Workspace/Environment","Set",
    "Name",
    "VERSION"
    );
_xmlcfg_set_attribute(handle,Node,"Value","8.0");
