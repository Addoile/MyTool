_command typeless dload(_str filename='')

Loads a DLL which uses our DLL Interface functions. The entry point function of the DLL registers DLL functions that can be bound to keys and called from Slick-C® macros. If DllFilename is not given, you are prompted for one. See simple.c for an example of a DLL that can be loaded.

This function is not yet available in the UNIX version.

Returns: typeless

Returns 0 if successful.

Category:

Macro Programming Functions

See Also:

dunload, _dllload
