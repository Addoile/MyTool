int p_tile_id

Determines the tile id of the window. This property is used to keep track of tile groups. Windows with the same tile id don't overlap (unless a macro did something wrong). In addition, when the editor sets the focus to an edit window, all windows with the same tile id are moved in front in the Z order. This property is only available when the object is displayed.

Applies To:

Edit Window

Category:

Edit Window Properties
