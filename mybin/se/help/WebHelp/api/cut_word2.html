int cut_word2(boolean push, boolean doCopy=false, int subwordOption=0, boolean skipTrailing=true)

If create_clipboard is true, then a new clipboard is created containing the text from the cursor to the end of the current word.

If create_clipboard is false, then the text from the cursor to the end of the current word is appended to the current clipboard.

Regardless of the value of the create_clipboard parameter, if the second argument is not 'C', the text that is copied is deleted from the buffer.

Parameters:

push -

doCopy -

subwordOption -

skipTrailing -

Returns: int

Returns 0 if successful. Common return codes are 1 (no line at cursor), TOO_MANY_SELECTIONS_RC, and STRING_NOT_FOUND_RC.

Applies To:

Edit Window, Editor Control

Categories:

Clipboard Functions, Edit Window Methods, Editor Control Methods
