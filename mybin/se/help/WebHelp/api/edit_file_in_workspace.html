_command typeless edit_file_in_workspace(_str filenames='')

Command line version of the project_load command. Edits the file(s) specified. Only the file names need to be supplied, not their entire paths.

If you need to edit a file whose name contains space characters, place double quotes around the name.

This command accepts most of the same options as the edit() command.

Returns: typeless

Returns 0 if successful.

Applies To:

Edit Window

Category:

File Functions

See Also:

edit, project_load
