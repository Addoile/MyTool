VSPSZ VSAPI vsHvarFieldName(VSHVAR hvar, int index)

Parameters:

hvar - handle for interpreter variable

index - index >=0 of non-static data member

Returns: VSPSZ

Return the name of the given non-static member of a class or struct instance. Returns INVALID_ARGUMENT_RC if the field index is out of range. Returns INVALID_HVAR_ARGUMENT_RC if 'hvar' is not a struct or class instance.

Category:

Miscellaneous Functions

See Also:

vsHvarFieldIndex, vsHvarGetField
