int _spell_check_area(_str &word, _str &replace_word, int &col, int start_col, int width=-1)

Spell checks a limited area of the current line. start_col is an imaginary column position and specifies the column to start spell checking from. width specifies the number of characters in the area to spell check. This function is intended to used to assist in spell checking a selection.

Parameters:

word -

replace_word -

col -

start_col -

width -

Returns: int

Return values are as follows:

SPELL_NO_MORE_WORDS_RC
End of buffer reached.
SPELL_REPEATED_WORD_RC
Found two of the same words found in a row.
SPELL_WORD_NOT_FOUND_RC
Possible misspelled word. word variable set to the misspelled word.
SPELL_REPLACE_WORD_RC
History word found. Replace word at cursor with replace_word.
SPELL_ERROR_READING_MAIN_DICT_RC

Applies To:

Editor Control, Edit Window

Categories:

Edit Window Methods, Editor Control Methods
