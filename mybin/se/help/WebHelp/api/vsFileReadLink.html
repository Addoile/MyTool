const char*VSAPI vsFileReadLink(const char *pszSourceFilename)

Resolve link for pszSourceFilename. UNIX only. If filename does not exist, then "" is returned. If filename is not a link, then original filename is returned. If filename is not an absolute path, then it is assumed relative to current directory.

Parameters:

pszSourceFilename - Filename to resolve link for

Returns: const char*VSAPI

Resolved link filename.

Category:

File Functions
