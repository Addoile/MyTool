void _xmlcfg_copy_children_as_siblings(int dest_handle, int dest_index, int src_handle, int src_index, boolean copyBefore=false)

Copies the subtree underneath an XML node to another location, where they are siblings of a destination node.

Parameters:

dest_handle - handle to destination tree

dest_index - index of destination node

src_handle - handle to source tree

src_index - index of source node (parent of subtree to copy)

copyBefore - true to copy the subtree before the dest_index, false to copy them after

Returns: void

Category:

XMLCFG Functions
