int _setext(_str extension, _str field_name, typeless value)

Deprecated. Use _SetLanguageOption()

Sets default extension specific options. Use the _update_buffers function to change extension specific options for buffers already loaded.

Parameters:

field_name - may be one of the following:

p_mode_name or MN
Value specifies mode name.
p_tabs or TABS
Value specifies tab settings.
p_margins or MA
Value specifies margins.
p_mode_eventtab or KEYTAB
Value specifies mode event table.
p_word_wrap_style or WW
Value specifies word wrap style.
p_indent_with_tabs or IWT
Value specifies indent with tabs.
p_indent_style or IN
Value specifies indent style.
p_word_chars or WC
Value specifies word chaacters.
p_lexer_name or LN
Value specifies lexer name.
p_color_flags or CF
Value specifies color flags.

For the above options, value may not contain a comma character.

Additional field_name values:

BEGINEND
Value specifies begin/end pairs.
OPTIONS
Value specifies extension specific options. We recommend that you use macro recording to correctly generate calls that set this option.
SETUP
Value specifies miscellaneous extension specific options. This option is very sensitive to the order of the options. We recommend that you use macro recording to correctly generate calls that set this option.
ALIAS
Value specifies alias filename.
SMARTTAB
Set the smart tab value for this extension, one of the following:
  • VSSMARTTAB_INDENT
  • VSSMARTTAB_MAYBE_REINDENT_STRICT
  • VSSMARTTAB_MAYBE_REINDENT
  • VSSMARTTAB_ALWAYS_REINDENT

Example:

#include "slick.sh"
defmain()
{
   // Can specify "" instead of "fundamental" to refer to buffers with no extension
   // Change the default margins for fundamental mode.
    _setext("fundamental","p_margins","1 70 1");
   // Update buffers already being edited
    _update_buffers("fundamental","p_margins=1 70 1");
}
