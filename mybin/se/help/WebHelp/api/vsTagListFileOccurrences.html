int VSAPI vsTagListFileOccurrences(int tree_wid, int tree_index, const char *tag_name, int exact_match, int case_sensitive, int &num_refs, int max_refs, VSPSZ restrictToLangId=NULL)

List the files containing 'tag_name', matching by prefix or case-sensitive, as specified. Terminates search if a total of max_refs are hit. Items are inserted into the tree, with the user data set to the file path.

Parameters:

tree_wid - window id of tree control to insert into

tree_index - index, usually TREE_ROOT_INDEX to insert under

tag_name - name of tag to search for

exact_match - exact match or prefix match

case_sensitive - case sensitive match, or case-insensitive

num_refs - (reference) number of references found so far

max_refs - maximum number of items to insert

restrictToLangId - restrict references to files of the given language, or related language.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
