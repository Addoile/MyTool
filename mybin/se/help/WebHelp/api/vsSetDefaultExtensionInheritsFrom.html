void VSAPI vsSetDefaultExtensionInheritsFrom(const char *pszExtension, const char *pszInheritFrom)

Deprecated. Use vsSetLanguageInheritsFrom.

Allows you to create a language which has it's own set of options, but can inherit support callbacks from another language.

Parameters:

pszExtension - File extension. For list of file extension types, use our File Extension Manager dialog ("Tools", "Options...", "File Extension Manager")

pszInheritsFrom - pszExtension is set to inherit language specific callbacks from the language specified by pszInheritFrom.

Applies To:

Edit Window, Editor Control

Category:

Tagging Functions

See Also:

vsSetDefaultLanguageOptions, vsSetDefaultExtensionReferTo, vsDeleteExtensionOptions, vsDeleteLanguageOptions, vsUpdateExistingBufferExtensionOptions, vsUpdateExistingBufferLanguageOptions, vsLanguageInheritsFrom, vsFindLanguageCallbackIndex
