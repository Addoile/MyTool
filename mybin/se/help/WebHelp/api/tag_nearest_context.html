int tag_nearest_context(int linenum, int filter_flags=VS_TAGFILTER_ANYTHING, boolean find_tag_below=false, boolean filter_by_types=false)

Return the index of the nearest context item.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Parameters:

linenum - line number to check context on

filter_flags - Tag filter flags. See tag_filter_type for more details.

find_tag_below - find the nearest context item below the current line

filter_by_types - filter out symbols turned off by 'def_cb_filter_by_types'

Returns: int

Return the index of the context tag starting closest to the given line number. This differs from the current context because the nearest context is not necessarily the context that you are currently inside. For example, if the given linenum falls between two class members, the current context would be the class, but the nearest context would be the second class member. Returns <0 on error or 0 if no current context.
