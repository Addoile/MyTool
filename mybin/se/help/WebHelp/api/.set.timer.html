int _set_timer(int milli_interval, typeless pfnCallback=0, typeless callbackArgument=null)

Creates a timer and optionally calls a function you specify. interval is specified in milliseconds. callback is a pointer to a function or names table index to a global function returned by find_index. arg1 is a an optional argument to pass to the call back function. If only one argument is given, a get_event timer is created. This allows you to use get_event in a loop and wait for a key or mouse event, or until the on_timer event occurs. When the callback argument is not null, get_event is not effected. Instead, the callback function is called with the arg1 argument if given at intervals specified by interval.

Returns: int

Returns a timer handler >=0. On error, negative error code is returned.

Example:

_set_timer(1000);
for(;;){
   event=get_event();
   if (event==on_timer){
       message("on_timer event");
   } else {
       message(event2name(event));
   }
}
