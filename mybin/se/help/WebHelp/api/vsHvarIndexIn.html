VSHVAR VSAPI vsHvarIndexIn(HVAR hvarHashtab, VSPSZ pBuf, int BufLen=-)

Parameters:

hvarHashtab - hash table to test

pBuf - index or key corresponding to item to look for

BufLen - length of hash key

Returns: VSHVAR

Returns a reference to element if element exists at index given.

Example:

   vsHvarMakeEmpty(VSHVAR_RC);
   hvar=vsHvarHashtabEl(VSHVAR_RC,"styles");
   vsHvarSetI(hvar,1);
   hvar=vsHvarIndexIn(VSHVAR_RC,"styles");  // will be non-zero
   hvar=vsHvarIndexIn(VSHVAR_RC,"color");   // will be zero

Category:

Miscellaneous Functions
