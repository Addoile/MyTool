_str _fieldname(int index)

Parameters:

index - position of field to look up.

Returns: _str

Returns the name of the given given class field. Indexes start at zero for the first field of the base (least derived) class.

Example:

class AAAA { int x; int y; };
class BBBB : AAAA { int z; };
void testFieldName() {
   BBBB b;
   say("field 0 is named "b._fieldname(0));  // x
   say("field 1 is named "b._fieldname(1));  // y
   say("field 2 is named "b._fieldname(2));  // z
}
