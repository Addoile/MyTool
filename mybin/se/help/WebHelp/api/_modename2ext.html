_str _modename2ext(_str mode_name, int &setup_index, boolean doAutoLoadSupport=true)

Deprecated. Use _Modename2LangId().

Converts the unique display name for a language to it's language ID (canonical file extension).

Parameters:

mode_name - Display name for language type

setup_index - (reference) set to the 'def-language-[lang]' name info entry for the language

doAutoLoadSupport - Automatically load language support if not already loaded.

Returns: _str

The language ID for the language corresponding to 'mode_name'.

Category:

Miscellaneous Functions
