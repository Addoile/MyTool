_str dbg_get_capability_name(int capability)

Parameters:

capability - capability type (DEBUG_CAPABILITY_*)

Returns: _str

the string representation of the given debugger capability

Since:

10.0

Example:

To look up the call back name expected for update registers 

   _str cap = dbg_get_capability_name(DEBUG_CAPABILITY_UPDATE_REGISTERS);
   _message_box("registers = " :+ cap);

Category:

Debugger Functions
