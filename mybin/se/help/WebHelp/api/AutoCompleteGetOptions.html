int AutoCompleteGetOptions(_str lang='')

Get the auto complete options wich are a bitset of AUTO_COMPLETE_*.

The options are stored per extension type. If the options are not yet defined for an extension, then use def_auto_complete_options as the default.

Parameters:

lang - language ID -- see p_LangId

Returns: int

bitset of AUTO_COMPLETE_* options.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Category:

Completion Functions
