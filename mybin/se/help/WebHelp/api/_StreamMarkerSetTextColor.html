void _StreamMarkerSetTextColor(int StreamMarkerIndex, int ColorIndex)

Sets the color index for the text in the stream marker specified.

Parameters:

StreamMarkerIndex - Index of stream marker returned by a _StreamMarkerAdd function.

ColorIndex - Index of color allocated by vsAllocColor(). Specify 0 for no color (null).

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
