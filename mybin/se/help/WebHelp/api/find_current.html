_command int find_current()

Repositions the cursor on the item currently selected in the search or references tool window. If neither window is active, it will do a find-prev, followed by a find-next to attempt to reposition on the current match.

Parameters:

doPrevious - If true, repeats search in the oposite direction.

Returns: int

Returns 0 if successful. Otherwise STRING_NOT_FOUND_RC is returned. On error, message is displayed.

Applies To:

Edit Window,Editor Control

Categories:

Editor Control Methods, Edit Window Methods, Search Functions
