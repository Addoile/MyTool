typeless hex2dec(_str hex, int base=16)

Parameters:

hex - 0xNNNN, xNNNN, or NNNN represent a valid hex number 0NNNN for an Octal number 0bNNNN, bNNNN represent a valid binary number

base - input base. Defaults to 16.

Returns: typeless

Returns number of base 16 or base specified in base 10. Only bases 2, 8, and 16 are supported. If number is invalid, '' is returned. Binary numbers may start with 'b'. Hex numbers may start with '0x' or 'x'. Octal numbers may start with the digit '0'.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
