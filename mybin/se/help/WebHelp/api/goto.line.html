_command int goto_line(_str lineNumber='')

Moves the cursor to the line number specified. Cursor column position is not changed. Another way to go to a line is to type the line number on the SlickEdit command line. Press ESC to get to the command line.

Parameters:

LineNumber -

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods, Search Functions
