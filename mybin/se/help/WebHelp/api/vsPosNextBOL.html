seSeekPosRet VSAPI vsPosNextBOL(seSeekPos Noflines)

You must call vsPosInit before calling this function. Once you use a non vsPosXXX function which accesses buffer data you MUST call vsPosInit again to avoid crashing the editor. Only use this function if you want your searching to be as fast as possible.

Moves the Pos cursor up or down the number of lines specified. The Pos cursor is always place at the beginning of the line even when 0 is given.

Parameters:

Noflines - Number of lines to move Pos cursor up or down. The Pos cursor is always place at the beginning of the line even when 0 is given.

Returns: seSeekPosRet

Returns the actually number of lines the Pos cursor went up or down.

Applies To:

Edit Window, Editor Control

Category:

Search Functions
