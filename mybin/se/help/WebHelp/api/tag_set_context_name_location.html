int tag_set_context_name_location(int context_id, int name_line_no, int name_seekpos)

Set the name and line number positions for the symbol name of the context with the given context ID.

For synchronization, macros should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

context_id - Context ID [ from 1 to tag_get_num_of_context() ]

name_line_no - line number that symbol name is located on

name_seekpos - Seek position of the first character of the symbol name

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions

tag_insert_context, tag_end_context, tag_get_detail2
