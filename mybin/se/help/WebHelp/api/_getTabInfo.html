boolean _getTabInfo(int tabIndex, SSTABCONTAINERINFO &info)

Retrieves the tab information for the tab at the specified tab-index.

The active window MUST be a tab control (OI_SSTAB).

Parameters:

tabIndex - Tab-index to retrieve info for.

info - (out) SSTABCONTAINERINFO object.

Returns: boolean

Returns true on success, false on error (e.g. tab-index does not exist).

Applies To:

SSTab

Category:

SSTab Methods
