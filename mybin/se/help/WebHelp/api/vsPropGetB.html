int VSAPI vsPropGetB(int wid, int prop_id, void *pBuf, int BufLen, int *pBufLen=a)

Copies string value of property into pBuf. No more than BufLen bytes are copied.

Parameters:

wid - Window id of object or index to resource returned by vsFindIndex, vsNameMatch, VSP_CHILD, or VSP_NEXT. 0 specifies the current object.

prop_id - One of the VSP_??? constants in "vs.h". pBuf Output buffer for string value. Buffer is not null terminated. This can be 0.

BufLen - Number of bytes allocated to pBuf.

pBufLen - If this is not 0, this is set to the number of characters you need to allocate to pBuf.

Returns: int

Returns number of bytes copied

Example:

     int len;
     char bufname[100];

     len=vsPropGetB(0,VSP_BUFNAME,bufname,100);

Applies To:

All Window Objects

Categories:

Edit Window Methods, Editor Control Methods
