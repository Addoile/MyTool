int _lbfind_selected(boolean find_first)

Places the cursor on the next or first selected item of a list box. Specify a non-zero value for the find_first parameter to find the first selected item. This function is intended for list boxes with the p_multi_select property set to MS_EXTENDED or MS_SIMPLE_LIST. If p_multi_select is set to MS_NONE, the line at the cursor is the only line that can be selected (unless a macro has a bug).

Returns: int

Returns 0 if a selected item is found.

Applies To:

List Box

Category:

List Box Methods
