_str getSaveEOLFormat(_str langID)

Gets the format used to translate EOL characters when files in this language are saved. This value is available on the File Options form for each applicable language.

Parameters:

langID - language ID (see p_LangId)

Returns: _str

how EOL characters are translated when a file is saved values:

                           A - Automatic
                           D - DOS
                           M - Mac
                           U - Unix

Category:

LanguageSettings API
