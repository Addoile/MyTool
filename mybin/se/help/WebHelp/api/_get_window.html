void _get_window(int &x, int &y, int &width, int &height, _str option='C', int &icon_x=0, int &icon_y=0)

Get the geometry of the current window.

(x,y) specify the top-left corner. (width,height) specify width and height. icon_x,icon_y specify the iconized position. All values returned are in the parent scale mode (p_xyscale_mode) except for the 'O' option which returns the values in pixels. option can be 'C' for current size, 'N' for normalized size (before it was iconized), or 'O' for outer-frame geometry.

Parameters:

x - (reference) x position

y - (reference) y-position

width - (reference) width

height - (reference) height

option - option, see above

icon_x - (reference) icon x-position

icon_y - (reference) icon y-position

Applies To:

All Window Objects
