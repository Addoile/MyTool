_command int goto_bookmark(_str bookmarkName='')

Places the cursor at the file and cursor position stored in the bookmark given by bookmarkName.

Note: This command no longer supports the + and - options.

Parameters:

bookmarkName - Name of bookmark on which to place cursor.

Returns: int

Returns the bookmark ID on success. <0 on error.

Applies To:

Edit Window

Category:

Bookmark Functions

See Also:

set_bookmark, toggle_bookmark, old_goto_bookmark, bookmark_stack
