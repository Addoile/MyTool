int VSAPI vsHvarGetWID(VSHVAR hVar)

Returns: int

Returns integer value representing a window ID of interpreter variable corresponding to hvar. If the value of the variable is not a valid integer, an unknown value is returned.

Example:

       VSHVAR hvar;
       int i;
       int index;

       // The rc variable is a global variable which always exists.
       index=vsFindIndex("rc",VSTYPE_VAR);
       // Get handle to interpreter variable
       hvar=vsGetVar(index);
         i=vsHvarGetWID(hvar);

Miscellaneous Functions
