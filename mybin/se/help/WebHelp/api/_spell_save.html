int _spell_save(_str user_dict1="", _str user_dict2="")

Writes the user dictionary 1 and 2 information in memory to the files specified. user_dict1 and user_dict2 specify output file names. The default user dictionary name set by the _spell_option function is used if a dictionary name is not specified.

Returns: int

Returns 0 if successful.

Category:

File Functions
