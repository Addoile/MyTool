int VSAPI vsHvarNumFields(VSHVAR hvar)

Parameters:

hvar - handle for interpreter variable

Returns: int

Return the number of non-static data members in a class or struct type instance. Returns CLASS_NAME_NOT_FOUND_RC if the class is not found. Returns INVALID_HVAR_ARGUMENT_RC if 'hvar' is not a struct or class instance.

Category:

Miscellaneous Functions

See Also:

vsHvarFieldName, vsHvarFieldIndex, vsHvarClassFieldName, vsHvarClassFieldIndex, vsHvarClassNumFields
