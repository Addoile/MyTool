int dbg_session_eval_condition(int session_id, int thread_id, int frame_id, _str expr)

Parse a conditional expression in the current thread and frame.

Parameters:

session_id - debugger session ID (1 .. max_session)

thread_id - current thread

frame_id - current stack frame

expr - expression to parse

Returns: int

0 if the expression evaluates to false, 1 for true, <0 on error.

Since:

10.0

Category:

Debugger Functions
