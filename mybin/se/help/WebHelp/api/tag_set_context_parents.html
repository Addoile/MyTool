int tag_set_context_parents(int context_id, _str parents)

Set the class inheritance for the given context tag.

For synchronization, macros should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

context_id - Context ID [ from 1 to tag_get_num_of_context() ] of context tag representing class or struct to set inheritance relationships for.

parents - Delimited list of immediate class ancestors. It is important to remember that for some languages, all classes have a default ancestor, such as java.lang/Object in Java. Otherwise, use the empty string to indicate that class_name is a base class. See VS_TAGSEPARATOR_parents for more details on constructing this string.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
