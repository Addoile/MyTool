int p_buf_id

Determines buffer displayed in current window. This property is only available when the control is displayed.

Example:

// Remember the original buffer
orig_buf_id=p_buf_id;
// Create a new buffer.  -q is a quit option
status=load_files('-q main.e');
if (status) {
   first_line='';
   if (status== NEW_FILE_RC) {
       _delete_buffer();
   }
} else {
   top();   // Just in case the file was already loaded.
   get_line(first_line);
   _delete_buffer();
}
// Make sure were are back to the original buffer.
p_buf_id=orig_buf_id;
message('first_line='first_line);

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
