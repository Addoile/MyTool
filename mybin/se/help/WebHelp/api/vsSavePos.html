void VSAPI vsSavePos(int wid, VSSAVEPOS *pbuf, int reserved=a)

Saves the editor cursor position and scroll position information so that it can be restored later by vsRestorePos. NO EDITS should be made to this buffer inbetween the vsSavePos and vsRestorePos calls. Use vsSavePos2 and vsRestorePos2 to save and restore where edits are made.

IMPORTANT: You do not have to call vsRestorePos after calling this function or free and memory.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

pbuf - Structure filled in with editor cursor position and scroll position information.

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods
