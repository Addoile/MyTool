void VSAPI vsNextBuffer(int wid, int nbflags)

Switches to the next buffer.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

nbflags - zero or more of the following flags:

VSNBFLAG_FIND_HIDDEN
When specified, buffers with (VSP_BUFFLAGS & VSBUFFLAG_HIDDEN) are found.
VSNBFLAG_NO_UPDATE_BUFFER_VIEW
When specified, the buffers old cursor position (line number, col, etc.) information is not updated.
VSNBFLAG_NO_REFRESH
When specified, window refresh flags are not updated to avoid unnecessary screen painting.

Applies To:

Edit Window, Editor Control

Categories:

Buffer Functions, Edit Window Methods, Editor Control Methods

See Also:

vsPrevBuffer, vsDeleteBuffer
