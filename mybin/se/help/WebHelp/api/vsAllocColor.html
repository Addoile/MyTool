VSCOLORINDEX VSAPI vsAllocColor(int wid, int fg=0x000000, int bg=0xffffff, int fontFlags=0, VSCOLORINDEX parentColor=0)

Parameters:

wid - Window id of editor control. 0 specifies the current object.

fg - [optional] foreground color to set, default is black

bg - [optional] background color to set, default is white

fontFlags - [optional] bitset of VSFONTFLAG_*, default is plain

parentColor - [optional] color to inherit fg, bg, or font attributes from

Returns: VSCOLORINDEX

If successful, new color index is returned. Otherwise 0 is returned. Use the vsSetDefaultColor function to set the color attributes. Color indexes are passed to the vsSetTextColor function to set color. There is a limit of 255 colors so make sure to free the color when you are done.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

vsFreeColor, vsSetTextColor, vsSetDefaultColor, vsGetDefaultColor
