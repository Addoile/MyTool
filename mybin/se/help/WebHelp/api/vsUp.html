int VSAPI vsUp(int wid, seSeekPos Noflines)

Moves cursor up the number of lines specified by Noflines. If the destination line is not in view, it is center scrolled or smooth scrolled into view. wid identifies the object to be operated on. 0 specifies the current object.

Returns: int

Returns 0 if successful. Otherwise TOP_OF_FILE_RC is returned.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods
