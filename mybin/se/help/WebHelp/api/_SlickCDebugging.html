boolean _SlickCDebugging(int on_off, _str port)

Start or stop the Slick-C debugging protocol. Slick-C implements a JDWP (Java Debug Wire Protocol) server that hooks into the interpreter loop when enabled. Turning on debugging will allow a JDWP enabled debugger (SlickEdit) to connect to us. Turning off debugging will terminate the debugger connection.

Parameters:

on_off - see SLICKC_DEBUG_* options above.

port - port number to connect to

Returns: boolean

'true' if Slick-C® debugging was originally enabled, 'false' otherwise.
