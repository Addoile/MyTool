int _TreeBeginUpdate(int ItemIndex, _str options="")

Start effecient update of tree contents. Using this function saves time and reduces the amount of memory allocation and redraws that get triggered when the contents of a tree are updated.

Do not set column flags (_TreeSetColButtonInfo) between calls to _TreeBeginUpdate and _TreeEndUpdate, they will be reset.

Parameters:

ItemIndex - Specifies a tree item (parent node to update children of)

options - if 'T', update the node and all child nodes recursively

Applies To:

Tree View

Category:

Tree View Methods

See Also:

_TreeAddItem, _TreeEndUpdate
