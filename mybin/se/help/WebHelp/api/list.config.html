_command typeless list_config()

Runs the external batch macro 'listcfg.e' with no arguments. This command creates a new buffer and inserts Slick-C® source code representing the configuration of insert mode, colors, search case sensitivity, cache size, spill file path, scroll style, language syntax options, file load/save options, tabs, margins, and more. For all variables that start with the four letters 'def_', an assignment statement (variable= current value) will be inserted into this buffer. This command does NOT list key bindings or modified dialog box templates. The commands list_objects, list_usersys_objects, and list_source generate other information.

Returns: typeless

Returns 0 if successful.

Category:

Miscellaneous Functions
