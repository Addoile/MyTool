int VSAPI SETagOccurrencesEnd(int dbHandle, const slickedit::SEString &file_name, slickedit::SEHashTable&occurHashTable)

Clean up after inserting a series of occurrencess from a single file for update. Doing this allows the tag database engine to remove any occurrences from the database that are no longer valid.

Parameters:

dbHandle - database handle

file_name - full path of file

occurHashTable - hash table mapping file names to a hash set of symbol names this is used to cache results and avoid inserting the same symbol twice.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
