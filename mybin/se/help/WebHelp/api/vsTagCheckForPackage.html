int vsTagCheckForPackage(const char *symbol, VSHREFVAR tag_files, int exact_match, int case_sensitive, char *aliased_to=a, int aliased_to_len=a)

Look up 'symbol' and see if it is a package, namespace, module or unit.

For synchronization, threads should perform a tag_lock_context(false) and tag_lock_matches(true) prior to invoking this function.

Parameters:

symbol - current symbol to look for

tag_files - (reference to _str[]) list of tag files to search

exact_match - look for exact match rather than prefix match

case_sensitive - case sensitive comparison?

aliased_to - set to namespace alias name, if applicable

aliased_to_len - number of bytes allocated to 'aliased_to'

Returns: int

>0 if 'symbol' or prefix of matches package, otherwise returns 0. A value of 1 indicates the item was found in a tag file, a value of greater than 1 indicates that that item was found in the context.

Category:

Tagging Functions
