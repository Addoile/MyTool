14 VSCALLBACK_AR_BUF_DRAW_LINE_START_FINISH

void (VSAPI *pfnDrawLineStartFinish)(int buf_id, int wid, int start)

The DRAW_LINE_START_FINISH callback is called before starting to draw lines for a buffer (start=1), and when finished drawing lines for a buffer (start=0).

This callback is useful when you have some GUI widget (e.g. gutter, trough, etc.) that you must keep in sync line-by-line with the editor control window. See also VSCALLBACK_AR_BUF_DRAW_LINE callback.

WARNING: It is not safe to access editor buffer data during this callback.

Parameters:

buf_id - Buffer id.

wid - Window id of editor control. 0 specifies the current object.

start - (boolean). Set to true (1) when starting to draw lines. Set to false (0) when finished drawing lines.

Category:

Editor Control Callback Functions
