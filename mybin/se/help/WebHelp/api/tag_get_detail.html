void tag_get_detail(int tag_detail, var result)

Retrieve specific information about the current tag (as defined by calls to tag_find_equal, tag_find_prefix, tag_find_in_file, tag_find_in_class, etc.). This function complements tag_get_info, or can be used in its place for efficiency when only very specific information is needed.

Parameters:

tag_detail - ID of default to extract. One of the following:

VS_TAGDETAIL_name
(string) tag name
VS_TAGDETAIL_type
(string) tag type name, see tag_get_type.
VS_TAGDETAIL_type_id
(int) unique id for tag type.
VS_TAGDETAIL_file_name
(string) full path of file the tag is in
VS_TAGDETAIL_file_date
(string) date of file when tagged
VS_TAGDETAIL_file_line
(int) line number of tag within file
VS_TAGDETAIL_file_id
(int) unique id for file the tag is located in
VS_TAGDETAIL_class_simple
(string) name of class the tag is present in
VS_TAGDETAIL_class_name
(string) name of class with outer classes
VS_TAGDETAIL_class_package
(string) package/namespace tag belongs to
VS_TAGDETAIL_class_id
(int) unique id for class tag belongs to
VS_TAGDETAIL_flags
(int) tag flags, see tag_insert_simple
VS_TAGDETAIL_return
(string) value or type of var/function
VS_TAGDETAIL_arguments
(string) function or template arguments
VS_TAGDETAIL_tagseekloc
(int) PRIVATE
VS_TAGDETAIL_fileseekloc
(int) PRIVATE
VS_TAGDETAIL_classseekloc
(int) PRIVATE
VS_TAGDETAIL_num_tags
(int) number of tags in database
VS_TAGDETAIL_num_classes
(int) number of classes in database
VS_TAGDETAIL_num_files
(int) number of files in database
VS_TAGDETAIL_num_types
(int) number of types in database
VS_TAGDETAIL_num_refs
(int) number of references in database
VS_TAGDETAIL_throws
(string) function/proc exceptions.
VS_TAGDETAIL_included_by
(string) file including this macro.
VS_TAGDETAIL_return_only
(string) just the return type of variable.
VS_TAGDETAIL_return_value
(string) default value of variable.

result - (Output) Set to tag detail after successful completion of function. If the current tag is not defined, this will be set to either '' or 0, depending on if the detail is a string or numeric.

Category:

Tagging Functions
