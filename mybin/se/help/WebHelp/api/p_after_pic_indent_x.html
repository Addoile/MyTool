int p_after_pic_indent_x

Gets or sets the amount to indent in twips after a bitmap in a list box, text box, or combo box (text box not list). This property is not available when the control is not displayed.

Example:

#include 'slick.sh'
defeventtab form1;
combo1.on_create()
{
   p_picture=_pic_drremov;
   // Set indent before picture.
   p_pic_indent_x=100;
   // Set indent after picture.
   p_text='A:';
   p_after_pic_indent_x=200;
   // Extra y spacing. Half displayed above and half below picture
   p_pic_space_y=100;
}

Applies To:

List Box, Combo Box, Text Box

Categories:

List Box Properties, Combo Box Properties, Text Box Properties
