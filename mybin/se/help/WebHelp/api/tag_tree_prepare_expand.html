int tag_tree_prepare_expand(int form_wid, int tree_index, int tree_wid, int in_refresh, _str class_filter, _str member_filter, _str exception_name, int filter_flags, var icons, int filter_flags2=0)

This function is called to prepare this module for inserting a large number of tags from a common class or category. It copies a number of options and parameters into globals where they are accessed only by vsTagTreeAddClassMember(). This mitigates the parameter passing overhead normally required by vsTagTreeAddClassMember.

Parameters:

form_wid - Window ID of form containing the class tree view

tree_index - Tree parent node index which items will be inserted under.

tree_wid - Window ID of symbol browser tree control.

in_refresh - non-zero if we are in a refresh operation

class_filter - Regular expression for class filtering

member_filter - Regular expression for member filtering

exception_name - Name of tag to be allowed as an exception even if class/member/attribute filtration fails.

filter_flags - Attribute (flag-based) filtration flags. For details, see tag_tree_filter_member2.

icons - Two-dimensional array of tag bitmaps.

filter_flags2 - Extended filter flags. For details, see tag_tree_filter_member2.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions

See Also:

tag_tree_add_members_in_category, tag_tree_add_members_in_section, tag_tree_add_members_of, tag_tree_insert_tag, tag_tree_select_bitmap
