int tag_get_date(_str file_name, _str &modify_date, _str included_by=null)

Retrieve the date of tagging for the given file. The string returned by this function is structured such that consecutive dates are ordered lexicographically, and is reported in local time cooridinates (YYYYMMDDHHMMSSmmm). This function has the side effect of finding and position the file iterator on the given file name, returns BT_RECORD_NOT_FOUND_RC if file_name is not in the database.

Parameters:

file_name - name of file to update date of tagging for

modify_date - (Output) Modification date of file when tagged. Format is YYYYMMDDHHMMSSmmm (year, month, day, hour, minute, second, ms).

included_by - name of file that 'file_name' was included by

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if the file is not found.

Category:

Tagging Functions
