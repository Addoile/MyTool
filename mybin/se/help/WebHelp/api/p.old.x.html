int p_old_x

Determines the old x position in pixels before the x position of the window was changed. This property is used to determine what other tiled edges need to be adjusted to keep the tiled windows tiled. The editor sets this property when _move_window, _tile_windows, or _cascade_windows is called. This property is available only when the object is displayed.

Applies To:

All Window Objects

Category:

All Windows Properties
