int dbg_session_update_thread_states(int session_id)

Update the status flags of all threads in this debugging session.

NOTE: You must update the threads first.

Parameters:

session_id - debugger session ID (1 .. max_session)

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

   int status = dbg_session_update_threads(session);
   if (status) {
      return status;
   }
   status = dbg_session_update_thread_states(session);
   if (status) {
      return status;
   }

Category:

Debugger Functions
