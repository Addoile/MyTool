_str stranslate(_str original, _str replace_with, _str search_for, _str search_options="")

Replaces all occurrences of a search string within a string.

Parameters:

original - String to modify

replace_with - String to replace occurrences of search_for with

search_for - String to search for. May be a regular expression

search_options - Search options. See pos for valid options.

Returns: _str

Returns original with all occurrences of search_for replaced with replace_with. See pos function for information on valid search search_options.
