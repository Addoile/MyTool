_command int forward_up_sexp()

Moves forward up into the parent S-expression starting from the cursor position. Cursor position is unchanged if not found.

Returns: int

Returns 0 if successful, <0 on error.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods

See Also:

next_sexp, prev_sexp, backward_up_sexp, backward_down_sexp, forward_down_sexp
