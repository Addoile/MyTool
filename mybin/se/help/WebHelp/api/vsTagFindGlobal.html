int VSAPI vsTagFindGlobal(int type_id, int mask=a, int nzero=a, int find_first=a)

Retrieve the first tag included in this tag database with global scope that is one of the given type (type_id) and that matches the given tag flag mask ((mask & tag_flags) != 0) == non_zero. Use vsTagGetInfo and/or vsTagGetDetail to extract the details about the tag.

Parameters:

type_id - Type id (VS_TAGTYPE_*) If (type_id<0), returns tags with any user defined tag types.

mask - Tag attribute flags set (VS_TAGFLAG_*). The intersection of this set of flags and those for each tag found (tag_flags) will be calculated.

nzero - Accept tag if (mask & tag_flags) is non-zero

find_first - find first match or next match

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Example:

find the first inline global function. 

     vsTagFindGlobalOfType(VS_TAGTYPE_function, VS_TAGFLAG_inline, 1);

Category:

Tagging Functions
