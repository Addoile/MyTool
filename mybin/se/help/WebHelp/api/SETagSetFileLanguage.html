int VSAPI SETagSetFileLanguage(int dbHandle, const slickedit::SEString &fileName, const slickedit::SEString &langId)

Set the language ID for the given file.

Parameters:

dbHandle - database handle

fileName - File path of source file to setting tagging date for.

langId - p_LangId property for file_name

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if the file is not found.

Category:

Tagging Functions
