int VSAPI vsRefactorConfigJavaAdd(const char *configName, const char *sourcePath, const char *jarPath)

Add a new java compiler configuration to the master refactoring configuration file.

Parameters:

configName - name for this configuration

sourcePath - root dir for this configuration

jarPath - PATHSEP deliminated list of the system jars

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions

See Also:

vsRefactorConfigCount, vsRefactorConfigGetName, vsRefactorConfigSetName, vsRefactorConfigCountJARS, vsRefactorConfigGetJAR, vsRefactorConfigAddJAR, vsRefactorConfigDeleteJAR, vsRefactorConfigDeleteAllJARs
