int tag_read_db(_str file_name)

Open an existing tag database and return a handle to the database. This function opens the database for read-only access. The database type (tags, references, BSC file) is automatically detected.

Parameters:

file_name - file name of tag database to open.

Returns: int

database handle >= 0 on success, <0 on error.

Category:

Tagging Functions
