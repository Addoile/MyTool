boolean getInsertRealIndent(_str langID, boolean defaultValue=null)

Gets whether we insert real indent for this language. This value is available on the GUI on the Indent form for each applicable language.

Parameters:

langID - language ID (see p_LangId)

defaultValue - if the value is not available, then we just return this value. If this parameter is not specified, then the default value for the overall def-var is used.

Returns: boolean

whether we insert real indent for this language

Category:

LanguageSettings API
