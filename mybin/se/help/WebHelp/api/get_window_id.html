void get_window_id(int &window_id)

Determines the current object/current window. This function has the same effect as setting the p_window_id property.

Example:

get_window_id(window_id);   // Remember current window
_next_window();           // Switch to next window
activate_window(window_id); // Go back to orig. window

Applies To:

All Window Objects

Categories:

Buffer Functions, Window Functions

See Also:

activate_window, load_files, _next_window, _prev_window
