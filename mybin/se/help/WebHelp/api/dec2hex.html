_str dec2hex(long number, typeless base="")

Parameters:

number - decimal number

base - may be 2 or 8 or 16.

Returns: _str

Returns number converted to base 16 or base specified. base must be one of the values 2, 8, or 16. If number is invalid, '' is returned. Binary numbers are prefixed with the string "0b". Hex numbers are prefixed with "x". Octal numbers are prefixed with the digit '0'.

Output: base=16 0xNNNN base=8 0NNNN base=2 0bNNNN

Category:

String Functions
