int VSAPI vsTagCreateDB(const char *pszFilename, int dbFlags=a, const char *dbDescription=a)

Create a tags database, with standard tables, index, and types.

Parameters:

pszFilename - File path where to create new database. If the file already exists, it will be truncated.

dbFlags - Bitset of VS_DBFLAG_

dbDescription - database comment

Returns: int

database handle >= 0 on success, < 0 on error.

Category:

Tagging Functions
