_str def_pmatch_chars

Charactors to use for parentheses pair matching.

By default, this only supports parentheses, square brackets, and curly braces.

To support < and > for template argument lists, simply append <> to this variable. Note that this can be easily confused with uses of < and > in comparison expressions.

Default:

"()[]{}"

Category:

Configuration Variables

See Also:

find_matching_paren, keyin_match_paren
