int p_pic_indent_x

Determines the amount to indent in twips before a picture displayed text box, or combo box (text not list).

Example:

#include 'slick.sh'
defeventtab form1;
combo1.on_create()
{
   p_picture=_pic_drremov;
   // Set indent before picture.
   p_pic_indent_x=100;
   // Set indent after picture.
   p_text='A:';
   p_after_pic_indent_x=200;
   // Extra y spacing. Half displayed above and half below picture
   p_pic_space_y=100;
}

Applies To:

Combo Box, Text Box

Categories:

Combo Box Properties, Text Box Properties
