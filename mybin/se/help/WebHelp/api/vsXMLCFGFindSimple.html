int VSAPI vsXMLCFGFindSimple(int iHandle, VSPSZ pszQueryStr, int NodeIndex=VSXMLCFG_ROOT_INDEX, int reserved=a)

Parameters:

iHandle - iHandle Handle to an XMLCFG tree returned by vsXMLCFGOpen(), vsXMLCFGCreate(), or vsXMLCFGOpenFromBuffer().

pszQueryStr - This is a small subset of an XPath expression. See _xmlcfg_find_simple() for information on this parameter.

NodeIndex - Tree node index to start search from.

reserved -

Returns: int

Returns node index (>=0) found if successful. Otherwise a negative error code is returned.

Example:

// Find the first file element node under /project with name equal to
// "main.c"
FileIndex=vsXMLCFGFindSimple(handle,
    "/project/file[file-eq(@name,'main.c')]");

// Find the attribute node under /project/file with name equal
// to "main.c"
AttrIndex=vsXMLCFGFindSimple(handle,
    "/project/file/@name[file-eq(.,'main.c')]");
