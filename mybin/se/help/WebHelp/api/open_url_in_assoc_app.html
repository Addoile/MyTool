_command void open_url_in_assoc_app ()

The open_url_in_assoc_app command attempts to intelligently open the URL the cursor is in:

  • First, attempt to use the associated application found in the File Extension Manager.
  • Second, attempt to use the associated application found in the registry (on Windows systems).
  • Third, open with the default web browser
