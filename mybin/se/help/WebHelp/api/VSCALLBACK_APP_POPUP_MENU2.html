31 VSCALLBACK_APP_POPUP_MENU2

void (VSAPI *pfnPopupMenu2)(const char *pszMenuName,int vtpm_flags,int x,int y);
The POPUP_MENU2 callback is called when a right click menu would be displayed. When this callback is defined no, menu is displayed. Later we may expand this hook to support when any pop-up menu is displayed.

Parameters:

pszMenuName - Name of Slick-C® menu which would be displayed.

vtpm_flags - Windows VTPM_??? flags which would be given to the WIN32 TrackPopupMenu function.

x - Recommended x screen coordinate to display menu.

y - Recommended y screen coordinate to display menu.

Category:

Editor Control Callback Functions
