_str _message_box(_str string, _str title="SlickEdit", int mb_flags=MB_OK | MB_ICONEXCLAMATION, int default_button=0)

Creates a modal dialog box and displays string message and various buttons.

Parameters:

string - may contain carriage return characters to break a string message into multiple lines. By default, string is broken into multiple lines on word boundaries if necessary. The title of the dialog box is set to title if given.

title -

mb_flags - is a combination of zero or more ID button flags and MB_* flags:

Note: The MB_OK, MB_OKCANCEL, MB_ABORTRETRYIGNORE, MB_YESNOCANCEL, MB_RETRYCANCEL are combinations of the ID button flags (ex. MB_OKCANCEL is IDOK|IDCANCEL)

IDOK
Display an OK button
IDSAVE
Display a Save button
IDSAVEALL
Display a Save All button
IDOPEN
Display an Open button
IDYES
Display a Yes button
IDYESTOALL
Display a Yes to All button
IDNO
Display a No button
IDNOTOALL
Display a No to All button
IDABORT
Display an Abort button
IDRETRY
Display a Retry button
IDIGNORE
Display an Ignore button
IDCLOSE
Display a Close button
IDCANCEL
Display a Cancel button
IDDISCARD
Display a Discard button
IDHELP
Display a Help button
IDAPPLY
Display an Apply button
IDRESET
Display a Reset button
IDRESTOREDEFAULTS
Display Restore Defaults button
MB_OK
Display an OK button.
MB_OKCANCEL
Display an OK and Cancel button.
MB_ABORTRETRYIGNORE
Display Abort, Retry, Ignore buttons. Not supported on UNIX.
MB_YESNOCANCEL
Display Yes, No, and Cancel buttons.
MB_YESNO
Display Yes, and No buttons.
MB_RETRYCANCEL
Display a Retry and Cancel button. Not supported on UNIX.
MB_ICONHAND
Display a stop sign picture to the left of the message.
MB_ICONQUESTION
Display a question picture to the left of the message.
MB_ICONEXCLAMATION
Display an exclamation point picture to the left of the message.
MB_ICONINFORMATION
Display an 'i' picture to the left of the message.
MB_ICONSTOP
Display a stop sign picture to the left of the message.

default_button - If specified, determines which of the buttons is the default button. One of the following button constants:

  • IDOK
  • IDSAVE
  • IDSAVEALL
  • IDOPEN
  • IDYES
  • IDYESTOALL
  • IDNO
  • IDNOTOALL
  • IDABORT
  • IDRETRY
  • IDIGNORE
  • IDCLOSE
  • IDCANCEL
  • IDDISCARD
  • IDHELP
  • IDAPPLY
  • IDRESET
  • IDRESTOREDEFAULTS

Returns: _str

Returns the button id of the button pressed which closed the dialog box. One of the following constants is returned:

  • IDOK
  • IDSAVE
  • IDSAVEALL
  • IDOPEN
  • IDYES
  • IDYESTOALL
  • IDNO
  • IDNOTOALL
  • IDABORT
  • IDRETRY
  • IDIGNORE
  • IDCLOSE
  • IDCANCEL
  • IDDISCARD
  • IDHELP
  • IDAPPLY
  • IDRESET
  • IDRESTOREDEFAULTS

Example:

result=_message_box("Save changes?", '', MB_YESNO|MB_ICONQUESTION);
if (result==IDYES) {
    // save the file
} else {
    // Don't save it
}
// Same as above but using ID button flags instead of MB_OK.
// Also change the default button
result=_message_box("Save changes?", '', IDYES|IDNO|MB_ICONQUESTION,IDNO);
if (result==IDYES) {
    // save the file
} else {
    // Don't save it
}
