int VSAPI vsTagGetCurrentContext(char *cur_tag_name, int &cur_flags, char *cur_type_name, int &cur_type_id, char *cur_context, char *cur_class, char *cur_package)

Determine the name of the current class or package context. The current object needs to be an editor control.

For synchronization, threads should perform a tag_lock_context(false) and tag_lock_matches(true) prior to invoking this function.

Parameters:

cur_tag_name - name of the current tag in context (allocate VS_TAG_MAX_TAGNAME bytes)

cur_flags - type (VS_TAGTYPE_*) of the current tag

cur_type_name - type ID (VS_TAGTYPE_*) of the current tag (allocate VS_TAG_MAX_TYPENAME bytes)

cur_type_id - bitset of VS_TAGFLAG_* for the current tag

cur_context - class name representing current context (allocate VS_TAG_MAX_CLASSNAME bytes)

cur_class - cur_context minus the package name (allocate VS_TAG_MAX_CLASSNAME bytes)

cur_package - only package name for the current context (allocate VS_TAG_MAX_CLASSNAME bytes)

Returns: int

0 if no context, context ID >0 on success, <0 on error.

Category:

Tagging Functions
