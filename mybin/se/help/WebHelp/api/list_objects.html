_command typeless list_objects(_str args='', ...)

Opens a new buffer called "vusrobjs.e" and inserts Slick-C® source code for all dialog box templates (forms) created by the user. System dialog box templates are not inserted.

Returns: typeless

Returns 0 if successful. Returns 1 if no templates are defined. Other errors are non-zero.

Category:

Form Functions

See Also:

list_usersys_objects, list_sys_objects
