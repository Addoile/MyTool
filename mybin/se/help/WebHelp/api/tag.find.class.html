int tag_find_class(_str &class_name, _str search_class_name=null, boolean normalize=false, boolean case_sensitive=true, _str cur_class_name=null)

Retrieve the name of the first class included in this tag database, or optionally, the name of a specific class, either to check if the file is in the database or to position the class iterator at a specific point. If normalize is non-zero, then search_class_name is a simple class name and the search attempts to qualify it with its outer class name or package name, etc., before returning result. The default search is case-sensitive.

Parameters:

class_name - (Output) Name of class

search_class_name - Specific class to search for (prefix search)

normalize - Attempt to qualify search_class_name

case_sensitive -

cur_class_name - Name of current class in context

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
