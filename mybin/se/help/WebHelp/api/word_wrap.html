_command typeless word_wrap(_str yesno="")

If yes, the editor attempts to keep the cursor within the margins when entering text, moving the cursor, and deleting characters. The word_wrap command only affects the current buffer. To set the default word wrap style for specific file extension, use the Extension Options dialog box ("Tools", "Configuration..", "File Extension Setup...", select the Word Wrap tab).

Applies To:

Edit Window, Editor Control
