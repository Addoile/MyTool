int VSAPI vsGetText(int wid, int Nofbytes, seSeekPos seekpos, char *pszBuf)

Places Len bytes of data starting from the byte offset specified by SeekPos into pszBuf. The resulting string is always null terminated. The buffer pszBuf must be large enough to contain Len+1 bytes of data.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

Len - Number of characters to copy into pszBuf.

SeekPos - Specifies the seek position of from which to start copying. Specify VSNULLSEEK to start copying for the cursor position.

pszBuf - Output buffer for text. The buffer pszBuf must be large enough to contain Len+1 bytes of data.

Returns: int

Returns number of bytes copied not including null character.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

vsInsertLine, vsReplaceLine, vsDeleteLine, vsGetLine
