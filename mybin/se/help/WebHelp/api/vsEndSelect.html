int VSAPI vsEndSelect(int wid, int markid=-, int LockSelection=a)

If all input is valid and there is a selection, the cursor is placed at the end of the selection. For line selection, the column position of the cursor is not changed.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

mark_id - Handle to a selection returned by vsAllocSelection or vsDuplicateSelection. A mark_id of -1 identifies the active selection.

LockSelection - If this is 1, the selection is "locked" (moving the cursor will no longer remove or extend the selection).

Returns: int

Returns 0 if successful. Possible return codes are PROPERTY_OR_METHOD_NOT_ALLOWED_RC, INVALID_SELECTION_HANDLE_RC, TEXT_NOT_SELECTED_RC, or INVALID_OBJECT_HANDLE_RC.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
