int _process_info(_str option="")

This function returns information about the process running depending upon the input given as follows:

""
Returns 1 if the build window (concurrent process buffer) is running. Otherwise 0.
'C'
Returns start column of process request for input. If the current line is not reading input for a concurrent process, 0 is returned.
'B'
Returns 1 if the current buffer has the build window. Otherwise 0 is returned.
'R'
Returns 1 if the build window is running. Otherwise 0 is returned. A delay is performed and pending build window output is read into the build window.
'X'
Returns 1 if the command shell running in the build window has exited. Otherwise 0 is returned. A delay is performed and pending build window output is read into the build window.
