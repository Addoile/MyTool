_command typeless gui_print()

Non-UNIX platforms: Prints the current buffer or the selection. The Print dialog box is displayed, which allows you to specify various print options.

UNIX: Prints the current buffer or the selection. The Text Mode Print dialog box is displayed, which allows you to specify various print options.

Returns: typeless

Returns 0 if successful.

Applies To:

Edit Window

Category:

Miscellaneous Functions
