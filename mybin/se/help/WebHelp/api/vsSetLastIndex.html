void VSAPI vsSetLastIndex(char option, int index)

Parameters:

option - Indicates last index information to Set.

'C'
Set the index of the last command or function executed by a key, vsCallKey, or vsExecute.
'L'
Set the index of the last command or function executed by a key or the vsCallKey function.
'K'
Set the index of the last event table used to determine a key binding.
'W'
Set the command line/menu wait value. This flag is used by a command to determine whether it was invoke by vsExecute (command line or menu) or a key press. This flag is set to non-zero when vsExecute is called and 0 when a key is pressed or vsCallKey is called.

index - New value for index.

Returns: void

Sets a last index corresponding to option given.

Category:

Keyboard Functions
