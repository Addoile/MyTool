VSPSZ VSAPI vsHvarClassFieldName(VSPSZ class_name, int index)

Parameters:

class_name - Slick-C® class name

index - index >=0 of non-static data member

Returns: VSPSZ

Return the name of the given non-static member of a class or struct. Returns CLASS_NAME_NOT_FOUND_RC if the class is not found. Returns INVALID_ARGUMENT_RC if the field index is out of range.

Category:

Miscellaneous Functions
