_str _config_path(_str options="")

Deprecated. Use _ConfigPath().

Parameters:

options - (optional, UNIX only) set to 'n' pre-6.0 style config dir. This option is essentially obsolete at this point.

Returns: _str

Returns the directory used to store miscellaneous configuration files such as "vslick.sta" (UNIX: "vslick.stu") and "alias.als.xml".
