_str p_completion

Determines the completion which occurs when the space bar or question mark keys are pressed. Valid p_completion constants are:

NONE_ARG
No completion
FILE_ARG
One filename
MULTI_FILE_ARG
Multiple filenames
DIR_ARG
One directory name
MULTI_DIR_ARG
Multiple directories, separated by semicolons
BUFFER_ARG
Buffer name
COMMAND_ARG
Slick-C® command function
PICTURE_ARG
Already loaded picture
FORM_ARG
Dialog box template
MODULE_ARG
Loaded Slick-C® module
MACRO_ARG
User recorded Slick-C® command function
MACROTAG_ARG
Slick-C® tag name
VAR_ARG
Slick-C® global variable
ENV_ARG
Environment variable
MENU_ARG
Menu name
TAG_ARG
Tag name

Applies To:

Combo Box, Text Box

Categories:

Combo Box Properties, Text Box Properties
