int refactor_c_move_method(int nHandle, _str szSrcMethodName, _str szSrcClassName, _str szDstMethodName, _str szDstClassName, int nSrcIdx, int nDstIdx, _str tag_files[], int nReceiver, int nFlags)

Move a method. Move a method from one class to another and fix all references.

Parameters:

nHandle - Handle to transaction

szSrcMethodName - Name of method to move

szSrcClassName - Name of class to move method from.

szDstMethodName - Name of destination method

szDstClassName - Name of class to move method to.

nFlags - Flags for access modifiers

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
