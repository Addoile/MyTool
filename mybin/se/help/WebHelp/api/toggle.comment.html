_command void toggle_comment(boolean godown=false, boolean deselect=true)

HS2. Toggles comments of current line or selection. If there is no active selection, it comments the current line. If at least 1 uncommented line is contained in the selection, it's commented blank lines are omitted when doing commenting check.

Returns: void

void

Applies To:

Edit Window, Editor Control
