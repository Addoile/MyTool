_command int debug_step_into(boolean buildFirstDone=false, boolean doRestart=false)

Step into the next statement (or step into the first statement).

This command also doubles as the entry point for starting debugging. If we are not already in debug mode, this command will delegate to "_project_debug2" to start everything, and then immediately issue a 'step into' command.

Returns: int

0 on success, <0 on error.

Category:

Debugger Commands
