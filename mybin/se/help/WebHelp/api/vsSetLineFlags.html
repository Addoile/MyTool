void VSAPI vsSetLineFlags(int wid, int Flags, int Mask=a)

Sets the line status flags for the current line.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

Flags - Flags is one or more of the following constants ORed together:

VSLF_MLCOMMENTINDEX
Indicates which multi-line comment. Only two are allowed. Must know which multi-line comment we are in so we know what will terminate it.
VSLF_MLCOMMENTLEVEL
Indicates multi-line comment nest level.
VSLF_NOSAVE
Used by Difference Editor and Merge Editor. Lines with the VSLF_NOSAVE flag set are not saved in the file.
VSLF_VIMARK
Used by VI emulation to mark lines.
VSLF_MODIFY
Line has been modified.
VSLF_INSERTED_LINE
Line was inserted.
VSLF_HIDDEN
Indicates that this line should not be displayed.
VSLF_PLUSBITMAP
Display "+" bitmap to left of this line.
VSLF_MINUSBITMAP
Display "-" bitmap to left of this line.
VSLF_CURLINEBITMAP
Display current line bitmap.
VSLF_LEVEL
Bits used to store selective display nest level.

The MLCOMMENT flags can not be modified.

Mask - The Mask indicates which bits will be set according to Flags. If Mask is 0, Mask is set to the value of Flags.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

vsQLineFlags, vsSetLineFlags, vsResetModifiedLineFlags
