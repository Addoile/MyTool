int command_state()

Returns: int

Returns non-zero value if the current window (p_window_id) is the command line. Otherwise 0 is returned.

Example:

if ( command_state() ) { _delete_char(); } else { wordwrap_delete_char(''); }

Applies To:

Text Box, Combo Box

Categories:

Combo Box Methods, Command Line Functions, Text Box Methods

See Also:

command_toggle, execute, set_command, get_command
