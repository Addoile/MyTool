int dbg_session_do_command(int session_id, _str command, _str &reply, _str &errmsg, boolean parse_reply)

Send the given command to the debugger and get the result including both normal result ouput, and possible error output.

Parameters:

session_id - debugger session ID (1 .. max_session)

command - debugger command to send

reply - command reply data

errmsg - output skimmed from stderr

parse_reply - parse the command reply, or just return string?

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

   _str reply, errmsg;
   int status = dbg_session_do_command(session_id,
                                       "-insert-symbols vsapi.so",
                                       reply, errmsg);
   if (status) {
      _message_box("error = " :+ errmsg);
      return status;
   }
   _message_box("result = " :+ reply);
