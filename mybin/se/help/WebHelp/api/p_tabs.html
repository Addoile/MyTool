_str p_tabs

Determines the tabs for the current buffer. This property is available only when the control is displayed. p_tabs is a string in the format: t1 t2 t3 ... tN. Where t1... tN represent tab stop column positions in increasing order. When setting this property, you may specify a string of the format "+number" to set tab stops in increments of a number. Tab stops continue past the last tab stop by repeating the difference of the last two tab stop values.

Example:

p_tabs="+3";  // Set the tab stops in increments of 3.  "1 4 "
parse p_tabs with t1 t2
message('t1='t1' t2='t2);

Applies To:

Edit Window, Editor Control
