char*VSAPI vsUTF8CharBegin2(const char *pBuf, int BufLen, const char *pCurrent)

Finds the beginning of the current UTF-8 character.

Parameters:

pBuf - Buffer contain UTF-8 text.

BufLen - Number of UTF-8 characters in pBuf. Specify -1 to compute the length from a null terminated buffer.

pCurrent - Pointer to UTF-8 character in pBuf.

Returns: char*VSAPI

Returns pointer to the beginning of the UTF-8 character. pCurrent is returned if pCurrent >=pBuf+BufLen.

Category:

Unicode Functions
