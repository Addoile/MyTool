int _xmlcfg_save(int iHandle, int iIndentAmount, int iFlags, _str Filename=null, int encoding=-1, int NodeIndex=TREE_ROOT_INDEX)

Saves an XMLCFG tree to a XML file.

Parameters:

iHandle - Handle for tree to save

iIndentAmount - Amount to indent for each level. -1 to use tabs

iFlags - Formatting flags. flags may be zero or more of the following flags:

VSXMLCFG_SAVE_ALL_ON_ONE_LINE
Specifies that all attributes for a given element be written on the same line.
VSXMLCFG_SAVE_ONE_LINE_IF_ONE_ATTR
Specifies that attributes for a given element be written on the same line if there is only one attribute.
VSXMLCFG_SAVE_DOS_EOL
Specifies that carriage return and line feed characters separate each line. This is the default under Windows.
VSXMLCFG_SAVE_UNIX_EOL
Specifies that a line feed character separate each line. This is the default under UNIX.
VSXMLCFG_SAVE_SPACE_AROUND_EQUAL
Specifies that spaces be added before and after the '=' for attributes and their values.
VSXMLCFG_SAVE_CLOSE_BRACE_ON_SEPARATE_LINE
Specifies that the close brace be placed on a separate line.
VSXMLCFG_SAVE_PCDATA_INLINE
PCDATA will not be automatically indented on a new line. Ideal for cases like: Value

Filename - When given, specifies filename to write to. Otherwise, the filename used in the open or create call is used.

encoding - When -1 this parameter defaults to the value specified in the _xmlcfg_open() or _xmlcfg_create() call. For Microsort .NET project files, this parameter should be VSCP_ACTIVE_CODEPAGE. encoding may be one of the following: 

VSCP_ACTIVE_CODEPAGE
VSCP_EBCDIC
VSCP_CYRILLIC_KOI8_R
VSCP_ISO_8859_1
VSCP_ISO_8859_2
VSCP_ISO_8859_3
VSCP_ISO_8859_4
VSCP_ISO_8859_5
VSCP_ISO_8859_6
VSCP_ISO_8859_7
VSCP_ISO_8859_8
VSCP_ISO_8859_9
VSCP_ISO_8859_10
Any valid Windows code page
VSENCODING_UTF8
VSENCODING_UTF8_WITH_SIGNATURE
VSENCODING_UTF16LE
VSENCODING_UTF16LE_WITH_SIGNATURE
VSENCODING_UTF16BE
VSENCODING_UTF16BE_WITH_SIGNATURE
VSENCODING_UTF32LE
VSENCODING_UTF32LE_WITH_SIGNATURE
VSENCODING_UTF32BE
VSENCODING_UTF32BE_WITH_SIGNATURE

NodeIndex - The children of this node are written.

Returns: int

Returns 0 if successful. Otherwise a negative return code defined in "rc.sh" is returned.
