int dbg_set_frame_path(int thread_id, int frame_id, _str full_path)

Save the resolved (full) file path to the given stack frame.

Parameters:

thread_id - index of thread to get stack frame from

frame_id - index of frame in stack frame list

full_path - path to file that stack frame is positioned at

Returns: int

0 on success, <0 on error

Category:

Debugger Functions
