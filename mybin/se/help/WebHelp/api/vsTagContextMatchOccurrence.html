int VSAPI vsTagContextMatchOccurrence(VSHREFVAR errorArgs, const char *file_name, int rseekpos, const char *tag_name, int case_sensitive, int &num_matches, int max_matches=VS_TAG_MAX_MATCHES)

Use context_match_tags to locate the tag matches for the occurrence in the given file at the given seek position. If the seek position is 0, then use the current buffer, otherwise, open the given file in a temporary view and seek to the given position.

Parameters:

errorArgs - Array of strings for return code

file_name - Path of file occurrence is located in

rseekpos - Real seek position of occurrence

tag_name - Name of occurrence to match against

case_sensitive - Case sensitive tag search?

num_matches - (output) number of matches found

max_matches - maximum number of matches to find

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
