int dbg_session_suspend(int session_id, int thread_id)

Suspend the entire debugging session or suspend a specific thread.

Parameters:

session_id - debugger session ID (1 .. max_session)

thread_id - "0" to suspend all threads, otherwise, a specific thread ID to suspend.

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

To suspend all threads: 

   int status = dbg_session_suspend(session_id);
   if (status) {
      return status;
   }

To suspend only the "current" thread: 

   int thread_id = dbg_get_cur_thread();
   if (thread_id < 0) {
       return thread_id;
   }
   int status = dbg_session_suspend(session_id, thread_id);
   if (status) {
      return status;
   }

Category:

Debugger Functions
