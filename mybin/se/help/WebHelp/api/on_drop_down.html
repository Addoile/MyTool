event on_drop_down()

The on_drop_down event is sent to a combo box with a reason argument. The reason argument specifies one of the following conditions:

DROP_UP
After combo list box is made invisible.
DROP_DOWN
Before combo list box is made visible.
DROP_INIT
Before retrieve next/previous. Used to initialize list box before it is accessed.
DROP_UP_SELECTED
Mouse released while on valid selection in list box and list is visible.

Example:

#include "slick.sh"
defeventtab form1;
void combo1.on_drop_down(int reason)
{
    if (reason==DROP_INIT) {
       if (p_user=='') {
           p_user=1;   // Indicate that the list box has been filled.
           // Insert one heck of a lot of items
           _insert_name_list(COMMAND_TYPE);
           _lbsort();
           _lbtop();
       }
    }
}

Applies To:

Combo Box

Combo Box Events
