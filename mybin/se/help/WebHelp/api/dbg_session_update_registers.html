int dbg_session_update_registers(int session_id)

Update the registers and register values.

Parameters:

session_id - debugger session ID (1 .. max_session)

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

   int status = dbg_session_update_registers(session_id);
   if (status) {
      return status;
   }

Category:

Debugger Functions
