_command typeless project_user2()

Executes a user defined command. The command may be specified per project or per file extension. Parts of the buffer name may be parsed into the command before it is executed. Use the new command ("Project", "New...") to create a new workspace or project. Once a project is open, you can use the Tools Tab ("Project", "Project Properties...", select Tools Tab) to modify project commands, files, and other project setup information. If the current project has a resource editor command defined, the extension specific project compile command will be ignored.

Returns: typeless

Returns 0 if command queued successfully. On error, message is displayed.
