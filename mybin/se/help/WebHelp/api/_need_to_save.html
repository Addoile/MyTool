boolean _need_to_save()

Use this function if you only want to check for MDI buffers.

Returns: boolean

Returns non-zero if the current buffer needs to be saved. The p_modify property should be checked as well. A non-zero p_modify property indicates the buffer has been modified. However, certain special buffer names and buffers with the (p_buf_flags & (VSBUFFLAG_THROW_AWAY_CHANGES|VSBUFFLAG_HIDDEN)) true, do not need to be saved. An example, will better describe the correct usage of this function.

Example:

if (p_modify && _need_to_save()) {
    // Prompt the user here or automatically save this file.
}

Applies To:

Edit Window, Editor Control

Categories:

Buffer Functions, Edit Window Methods, Editor Control Methods
