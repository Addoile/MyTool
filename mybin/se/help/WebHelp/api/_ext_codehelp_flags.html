int _ext_codehelp_flags(_str lang='')

Deprecated. Use _GetCodehelpFlags()

Get the codehelp options wich are a bitset of VSCODEHELPFLAG_*.

The options are stored per extension type. If the options are not yet defined for an extension, then use def_codehelp_flags as the default.

Parameters:

lang - language ID

Returns: int

bitset of VSCODEHELPFLAG_* options.

Applies To:

Edit Window, Editor Control

Category:

Tagging Functions
