_command int debug_attach(_str debug_cb_name="", _str attach_info="", _str session_name="")

Attach the debugger to a process not managed by the editor, such as a Java application being ran remotely, or a running process.

The attach_info depends on the target debugger.

  • GDB -- pid=[process_id],app=[program_path]
  • Java -- host=[host_name],port=[port_number]

Parameters:

debug_cb_name - debugger system callback name (gdb, jdwp, or dotnet)

attach_info - parameters for attaching to the process

Category:

Debugger Commands
