typeless eq_name2value(_str name, _str string, boolean preserveCase=false)

Parameters:

name - key to search for

string - string to search within

preserveCase - whether to preserve the case of the returned value (as it is in the string)

Returns: typeless

Returns word in namesNvalues to right of the string expression (lowcase(name)'='). Now that Slick-C has arrays and hash tables, you don't need to use this function.
