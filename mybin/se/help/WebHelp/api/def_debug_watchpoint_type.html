int def_debug_watchpoint_type

The default type of watchpoint to set.

  • VSDEBUG_WATCHPOINT_READ -- stop on read access
  • VSDEBUG_WATCHPOINT_WRITE -- stop on write access
  • VSDEBUG_WATCHPOINT_ANY -- stop on any access (read or write)

Default:

VSDEBUG_WATCHPOINT_WRITE

Category:

Configuration Variables
