int VSAPI vsTagInsertSimpleTag(const char *tag_name, int type_id, const char *file_name, int line_no, const char *class_name, int tag_flags, const char *signature=a, const char *class_parents=a, const char *template_signature=a)

API function for inserting a tag entry with supporting info

Parameters:

tag_name - name of entry

type_id - type of tag, (see VS_TAGTYPE_*)

file_name - path to file that is located in

line_no - line number that tag is positioned on

class_name - name of class that tag belongs to

tag_flags - tag attributes (see VS_TAGFLAG_*)

signature - (optional) arguments and return type

class_parents - (optional) class parents

template_signature - (optional) template signature

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
