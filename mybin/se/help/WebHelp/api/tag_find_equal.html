int tag_find_equal(_str tag_name, boolean case_sensitive=false, _str class_name=null)

Retrieve the first tag with the given tag name. Use tag_get_info to extract the details about the tag.

Parameters:

tag_name - Name of tag to search for

case_sensitive - Case sensitive tag name comparison

class_name - Class name to search for tag in

Returns: int

0 on success, <0 on error, or BT_RECORD_NOT_FOUND_RC if no such tag.

Category:

Tagging Functions
