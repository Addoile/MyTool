int dbg_add_disassembly_line(_str source_file, int line_number, _str(&addresses)[], _str(&instructions)[], _str(&func_names)[], _str(&offsets)[])

Add the disassembly for the indicated line in the indicated source file.

Parameters:

source_file - source file the line is in

line_number - line number to disassemble

addresses - (reference) array of instruction addresses

instructions - (reference) array of disassembled instructions

func_names - (reference) array of function names

offsets - (reference) array of function offsets

Returns: int

the line ID > 0 on success, < 0 on error.

Category:

Debugger Functions
