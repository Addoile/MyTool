int _StreamMarkerAddB(_str BufName, long StartOffset, long Length, boolean isRealOffset, int BMIndex, int type, _str pszMessage)

Adds a new stream marker.

Parameters:

BufName - Name of buffer to add marker to.

RealLineNum - Line number or real line number (not counting no save lines).

isRealLineNum - Indicates whether RealLineNum is a real line number or not.

NofLines - Number of lines in this item. Specify zero if the type you are using does not have the VSMARKERTYPEFLAG_DRAW_BOX flag set.

BMIndex - Bitmap to display at the line specified. Specify 0 for no bitmap. The following are names of bitmaps used for debugging:

_breakpt.bmp
Indicates a line with an enabled breakpoint.
_breakpn.bmp
Indicates a line with a disabled breakpoint.
_execpt.bmp
Indicates current execution line.
_stackex.bmp
Indicates a line on the execution call stack.
_watchpt.bmp
Indicates a line with an enabled watchpoint.

type - Type allocated by _MarkerTypeAlloc().

Message - HTML message displayed when mouse is over bitmap. Only a subset of HTML supported by our mini-HTML control is supported.

Returns: int

Returns index of new stream marker.

Applies To:

Edit Window, Editor Control

Category:

Marker Functions

See Also:

_StreamMarkerAdd
