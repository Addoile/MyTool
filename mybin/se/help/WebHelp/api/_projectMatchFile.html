void _projectMatchFile(_str workspaceFile, _str projectFile, _str file, _str(&list)[], boolean append)

Searches for a file in a specified project using prefix matching. Attempts to match only the filename, even if a full path is sent

Parameters:

workspaceFile - full path to workspace file

projectFile - full path to project file

file - file to search for - only the stripped filename will be used

list - list of matching files

Category:

Project Functions
