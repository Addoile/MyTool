_str slick_path_search(_str name, _str options="")

Used for searching for SlickEdit executables, Slick-C® macros, help files, and miscellaneous SlickEdit files. The second and optional parameter specifies a program and batch macro search. Search order is as follows:

Try path given with filename
Non-UNIX platforms: Try current directory.
Try paths specified in VSLICKPATH environment variable.
Try paths specified in PATH environment variable.

Returns: _str

If successful, a complete file specification for filename is returned. Otherwise '' is returned.

Category:

File Functions
