void _TreeSortCol(int ButtonIndex=0, _str PrimarySortOptions="", _str SecondarySortOptionsDEPRECATED="")

Sort by a column of a multi-column tree control

Parameters:

ButtonIndex - column index, 0 is the first column. If not specified, find first column with TREEBUTTON_SORT flag on

PrimarySortOptions - String of one or more of the following options:

'I'
Ignore case
'F'
Sort filenames
'N'
Sort numbers
'E'
Exact Case.
'D'
Descending. Sort is ascending if this option is not specified.

SecondarySortOptions - Deprecated

Applies To:

Tree View

Category:

Tree View Methods
