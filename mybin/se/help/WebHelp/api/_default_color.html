_str _default_color(int ColorIndex, int fg_color=-1, int bg_color=-1, int font_flags=-1, int ParentIndex=0)

Sets the screen element corresponding to ColorIndex to the foreground and background colors given. The fg_color and bg_color parameters specify an RGB color. Use the _rgb function to build these colors.

Parameters:

ColorIndex - is one of the following constants defined in "slick.sh":

  • CFG_CLINE
  • CFG_CURSOR
  • CFG_SELECTION
  • CFG_SELECTED_CLINE
  • CFG_MESSAGE
  • CFG_STATUS
  • CFG_WINDOW_TEXT

fg_color - foreground color (RGB)

bg_color - background color (RGB)

font_flags - is one of the following font flags:

  • F_BOLD
  • F_ITALIC
  • F_STRIKE_THRU
  • F_UNDERLINE
  • F_INHERIT_STYLE

ParentIndex - parent color index to inherit color or font attributes from.

Returns: _str

Returns the current foreground, background, and font flags setting for the field specified each separated with a space.

Category:

Miscellaneous Functions
