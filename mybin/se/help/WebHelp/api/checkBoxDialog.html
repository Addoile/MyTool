int checkBoxDialog(_str formTitle, _str label, _str checkBoxLabel, int buttons, int defaultValue, _str retrieveName='', _str help='')

Displays a generic form with a label and a single checkbox control. This can be useful for asking a question that has a related setting that can be set with a checkbox (for example, a "Never prompt again" checkbox).

Parameters:

formTitle - title text for form

label - any text for a label message

checkBoxLabel - label for the checkbox

buttons - MB_OK, MB_OKCANCEL, MB_YESNO, MB_YESNOCANCEL

defaultValue - default checkbox value (0 or 1)

retrieveName - a retrieve name to be used to remember the last checkbox value. If a retrieved value is found, it overrides the defaultValue

help - help item for form

Returns: int

int IDOK, IDCANCEL, IDYES, IDNO

Category:

Forms
