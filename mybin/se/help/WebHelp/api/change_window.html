_command int change_window(typeless arg1="", typeless arg2="", typeless direction="")

Allows you to point with the cursor keys to the window you wish to change to.

Parameters:

direction - UP, DOWN, LEFT, or RIGHT

Returns: int

0 if successful. Otherwise 1 is returned indicating user cancelled or there is no window where the user pointed. On error, message is displayed.

Applies To:

Edit Window

Category:

Window Functions

See Also:

next_window, _prev_window, move_edge, window_left, window_right, window_above, window_below
