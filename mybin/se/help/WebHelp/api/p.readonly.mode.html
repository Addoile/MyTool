boolean p_readonly_mode

This property is identical to the p_ReadOnly property.

Edit Window, Editor Control: Determines whether the current buffer is read only. This property may have no effect depending on the setting of p_ProtectReadOnlyMode and _default_option(VSOPTION_PROTECT_READONLY_MODE). Use the _QReadOnly() method to test whether you should allow editing because it tests all necessary options.

Text Box: Determines whether the text in a text box can be modified. This is some times more useful than disabling the control because the user can still make a selection and copy the text to the clipboard. In addition, the color of the text is more readable when the text box is read only (p_ReadOnly=true) than when it is disabled (p_enabled=false).

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
