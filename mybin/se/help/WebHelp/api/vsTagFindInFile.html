int VSAPI vsTagFindInFile(const char *pszFilename, int find_first=a)

Find the first tag in the given file. Tags are returned unsorted. Use vsTagGetTagInfo to extract the details about the tag.

Parameters:

pszFilename - File name as stored in database.

find_first - find first match or next match

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Category:

Tagging Functions
