int dbg_get_method(int class_id, int method_id, _str &name, _str &class_name, _str &signature, _str &return_type, _str &file_name, int &flags)

Get the details about a method of a class.

Parameters:

class_id - index of class in class table

method_id - index of method in list of classes within specified class

name - (reference) name of method

class_name - (reference) name of class

signature - (reference) parameters of method

return_type - (reference) return type of method

file_name - (reference) source file containing this method

flags - (reference) bitset of DEBUG_MODEL_FLAG_

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
