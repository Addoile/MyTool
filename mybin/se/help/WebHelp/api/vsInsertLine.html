void VSAPI vsInsertLine(int wid, const char *pBuf, int BufLen=-)

Inserts line into editor buffer after the current line.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

pBuf - Data for new line. Should not include line separator characters except for binary record files.

BufLen - Number of characters in pBuf. -1 specifies that pBuf is null terminate and length can be determined from it.

Example:

vsInsertLine(0,"add line after current line",-1);

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

vsGetLine, vsReplaceLine, vsDeleteLine, vsGetText
