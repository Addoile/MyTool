void parse_error_re(_str &filename, _str &line, _str &col, _str &err_msg, typeless view_id='', _str alt_word='')

Parse error lines using regular expressions read from the ErrorRE.xml configuration file

Parameters:

filename - Output parameter for the file containing the error

line - Output parameter for the line number in filename

col - Output parameter for the column position on line in filename

err_msg - Output parameter for the error message
