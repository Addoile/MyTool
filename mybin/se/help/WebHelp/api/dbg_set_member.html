int dbg_set_member(int thread_id, int frame_id, _str member_path, _str value)

Set the value of a member of 'this' object.

Parameters:

thread_id - index of thread

frame_id - index of stack frame

member_path - path to member in list of member variables

value - new value to set member to

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
