boolean p_tab_stop

Determines whether Tab and Shift+Tab keys switch focus to the control.

Applies To:

Text Box, Editor Control, Command Button, Radio Button, Check Box, Combo Box, List Box, Hscroll Bar, Vscroll Bar, Drive List, File List Box, Directory List Box, Gauge, Tree View
