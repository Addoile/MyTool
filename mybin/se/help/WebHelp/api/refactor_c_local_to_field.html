int refactor_c_local_to_field(int handle, _str filename, _str symbolName, int beginSeekPos, int endSeekPos, _str fieldName, int flags)

Convert local variable to a field

Parameters:

handle - Handle to transaction

filename - Filename to find symbol in

symbolName - Name of lvar to convert to field

beginSeekPos - Seek position of symbol or position to start search

endSeekPos - Seek position to end symbol search (0 if begin is actual symbol)

fieldName - New name for field

flags - Flags for access modifiers and make field static

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
