int _load_template(int index_or_wid, int parent, _str options, ...)

Load a Slick-C resource template by index.

Parameters:

index_or_wid - Resource template index or window id of already-created object. If 'W' or 'R' options are used, then it is the window id of the already-created object. Otherwise it is the resource template.

parent - Parent window id.

options - Load options.

  • 'H' = Load template hidden.
  • 'E' = Used internally by dialog editor. Load template in edit mode.
  • 'A' = Load template with ON_CREATE arguments passed on the interpreter stack. IMPORTANT: See VSLOAD_ARGPARAMS for more explanation before you attempt to use this option.
  • 'W' = Reload template with provided wid passed in index_or_wid parameter.
  • 'R' = Reinitialize template with provided wid passed in index_or_wid parameter. p_user, p_user2, * window geometry, default property values are reset.
  • 'C' = Do not call ON_CREATE events for loaded template. This includes ON_LOAD and ON_RESIZE. Use 'W' to call these events later.
  • 'P' = Load template as a child of parent wid.
  • 'N' = Load template as a child of parent wid and do not draw a border.
  • 'S' = Create children of parent. Parent wid must be a handle to the already created parent.
  • 'Y' = Load template with ON_CREATE arguments passed as an array. Next argument is the array of ON_CREATE arguments.
  • 'M' = Use MDI callbacks. Useful for setting status line text when window is not an MDI child.
  • 'T' = Load template as toolbar.

Returns: int

Window id of loaded resource. Returns <0 on error.

Category:
