void VSAPI vsPropSetB(int wid, int prop_id, const void *pBuf, int BufLen)

Sets value of string property to pBuf. For documentation on the properties, you must translate the VSP_??? constants into the Slick-C® property name. For example, VSP_BUFNAME corresponds to p_buf_name and VSP_LINE corresponds to p_line.

Parameters:

wid - Window id of object or index to resource returned by vsFindIndex, vsNameMatch, VSP_CHILD, or VSP_NEXT. 0 specifies the current object.

prop_id - One of the VSP_??? constants in "vs.h".

pBuf - Buffer containing new value.

BufLen - Number of charactes to copy from pBuf.

Example:

     vsPropSetB(0,VSP_BUFNAME,"c:\\autoexec.bat",15);

Applies To:

All Window Objects

Category:

Miscellaneous Functions

See Also:

vsPropGetB, vsPropGetZ, vsPropSetZ, vsPropSetB, vsPropGetI, vsPropGetI64, vsPropSetI, vsPropSetI64, vsPropGetL, vsPropSetL
