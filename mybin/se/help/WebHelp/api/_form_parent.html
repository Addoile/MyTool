int _form_parent()

Returns: int

Returns the parent window id of the active form. If the p_parent property of the active form is 0, the active MDI edit window is returned (_mdi.p_child). Some dialog boxes such as the Replace and Spelling dialog boxes determine what buffer they are operating on by calling this function. It is probably sufficient to use the expression (p_active_form.p_parent). We added this function just in case it is needed for portability.

Applies To:

All Window Objects

Category:

Form Functions
