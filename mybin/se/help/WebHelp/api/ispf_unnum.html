_command void ispf_unnum(_str arglist="")

The unnumber command immediately blanks out the line numbers in a file.

Syntax:

   UNNUMBER  [ STD ] [ COBOL ]

Arguments:

STD or STANDARD
Line numbers are in the standard sequence columns (73-80).
COB or COBOL
Line numbers are as in COBOL, columns 1-6.

Note: Make sure that the line number columns (1-6 for COBOL or 73-80) do not have data in them before unnumbering the buffer. Otherwise, data may be overwritten with line numbers. Normally, this is detectable and you will be warned that data was overwritten, but if the data was numeric, you may not be warned.

Category:

ISPF Primary Commands
