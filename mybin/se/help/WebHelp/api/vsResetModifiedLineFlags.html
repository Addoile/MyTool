void VSAPI vsResetModifiedLineFlags(int wid, int Reserved=a)

Turns off the VSLF_INSERTED_LINE and VSLF_MODIFY flags for all lines in the specified editor control.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

vsQLineFlags, vsSetLineFlags, vsResetModifiedLineFlags
