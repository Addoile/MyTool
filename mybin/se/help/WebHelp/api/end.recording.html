_command typeless end_recording(boolean quiet=false, _str suffix="")

Ends the definition of a recorded macro and names it last_recorded_macro.

Parameters:

quiet - Save macro without prompting.

suffix - Suffix to append to macro name

Returns: typeless

Returns 0 if successful and displays message. Otherwise a non-zero value is returned and a message or message box is displayed.

Category:

Macro Programming Functions
