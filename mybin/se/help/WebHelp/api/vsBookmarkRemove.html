void VSAPI vsBookmarkRemove(int i, int free_markid=a)

Deletes a bookmark.

Parameters:

i - Index of bookmark where 0<=i<vsBookmarkQCount().

free_markid - When non-zero, vsFreeSelection is called for the markid belonging to this bookmark.

Category:
