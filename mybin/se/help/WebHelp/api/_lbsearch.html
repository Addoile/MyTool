_str _lbsearch(_str text, _str options="")

Searches for a whole item in the list box which matches the text string given. If the options parameter contains the R option, this function will search for an item which contains the text string given. The search always starts from the top of the list.

Parameters:

options - is a string of zero or more of the following letters:

E
Match case exactly
I
(Default) Ignore case
R
Interpret text as a regular expression.
U
Interpret text as a UNIX regular expression.
B
Interpret text as a Brief regular expression.

Returns: _str

Returns 0 if match is found

Applies To:

List Box Combo Box

Categories:

List Box Methods, Combo Box Methods
