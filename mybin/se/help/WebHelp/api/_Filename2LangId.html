_str _Filename2LangId(_str filename, int options=0)

Returns the language mode associated with the given file.

The Language Options dialog box allows you to map an extension that to a language mode. This function performs that translation, and will also use the buffer name to determine the language mode.

If the file's actual extension matches a _[ext]_Filename2LangId() callback, it will first try the callback to see if the file, based on it's path or name should be referred to an alternate language.

Otherwise, if the file's actual extension matches a suffix_[ext] callback, it will open the file in a temporary view and try the callback to determine the file's actual language type.

Parameters:

file_name - source file name with path

options - one or more of the following flags ORed together:

F2LI_NO_CHECK_OPEN_BUFFERS
Do not go through open buffers to determine lang id. Normally, we see if the file is already open and then return its p_LangId property. If this flag is included, this step is skipped
F2LI_NO_CHECK_PERFILE_DATA
Do not go through data saved for previously opened files to determine lang id. Normally, we check the data stored in perfile.xml to see if this file has been opened before and has a stored lang id. If this flag is included, this step is skipped

Returns: _str

Returns the language ID for the given file.

Category:

Miscellaneous Functions
