_str _LexerName2LangId(_str lexer_name)

Locates the first language specification that uses the given lexer name. Note there is no guarantee that the mapping form lexer names to languages should be unique, so use this function sparingly, as it will *only* find the first match.

Parameters:

lexer_name - Language lexer name

Returns: _str

The language ID for the language corresponding to 'lexer_name'.

Category:

Miscellaneous Functions
