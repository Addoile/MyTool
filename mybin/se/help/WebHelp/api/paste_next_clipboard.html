_command void paste_next_clipboard(int nextOrPrev=1)

Cycles clipboards and pastes next clipboard item to buffer. Item remains selected and ignores deselect paste option, this allows for quickly cycling through all clipboards.

Parameters:

nextOrPrev - - option to use next (default [greater than 0]) or previous clipboard (less than or equal to 0)

Applies To:

Edit Window, Editor Control

Category:

Clipboard Functions
