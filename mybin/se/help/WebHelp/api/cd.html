_command typeless cd(_str cmdline="", _str quiet="")

Changes the current working directory to the drive and path if given. A current directory message is displayed. By default, this command supports specifying directory aliases for driveNpath and will change directory in the build window. Use the Change Directory dialog box ("File", "Change Directory...") to change these defaults and press the save settings button.

Parameters:

cmdline - is a string in the format: [+p | -p] [+a | -a] driveNpath 

 The options have the following meaning:
 +p      Change directory in the build window.
 -p      Don't change directory in the build window.
 +a      Support directory aliases for driveNpath.
 -a      Do not support directory aliases for driveNpath.

Returns: typeless

Returns 0 if successful. Common error code is PATH_NOT_FOUND_RC.

Category:

File Functions
