int _copy_to_cursor(_str markid="", int MarkFlags=-1)

Copies the selection specified by mark_id to cursor. Character or block selections are inserted before the character at the cursor. Line selections are always inserted after the current line. Resulting selection is always on destination text.

Returns: int

Returns 0 if successful. Common return codes are TEXT_NOT_SELECTED_RC, SOURCE_DEST_CONFLICT_RC, and INVALID_SELECTION_HANDLE_RC. On error, message is displayed.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
