typeless _justifyok.on_create(typeless word_wrap_style=0)

Displays Justification dialog box. See justify function for information on justify_option.

Returns: typeless

Returns '' if user cancelled the dialog box. Returns the justify option selected by the user. You may use the justify function to set the justification based on this return value.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
