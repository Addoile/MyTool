int VSAPI vsGetBufferFromView(char **ppBuffer, int *piLen, int iWindowId)

Get char buffer from window id. Use vsFree to free buffer data.

Parameters:

ppBuffer - Address of char * variable to store data in

piLen - Address of int variable to store length of the data in

iWindowId - Window id to store in ppBuffer

Returns: int

0 if successful

Category:

File Functions
