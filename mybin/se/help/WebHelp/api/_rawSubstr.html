_str _rawSubstr(_str &string, int StartCol, int Width=-1)

Returns: _str

Returns part of the input string starting from StartCol including the rest of the string or Width characters if specified. The input string is converted to the same format as the buffers raw data before the return result is computed. See "Unicode and SBCS/DBCS Macro Programming." This function is identical to the substr function except when a double byte or UTF-8 sequence is bisected, the remaining bisected characters are replaced with zeros. This function does not yet support the pad argument.

Example:

        // This will always mismatch if the character at index 5 is part of a
DBCS
        // or UTF-8 sequence.
        if (_rawSubstr(string,5,1)=='+') {
             ...
        }

Applies To:

Edit Window, Editor Control
