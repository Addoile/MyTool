int _create_temp_view(int &temp_wid, _str load_options='', _str buf_name='', boolean doSelectEditorLanguage=false, int more_buf_flags=0)

Creates a temporary window, and buffer. This function is used to create lists or process files you any editing feature without the buffer being displayed.

Parameters:

temp_wid - Window id of file that is opened

load_options - may be "" or only may contain the following

  • +bi <buf_id>
  • +d
  • +b

buf_name - Specified buffer name for temp window/buffer.

doSelectEditorLanguage - When true, _SetEditorLanguage() is called to initialize the language setup for the buffer. In addition, build_load_options() to generate all default load options which typically turns on undo.

more_buf_flags - Allows more buffer flags (typically VSBUFFLAG_THROW_AWAY_CHANGES) to be added to buffers loaded from disk.

Returns: int

If successful, temp_wid is set to the window created, the new window is active, and the original window id is returned. On error, 0 is returned and message box is displayed.

Example:

#include "slick.sh"
defmain()
{
    orig_wid=_create_temp_view(temp_wid);
    if (orig_wid=='') return(1);
    // The buffer and window allocated by _create_temp_view are active
    _lbadd_item('a');
    _lbadd_item('b');
    _lbadd_item('c');
    //  The original window must be activated before showing the _sellist_form
    activate_window(orig_wid);
    result=show('_sellist_form -mdi -modal',
                "Sample Selection List",
                SL_VIEWID|SL_SELECTCLINE, // Indicate the next argument is a window id
                temp_wid,
                "OK",
                "",  // Help item
                '',  // Use default font
                ""      // Call back function
               );
      if (result) {
           message("Item selected is "result);
      } else {
           message("Selection list cancelled");
      }
}
