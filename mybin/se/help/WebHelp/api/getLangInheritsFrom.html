_str getLangInheritsFrom(_str langID, _str defaultValue=null)

Gets the inheritance value for this language, which defines if a language inherits callbacks from another language.

Parameters:

langID - language ID (see p_LangId)

defaultValue - if the value is not available, then we just return this value. If this parameter is not specified, then the default value for the overall def-var is used.

Returns: _str

inheritance value for this language (language whose callbacks this language inherits from)

Category:

LanguageSettings API
