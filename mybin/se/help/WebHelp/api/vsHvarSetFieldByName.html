int VSAPI vsHvarSetFieldByName(VSHVAR hvar, VSPSZ name, VSHVAR value)

Modify the contents of the given non-static data member of a class or struct.

Parameters:

hvar - handle for interpreter variable

name - name of non-static data member

value - value to set field to

Returns: int

0 on success. Returns CLASS_MEMBER_NOT_FOUND_RC if the field is not found. Returns INVALID_HVAR_ARGUMENT_RC if 'hvar' is not a struct or class instance.

Category:

Miscellaneous Functions

See Also:

vsHvarFieldName, vsHvarSetField
