_command typeless case_indirect(_str key='')

Executes the key binding for the other case of the alphabetic key pressed. Bind this command to multiple key sequence key bindings that end with an alphabetic key. For example, if Ctrl+X 'k' is bound to the quit command and Ctrl+X 'K' is bound to the case_indirect command. When Ctrl+X 'K' is pressed, the quit command is executed. ALT KEYS ARE NOT CAPS LOCK SENSITIVE! This command should not be invoked from the command line.

Returns: typeless

If key != '' translated key is returned. Otherwise '' is returned.
