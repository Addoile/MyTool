void _SetDefaultExtensionOptions(_str extension, VS_LANGUAGE_OPTIONS &langOptions, int reserved)

Deprecated. Use _SetDefaultLanguageOptions

Sets language specific options for a specific file extension.

Parameters:

extension - File extension. For list of file extension types, use our File Extension Manager dialog ("Tools", "Options...", "File Extension Manager")

langOptions - New language specific options for pszExtension. Since all options are must be set, use the vsGetDefaultExtensionOptions first to query the existing value before setting new values.

reserved - unused

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions

See Also:

_GetDefaultExtensionOptions, _SetDefaultLanguageOptions, _SetExtensionReferTo, _DeleteExtensionOptions, _DeleteLanguageOptions
