typeless _callmethod(_str indexOrName, ...)

Call the given class method and return it's result.

Parameters:

indexOrName - Name or names table index of method to call.

argument1 - first argument for method

argument2 - second argument for method

Returns: typeless

Returns the return value of the given method.

Example:

class AAAA {
   void saySomething() {
      message("something");
   }
};
int testCallMethod() {
   AAAA a;
   a._callmethod("saySomething");
   index := a._findmethod("saySomething");
   if (index_callbale(index)) {
      a._callmethod(index);
   }
}
