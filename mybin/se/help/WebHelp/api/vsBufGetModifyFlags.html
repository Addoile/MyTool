int VSAPI vsBufGetModifyFlags(int buf_id)

Return the buffer modify flags for the given buffer.

Parameters:

buf_id - Buffer id returned by vsPropGetI(wid,VSP_BUF_ID), vsBufEdit, or vsBufMatch.

Returns: int

bitset of VSMODIFYFLAG_

Attention:

This function is thread-safe.

Category:

Buffer Functions
