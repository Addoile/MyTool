int _lbfind_and_select_item(_str item, _str options='', boolean selectTopIfNotFound=false)

Searches for an item in the list box. If the item is found, we select it.

Parameters:

item - item to search for.

options - 'i' for case-insensitive searching, 'e' for exact match, '' to use the existing p_case_sensitive value

selectTopIfNotFound - if the item is not found, the topmost item in the list is selected

Returns: int

int 0 if item is found, error code otherwise

Applies To:

List Box, Combo Box

Categories:

List Box Methods, Combo Box Methods
