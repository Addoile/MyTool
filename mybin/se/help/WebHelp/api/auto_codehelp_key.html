_command void auto_codehelp_key()

Command normally bound to keystrokes that should initiate auto-list-members. For example, in C and C++, this could be ".", ":", or ">" (last char of "->").

Does not invoke function help if on command line or in a comment or string, or if auto function help is turned off.
