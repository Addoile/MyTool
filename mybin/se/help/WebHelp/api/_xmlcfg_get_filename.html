_str _xmlcfg_get_filename(int handle)

Gets the filename for the XMLCFG tree specified. This is the name specified in the _xmlcfg_open and _xmlcfg_create calls.

Parameters:

iHandle - XMLCFG tree handle returned from _xmlcfg_open, _xmlcfg_open_from_buffer, or _xmlcfg_create

Category:

XMLCFG Functions
