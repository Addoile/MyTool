int p_ToolbarBorder

When this property is true, the caption at the top of a form is smaller. This property is typically used for floating tool bar forms. This property is not supported when the object is displayed and is ignored on all platforms except Windows 95/98 and Windows NT version >=4.0.

VSTBBORDER_BORDER
May display extra necessary border.
VSTBBORDER_GRABBARS
Displays double bars so that the toolbar (usually one where buttons can be added) can be dragged with the mouse.

Applies To:

Form

Category:

Form Properties
