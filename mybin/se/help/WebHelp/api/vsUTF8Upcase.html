void VSAPI vsUTF8Upcase(char *p, size_t buflen)

Converts UTF-8 text to upper case. Under Windows, this function relies on the WIN32 function CharUpperBuffW which only support characters below 65536. Characters above 65537 are not changed.

Parameters:

pBuf - Buffer to convert.

BufLen - Length of buffer to convert.
