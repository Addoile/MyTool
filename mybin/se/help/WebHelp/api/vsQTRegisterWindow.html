int VSAPI vsQTRegisterWindow(QWidget *pwidget, int oi=VSOI_FORM)

Registers a system window as a SlickEdit object. There are different reasons for registering form (outer) windows. One reason is so that a window created outside of SlickEdits dialog boxes will get disabled when SlickEdit displays a modal dialog box. Another reason is so that you can make a particular window the parent of one of our dialog boxes. Make sure you call vsUnregisterWindow before you destroy your window.

Parameters:

pwidget - QWidget to register

oi - Object type. One of VSOI_??? constants. For now this must be VSOI_FORM.

Returns: int

Returns window id of newly registered window.

Category:

Window Functions
