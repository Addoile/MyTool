int def_cw_analyze_lines_max

Maximum number of lines in a block comment or in a block of consecutive line comments that will be analyzed when trying to automatically determine the proper width on the comment. Comment wrapping will look analyze at most def_cw_analyze_lines_max number of lines above the cursor position and def_cw_analyze_lines_max number of lines below the cursor postion.

Default:

100

Category:

Configuration Variables
