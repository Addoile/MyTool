int def_hanging_statements_after_col

When deleting an open brace from a brace block that encloses a single line statement, leave the statement hanging instead of pulling it up to the same line if the current column is greater than or equal to this value. To always force hanging statements, set this number to 0.

Example:

if (condition)
   doSomething();
instead of 
if (condition) doSomething();

Default:

40

Category:

Configuration Variables
