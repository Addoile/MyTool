_command typeless save_all_forms()

Saves all modified forms currently being edited.

Returns: typeless

Returns 0 if successful. Common return codes are ACCESS_DENIED_RC, ERROR_OPENING_FILE_RC, ERROR_WRITING_FILE_RC, INSUFFICIENT_DISK_SPACE_RC, DRIVE_NOT_READY_RC, and PATH_NOT_FOUND_RC. On error, message box is displayed.

Category:

File Functions
