int p_RLine

Determines the current line number. Note that p_RLine (short for "real line") does not include lines which are non-savable (lines with NOSAVE_LF flag set). Use the p_line property to count lines which will not be saved in your file. The Top of File line is line 0. A list box has a line 0, but it is never displayed. If the current line number is not already known, it is determined. If the p_RLine property is set to a value greater than the number of lines in the buffer, the cursor is placed on the last line. This property is available only when the control is displayed.

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
