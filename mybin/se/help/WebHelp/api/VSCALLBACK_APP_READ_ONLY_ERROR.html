17 VSCALLBACK_APP_READ_ONLY_ERROR

int (VSAPI *pfnReadOnlyError)();
This callback is called when the user attempts to modify a read-only file. By default, this callback displays a file is read-only message box and/or optional checks out a file.

Returns: 17

Returns 0 to indicate that no default processing is necessary. A non- zero value indicates that the default processing should take place.
