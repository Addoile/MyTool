_command typeless reflow_selection(typeless new_right_margin="")

Reflows the text within the selection according to the margin settings. Character selection is not supported.

Returns: typeless

Returns 0 if successful. Common return codes are 1 (buffer mark must be active), TOO_MANY_SELECTIONS_RC, TEXT_NOT_SELECTED_RC, and LINE_OR_BLOCK_SELECTION_REQUIRED_RC. On error, message is displayed.

Applies To:

Edit Window, Editor Control

Category:

Selection Functions

See Also:

margins, gui_margins, reflow_paragraph, gui_justify, justify
