int dbg_session_enable_breakpoint(int session_id, int breakpoint_id)

Enable the given breakpoint

Parameters:

session_id - debugger session ID (1 .. max_session)

breakpoint_id - Breakpoint to enable

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

   int status = dbg_session_enable_breakpoint(session_id, 1);
   if (status) {
      return status;
   }

Category:

Debugger Functions
