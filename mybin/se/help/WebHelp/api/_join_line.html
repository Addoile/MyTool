int _join_line()

Joins the next line to the end of the current line.

Returns: int

Returns 0 if successful. The join is aborted if the join will create a line longer than the truncation length and the truncation length is non-zero (see p_TruncateLength property). A non-zero value is returned if the join is aborted.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
