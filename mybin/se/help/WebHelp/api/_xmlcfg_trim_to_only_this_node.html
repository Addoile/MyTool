int _xmlcfg_trim_to_only_this_node(int handle, int indexToSave)

Removes all the nodes in the XML DOM that are not direct parents or children of the given node.

Parameters:

handle - Handle to an XMLCFG tree returned by _xmlcfg_open(), _xmlcfg_create(), or _xmlcfg_open_from_buffer().

indexToSave - Node index of element to save parents and children of

Returns: int

0 on success, <0 on error.

Category:

XMLCFG Functions
