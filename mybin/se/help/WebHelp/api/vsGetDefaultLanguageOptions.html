int VSAPI vsGetDefaultLanguageOptions(const char *pszLangId, const char *pszExtension, VS_LANGUAGE_OPTIONS *pLangOptions, int version=VS_LANGUAGE_OPTIONS_VERSION)

Gets language specific options for a specific language type.

Parameters:

pszLangId - Language ID (see p_LangId). For list of language types, use our Language Manager dialog ("Tools", "Options...", "Language Manager")

pszExtension - File extension referred to pszLangId

pLangOptions - Initialized to language specific options for pszLangId.

version - normally VS_LANGUAGE_OPTIONS_VERSION

Returns: int

Returns 0 if successful.

Since:

13.0

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions

See Also:

vsSetDefaultLanguageOptions, vsSetDefaultExtensionReferTo, vsDeleteLanguageOptions, vsUpdateExistingBufferLanguageOptions, vsSetDefaultLanguageOptions
