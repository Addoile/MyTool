_command int prev_char()

Move the cursor left one character. If the cursor is at the beginning of the line, move down to the end of the previous line.

Returns: int

0 on success, <0 if at then beginning of the file.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods

See Also:

left, up, next_char, traverse_char
