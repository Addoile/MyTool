char*VSAPI vsGetEmulation(char *pszResult, int MaxResult, int *pMaxResultLen=a)

Parameters:

pszResult - Ouput buffer for current emulation string. Currently supported emulations are "" (SlickEdit), "brief-keys", "emacs-keys", "vi- keys", "gnuemacs-keys", "vcpp-keys" (Visual C++), "ispf-keys" and "windows- keys" (CUA). This can be 0.

MaxResult - Number of characters allocated to pszResult. We recommend VSMAXEMULATION.

pMaxResult - If this is not 0, this is set to the number of characters you need to allocated.

Returns: char*VSAPI

Returns pszResult. To set the emulation, call vsExecute with the command "emulate emu_mode", where emu_mode is one of the following value, for example, "emulate vi" switches the editor into vi emulation.
