void VSAPI vsBookmarkRestore(int wid, int vsbmflags=-)

Restores inactive bookmarks for the file specified by wid. This function is typically called when a new file is loaded, so that inactive bookmarks may be restored.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

vsbmflags - One or more of these flags must be in the bookmark for the bookmark to be restored.

Category:

Bookmark Functions

See Also:

vsBookmarkAdd, vsBookmarkRemove, vsBookmarkRestore, vsBookmarkGetInfo, vsBookmarkFind, vsBookmarkQCount
