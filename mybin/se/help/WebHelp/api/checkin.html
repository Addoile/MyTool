_command int checkin(_str filename="")

The checkin command checks in the files specified into the version control system specified by the current project. If cmdline is not given or '', the Check In dialog box is displayed.

Returns: int

Returns 0 if successful.

Paraam:

options is a string of one or more of the following switches: 

+ or -c  Comment for all files.
+ or -p  Prompt for each comment
+ or -d  Discard changes
+ or -new   Create a new archive file

You may specify '.' to check in the current buffer ("checkin .").

Example:

cmdline is a string in the format: [options] file1 file2 file3 ...
