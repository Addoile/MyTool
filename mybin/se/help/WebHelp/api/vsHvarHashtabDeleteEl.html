void VSAPI vsHvarHashtabDeleteEl(HVAR hvarHashtab, VSPSZ pBuf, int BufLen=-)

Deletes an element from hash table.

Parameters:

hvarHashtab - hash table to delete item from

pBuf - index or key corresponding to item to look for

BufLen - length of hash key

Example:

   vsHvarMakeEmpty(VSHVAR_RC);
   hvar=vsHvarHashtabEl(VSHVAR_RC,"styles");
   vsHvarSetI(hvar,1);
   vsHvarHashtabDeleteEl(VSHVAR_RC,"styles");

Category:

Miscellaneous Functions

See Also:

vsHvarNextEl, vsHvarMakeEmpty, vsHvarIsEmpty, vsHvarFormat, vsHvarIndexIn
