VSSTATIC int vsUTF8CharWrite(unsigned wch, char *pdest)

Converts a UTF-32 character range 0x0-0x7fffffff to UTF-8. The caller must handle converting UTF-16 (windows UNICODE) surrogates into the UTF-32 character equivalent. That is, the caller must convert 0xD800,0x0DC00 to 0x10000.

Parameters:

wch - UTF-32 character in range 0x0-0x7fffffff.

pdest - Pointer to UTF-8 destination buffer Must contain enough space for UTF-8 character. In the worst case, 6 bytes are required.

Returns: VSSTATIC int

Returns number of UTF-8 bytes required to encode the UTF-32 character given. 0 is returned if the UTF-32 character is too large (above 0x7fffffff).
