void VSAPI vsMarkerTypeSetColorIndex(int type, int ColorIndex)

Sets the marker type color index for the marker type specified. Currently only supported by stream markers with VSMARKERTYPEFLAG_DRAW_SCROLL_BAR_MARKER flag.

Parameters:

type - Index of marker type.

ColorIndex - Color index to use for markers of type type.

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
