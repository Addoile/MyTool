_command int expand_surround_with(_str text=' ', boolean delete_on_error=false, _str prefix='', boolean prompt=true)

Inserts the specified text, selects it, and then runs surround_with

Parameters:

text - The text to insert and surround

delete_on_error - If true and there is an error expanding the template, delete the inserted text

prefix - Specifies the name of the surround-with template to use. If no argument is given or if multiple possible templates are found, the user may be prompted to select the template they wish to use.

prompt - if false and multiple matches are found, it is handled as if no matches were found

Returns: int

zero if a code template was expanded

Editor Control Methods
