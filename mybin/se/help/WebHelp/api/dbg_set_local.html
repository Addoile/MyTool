int dbg_set_local(int thread_id, int frame_id, _str local_path, _str value)

Set the value of a local variable in the current stack frame.

Parameters:

thread_id - index of thread

frame_id - index of stack frame

local_path - path to local in list of local variables

value - new value to set local variable to

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
