int VSAPI vsFileWrite(int fh, const void *pBuf, int BufLen)

Write number of bytes specified by BufLen to the file specified by fh. pBuf points to the data to be written.

Returns: int

If successful, the number of bytes written is returned. Otherwise a negative error code is returned.

Category:

File Functions
