boolean p_ProtectReadOnlyMode

Determines whether how read only mode should be protected. Unlike the p_readonly_mode property which is per buffer, this property is per window.

VSPROTECTREADONLYMODE_OPTIONAL
If the buffer is in read only mode and the user has selected to protect read only mode (_default_option (VSOPTION_PROTECT_READONLY_MODE) )

VSPROTECTREADONLYMODE_ALWAYS
If the buffer is in read only mode, the buffer is protected.

VSPROTECTREADONLYMODE_NEVER
Read only mode is never protected.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Properties, Editor Control Properties
