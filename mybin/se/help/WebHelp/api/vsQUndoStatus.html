int VSAPI vsQUndoStatus(int wid, char undo_option='U')

Parameters:

wid - Window id of editor control. 0 specifies the current object.

undo_option - Specify 'U' to test for undo information. Specify 'R' to test for redo information.

Returns: int

If undo_option is 'U', NOTHING_TO_UNDO_RC if there is no undo information. If undo_option is 'R', NOTHING_TO_REDO_RC if there is no redo information. Use this function to determine whether menu items which execute the undo, undo_cursor, or redo commands should be enabled.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
