typeless _GetCommentWrapFlags(int commentWrapOption, _str lang='')

Get the comment wrap flags.

The options are stored per language. If the options are not yet defined for the specified language, then use CFcommentWrapDefaults as the default.

Parameters:

commentWrapOption - comment wrap option to extract

lang - language ID, see {@p_LangId}

Returns: typeless

structure of type CommentWrapSettings_t.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
