int VSAPI vsQLastIndex(char option)

Parameters:

option - Indicates last index information to return.

'C'
Returns the index of the last command or function executed by a key, vsCallKey, or vsExecute.

Note that this is different than the Slick-C® last_index function which updates this value when call_index is called. Call vsSetLastIndex to explicitly update this value.
'L'
Returns the index of the last command or function executed by a key or the vsCallKey function.
'K'
Returns the index of the last event table used to determine a key binding.
'W'
Returns the command line/menu wait value. This flag is used by a command to determine whether it was invoke by vsExecute (command line or menu) or a key press. This flag is set to non-zero when vsExecute is called and 0 when a key is pressed or vsCallKey is called.

Returns: int

Returns a last index corresponding to option given.

Category:

Keyboard Functions
