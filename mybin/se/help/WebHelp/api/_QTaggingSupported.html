boolean _QTaggingSupported(int wid=0, _str lang=null)

Parameters:

wid - Window id of MDI frame window.

lang - current language ID (default=p_LangId)

Returns: boolean

Returns non-zero value if this editor controls language mode supports tag navigation commands include show_procs, list_tags, next_proc, prev_proc, find_tag, and push_tag.

Since:

13.0

Applies To:

Edit Window, Editor Control

Category:

Tagging Functions
