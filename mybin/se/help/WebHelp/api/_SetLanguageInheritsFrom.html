void _SetLanguageInheritsFrom(_str lang, _str parent)

Allows you to create a language which has it's own set of options, but can inherit support callbacks from another language.

Parameters:

lang - Language ID (see p_LangId

parent - lang is set to inherit language specific callbacks from the language specified by parent. If NULL, remove language inheritance for lang.

Since:

13.0

Applies To:

Edit Window, Editor Control

Category:

Tagging Functions

See Also:

_SetDefaultLanguageOptions, _DeleteLanguageOptions, _LanguageInheritsFrom, _FindLanguageCallbackIndex
