int tag_list_class_locals(int tree_wid, int tree_index, _str(&tag_files)[], _str tag_name, _str class_name, int filter_flags, int context_flags, var vnum_matches, int max_matches, boolean exact_match, boolean case_sensitive, ...)

Attempt to locate the given symbol in the given class by searching local variables. Recursively looks for symbols in enumerated types and anonymous unions (designated by having *both* the anonymous and 'maybe_var' tag flags).

Look at num_matches to see if any matches were found. Generally if (num_matches >= max_matches) there may be more matches, but the search terminated early.

For synchronization, macros should perform a tag_lock_matches(true) prior to invoking this function.

Parameters:

tree_wid - window id of tree control to insert into, 0 indicates to insert into a match set

tree_index - tree index to insert matches under

tag_files - symbol prefix to match

tag_name - name of class to search for matches

class_name - (reference to _str[]) tag files to search

filter_flags - Tag filter flags, only insert tags passing this filter. See tag_filter_type for more details.

context_flags - VS_TAGCONTEXT_*, tag context filter flags

vnum_matches - (Input, Output) number of matches

max_matches - maximum number of matches allowed

exact_match - exact match or prefix match (0)

case_sensitive - case sensitive (1) or case insensitive (0)

friend_list - (optional) List of friends to the current context

visited - (optional) hash table of prior results

depth - (optional) depth of recursive search

Returns: int

1 if the definition of the given class class_name is found, othewise returns 0, indicating that no matches were found.

Category:

Tagging Functions
