void VSAPI vsRubout(int wid, int Reserved=a)

Deletes a character if any to left of cursor and moves the cursor to left. When the left edge is hit, the text area is smooth scrolled or center scrolled depending on the scroll style.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods
