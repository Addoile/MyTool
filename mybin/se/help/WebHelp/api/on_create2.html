event on_create2()

All objects are sent an on_create2 event when a dialog box template is loaded with the show or _load_template function. No parameters are passed to the on_create2 event. We recommend that only user level 2 inheritance catch this event. User level 1 inheritance should catch the on_create event instead.

Applies To:

Text Box, Combo Box, List Box, Directory List Box, File List Box, Drive List, Command Button, Check Box, Radio Button, Frame, Hscroll Bar, Vscroll Bar, Spin, Gauge, Image, Form, Tree View

Categories:
