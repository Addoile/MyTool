void tag_tree_select_bitmap(int i_access, int i_type, int &leaf_flag, int &pic_member)

This function is used to get the picture indexes of the icons corresponding to the given i_access level and i_type category. You must call tag_tree_prepare_expand() prior to calling this function. i_access and i_type are typically obtained from tag_tree_filter_member.

Parameters:

i_access - Nenber access level, see tag_tree_filter_member2.

i_type - Item type for selecting bitmap, see tag_tree_filter_member2.

leaf_flag - (Output) is this item a container or leaf?

pic_member - (Output) picture index for bitmap

Category:

Tagging Functions

See Also:

_TreeAddItem, tag_tree_prepare_expand, tag_tree_insert_tag
