_command typeless prev_level()

Used in EMACS emulation. Searches for close parenthesis character ')', ']', or '}' and places the cursor on the matching open parenthesis. Searching for close parenthesis starts at character before cursor. Cursor position is unchanged if matching open parenthesis is not found. Comment or string literals may cause this command to match parenthesis incorrectly.

Returns: typeless

Returns 0 if successful. Common return values is TOO_MANY_MARKS_RC. On error, message displayed.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods

See Also:

next_level, cut_level, cut_prev_level
