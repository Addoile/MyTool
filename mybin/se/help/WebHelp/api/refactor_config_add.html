int refactor_config_add(_str configName, _str configHeader, _str includePath)

Add a new C++ compiler configuration to the master refactoring configuration file.

Parameters:

configName - name for this configuration

configHeader - header file to configure the compiler

includePath - PATHSEP deliminated include file path

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
