_str set_scroll_pos(int left_edge, int cursor_y)

Sets the left edge scroll position and the cursor y position for viewing the current line. The current line is unchanged. left_edge and cursor_y are in pixels. The p_cursor_y property may also set the cursor y position. However, the current line will not remain the same.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods
