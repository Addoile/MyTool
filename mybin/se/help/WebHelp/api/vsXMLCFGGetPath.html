const char*VSAPI vsXMLCFGGetPath(int iHandle, VSPSZ pszQueryStr, VSPSZ pszAttrName, int *pAttrValueLen=a, VSPSZ pszDefaultValue="")

Gets the value of an attribute

Parameters:

iHandle - Handle to an XMLCFG tree returned by vsXMLCFGOpen(), vsXMLCFGCreate(), or vsXMLCFGOpenFromBuffer().

pszQueryStr - This is a small subset of an XPath expression. See {@link_xmlcfg_find_simple}() for information on this parameter.

pszAttrName - Name of attribute to get value of.

pAttrValueLen - Number of bytes in returned attribute value.

pszDefaultValue - Value to return if attribute does not exist or an error occurs.

Returns: const char*VSAPI

attribute value if successful. Otherwise pszDefaultValue is returned.

Example:

// Find the first file element node under /project with name equal to
// "main.c" and get the "options" attribute value.
char *pszOptions=vsXMLCFGGetPath(handle,
    "/project/file[file-eq(@name,'main.c')]",
    "options"
    );
