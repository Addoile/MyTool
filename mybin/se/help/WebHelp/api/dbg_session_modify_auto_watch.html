int dbg_session_modify_auto_watch(int session_id, int thread_id, int frame_id, _str autovar_path, _str new_value)

Modify the value of the specified auto watch expression in the specified stack frame in the specified thread.

Parameters:

session_id - debugger session ID (1 .. max_session)

thread_id - index of thread in table

frame_id - index of frame in thread's stack

watch_path - path to local variable to expand

new_value - value to set variable to

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

To change the value of the the auto watch with ID 8 in the current thread and the top-most stack frame to 48: 

  int thread_id = dbg_get_cur_thread();
  int status = dbg_session_modify_auto_watch(session_id, thread_id, 1, "8", "48");
  if (status) {
     return status;
  }
