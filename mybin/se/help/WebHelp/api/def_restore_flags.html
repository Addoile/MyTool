int def_restore_flags

Restore option flags. Composed of a bitset of RF_*. This effects what aspects of the editor are restored when the editor is closed and restarted or when we switch workspaces (in the case of RF_PROJECTFILES).

  • RF_CLIPBOARDS -- Restore clipboards
  • RF_CWD -- Restore current working directory
  • RF_PROCESS -- Restore Build Window
  • RF_PROJECTFILES -- Restore files on a per-workspace basis
  • RF_LINEMODIFY -- Restore line modification colors
  • RF_NOSELDISP -- If this flag is not set, restore selective display
  • RF_CBROWSER_TREE -- Restore expanded nodes in Symbols tool window

Default:

RF_CWD|RF_PROJECTFILES

Category:

Configuration Variables
