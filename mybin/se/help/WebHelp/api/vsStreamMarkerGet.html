int VSAPI vsStreamMarkerGet(int StreamMarkerIndex, VSSTREAMMARKERINFO &info, int version=0)

Retrieves properties for stream marker.

Parameters:

StreamMarkerIndex - Index of stream marker returned by a vsStreamMarkerAdd function.

info - Returns all properties for stream marker.

version -

Returns: int

0 if stream marker index is valid.

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
