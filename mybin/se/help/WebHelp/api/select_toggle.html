_command void select_toggle()

Toggle between successive, increasingly large selections with each invocation.

Starting with no selection, creates an empty character selection, then selects the current word, then the current line, then the current code block, then a larger code block, then the current function, then the entire file, then deselect and start all over again.

Except for empty character selections and line selections, the selections are all locked so that the cursor remains stationary.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
