_command void find_error()

Finds the last SlickEdit interpreter run time error. The module with the error is loaded and the cursor is placed on the line causing the error. By default, only errors that cause a dialog box with the title "Slick-C® Error" can be found. However, if you want mild errors displayed, which occur in some built-in functions, turn on the weak errors option (see _default_option function for more information).
