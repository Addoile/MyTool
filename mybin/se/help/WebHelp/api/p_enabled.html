boolean p_enabled

Determines whether the object is enabled. When an object is disabled (p_enabled=0), the object can not receive focus. The font foreground color for a Label, Text Box, and Combo Box is gray (the system disabled color) when the control is disabled. Objects may receive focus when disabled if they are being edited (p_edit!=0).

Example:

// This example illustrates how to disable a list box and make the items in the list box
// appear grayed (not so obvious)
#include "slick.sh"
defeventtab form1;
list1.on_create()
{
   _lbadd_item('item1');
   _lbadd_item('item2');
   p_no_select_color=1;
   p_enabled=0;
   p_forecolor=_rgb(80,80,80);
}

Applies To:

All Window Objects

All Windows Properties
