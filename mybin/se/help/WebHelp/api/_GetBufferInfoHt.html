typeless* _GetBufferInfoHt(_str key)

Retrieve a the per-file (per-buffer) data from a indexed by key.

Use _SetBufferInfoHt() to store data by hash table key.

Parameters:

key - A unique hash table key that represents a given item.

Returns: typeless*

Data found with hash table key.

Category:

Buffer Functions
