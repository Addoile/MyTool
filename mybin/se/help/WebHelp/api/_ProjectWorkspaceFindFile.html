_str _ProjectWorkspaceFindFile(_str filename, boolean searchIncludes=true, boolean strict=false)

First searches the current project for filename, then the current workspace. If that fails, it reduces filename to it's basename and searches again. If that fails, it searches include paths as defined by the current project.

Parameters:

filename - path of file to search for

searchIncludes - search the project include paths

strict - search only for exact matches, do not strip path from filename

Returns: _str

'' if not found, otherwise, the absolute path to filename.

Category:

Project Functions
