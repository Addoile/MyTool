seSeekPosRet VSAPI vsQOldLineNumberFromLine(int wid, seSeekPos LineNumber)

Parameters:

wid - Window id of editor control. 0 specifies the current object.

LineNumber - Line number (VSP_LINE).

Returns: seSeekPosRet

Returns old line number (VSP_OLDLINENUMBER) corresponding to LineNumber.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

vsQLineNumberFromReal, vsQLineNumberFromOld, vsQRealLineNumberFromLine, vsQOldLineNumberFromLine
