char*VSAPI vsBufGetName(int buf_id, char *pszBufName, int MaxBufName, int *pMaxBufNameLen=a)

Parameters:

buf_id - Buffer id returned by vsPropGetI(wid,VSP_BUF_ID), vsBufEdit, or vsBufMatch.

pszBufName - Name of buffer corresponding to buf_id specified. This can be NULL, in which case, only the file name length will be set.

MaxBufName - Number of characters allocated to pszBufName. We recommend VSMAXFILENAME.

pMaxBufName - If this is not 0, this is set to the number of characters needed to be allocated.

Returns: char*VSAPI

Returns buffer name corresponding to buffer id specified. Returns pszBufName.

Attention:

This function is thread-safe.

Category:

Buffer Functions
