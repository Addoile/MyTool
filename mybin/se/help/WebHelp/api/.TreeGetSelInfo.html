void _TreeGetSelInfo(int &NofSelected, int &startSelectedItem=0, int &endSelectedItem=0)

Deprecated. Use _TreeGetNumSelectedItems or _TreeGetNextSelectedIndex

Parameters:

NofSelected - (Output only) Set to number of selected items in the tree.

firstSelectedItem - (Ouput only) Set to the tree item that started the selection.

endSelectedItem - (Ouput only) Set to the tree item that ends the selection.

Returns: void

Returns information about the selected items in the tree. Only use this function if p_multi_select is set to MS_SIMPLE_LIST.

Applies To:

Tree View

Category:

Tree View Methods
