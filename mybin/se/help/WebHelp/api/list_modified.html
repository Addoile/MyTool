_command int list_modified(_str title='', boolean quiet=false, boolean hideCurrent=false, int(*bufIdList)[]=null, boolean show_using_current_wid=false)

Displays a list of buffers that are modified. The Modified Buffers dialog box is displayed which allows you to select the files you wish to save, save all files, or save none. The title argument specifies the title of the dialog box and defaults to "Modified Buffers" if not specified or ''.

Returns: int

Returns 0 if successful.

Applies To:

Edit Window

Categories:

Buffer Functions, Window Functions
