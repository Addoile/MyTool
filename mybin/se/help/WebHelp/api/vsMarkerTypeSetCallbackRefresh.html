int VSAPI vsMarkerTypeSetCallbackRefresh(int type, void(VSAPI*pfnRefreshMarkers)(int MarkerRefreshFlags))

Parameters:

type - Index of marker type.

pfnRefreshMarkers - Callback.

Returns: int

Returns 0 if successful. Sets the refresh callback for the marker type specified. This callback is called when a line marker line numbers change or a marker is removed because all text for the marker was removed.

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
