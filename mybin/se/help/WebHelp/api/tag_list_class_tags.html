int tag_list_class_tags(int tree_wid, int tree_index, _str(&tag_files)[], _str tag_name, _str class_name, int filter_flags, int context_flags, var vnum_matches, int max_matches, boolean exact_match, boolean case_sensitive, ...)

Attempt to locate the given symbol in the given class by searching the given tag files. Recursively looks for symbols in enumerated types and anonymous unions (designated by having *both* the anonymous and 'maybe_var' tag flags).

Look at num_matches to see if any matches were found. Generally if (num_matches >= max_matches) there may be more matches, but the search terminated early.

For synchronization, macros should perform a tag_lock_matches(true) prior to invoking this function.

Parameters:

tree_wid - Window ID of tree control to insert into, 0 indicates to insert into a match set, see tag_insert_match

tree_index - Parent tree node index to insert item under, ignored if tree_wid is zero.

tag_files - (Input) List of tag files to search.

tag_name - Tag name or prefix of tag name to search for.

class_name - Name of class context to search for tags belonging to.

filter_flags - Tag filter flags, only insert tags passing this filter. See {@l;ink tag_filter_type} for more details.

context_flags - Context flags representing what to allow or restrict to. See tag_check_context_flags for more details.

vnum_matches - (Input, Output) Number of matches

max_matches - Maximum number of matches allowed

exact_match - Specifies exact match or prefix match (0)

case_sensitive - Specifies search case sensitivity.

visited - (optional) hash table of prior results

depth - (optional) depth of recursive search

Returns: int

1 if the definition of the given class class_name is found, othewise returns 0, indicating that no matches were found.

Category:

Tagging Functions
