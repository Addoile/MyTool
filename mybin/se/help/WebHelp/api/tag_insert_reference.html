int tag_insert_reference(int refto_id, int refby_id, _str ref_file, int ref_type, _str file_name, int line_no)

Add a tag reference located in the given file name and line number representing that the tag instance (ref_to_id) used within the context (ref_by_id), also a tag instance. See tag_insert_instance for an example showing how this function is used.

Parameters:

refto_id - unique ID of tag referenced from tag_insert_instance or tag_match_instance.

refby_id - Unique ID of the context (where the reference occurred) of from tag_insert_instance or tag_match_instance. Use 0 to represent the global context or if the context is unknown.

ref_file - name of references (browse database or object) file

ref_type - type of reference. One of the following:

VS_REFTYPE_unknown
unspecified type of reference
VS_REFTYPE_macro
use of #define'd macro
VS_REFTYPE_call
function or procedure call
VS_REFTYPE_var
use of a variable
VS_REFTYPE_import
use/import of a package
VS_REFTYPE_derive
class derivation
VS_REFTYPE_type
use of abstract type
VS_REFTYPE_class
instantiation of class
VS_REFTYPE_constant
use of constant value or enum value
VS_REFTYPE_label
use of label for goto

file_name - name of file where reference occurs

line_no - Line number where the reference begins in file_name. Use 0 to represent an unknown line number.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions

See Also:

tag_insert_file_end, tag_insert_file_start, tag_insert_instance, tag_insert_tag
