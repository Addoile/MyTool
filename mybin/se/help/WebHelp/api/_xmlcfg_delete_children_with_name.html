void _xmlcfg_delete_children_with_name(int handle, int ParentNode, _str name)

Deletes all children of ParentNodeIndex with the specified Name.

Parameters:

handle - Handle to an XMLCFG tree returned by _xmlcfg_open(), _xmlcfg_create(), or _xmlcfg_open_from_buffer().

ParentNode - Tree node index.

name - Name of elements to delete.

Category:

XMLCFG Functions
