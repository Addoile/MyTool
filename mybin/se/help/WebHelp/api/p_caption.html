_str p_caption

Determines the title or text displayed.

Menu, Menu Item - Determines the title of a submenu or menu item. When menus are displayed, there is no instance handle. Menu and menu item properties can only be accessed in menu resources. The _menu_get_state and _menu_set_state function may be used to get and set menu properties of displayed menus.

IMPORTANT: For menu items, set the p_caption property to "-" if you want a line separating menu items which follow.

Form, Frame - Determines the title of the window.

Picture Box, Image - Determines the caption displayed depending on p_style setting.

Label, Command Button, Radio Button, Check Box - Determines the text displayed in the window.

Example:

   #include "slick.sh"
   defmain()
   {
      // Find menu resource in names table
      tindex=find_index("menu1",oi2type(  OI_MENU));
      if (!tindex){
           messageNwait("menu1 not found");
           return('');
      }
      // Get resource handle to first child of this menu
      // The first child of a menu designed to be a menu bar is typically the File menu
      child=tindex.p_child;
      // Display the caption of the first child of this menu.
      messageNwait("caption="child.p_caption);
   }

Applies To:

Form, Label, Frame, Command Button, Radio Button, Check Box, Picture Box, Image

Categories:

Label Properties, Command Button Properties, Picture Box Properties, Image Properties
