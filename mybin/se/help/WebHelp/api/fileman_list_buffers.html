_command typeless fileman_list_buffers(_str wildcard=ALLFILES_RE)

Creates a buffer and inserts a list of the files that are currently open specified with columns for size, date, time, attributes (UNIX: permissions), and name. The mode is changed to fileman mode. When in fileman mode, there are some additions and changes to the key bindings. See help on fileman_mode.

Parameters:

wildcards - may contain operating system wild cards such as '*' and '?'.

Returns: typeless

Returns 0 if successful.

Category:

File Functions
