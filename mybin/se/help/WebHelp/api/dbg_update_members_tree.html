int dbg_update_members_tree(int tree_wid, int tree_index, int thread_id, int frame_id, _str member_path=null)

Update the list of members of the current class for the specified stack frame within the given thread. Each entry has three columns:

  • bitmap (variable or group)
  • name of variable
  • class name
  • value of variable (editable)

Parameters:

tree_wid - window ID of tree control

tree_index - index to insert under (TREE_ROOT_INDEX)

thread_id - thread ID to inspect

frame_id - stack frame ID to get members in

member_path - (optional) position to start updating tree at

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
