int tag_get_language(_str file_name, _str &lang)

API function for retrieving the language type for the given filename. This corresponds to the p_LangId property of file_name, not necessarily the literal file extension.

Parameters:

file_name - Name of file to set language property for

lang - (Output) p_LangId property for file_name

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if the file is not found.

Since:

13.0

Category:

Tagging Functions
