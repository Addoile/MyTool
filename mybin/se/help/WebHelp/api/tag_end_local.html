int tag_end_local(int context_id, int end_line_no, int end_seekpos)

Set the end positions of the local with the given local ID.

For synchronization, macros should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

local_id - Local ID [ from 1 to tag_get_num_of_locals() ] of local tag to set end position of.

end_line_no - ending line number of tag within file

end_seekpos - seek position where the tag ends. Technically, this is the seek position of the last character in the tag. Using a C local variable, "int a;" for example, this is the seek position of the of the semicolon.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions

See Also:

tag_insert_local, tag_insert_local2
