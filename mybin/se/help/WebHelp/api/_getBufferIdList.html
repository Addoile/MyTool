void _getBufferIdList(int next_option, int(&idList)[])

Retrieves the buffer ids of the currently open buffers in the same order as _next_buffer/_prev_buffer. This has the advantage of the getting the list of buffer ids in order without actually switching buffers and potentially mucking up scroll position information.

Parameters:

next_option - 1 to go in the same order as _next_buffer, 0 to use _prev_buffer order

idList - list of buffer ids

Category:

Buffer Functions
