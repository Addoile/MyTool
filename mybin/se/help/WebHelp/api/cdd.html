_command typeless cdd(_str param="")

Changes the current working directory to the drive and path if given. A current directory message is displayed. This command will also change directory in the build window if a path is given.

Returns: typeless

Returns 0 if successful. Common error code is PATH_NOT_FOUND_RC. This function can not detect an error which occurs while changing directory in the build window.

Category:

File Functions
