void VSAPI vsBufSetModifyFlags(int buf_id, int ModifyFlags, int resetFlags=false)

Set the buffer modify flags for the given buffer.

Parameters:

buf_id - Buffer id returned by vsPropGetI(wid,VSP_BUF_ID), vsBufEdit, or vsBufMatch.

BufFlags - Bitset of VSMODIFYFLAG_

resetFlags - reset the flags rather than just or'ing in the new flags

Attention:

This function is thread-safe.

Category:

Buffer Functions
