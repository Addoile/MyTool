_b2k_form

Syntax _str show('-modal _b2k_form' [, allow_editrun [ ,title [, initial_command [, completion]]]] )

allow_editrun defaults to '' title defaults to "Bind Command to Key" initial_command defaults to '' completion defaults to all commands

Displays Bind Command to Key dialog box. If allow_editrun is 0, the Edit and Run buttons of this dialog box are disabled. title specifies the title of the dialog box. If initial_command is not '', the command combo box text is initialized to this value. The completion argument may be MACRO_ARG (defined in "slick.sh") to list user defined macros.

Returns:

If the return value is not '', it should be executed. This occurs when the user presses the edit or run buttons of this dialog box.

Applies To:

Edit Window, Editor Control

Categories:

Forms, Keyboard Functions
