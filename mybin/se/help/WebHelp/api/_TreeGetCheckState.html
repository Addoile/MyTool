int _TreeGetCheckState(int iHandle, ...)

Get the check state of a tree node

Parameters:

ItemIndex - Handle to tree node to get check state for

ColIndex - Column of tree node to set checkable (defaults to 0)

Returns: int

int State of check box, one the following members of the TreeCheckVal enum: * TCB_UNCHECKED * TCB_CHECKED * TCB_PARTIALLYCHECKED

Applies To:

Tree View

Category:

Tree View Methods
