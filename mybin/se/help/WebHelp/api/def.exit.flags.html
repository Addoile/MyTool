int def_exit_flags

Exit options. Bitset of EXIT_

  • EXIT_CONFIG_ALWAYS -- always save configuration
  • EXIT_CONFIG_PROMPT -- always prompt to save configuration
  • EXIT_CONFIRM -- confirm exit before closing editor
  • SAVE_CONFIG_IMMEDIATELY -- save configuration changes immediately instead of waiting for exit.

Default:

EXIT_CONFIG_ALWAYS

Category:

Configuration Variables
