char**VSAPI vsWorkspaceListProjects(int *pNofProjects, int *pActiveProject=a)

Parameters:

pNofProjects - Set to the number of projects in the current workspace. Set to zero of there is not a workspace currently open.

pActiveProject - Set to the index (0..NofProjects-1) of the active project. Set to -1 if no project is active.

Returns: char**VSAPI

Returns an array of ASCIIZ strings which are the relative projects filenames in the current workspace. 0 is returned if an error occurs or there is no workspace currently open.

Category:

Project Functions
