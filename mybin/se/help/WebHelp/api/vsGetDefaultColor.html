int VSAPI vsGetDefaultColor(int cfgfield, int *pFGColor, int *pBGColor, int *pFontFlag=NULL, int *pcfgParentColor=NULL)

Gets rgb color values (always in Windows format) and font style flag for the cfgfield specified.

Parameters:

cfgfield - One of the VSCFG_??? constants or a color index allocated by vsAllocColor.

pFGColor - Ouput only. Set to rgb foreground color.

pBGColor - Ouput only. Set to rgb background color.

pFontFlag - Output only. Set to one of the VSFONTFLAG_?? constants or 0.

pcfgParentColor - Output only. Set to the color ID of the color which 'cfgfield' inherits properties from. One of the VSCFG_??? constants or a color index allocated by vsAllocColor.

Returns: int

Returns 0 if successful.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions

See Also:

vsAllocColor, vsFreeColor, vsSetTextColor, vsSetDefaultColor
