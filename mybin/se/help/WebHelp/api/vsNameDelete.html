void VSAPI vsNameDelete(int index, int reserved=a)

Delete the names table entry with the given index.

Parameters:

index - Index into names table of item to delete. For safety, Slick-C® functions or DLL functions which have explicit calls from Slick-C® are not removed. In addition, variables (VSTYPE_VAR) and modules (VSTYPE_MODULE) may not be deleted. Removes name entry corresponding to name table index.

Category:

Names Table Functions
