void VSAPI vsCommandName(int wid, const char *pszFilename)

This function changes the buffer name. Use this function instead of the VSP_BUFNAME property if you want various edit hook callbacks and/or your filename converted to absolute.

This function is almost identical to making the call vsExecute(wid,"name \"Filename\" ") except that vsRefresh is not called, a new undo step is never started, and code is not generated by macro recording.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

pszFilename - New buffer name. Filename is converted to absolute.
