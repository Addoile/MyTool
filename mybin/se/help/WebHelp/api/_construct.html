void _construct(_str className, ...)

Construct an instance of the specified type and pass the given arguments to it's constructor. Will make the instance 'null', like _makeempty(), if className is not a known Slick-C class type.

This class is useful for dynamically creating class instances. It does not actually allocate memory, it just initializes the variable it operates on. On account of this, Slick-C has no need for a "delete" or _destruct() operator.

Parameters:

className - Name of class to construct

argument1 - first argument for constructor

argument2 - second argument for constructor

Example:

class AAAA {
   int x; int y;
   AAAA(int ax=0, int ay=0) {
      x=ax;
      y=ay;
   }
};
AAAA AFactory(int x, int y) {
   AAAA r = null;
   r._construct("AAAA", x, y);
   return r;
}
