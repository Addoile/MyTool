int event2index(_str string)

Returns: int

Returns the event index corresponding to the binary event string given. The event index returned by this function may be used by the functions eventtab_index, or index2event. event may have been returned by get_event(), test_event(), and last_event().

Example:

        key=get_event();
        root_keys= find_index('root_keys',EVENTTAB_TYPE);
        index=eventtab_index(root_keys,p_mode_eventtab,EVENT2INDEX(key));
        // name could be type nothing, proc, command, macro, or key table.
        message('Key is bound to 'name_name(index));
