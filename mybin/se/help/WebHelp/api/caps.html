_command void caps()

Turn on or off uppercase mode. The caps command controls whether alphabetic data that you type is automatically converted to uppercase as you edit.

Note that if Auto Caps is turned on for the current file, the editor will set caps mode according to the data in the file when it is opened. If the data contains uppercase letters and no lowercase letters, caps mode is turned on. Otherwise, caps mode is off.
