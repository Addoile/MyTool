_command typeless cut_full_word()

Same as cut_word, except ignores the def_subword_nav setting and deletes from the cursor to the end of the full word at the cursor or the next word, and copies it to the clipboard.

Returns: typeless

Returns 0 if successful.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Clipboard Functions, Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods

See Also:

cut_word, cut_subword
