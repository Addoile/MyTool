_command int make_tags(_str params="")

This command runs the external macro "maketags.e".

The make_tags command builds a file called "tags.vtg" (UNIX: "utags.vtg") which is a binary database file. This information is used by the push_tag command to quickly find one of your procedures. Multiple files with wild cards may be specified.

See "tags.e" for information on adding support for other languages.

Parameters:

cmdline - a string with the following possible options

-L
specifies list file
@ [file]
specifies a file which contains a list of file names. Use the write_list command to generate a list of file names.
-R
Update tag file (checks dates)
-D
delete tags
-C
allow cancellation
-O [filename]
specifies a output tag file, rather than using the default of tags.vtg (UNIX: utags.vtg)
-X
build symbol cross-reference
-N [desc]
Adds a description of the tag file
-U [filename]
rebuild the specified tag file
-P
report the time required to tag
-B
use background thread if possible
-T
find files recursively in subdirectories
-Q
quiet
-E [dir]
exclude files under directory

OutputFile - ] [@]file1 [@]file2 ...

Returns: int

0 on success, nonzero on error.

Example:

make-tags   -t *.c *.asm
make-tags   @list1 *.pas

Category:

Search Functions

See Also:
