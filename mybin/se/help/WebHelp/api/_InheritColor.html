int _InheritColor(int cfg, int parent_color=0)

If 'parent_color' is not specified, just return what color index 'cfg' inherits color or font attributes from. If 'parent_color' is given, set the parent for this color to the specified color.

Parameters:

cfg - color index (note you can not set the parent color for default CFG_* color constants, however, these all implicitly consider CFG_WINDOW_TEXT as their parent color.

parent_color - color index for 'cfg' to inherit from.

Returns: int

Returns the parent color index.
