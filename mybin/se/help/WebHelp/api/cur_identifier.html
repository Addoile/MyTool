_str cur_identifier(int &start_col, int option=VSCURWORD_WHOLE_WORD)

Returns the current identifier. "" is returned if there is no identifier at the cursor. start_col is set to the physical position within the current line of the word returned.

Parameters:

start_col - (output) start column for identifier. The start column is returned as imaginary columns, not physical columns.

option - one of the following:

  • VSCURWORD_WHOLE_WORD - get the entire identifier
  • VSCURWORD_FROM_CURSOR - get part of identifier before cursor
  • VSCURWORD_BEFORE_CURSOR - get part of identifier after cursor
  • VSCURWORD_AT_END_USE_PREV - not supported

Returns: _str

Returns the current word.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
