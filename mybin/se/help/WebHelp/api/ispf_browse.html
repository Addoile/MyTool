_command void ispf_browse(_str filename='')

Browse a data set or member. If the member is a wildcard or directory, this command will generate a list of files to browse in fileman mode. Otherwise the given file or PDS member is opened for viewing and/or editing.

Parameters:

filename - Filename or path which may contains wildcards.

Category:

ISPF Primary Commands

See Also:

edit, ispf_end, ispf_cancel
