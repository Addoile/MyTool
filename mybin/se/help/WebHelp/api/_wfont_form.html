void _okay.on_create()

Displays Font dialog box which allows you to change the font for the current MDI edit window or all MDI edit windows.

Parameters:

options - is a string of zero or more of the following option letters:

F
Display fixed pitch fonts only.
S
(Default) Display screen fonts.
P
Display printer fonts.

Printer and screen fonts can not be displayed at the same time.

font - specifies the font that should be used to initialize the dialog box. It is a string in the format:
font_name,font_size, font_flags

font_size - is the point size of the font. font_flags is zero or constants ORed together. The font flag constants are defined in "slick.sh" and have the prefix "F_" (ex. F_BOLD).

Example:

void show('_wfont_form', options , font)

Category:

Forms

See Also:

_choose_font, wfont
