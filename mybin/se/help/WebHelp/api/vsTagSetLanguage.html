int VSAPI vsTagSetLanguage(VSPSZ pszFilename, VSPSZ pszLangId)

API function for setting the language type for the given filename. This corresponds to the p_LangId property of 'file_name', not necessarily the literal file extension.

Parameters:

file_name - name of file to set language type for

lang - p_LangId property for file_name

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
