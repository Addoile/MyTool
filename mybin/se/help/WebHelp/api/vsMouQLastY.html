int VSAPI vsMouQLastY(int wid=a, char option='D')

Parameters:

wid - Window id of editor control. 0 specifies the current object.

option - 'D' specifies to return the position of mouse in pixels relative to the screen. '\0' specifies to return the position of the mouse in pixels relative to the window specified. 'M' specifies to return the position in the scale mode (VSP_SCALEMODE) of wid relative to the window specified.

Returns: int

Returns y position of mouse for last mouse event. This function is typically used by a command or function which is bound to a mouse event, to determine where the mouse was when the event occurred.

Applies To:

Edit Window, Editor Control

Category:

Mouse Functions
