_str _bufnameNmode2ext(_str file_name, _str buf_mode_name)

Deprecated. Use _Filename2LangId() or _Modename2LangId()

Returns the language that should be used to lookup language specific information for the mode buf_mode_name specified. If no language data has a mode name of buf_mode_name the language associated with the file_name specified is returned.

Many macros ignore the mode name property and use the extension on the buffer name to determine language specific options. It is more correct to use this function to determine the language ID to used to lookup language specific information.

Parameters:

file_name - File name (and path)

buf_mode_name - Expected mode name, see p_mode_name

Returns: _str

The language ID that should be used to lookup language specific information for the mode buf_mode_name specified. If no language data has a mode name of buf_mode_name the language associated with the file_name specified is returned.
