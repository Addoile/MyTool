void _insert_text_raw(_str string, boolean binary=false, _str newlineChars="\r\n")

Parameters:

binary - defaults to false.

NewLineChars - defaults to "\r\n".

Insert string at cursor position. Specify true for the binary option if you want to allow splitting of newline characters (i.e. CR<insert here>LF). You may use this function to insert multiple lines of text. The NewLineChars string specifies a 1 or two byte sequence which indicates the line separation characters for the string argument. The NewLineChars argument is ignored if the binary option is true.

When you use this function on a record file (p_buf_width!=0) and the binary option is true, all data is inserted into current line and not broken up into multiple lines.

NOTE: If you are inserting at the end of a file and the last character of string is a new line, you can end up with a blank line that has no new line characters in it. You can test for this condition by calling _line_length(1) which will return 0. Use delete_line() to delete the line.

Applies To:

Edit Window, Editor Control
