12 VSCALLBACK_AR_APP_RENAME_BUFFER

void (VSAPI *pfnRenameBuffer)(int buf_id,
                                   const char *pszOldBufName,
                                   const char *pszNewBufName,
                                   int buf_flags);
The RENAME_BUFFER callback is called when the vsName function is called. Note that this function does not hook the VSP_BUFNAME property.

Parameters:

buf_id - Buffer id.

pszOldBufName - Old name of buffer.

pszNewBufName - New name of buffer.

buf_flags - Current value of VSP_BUFFLAGS property. You might to use this to check whether the buffer is hidden or not.

Category:

Editor Control Callback Functions
