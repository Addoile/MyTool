boolean tag_is_same_package(_str class1, _str class2, boolean case_sensitive)

Check if the given class is in the same package as the other class.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Parameters:

class1 - first class name, qualified

class2 - second class name, qualified

case_sensitive - case sensitive comparison for package names?

Returns: boolean

true if two classes are in the same package scope, otherwise false.

Category:

Tagging Functions
