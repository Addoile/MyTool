int dbg_session_update_disassembly(int session_id, _str file_name, int start_line, int num_lines)

Update the dissassembler output associated with the given stack frame.

Parameters:

file_name - Name of source file to disassemble

start_line - First line to disassemble

num_lines - Number of lines in the source file

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

   int status = debugger->updateDisassembly("myfile.cpp", 1, 100);
   if (status) {
      return status;
   }

Category:

Debugger Functions
