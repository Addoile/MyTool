int tag_open_db(_str file_name)

Open an existing tag database and return a handle to the database. This function opens the database for read-write access. . BSC files can not be opened read-write, use tag_read_db instead.

Parameters:

file_name - File name of tag database to open.

Returns: int

database handle >= 0 on success, <0 on error.

Since:

3.0

Category:

Tagging Functions
