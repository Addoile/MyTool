_find_file_form

Displays Find File dialog box used to search for open files.

Returns:

Returns string which should be used as input to the edit command. If '' is returned, the user cancelled the dialog box.

Example:

#include "slick.sh"
defmain()
{
    result = show('-modal _find_file_form');
    if (result == '') {
       return(COMMAND_CANCELLED_RC);
    }
    return(edit(result))
}
