int VSAPI vsBookmarkGetInfo(int i, char *pszBookmarkName=a, int *pmarkid=a, int *pvsbmflags=a, int *pbuf_id=a, int determineLineNumber=a, seSeekPos *pRealLineNumber=a, int *pCol=a, seSeekPos *pBeginLineROffset=a, char *pszLineData=a, char *pszFilename=a, char *pszDocumentName=a)

Retrieves information for a bookmark.

Parameters:

i - Bookmark index where 0<=i<vsBookmarkQCount().

pszBookmarkName - Set to name of bookmark. Must have VSMAXBOOKMARKNAME chacters allocated. Specify NULL if you don't need this value.

pmarkid - Set to selection id previous given to vsBookmarkAdd. Specify NULL if you don't need this value.

pvsbmflags - Set to combination of VSBMFLAGS_??? ORed together. Specify NULL if you don't need this value.

pbuf_id - Set to buffer id of bookmark. Specify NULL if you don't need this value.

pRealLineNumber - Set to real line number of bookmark. Specify NULL if you don't need this value.

pcol - Set to column of bookmark. Specify NULL if you don't need this value.

pBeginLineROffset - Set to real offset to beginning of line. Imaginary line data not counted. Specify NULL if you don't need this value.

pszLineData - Set to text on the line of the bookmark. Must have VSMAXBOOKMARKLINEDATA allocated. Specify NULL if you don't need this value.

pszFilename - Set to filename the bookmark is in. Must have VSMAXFILENAME allocated. Specify NULL if you don't need this value.

pszDocumentName - Set to document name the bookmark is in. Must have VSMAXDOCUMENTNAME allocated. Specify NULL if you don't need this value.

Returns: int

Returns 0 or TEXT_NOT_SELECTED_RC if successful. TEXT_NOT_SELECT_RC indicates that the bookmark is not active. This occurs when a buffer with bookmarks is closed.

Category:

Bookmark Functions
