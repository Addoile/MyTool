_command void ctab()

If Indent With Tabs ("Document", "Indent With Tabs") is on and the cursor is on or before the first non-blank character, one more syntax indent level is added to the line.

If indent with tabs is on and the cursor is after the first non-blank character, a tab character is inserted.

If indent with tabs is off, the cursor is moved to the next indent level.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Combo Box Methods, CursorMovement Functions, Edit Window Methods, Editor Control Methods, Text Box Methods
