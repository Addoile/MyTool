_command void spell_check_files()

Displays dialog for spell checks multiple source files. This command always does a language sensitive spell check. For HTML, markup that is not literal text is ignored. For source languages where color coding is provided, only comments and strings are spell checked.

Applies To:

Editor Control, Edit Window

Categories:

Edit Window Methods, Editor Control Methods

See Also:

spell_check, spell_check_source, spell_check_files
