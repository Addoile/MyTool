void _makeempty()

This method operates on _str, array ( [] ) , hash table ( :[] ) , struct, and union types. Sets variable to point to nothing. Slick-C® allocates spaces for the variable when an assignment is made. This function is very specific to the required implementation of Slick-C®. The changes we plan to make in the future will not effect this function.

Example:

t:["a"]=1;
t:["b"]=2;
t:["c"]=3;
// Traverse the elements in hash table
for (i._makeempty();;) {
   t._nextel(i);
   if (i._isempty()) break;
   messageNwait("index="i" value="t:[i]);
}
