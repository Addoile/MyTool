void _select_match(_str markid="")

Uses the selection id specified to select the last string matched by one of the built-in functions search or repeat_search.

Parm:

mark_id mark_id is a selection handle allocated by the _alloc_selection built-in. A mark_id of '' specifies the active selection or selection showing and is always allocated.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Search Functions

See Also:

match_length, search, repeat_search, search_replace, save_search, restore_search, _search_case
