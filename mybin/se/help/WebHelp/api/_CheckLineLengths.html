boolean _CheckLineLengths(int AllowedLineLen, _str &LineNumbers, int FromCursor=0, int &MaxLineLen=0)

Parameters:

AllowedLineLen - Lines lengths greater than this length are flagged.

LineNumbers - (Ouput only) ASCIIZ string. Set to a space delimited list of line numbers of lines which are too long. Lines with the VSLF_NOSAVE flag set are ignored.

FromCursor - (optional)

MaxLineLen - (optional)

Returns: boolean

Returns 0 if there are no lines longer than the AllowedLineLen specified. At the moment only the physical line length is checked as if tab characters count as 1 character. We may change this in the future.

Applies To:

Edit Window, Editor Control

Categories:

Editor Control Methods, File Functions
