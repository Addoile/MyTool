int _minihtml_command(_str command)

Send control command to the HTML control.

Parameters:

command - may be one of the following:

top
Go to the top of the document
bottom
Go to the bottom of the document
pagedown
Scroll to next page
pageup
Scroll to previous page
scrollup
Scroll up as if pressing the scroll up button on the vertical scroll bar.
scrolldown
Scroll down as if pressing the scroll down button on the vertical scroll bar.
pageleft
Scroll left one page.
pageright
Scroll right one page.
scrollleft
Scroll left as if pressing the scroll left button on the horrizontal scroll bar.
scrollright
Scroll right as if pressing the scroll right button on the horrizontal scroll bar.
copy
Copy selection or all text to clipboard if nothing selected
copyall
Copy all text to clipboard

Applies To:

Mini HTML

Category:

Mini HTML Methods
