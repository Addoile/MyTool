_command typeless cut_line(_str cbname='')

Deletes the current line and copies it to the clipboard. Invoking this command from the keyboard multiple times in succession creates one clipboard. You may retrieve the deleted line(s) with the command paste.

Returns: typeless

Returns 0 if successful. Common return codes are 1 (no line at cursor) and TOO_MANY_SELECTIONS_RC. On error, message is displayed.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Clipboard Functions, Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods
