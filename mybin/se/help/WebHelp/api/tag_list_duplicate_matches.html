int tag_list_duplicate_matches(_str proc_name, _str(&tag_files)[])

Find all tags in the given tag databases matching the given tag name, and possibly class or type specification. Places matches in the current match set.

For synchronization, macros should perform a tag_lock_matches(true) prior to invoking this function.

Parameters:

taglist - List of tag information, composed

filelist - List of corresponding filenames

proc_name - Composed tag name to search for

tag_files - List of tag files to search

Returns: int

0 on succes, <0 on error.

Category:

Tagging Functions
