char*VSAPI vsUTF8CharBegin(const char *pBuf, int BufLen, const char *pCurrent, int *pCharLen=0, int beginComposite=1)

Finds the beginning of the current composite character or UTF-8 character.

Parameters:

pBuf - Buffer contain UTF-8 text.

BufLen - Number of UTF-8 characters in pBuf. Specify -1 to compute the length from a null terminated buffer.

pCurrent - Pointer to UTF-8 character in pBuf.

pCharLen - Set to number of bytes in the current character.

BeginComposite - When non-zero, finds the beginning of the composite character.

Returns: char*VSAPI

Returns pointer to the beginning of the current composite or UTF-8 character. pCurrent is returned if pCurrent >=pBuf+BufLen.

Category:

Unicode Functions
