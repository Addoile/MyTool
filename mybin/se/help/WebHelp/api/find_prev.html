_command int find_prev()

Ctrl+Shift+G or "Search", "Previous Occurrence"

Repeats a search initiated by a search command in the opposite direction.

Returns: int

Returns 0 if successful. Otherwise STRING_NOT_FOUND_RC is returned. On error, message is displayed.

Applies To:

Edit Window,Editor Control

Categories:

Editor Control Methods, Edit Window Methods, Search Functions
