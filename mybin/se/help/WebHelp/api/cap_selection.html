_command typeless cap_selection()

Capitalizes the first character of the current selection and the first character of each word in the current selection and leaves the cursor at its current location. If there is no current selection, then the command is identical to cap_word().

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods
