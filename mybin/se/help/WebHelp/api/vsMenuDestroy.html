int VSAPI vsMenuDestroy(int menu_handle)

Destroy menu loaded by the vsMenuLoad function. This function is typically used to destroy the old menu bar when replacing a menu bar or to destroy a pop-up menu after it has been displayed.

Parameters:

menu_handle - Handle of loaded menu returned by VSP_MENUHANDLE, vsMenuLoad, vsMenuFind, or vsMenuGetState.

Returns: int

Returns 0 if successful.

Category:

Menu Functions
