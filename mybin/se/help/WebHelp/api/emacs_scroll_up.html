_command typeless emacs_scroll_up()

Used in EMACS emulation.

When called without an arguent, calls page_down

Moves the lines of the current buffer one line up. The cursor is moved one screen row up if the cursor is not on the top row of the window.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods
