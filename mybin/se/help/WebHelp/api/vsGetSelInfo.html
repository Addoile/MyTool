int VSAPI vsGetSelInfo(int wid, int markid, int *pfirstcol, int *plastcol=0, int *pbuf_id=0, char *pszBufName=0, int MaxBufName=0, int *pMaxBufNameLen=0, bool mark_to_cursor=true)

Returns information about the selection specified.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

markid - Selection handle returned by vsAllocSelection or vsDuplicateSelection. -1 specifies current selection.

pfirstcol - Output for first column of the first line of the selection.

plastcol - Output for last column of the last line of the selection. For a character selection, you need to check whether the selection includes this column ( vsQSelectType(markid,'I') ).

pbuf_id - Indicates the buffer id of the buffer which has this selection.

pszBufName - Indicates the name of the buffer which has this selection. This can be 0.

MaxBufName - Number of bytes allocated to pszBufName. We recommend VSMAXFILENAME.

pMaxBufNameLen - If this is not 0, this is set to the number of characters you need to allocate.

Returns: int

Returns 0 if wid and markid are valid.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
