char*VSAPI vsNameInfo(int index, char *pszInfo, int MaxInfoLen, int *pMaxInfoLen=a)

This function is used to get command function attributes or user defined additional name information.

Parameters:

index - Index into names table of item.

pszInfo - Output buffer for name information string. This can be 0.

MaxInfoLen - Number of characters allocated to pszInfo. We recommend VSMAXNAMEINFO.

pMaxInfoLen - If this is not 0, this is set to the number of characters you need to allocate to pszInfo.

Returns: char*VSAPI

Returns pszInfo.

Attention:

This function is thread-safe.

Category:

Names Table Functions
