void _retrieve_list(_str object_name='')

Inserts text box history into list box. When no argument is given, the history is retrieved based on the active form name and active control name (p_name). If the cmdline argument does not contain a '.', the history is based on the active form name and cmdline specifies an alternate control name.

Parameters:

cmdline - is a string in the format: form_name[.ctl_name]

Example:

defeventtab form1;
ok.lbutton_up()
{
    // When OK button is pressed, you will want to save combo box
retrieve information.
    _append_retrieve(_control combo1,combo1.p_text);
}
ok.on_create()
{
     // Fill in the combo box list
     combo1._retrieve_list();
}

Applies To:

Combo Box, List Box

Categories:

Combo Box Methods, List Box Methods, Retrieve Functions
