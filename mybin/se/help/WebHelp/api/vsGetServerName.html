void VSAPI vsGetServerName(char *pszServerName, int MaxServerNameLen, int *pMaxServerNameLen=a)

SlickEdit uses the server name to communicate with other instances of the editor and for DDE.

Parameters:

pszServerName - Ouput buffer for server name. The server name is set by the vsInit call by VSINIT. pszServerNamePrefix. This can be 0.

MaxServerNameLen - Number of characters allocated to pszServerName. We recommend you this be VSMAXSERVERNAME.

pMaxServerNameLen - If this is not 0, this is set to the exact number of characters you need to allocate.

Returns: void

Returns pszServerName.

Category:

Miscellaneous Functions
