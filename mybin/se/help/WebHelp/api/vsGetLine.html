int VSAPI vsGetLine(int wid, char *pszBuf, int BufLen)

Places the current line into pszBuf. The resulting string is always null terminated. No more than BufLen bytes are written. Use the vsQLineLength function to determine the length of the current line.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

pszBuf - Output buffer for current line.

BufLen - Number of characters allocated to pszBuf.

Returns: int

Returns number of bytes copied not including null character.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

vsInsertLine, vsReplaceLine, vsDeleteLine, vsGetText
