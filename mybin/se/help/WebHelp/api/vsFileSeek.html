VSINT64 VSAPI vsFileSeek(int fh, VSINT64 seekpos, int option)

Changes the file position for the file specified by fh.

Parameters:

option - changes the meaning of SeekPos as follows:

0
SeekPos is absolute position.
1
SeekPos is relative to current position.
2
SeekPos is relative to end of file.

Returns: VSINT64

If successful, the new seek position is returned. Otherwise a negative error code is returned.

Category:

File Functions
