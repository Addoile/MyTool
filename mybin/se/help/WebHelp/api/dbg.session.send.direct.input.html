int dbg_session_send_direct_input(int session_id, _str input_str)

Send input text directly to debugger. This will be interpreted as debugee input by the debugger.

Parameters:

session_id - debugger session ID (1 .. max_session)

input_str - String to send to debugger tty.

Returns: int

0 on success, <0 on error.

Since:

10.0

Category:

Debugger Functions
