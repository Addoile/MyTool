_command typeless link_window()

Displays Link Window dialog box. Allows you to select a different file or buffer to display in the current window. This command is only useful when One File per Window is on. Use the list_buffers command, if you are NOT in one file per window mode because it is more powerful than this function.

Returns: typeless

Returns 0 if switched buffers successfully.

Applies To:

Edit Window

Category:

Window Functions
