void _next_control(_str direction="", int form_wid=0)

Switches to the next control in a dialog box. Identical to pressing the TAB key in a dialog box except text in text box and combo box controls is not selected.

Applies To:

Text Box, Check Box, Command Button, Radio Button, Frame, Label, List Box, Editor Control, File List Box, Directory List Box, Hscroll Bar, Vscroll Bar, Combo Box, Picture Box, Image, Gauge, Spin

Category:

Miscellaneous Functions
