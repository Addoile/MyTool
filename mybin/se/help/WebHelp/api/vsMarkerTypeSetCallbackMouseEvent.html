int VSAPI vsMarkerTypeSetCallbackMouseEvent(int type, int event, int(VSAPI*pfnCommand)(int wid, seSeekPos LineNum, int LineMarkerIndex, int BMIndex, int Type, const char*pszMessage, void*pUserData)=a)

Parameters:

type - Index of marker type.

event - Mouse event constant. Mouse event constant (ex VS_OFFSET_MEVENTS+VS_MK_LBUTTON_DOWN). Only the LBUTTON_DOWN, LBUTTON_DOUBLE_CLICK, RBUTTON_DOWN events are supported.

pfnCommand - Callback.

Returns: int

Returns 0 if successful. Sets the callback for the specified mouse event.

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
