int _buf_transfer(int src_buf_id, int start_linenum=1, int end_linenum=-1, int start_col=0, int end_col=0)

Copies specified area of source buffer to the current buffer. If start_col and end_col are not specified or '', the source lines are inserted after the current line. Otherwise the source text is inserted before the cursor position.

Parameters:

src_buf_id -

start_linenum - Source start line. Defaults to first line of source buffer

end_linenum - Source end line. Defaults to last line of source buffer

start_col - Source start column.

end_col - Source end column.

Returns: int

Returns 0 if successful. Otherwise, message box error is displayed.

Applies To:

Edit Window, Editor Control

Categories:

Buffer Functions, Edit Window Methods, Editor Control Methods
