int VSAPI vsTagFindLanguage(char *pszLangId, const char *pszSearchFor=a, int find_first=a)

Retrieve a language ID from the language name table. This function has the side effect of positioning the language iterator on the given language type, and may be used to test the existence of a language type in the database.

Parameters:

pszLangId - Language ID found

pszSearchFor - Language ID to search for

pszFindFirst - Find first language, or find next?

Returns: int

0 on success, <0 on error, BT_RECORD_NOT_FOUND_RC if the language is not found.

Category:

Tagging Functions
