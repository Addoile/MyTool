int VSAPI vsBPMGet(int BreakPointIndex, seSeekPos *pRealLineNum=0, int *pflags=a, void **ppuserdata=a, int *pBufID=a, void *preserved1=a, void *preserved2=a)

Deprecated. Use vsLineMarker* functions Retrieves various information for a break point.

Parameters:

BreakPointIndex - Index to a break point from vsBPMAdd(), vsBPMAddB(), or vsBPMNext().

pRealLineNum - Set to real line number (VSP_RLINE) of breakpoint.

pflags - Set to flags of break point.

ppuserdata - Set to pointer to user data.

PBufID - Set to buffer id or a negative number if this break point if for a file which has not been loaded.

Category:

Deprecated Breakpoint Functions
