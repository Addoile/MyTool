void _deleteel(typeless index, int Nofitems=1)

Deletes element from hash table or array variable. Nofitems specifies the number of elements to delete and is ignored for hash table variables. Nofitems defaults to 1.

Parameters:

index - index / key corresponding to item to delete

Nofitems - number of items to delete

Example:

t[0]=1;
t[1]=2;
t[2]=3;
t._deleteel(1);   // Delete t[1].  t[1] will now contain 3
t:["a"]=1;
t._deleteel("a");  // Delete "a" element

Category:

Miscellaneous Functions
