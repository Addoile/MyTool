void tag_get_detail2(int tag_detail, int item_id, var result)

Get the given detail type for the given item either as a context tag, local tag, or part of a match set.

For synchronization, macros should perform a tag_lock_matches(false) or tag_lock_context(false) prior to invoking this function.
