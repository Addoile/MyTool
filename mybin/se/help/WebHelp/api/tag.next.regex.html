int tag_next_regex(_str tag_regex, _str search_options='')

Retrieve the next tag with the a tag name matching the given regular expression with the given matching options. Should be called only after calling tag_find_regex().

Parameters:

tag_regex - Regular expression for tag name to search for

search_options - Options passed to vsStrPos() for evaluating regular expression.

Returns: int

0 on success, <0 on error, or if no such tag.

Category:

Tagging Functions
