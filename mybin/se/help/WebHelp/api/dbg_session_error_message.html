int dbg_session_error_message(int session_id, _str &errmsg)

Get the last error message returned by the debugger from a failed command.

Parameters:

session_id - debugger session ID (1 .. max_session)

errmsg - error message from debugger

Returns: int

0 on success, <0 on error.

Since:

10.0

Category:

Debugger Functions
