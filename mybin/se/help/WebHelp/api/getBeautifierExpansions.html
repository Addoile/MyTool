int getBeautifierExpansions(_str langID, int defaultValue=0)

Returns a set of BEAUT_EXPAND_* flags for the given language, if any.

Parameters:

langID - language ID (see p_LangId)

defaultValue - if the value is not available, then we just return this value. If this parameter is not specified, then the default value for the overall def-var is used.

Returns: int

Profile name.

Category:

LanguageSettings API
