int mou_last_y(_str option="")

Parameters:

option - (optional). '' means to return pixels. 'M' means to return value in current windows scale mode (p_scale_mode).

Returns: int

Last mouse y position relative to active window in pixels. If the 'M' option is given the return value is in current windows scale mode (p_scale_mode).

Applies To:

All Window Objects

Category:

Mouse Functions

See Also:

p_scale_mode, mou_last_x, mou_get_xy
