_str _WorkspaceFindProjectWithFile(_str filename, _str workspaceName=_workspace_filename, boolean isAbsolute=false, boolean quiet=false)

Search for a file in any project in the specified workspace

Parameters:

filename - File to search for

workspaceName - Workspace path

isAbsolute - T if the specified filename is an absolute path

quiet - T if we should just take the first match

Returns: _str
