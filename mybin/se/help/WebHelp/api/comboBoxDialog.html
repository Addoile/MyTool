int comboBoxDialog(_str formTitle, _str label, _str(&items)[], int flags=0, _str defaultItem='', _str help='', _str retrieveName='')

Displays a generic form with a single combo box control.

Parameters:

formTitle - title text for form

label - any text or prompt for the combo box

items - array of items in the combo box

flags - a combo of the following flags

COMBOFORM_ALLOW_EDIT
Allow the user to edit the textbox portion of the combo box. The default is to allow no editing.

defaultItem - default item to be initially selected, must be included in the items array

help - help item for form

retrieveName - for saving/retrieving the last combo value picked

Returns: int

int IDOK if user picked a value, IDCANCEL if form was cancelled

Category:

Forms
