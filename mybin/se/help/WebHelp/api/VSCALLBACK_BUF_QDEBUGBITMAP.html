13 VSCALLBACK_BUF_QDEBUGBITMAP

int (VSAPI *pfnQDebugBitmap)(int buf_id,seSeekPos LineNum,seSeekPos RealLineNum,seSeekPos OldLineNum,int wid);

This callback returns the index to bitmap for the line number specified. index must be an index into the names table of a registered bitmap. Use vsUpdatePicture to register a bitmap. 0 indicates that the line has no bitmap.

WARNING: It is not safe to modify this buffer during this call.

Parameters:

buf_id - Buffer id.

RealLineNum - Real line number (VSP_RLINE).

OldLineNum - Old line number set by vsSetAllOldLineNumbers.

Category:

Editor Control Callback Functions
