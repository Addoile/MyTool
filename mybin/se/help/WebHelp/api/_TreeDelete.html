void _TreeDelete(int ItemIndex, _str option="")

Deletes the tree item specified and all its descendants. option may be "C" or "". When the "C" option is given, only the children of this tree item are deleted.

Parameters:

ItemIndex - Specifies a tree item.

option - may be "C" or "". When the "C" option is given, only the children of this tree item are deleted.

Applies To:

Tree View

Category:

Tree View Methods
