int tag_find_context(_str tag_name, boolean exact, boolean case_sensitive, boolean pass_through_anonymous=false, _str class_name=null)

Deprecated. Use tag_find_context_iterator(), which is re-entrant.

Find the first context entry with the given tag prefix. Use tag_get_context to extract defaults about the tag.

For synchronization, threads should perform a tag_lock_context(true) prior to invoking this function. This is considered as a write operation because it uses an iterator to traverse the items in the context.

Parameters:

tag_name - Tag name or prefix of tag name

exact - Search for exact match or prefix match

case_sensitive - Case sensitive string comparison?

pass_through_anonymous - Pass through anonymous classes

class_name - Class to find item in

Returns: int

Returns context ID of tag if found, <0 on error or not found.

Category:

Tagging Functions
