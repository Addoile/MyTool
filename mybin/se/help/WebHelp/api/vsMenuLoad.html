int VSAPI vsMenuLoad(int wid, int index, char menu_option)

Parameters:

wid - Window id of SlickEdit form or MDI frame.

menu_index - Names table index of menu returned by vsFindIndex or vsNameMatch.

menu_option - Specify 'P' for a popup-menu or 'M' for menu bar.

Returns: int

If successful, loads menu resource and returns handle of loaded menu. Otherwise a negative error code is returned. Displaying a menu resource as a pop-up menu with the vsMenuShow function requires that you specify 'P' as the second argument to this function. menu_index is a handle to a menu resource returned by vsFindIndex or vsNameMatch.

Example:

void VSAPI dllproc ()
{
    int index;
    // Find index of SlickEdit MDI menu resource
    index=vsFindIndex("_mdi_menu",vsoi2type(VSOI_MENU));
    // Load this menu resource
    int menu_handle;
    menu_handle=vsMenuLoad(VSWID_MDI,index,'M');

    int old_menu_handle;

    old_menu_handle=vsPropGetI(VSWID_MDI,VSP_MENUHANDLE);

    // _set_menu will fail if the form has a dialog box style border.
    // Put a menu bar on this form.
    vsMenuSet(VSWID_MDI,menu_handle);
    // You DO NOT need to call vsMenuDestroy.  This menu is
    // destroyed when the MDI window is deleted.
    vsMenuDestroy(old_menu_handle);
}
