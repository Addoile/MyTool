_command typeless end_select(_str markid='')

Places cursor on last character of selection. If the current selection type is LINE, the cursor is moved to the last line of the selection and the column position is unchanged.

Returns: typeless

Returns 0 if successful. Otherwise TEXT_NOT_SELECTED_RC is returned. On error, message is displayed.

Applies To:

Edit Window

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
