int refactor_c_extract_method(int handle, _str filename, _str symbolName, int &createCall, int beginSeekPos, int endSeekPos, _str &returnType, _str ¶mInfo, int flags, int syntaxIndent)

Analyze what is necessary to do in order to extract the code between 'beginSeekPos' and 'endSeekPos' into a separate function and replace the code with a call to the mew function.

Each line of parameter inforomation is a string of the form: 

   name [tab] return_type [tab] reference [tab] required [newline]

Parameters:

handle - Handle to transaction

filename - Filename to find symbol in

symbolName - Name of symbol to extract method from

createCall - [reference] Set to 1/0, if function call can be created

beginSeekPos - Seek position to begin extraction at

endSeekPos - Seek position to end extraction at

returnType - [reference] Set to return type of extracted function

paramInfo - [reference] Set to list of parameters for new function

flags - Optional flags, bitset of VSREFACTOR_FORMAT_* (above)

syntaxIndent - Syntax indent amount

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
