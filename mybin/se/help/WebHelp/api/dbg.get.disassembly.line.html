int dbg_get_disassembly_line(_str source_file, int line_number, _str(&addresses)[], _str(&instructions)[], _str(&func_names)[], _str(&offsets)[])

Get the details about a disassembled source line.

Parameters:

source_file - source file

line_number - line number

addresses - (reference) array of instruction addresses

instructions - (reference) array of disassembled instructions

func_names - (reference) array of function names

offsets - (reference) array of function offsets

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
