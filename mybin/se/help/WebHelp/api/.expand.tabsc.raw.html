int _expand_tabsc_raw(int start=1, int count=-1, _str option="E")

This function is identical to the _expand_tabsc function, except that the resulting string is always in the same format as the internal buffer data which can be SBCS/DBCS or UTF-8. See "Unicode and SBCS/DBCS Macro Programming".

Edit Window, Editor Control
