_command void ispf_autosave(_str cmdline='')

Turn on or off prompting to save changes when you issue an ispf_end command. If autosave is turned on, you will not be prompted, otherwise, you will be prompted. This only effects the ISPF command ispf_end, and does not effect the built-in quit() command. The ISPF emulation does not support the NOPROMPT option.

Parameters:

cmdline - Syntax of cmdline: 

   on| off|prompt

Category:

ISPF Primary Commands

See Also:

ispf_number, ispf_renumber, ispf_nonumber, ISPF Emulation Options
