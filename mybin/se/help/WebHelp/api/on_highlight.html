event on_highlight(int index, _str caption="")

This event is triggered when the mouse passes over an item in a collection control and the item is highlighted.

Combo Box -Occurs when an item is highlighted when you move over it with the mouse. An index is passed in to indicate which line was highlighted, followed by the text for the item which was highlighted.

Parameters:

index - index of item which was highlighted. -1 indicates that the mouse has left the control and nothing is being highlighted.

caption - string caption of the item highlighted this is redundant but passed in for convenience.

Applies To:

Combo Box, List Box, SSTab, Tree View

Categories:

Combo Box Events, List Box Events, SSTab Properties, Tree View Events
