int tag_match_occurrences_in_file_get_positions(var errorArgs, var seekPositions, _str tag_name, boolean case_sensitive, _str file_name, int line_no, int filter_flags, int start_seekpos, int stop_seekpos, int &num_matches, int max_matches, typeless &visited=null, int depth=0)

Match occurrences of the given tag name in the current buffer, starting at the specified start seek position and continuing until the specified stop seek position.

For synchronization, macros should perform a tag_lock_matches(true) prior to invoking this function.

Parameters:

errorArgs - (Output) error message parameters

seekPositions - (Output) seek positions of occurrences

tree_index - index of item in tree to insert under

tag_name - name of tag to search for

case_sensitive - case sensitive search?

file_name - path of file match tag is in

line_no - real line number match tag is located on

filter_flags - Tag filter flags, only insert tags passing this filter. See tag_filter_type for more details.

start_seekpos - starting seekpos, 0 means beginning of file

stop_seekpos - ending seekpos, 0 means EOF

num_matches - (Output) number of occurrence matches found

max_matches - maximum number of occurrences to find

visited - (optional) hash table of prior results

depth - (optional) depth of recursive search

Returns: int

0 on success, <0 on error, >0 on Context Tagging® error.

Category:

Tagging Functions
