_str get_text_raw(int count=1, int seek_pos=-3)

This function is identical to the get_text function, except that the resulting string is always in the same format as the internal buffer data which can be SBCS/DBCS or UTF-8. See "Unicode and SBCS/DBCS Macro Programming."

Applies To:

Edit Window, Editor Control
