int tag_find_ext_callback(_str callback_name, _str ext=null)

Deprecated. use _FindLanguageCallbackIndex()

This function is used to look up an extension-specific callback function.

Return the names table index for the callback function for the current language, or a language we inherit behavior from. The current object should be an editor control.

Parameters:

callback_name - name of callback to look up, with a '%s' marker in place where the extension would be normally located.

ext - current language (default=p_LangId)

Returns: int

Names table index for the callback. 0 if the callback is not found or not callable.

Category:

Tagging Functions
