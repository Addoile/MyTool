_str _xmlcfg_get_path(int iHandle, _str QueryStr, _str AttrName, _str DefaultValue='')

Gets the value of an attribute

Parameters:

iHandle - Handle to an XML tree

QueryStr - This is a small subset of an XPath expression. See _xmlcfg_find_simple() for information on this parameter.

AttrName - Name of attribute to get value of.

DefaultValue - Value to return if attribute does not exist or an error occurs.

Returns: _str

Returns value of attribute if successful. Otherwise DefaultValue is returned.

Example:

// Find the first file element node under /project with name equal to
// "main.c" and get the "options" attribute value.
options=_xmlcfg_get_path(handle,
    "/project/file[file-eq(@name,'main.c')]",
    "options");

Category:
