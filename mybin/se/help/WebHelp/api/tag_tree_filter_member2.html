int tag_tree_filter_member2(int filter_flags_1, int filter_flags_2, _str type_name, int in_class, int tag_flags, int &i_access, int &i_type)

Filter the given item based on the given filtering flags, and determine the access level and member type. Though this function is typically used by the symbol browser for filtering items, it can also be used simply to determine the access level and bitmap type for the tag, just ignore the return value in this case.
