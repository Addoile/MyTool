int p_pic_point_scale

Determines the scaling (in points) of a picture with respect to the font size (p_font_size). If p_pic_point_scale is 0, the picture (p_picture) is not scaled. The height and width of the picture are multiplied by (p_font_size/p_pic_point_scale).

Example:

#include 'slick.sh'
defeventtab form1;
combo1.on_create()
{
   // Let this picture be scaled based on an 8 point font
   p_pic_point_scale=8;
   p_font_size=18;
   p_picture=_pic_drremov;
   // Set indent before picture.
   p_pic_indent_x=100;
   // Set indent after picture.
   p_text='A:';
   p_after_pic_indent_x=200;
   // Extra y spacing. Half displayed above and half below picture
   p_pic_space_y=100;

}

Applies To:

Combo Box, Text Box, List Box

Combo Box Properties, Text Box Properties, List Box Properties
