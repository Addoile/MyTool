_command typeless copy_word()

Ctrl+K or "Edit", "Copy Word"

Copies word at cursor to the clipboard. Invoking this command from the keyboard multiple times in succession creates one clipboard.

Returns: typeless

Returns 0 if word exists at cursor. Common return codes are STRING_NOT_FOUND_RC and TOO_MANY_SELECTIONS_RC. On error, message is displayed.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Clipboard Functions, Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods
