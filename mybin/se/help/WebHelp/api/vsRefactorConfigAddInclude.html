int VSAPI vsRefactorConfigAddInclude(const char *configName, const char *includePath, int includeIndex=-)

Add a path to the include path

Parameters:

configName - name of configuration to modify

includePath - include path to add

includeIndex - where to insert path, -1 to insert at end of list

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions

See Also:

vsRefactorConfigAdd, vsRefactorConfigCountIncludes, vsRefactorConfigDeleteInclude, vsRefactorConfigDeleteAllIncludes
