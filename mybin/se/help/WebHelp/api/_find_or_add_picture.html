int _find_or_add_picture(_str filename)

Finds a picture in the names table. If the picture is not already in the names table, it is added.

filename is the filename of a bitmap (.bmp) or pixmap (.xpm) . Currently only bitmaps and pixmap files are supported. After a picture is inserted into the names table, the path information is removed (name_name(p_picture) has no path). Therefore, if you try to load two pictures from different directories with the same name part into the names table, the second copy will replace the first.

Important: Pictures are deleted from the names table when you save the configuration if the name of the picture does not start with an underscore ('_') and no forms reference the picture (p_picture).

Returns: int

Returns the names table index of the picture if successful. This index can be used by the p_picture property. On error, a negative error code is returned. Common error codes are FILE_NOT_FOUND_RC are PATH_NOT_FOUND_RC.
