int tag_check_for_typedef(_str symbol, _str(&tag_files)[], boolean case_sensitive, _str namesapce_name=null)

Lookup tag_name and see if it could be a typedef symbol. If so, return 1, otherwise return 0.

This function ignores class scope. Simply put, if tag_name is a typedef, anywhere, this function may return true. Thus, it's should really be used only as an arbiter prior to attempting to match symbol in context as a typedef. The reason that this function behaves in this was is for speed and simplicity. Otherwise, it would have to search all parent and outer class scopes, in addition to locals, context and tag files.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Parameters:

tag_name - Tag name to check if it is a typedef

tag_files - (Input) Tag files to search

case_sensitive - use case-sensitive comparisons?

namespace_name - namespace or class to search

Returns: int

Returns 1 if tag_name could be a typedef. Returns 0 if tag_name is not a typedef. <0 on error.

Category:

Tagging Functions
