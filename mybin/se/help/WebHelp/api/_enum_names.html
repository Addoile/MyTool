void _enum_names(_str enum_name, _str(&a)[])

Get all the enumerator constants for an enumerated type. Note the enumerator name needs to be qualified with namespace and class names as necessary. The enumerator constants will returned qualified with the same namespace and class as the enumerator.

Parameters:

enum_name - enumerator_name

a - (output) array of enumerator names

Category:

Miscellaneous Functions
