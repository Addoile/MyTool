_command int c_format(int in_wid=0, int start_indent=0, boolean use_form_options=false, boolean quiet=false)

Beautifies the current buffer using the current options. Use the C/C++/Java/JavaScript/C# Beautifier dialog box set beautifier options used by this command.

Parameters:

in_wid - Input window id to format. If 0, then current window is formatted. Defaults to 0.

start_indent - Starting indent for formatted lines. Defaults to 0.

use_form_options - Set to true to use globally stored options filled in on form by

quiet - (optional). Set to true if you do not want to see status messages or be prompted for options (e.g. tab mismatch). More serious errors (e.g. failed to save default options, etc.) will still be displayed loudly. Defaults to false.

Returns: int

a status value. Return value=2 means there was an error beautifying and calling function should get the error message with vscf_iserror().

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
