_command typeless read_list(_str filename='', _str arg2='', _str options='')

Appends the list specified to the current file. All columns are updated to the current file info. Files read by this command can be written using write_list, put, or any save command. A file list of absolute or relative file names starting in column one is accepted. This command is intended to be used when in fileman mode.

Returns: typeless

Returns 0 if successful. Common return codes are FILE_NOT_FOUND_RC, and PATH_NOT_FOUND_RC. On error, message is displayed.

Applies To:

Edit Window

Category:

File Functions
