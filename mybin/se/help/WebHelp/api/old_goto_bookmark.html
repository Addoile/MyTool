_command typeless old_goto_bookmark(_str bookmarkName='')

Deprecated. Use goto_bookmark().

Places the cursor at the buffer cursor position stored in the bookmark given by bookmarkName.

Note: This command no longer supports the + and - options.

Parameters:

bookmarkName - Name of bookmark on which to place cursor.

Returns: typeless

Zero on success.

Applies To:

Edit Window

Category:

Bookmark Functions

See Also:

push_bookmark, pop_bookmark, set_bookmark, toggle_bookmark, goto_bookmark
