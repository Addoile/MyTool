int VSAPI vsRefactorGetModifiedFileContents(int wid, int handle, const char *filename)

Get the contents of the modified file from the refactoring transaction and insert it into the given editor control.

Parameters:

wid - Editor control window ID

handle - Handle to transaction

filename - Filename of file that was modified

Returns: int

Length of error string on success, <0 on error

Category:

Refactoring Functions
