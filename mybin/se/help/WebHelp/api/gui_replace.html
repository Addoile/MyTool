_command int gui_replace(...)

Searches for a string you specify and replaces it with another string you specify. The Find and Replace tool window is displayed to prompt you for the search and replace options.

Returns: int

Returns 0 if successful.

Applies To:

Edit Window, Editor Control

Categories:

File Functions, Edit Window Methods, Editor Control Methods, Search Functions
