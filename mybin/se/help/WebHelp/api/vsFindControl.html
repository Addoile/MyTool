int VSAPI vsFindControl(int wid, VSPSZ pszName)

Parameters:

wid - Window id of object. 0 specifies the current object.

pszName - Name of control to search for

Returns: int

Returns instance handle (window id) of control, control_name, in the current form. If the current form does not have a control with name (p_name), ctlName, 0 is returned.

Applies To:

All Window Objects

Category:

Form Methods
