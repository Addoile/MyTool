_str _dbcsSubstr(_str string, int StartCol, typeless Width="")

Returns: _str

Returns part of the input string starting from StartCol including the rest of the string or Width characters if specified. This function is identical to the substr function except when double byte characters are bisected, they are replaced with zeros. This function does not yet support the pad argument.

Example:

        // This will always mismatch if index 5 is the first or second character of
        // a double byte character.
        if (_dbcsSubstr(string,5,1)=='+') {
             ...
        }
