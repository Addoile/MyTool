boolean p_auto_size

Determines whether a control automatically sizes its window based on its contents.

Label - Determines whether the window size is determined by the font and the p_caption property. When p_auto_size is non-zero, the p_alignment property has no effect.

Text Box - Determines whether the window size is determined by the fond and the p_text property.

Editor Control, List Box, File List Box, Directory List Box - Determines whether the window height is rounded to an integral of the font height.

Hscroll Bar- Determines whether the window height is set to the default system scroll bar height.

Vscroll Bar- Determines whether the window width is set to the default system scroll bar width.

Picture Box, Image - Depending on the p_style setting, determines whether the window size is set to the size of the picture defined by the p_picture property. If the p_picture property is zero and the p_caption property is '', the p_auto_size property has no effect.

Applies To:

Label, Text Box, Editor Control, List Box, File List Box, Directory List Box, Hscroll Bar, Vscroll Bar, Picture Box, Image, Spin
