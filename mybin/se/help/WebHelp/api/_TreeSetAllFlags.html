void _TreeSetAllFlags(int flags, int Mask=flags)

Sets all node flags in the tree. Use this function only if p_multi_select is set to MS_SIMPLE_LIST.

Parameters:

flags - New flags to set.

mask - Indicates which flags will be effected.

Example:

 // Select all the items in the tree
 tree1._TreeSetAllFlags(TREENODE_SELECTED);
 // Deselect all the items in the tree
 tree1._TreeSetAllFlags(0,TREENODE_SELECTED);

Applies To:

Tree View

Category:

Tree View Methods
