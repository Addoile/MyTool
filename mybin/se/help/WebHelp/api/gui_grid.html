_command typeless gui_grid()

Displays and optionally sets the grid width and height used by the dialog editor. This affects the distance displayed between the dots displayed on a form that is being edited. The Grid dialog box is displayed which prompts you for the width and height. The width and height parameters are in twips (1440 twips are one inch on the display).

Miscellaneous Functions
