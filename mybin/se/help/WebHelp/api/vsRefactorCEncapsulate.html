int VSAPI vsRefactorCEncapsulate(int handle, const char *filename, const char *symbolName, const char *getterName, const char *setterName, const char *methodName, seSeekPos startSeekPos, seSeekPos endSeekPos, int formattingFlags, int syntaxIndent)

Encapsulate a field. Make a field private and make public getter and setter functions.

Parameters:

handle - Handle to transaction

filename - Filename to find symbol in

symbolName - Name of symbol to encapsulate

getterName - Name of getter function to create

setterName - Name of setter function to create

startSeekPos - Seek position of symbol or position to start search

endSeekPos - Seek position to end symbol search (0 if begin is actual symbol)

formattingFlags - Flags for how generated code should be formatted

syntaxIndent - Syntax indent amount

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
