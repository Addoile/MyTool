void _SetDefaultLanguageOptions(_str lang, _str origExtension, VS_LANGUAGE_OPTIONS &langOptions)

Sets language specific options for a specific language.

Parameters:

lang - File language ID (see p_LangId). For list of language types, use our Language Options dialog ("Tools", "Options...", "Language Manager")

origExtension - File extension referred to 'lang'. This is necessary for retrieving options which are genuinely per extension, such as encoding and associated open application.

langOptions - New language specific options for pszExtension. Since all options are must be set, use the vsGetDefaultLanguageOptions first to query the existing value before setting new values.

Since:

13.0

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions

See Also:

_SetExtensionReferTo, _DeleteLanguageOptions
