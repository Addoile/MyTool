int _AllocColor(int fg=0x000000, int bg=0xffffff, int fontFlags=0, int parent_color=0)

Returns an index to a color. Use the _default_color function to set the color attributes. Color indexes are passed to the _SetTextColor function to set color. There is a limit of 255 colors so make sure to free the color when you are done.

Parameters:

fg_color - foreground color (RGB)

bg_color - background color (RGB)

font_flags - is one of the following font flags:

  • F_BOLD
  • F_ITALIC
  • F_STRIKE_THRU
  • F_UNDERLINE
  • F_INHERIT_STYLE

parent_color - color to inherit attributes from

Example:

int gColorIndex;
definit()
{
     if (arg(1)=='L' && gColorIndex) return;
     gColorIndex=_AllocColor(0xffffff,0xFF0000);
     //_default_color(gColorIndex,0xffffff,0xFF0000);
}
defmain()
{
     // Change the color of 5 bytes of text starting from cursor
     _SetTextColor(gColorIndex,5);
}

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
