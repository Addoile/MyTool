_str strip(_str string, _str ltb="B", _str strip_char=" \t")

Parameters:

string - input string

ltb - leading, trailing, or both

strip_char - character to strip

Returns: _str

Returns string stripped of leading and/or trailing strip_char. 'ltb' may be one of 3 values with the following meaning:

  1. 'L' -- Remove leading strip_char
  2. 'T' -- Remove trailing strip_char
  3. 'B' -- (Default) Remove both leading and trailing strip_char

If strip_char is omitted, both tabs and spaces are stripped.

Example:

   strip(' abc ') :== 'abc'
   strip(' abc ','L')   :== 'abc '
   strip(' abc ','T')   :== ' abc'
   strip('  '\t'abc '\t,'B',' ') :== \t'abc '\t
