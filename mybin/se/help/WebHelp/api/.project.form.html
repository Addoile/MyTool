void _ok.on_create(_str PropertiesForName='', int project_handle=-1, _str activetab='', boolean MakeCopyFirst=true, boolean doSaveIfNecessary=true, boolean IsProjectPackage=false)

Displays the Project Properties dialog box. The project's values are updated before the dialog box is closed unless the user cancels the dialog box.

Returns: void

Returns '' if the dialog box is cancelled. Otherwise, 1 is returned.
