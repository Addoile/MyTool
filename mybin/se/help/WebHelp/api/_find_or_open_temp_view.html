int _find_or_open_temp_view(_str buf_name, int &temp_wid, int &orig_wid, _str load_options='', boolean &buffer_already_exists=false, boolean doClear=false, boolean doSelectEditorLanguage=false, int more_buf_flags=0, boolean doCreateIfNotFound=false)

Activates the temp window/buffer for buffer name specified. If the buffer does not exists, a temp window/buffer is created.

Parameters:

buf_name - Name of buffer to search for.

temp_wid - Window id of file that is opened

orig_wid - Original window id

load_options - Load options supported by _open_temp_view().

buffer_already_exists - Set to true if the buffer already existed.

doClear - Indicates whether the contents of the buffer should always be cleared.

doSelectEditorLanguage - When true, _SetEditorLanguage() is called to initialize the language setup for the buffer. In addition, build_load_options() to generate all default load options which typically turns on undo.

more_buf_flags - Allows more buffer flags (typically VSBUFFLAG_THROW_AWAY_CHANGES) to be added to buffers loaded from disk.

doCreateIfNotFound - When true, a file (not buffer) is specified, and the file is not found, a new files is created.

Returns: int

Returns 0 if successful. Otherwise, a non-zero value is returned.

Applies To:

Edit Window

Category:

Buffer Functions
