int VSAPI vsQSelectedTextLength(int wid, int markid=-, int flags=-, const char *pEOLChars=a, int EOLCharsLen=-)

Parameters:

wid - Window id of editor control. 0 specifies the current object.

markid - mark_id is a selection handle allocated by the vsAllocSelection or vsDuplicateSelection function. A mark_id of -1 specifies the active selection or selection showing and is always allocated.

flags - Must be -1.

pEOLChars - Line separator characters. When this is 0, "\r\n" (UNIX: "\n") is used.

EOLCharsLen - Number of characters in pEOLChars. -1 means pEOLChars is null terminated.

Returns: int

Returns the number of characters in the selected text. 0 is returned if there is no selection.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions

See Also:

vsGetSelectedText
