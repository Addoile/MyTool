int _ntRegFindLatestVersion(int root, _str subkey, _str &version, int requiredMajor=0)

Finds the latest version subkey underneath the given subkey.

Parameters:

root - one of the HKEY_* constants

subkey - path of subkey

version - (by-ref) latest version found

requiredMajor - (optional) if you require a particular major version, send it here, otherwise use 0.

Returns: int

int 0 on success, non-zero on error

Category:

Miscellaneous Functions
