int VSAPI vsTagGetExtension(const char *pszFilename, char *pszLangId=a)

Deprecated. Use vsTagGetLanguage

Retrieve the p_LangId property for the given file. This function has the side effect of positioning the file iterator on the given file name, and may be used to test for the existence of a file in the database.

Parameters:

pszFilename - File path of source file

pszLangId - (256 bytes) Set to language ID for the file

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if the file is not found.

Category:

Tagging Functions
