int dbg_add_thread(int thread_id, int thread_flags, _str thread_name, int parent_group)

Add a thread to the debugger model

Parameters:

thread_id - Thread id from native debugger. VSE will have its own index to refer to this thread by

thread_flags - Combination of

  • VSDEBUG_THREAD_FLAG_ZOMBIE
  • VSDEBUG_THREAD_FLAG_RUNNING
  • VSDEBUG_THREAD_FLAG_SLEEPING
  • VSDEBUG_THREAD_FLAG_MONITOR
  • VSDEBUG_THREAD_FLAG_WAIT
  • VSDEBUG_THREAD_FLAG_SUSPENDED
  • VSDEBUG_THREAD_FLAG_FROZEN

thread_name - Thread name from native debugger

parent_group -

Returns: int

thread_id > 0 on success, < 0 on error.

Example:

int dbg_mydebugger_update_threads()
{
   dbg_clear_threads();
   dbg_add_thread(1,VSDEBUG_THREAD_FLAG_SUSPENDED,"main",0);
   dbg_add_thread(2,VSDEBUG_THREAD_FLAG_SUSPENDED,"cleanup",0);
   dbg_add_thread(3,VSDEBUG_THREAD_FLAG_SUSPENDED,"gc",0);
   return 0;
}
