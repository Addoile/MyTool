void _SetSymbolColoringOptions(_str lang, int flags)

Set the symbol coloring options which are a bitset of SYMBOL_COLOR_*.

The options are stored per extension type using def_symbolcoloring_[ext].

Parameters:

lang - language ID -- see p_LangId

flags - bitset of SYMBOL_COLOR_* options

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Category:

Completion Functions
