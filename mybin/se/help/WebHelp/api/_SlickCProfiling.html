boolean _SlickCProfiling(boolean on_off)

Start or stop Slick-C® profiling data collection. Turning on profiling will clear any currently collected profiling data.

Parameters:

on_off - 'true' to turn on profiling; 'false' to turn it off.

Returns: boolean

'true' if Slick-C® profiling was originally enabled, 'false' otherwise.

Category:

Macro Programming Functions
