void _BookmarkAdd(_str BookmarkName, int markid, int vsbmflags=VSBMFLAG_SHOWNAME | VSBMFLAG_STANDARD, int RealLineNumber=-1, int col=0, long BeginLineROffset=0, _str LineData="", _str Filename="", _str DocumentName="")

Adds a new bookmark.

Parameters:

BookmarkName - Name of bookmark to add.

markid - Selection id returned from vsAllocSelection or vsDuplicateSelection.

vsbmflags - Combination of VSBMFLAGS_??? ORed together.

RealLineNumber - Real line number of bookmark. Specify -1 if markid is an active bookmark.

col - Column of bookmark. Specify 0 if markid is an active bookmark.

BeginLineROffset - Real offset to beginning of line. Imaginary line data not counted. Specify 0 if markid is an active bookmark.

LineData - Text on the line of the bookmark. Specify 0 if markid is an active bookmark.

Filename - Filename the bookmark is in. Specify 0 if markid is an active bookmark.

DocumentName - Document name the bookmark is in. Specify 0 if markid is an active bookmark.

Category:

Bookmark Functions

_BookmarkAdd, _BookmarkRemove, _BookmarkRestore, _BookmarkGetInfo, _BookmarkFind, _BookmarkQCount
