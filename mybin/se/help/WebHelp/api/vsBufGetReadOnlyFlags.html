int VSAPI vsBufGetReadOnlyFlags(int buf_id)

Return the read only flags for the given buffer.

Parameters:

buf_id - Buffer id returned by vsPropGetI(wid,VSP_BUF_ID), vsBufEdit, or vsBufMatch.

Returns: int

bitset of the following flags

  • VSREADONLY_ON -- The file is in read only mode.
  • VSREADONLY_MANUALLY_SET -- Read only mode was set by the user.

Attention:

This function is thread-safe.

Category:

Buffer Functions
