void _remove_duplicates(_str ignore_case='', int start_line=0, int last_line=MAXINT)

Removes duplicate adjacent lines from the current buffer. Usually the buffer is sorted before calling this function.

Parameters:

ignore_case - 'i' for case sensitive

start_line - starting line (inclusive)

last_line - last line (inclusive)

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
