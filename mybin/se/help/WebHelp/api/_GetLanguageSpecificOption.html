typeless _GetLanguageSpecificOption(_str lang, _str fieldName, typeless defaultValue=null)

Get the specified option stored for the given language.

The options are stored per language mode. If the options are not yet defined for a language, then return defaultValue as the default.

Parameters:

lang - Language ID (see p_LangId For list of languages, use our Language Manager dialog ("Tools", "Options...", "Language Manager").

fieldName - may be one of the following:

p_mode_name or MN
Specifies mode name.
p_tabs or TABS
Specifies tab settings.
p_margins or MA
Specifies margins.
p_mode_eventtab or KEYTAB
Specifies mode event table.
p_word_wrap_style or WW
Specifies word wrap style.
p_indent_with_tabs or IWT
Specifies indent with tabs.
p_show_tabs or ST
Specifies to show tabs.
p_indent_style or IN
Specifies indent style.
p_word_chars or WC
Specifies word chaacters.
p_lexer_name or LN
Specifies lexer name.
p_color_flags or CF
Specifies color flags.
p_line_numbers_len or LNL
Specifies line number area size
p_softwrap or SW
Specifies soft wrap options
p_softwraponword or SOW
Specifies soft wrap on word

For the above options, value may not contain a comma character.

Additional field_name values:

LANGUAGE
General language setup data.
This option is very sensitive to the order of the options. We recommend that you use macro recording to correctly generate calls that set this option.
ALIAS
Language specific alias file
SMARTPASTE
Specifies Smart Paste(TM) options
SMARTTAB
Specifies tab expansion options
SMARTTAB
Set the smart tab value for this language, one of the following:
  • VSSMARTTAB_INDENT
  • VSSMARTTAB_MAYBE_REINDENT_STRICT
  • VSSMARTTAB_MAYBE_REINDENT
  • VSSMARTTAB_ALWAYS_REINDENT
BEGINEND
Specifies begin/end pairs
OPTIONS
Specifies extension specific options. We recommend that you use macro recording to correctly generate calls that set this option.
NUMBERING
Specifies line numbering options
SURROUND
Specifies dynamic surround flags
AUTOCOMPLETE
Specifies auto-complete flags
AUTOCOMPLETEMIN
Min word length for auto-complete
SYMBOLCOLORING
Symbol Coloring option flags/dd>
INDENT
Indentation options
CODEHELP
Context Tagging option flags
COMMENTEDITING
Comment editing option flags
DOCCOMMENT
Documentation comment options
COMMENT-WRAP
Specifies comment wrap options
XML-WRAP
Specifies XML wrap options
AUTOVALIDATE
Specifies XML validation options
INHERIT
Specifies parent language for tagging callback inheritance
ADAPTIVE-FLAGS
Specifies adaptive flags
ADAPTIVE-FORMATTING
Adaptive formatting flags

The following additional field_name options are actually stored per file extension rather than per language.

LANG-FOR-EXT
Maps file extension to language
ENCODING
Extension-specific encoding
ASSOCIATION
Extension-specific open application

defaultValue - Value to use if the option is not already set for this langauge.

Category:

Miscellaneous Functions
