char*VSAPI vsBufGetDocumentName(int buf_id, char *pszDocumentName, int MaxDocumentName, int *pMaxDocumentNameLen=a, int defaultToBufferName=a)

Gets the document name to be displayed to the user.

Parameters:

buf_id - Buffer id returned by vsPropGetI(wid,VSP_BUF_ ID), vsBufEdit, or vsBufMatch.

pszDocumentName - Set to document name (VSP_DOCUMENTNAME). If defaultToBufferName is non-zero and the document name is "" then this parameter is set to the buffer name (VSP_BUFNAME).

MaxDocumentName - Number of characters allocated to pszDocumentName.

defaultToBufferName - Indicates that the pszDocumentName parameter should be set to the buffer name if the document is "".

Returns: char*VSAPI

Returns pszDocumentName.

Attention:

This function is thread-safe.

Category:

Buffer Functions
