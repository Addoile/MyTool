void VSAPI vsSavePos2(int wid, int *pmarkid)

Saves the editor cursor position and scroll position information so that it can be restored later by vsRestorePos2. It is safe to make edits in between calls to vsSavePos2 and vsRestorePos2.

IMPORTANT: If you do not call vsRestorePos2, you need to call vsFreeSelection to free the selection allocated.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

pmark_id - Ouput for mark id.

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods
