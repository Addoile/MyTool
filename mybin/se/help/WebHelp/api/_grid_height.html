int _grid_height(int new_height=-1)

Returns: int

Returns and optionally sets the grid height used by the dialog editor. This affects the distance displayed between the dots displayed on a form that is being edited. If the new_height parameter is given, the grid height is set to this value. The new_height parameters is in twips (1440 twips are one inch on the display).
