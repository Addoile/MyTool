void VSAPI vsPropSetHvar(int wid, int prop_id, VSHVAR hvar)

Sets value of HVAR property to hvar. This can only be used with VSP_USER, VSP_USER2, and VSP_EMBEDDED_ORIG_VALUES.

Parameters:

wid - Window id of object or index to resource returned by vsFindIndex, vsNameMatch, VSP_CHILD, or VSP_NEXT. 0 specifies the current object.

prop_id - VSP_USER, VSP_USER2, or VSP_EMBEDDED_ORIG_VALUES

hvar - New value for property.

Example:

     // Go to line 5 in current buffer
       vsPropSetHvar(0,VSP_LINE,hvar);

Applies To:

All Window Objects

Category:

Miscellaneous Functions

See Also:

vsPropGetB, vsPropGetZ, vsPropSetZ, vsPropSetB, vsPropGetI, vsPropGetI64, vsPropSetI, vsPropSetI64, vsPropGetL, vsPropSetL
