int tag_current_version()

Return the version of the tags database currently open.

Returns: int

Return the version of the tags database currently open. Returns VS_TAG_USER_VERSION or higher. If the current version is VS_TAG_LATEST_VERSION, then the tag database was created by the most recent version of tagsdb.dll installed. The latest version of the references database is represented by the constant See VS_REF_LATEST_VERSION.
