_str tag_tree_format_args(_str signature)

Pretty-print function arguments to output buffer

Parameters:

signature - Return type and parameters of tag, as stored in the database, including VS_TAGSEPARATOR_args to separate the return type from the parameters.

Returns: _str

The output is returned in a staticly allocated string pointer.

Category:

Tagging Functions
