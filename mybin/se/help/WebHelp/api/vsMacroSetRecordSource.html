void VSAPI vsMacroSetRecordSource(int value)

Determines what vsMacroQRecordSource will return to a command called by vsExecute or vsCallIndex. By default, will Use vsMacroQRecordSource and the vsMacroQRecording function to determine whether your macro command needs to generate source code. If you have written a command, you should use vsMacroQRecordSource to check whether you need to generate source. If your command executes another command with vsExecute or vsCallIndex you may want to use vsMacroSetRecordSource to turn on or off macro recording before making the call.

Parameters:

value - Set to 1 if you want command called by vsExecute or vsCallIndex to do macro recording.

Example:

void VSAPI mycommand()
{
     vsMacroSetRecordSource(0);
     vsExecute(0,"save_as c:\\temp\\junk.c",
                 "" // DONT specify M option here
               );
}
