_str _spell_option(_str option, _str newValue=null)

If newValue is not null, the corresponding option is set to vewValue.

Returns: _str

Returns current/set value.

The options have the following meaning:

'R'
Indicates whether repeated words should be flagged as an error. Returns non-zero value when on.
'U'
Indicates whether upper case words should be ignored. Returns non-zero value when on.
'C'
Specifies name of common word dictionary. Returns name of dictionary.
'M'
Specifies name of main dictionary. Returns name of dictionary.
'1'
Specifies name of user 1 dictionary. Returns name of dictionary.
'2'
Specifies name of user 2 dictionary. Returns name of dictionary.

Category:

Miscellaneous Functions
