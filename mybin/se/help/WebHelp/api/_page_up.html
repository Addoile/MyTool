int _page_up()

Moves text under the cursor p_char_height - 1 lines down. The scroll style has no effect on this function.

Returns: int

If top of buffer is hit, TOP_OF_FILE_RC is returned. Otherwise, 0 is returned.

Applies To:

Edit Window, Editor Control

Categories:

CursorMovement Functions, Edit Window Methods, Editor Control Methods
