void _InsertSlickCProfilingData(int wid)

Insert Slick-C® profiling data to the designated editor control. The data is inserted one line per function, and tab-separated. The columns are as follows:

  1. module name -- loaded module function comes from
  2. offset -- offset of function within module
  3. function name -- optional
  4. number of calls
  5. mininimum time in function and descendants
  6. maxinimum time in function and descendants
  7. total time in function and descendants
  8. mininimum time in function only
  9. maxinimum time in function only
  10. total time in function only
  11. future data

Parameters:

wid - Editor control to insert into.

Category:

Macro Programming Functions
