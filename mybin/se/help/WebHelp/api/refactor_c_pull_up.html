int refactor_c_pull_up(int handle, _str filename, _str className, int beginSeekPos, int endSeekPos, _str superClass, _str membersToMove, _str &memberWorkingOn, _str superClassDefFilename)

Pull Up class member(s) to a super class

Parameters:

handle - Handle to transaction

className - Name of class

filename - Name of file containing class

beginSeekPos - Seek position of beginning of class reference

endSeekPos - Seek position of end of class reference

superClass - Name of superclass to move symbol to

membersToMove - Class members to move to super class

memberWorkingOn - (out)This is used for errors. This is the member being worked when error occurs.

superClassDefFilename - File to move function definiitions into. membersToMove is in the form: number of members [@] member [@] member [@] ...

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
