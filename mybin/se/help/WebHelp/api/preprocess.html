_command void preprocess(_str define_names="")

Preprocesses current buffer based on define_names specified. Lines of code which have no effect based on the preprocessing are hidden. Currently this command only supports C, C++, and Slick-C®.

The syntax for define_names is:

[+w | -w]  name1[=value1]
name2[=value2] ...

The -w option indicates that no warning should be given if a define name does not have a value in a preprocessor expression. The +w option currently has no effect.

Example:

preprocess  VSWINDOWS=1  WIN32S  UNIX=0

Applies To:

Edit Window, Editor Control

Categories:

Search Functions, Selective Display
