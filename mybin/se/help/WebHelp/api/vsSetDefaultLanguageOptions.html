void VSAPI vsSetDefaultLanguageOptions(const char *pszLangId, const char *pszExtension, const VS_LANGUAGE_OPTIONS *pLangOptions, int version=VS_LANGUAGE_OPTIONS_VERSION)

Sets language specific options for a specific language.

Parameters:

pszLangId - File language ID (see p_LangId). For list of language types, use our Language Options dialog ("Tools", "Options", "Language Setup...")

pszExtension - File extension referred to pszLangId

pLangOptions - New language specific options. Since all options are must be set, use the vsGetDefaultLanguageOptions first to query the existing value before setting new values.

reserved - unused

version - normally VS_LANGUAGE_OPTIONS_VERSION

Since:

13.0

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions

See Also:

vsSetDefaultExtensionReferTo, vsDeleteLanguageOptions, vsUpdateExistingBufferLanguageOptions
