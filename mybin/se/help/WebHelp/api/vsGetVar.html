VSHVAR VSAPI vsGetVar(int index)

Parameters:

index - Names table index returned by vsFindIndex or vsNameMatch.

Returns: VSHVAR

Returns handle to global variable corresponding to the name table index given.

Example:

       VSHVAR hvar;
       VSPLSTR plstr;
       char buffer[100];

       // The rc variable is a global variable which always exists.
       index=vsFindIndex("rc",VSTYPE_VAR);
       // Get handle to interpreter variable
       hvar=vsGetVar(index);
       // Convert contents of typeless variable into ASCIIZ
buffer.
       vsZLstrcpy(buffer,vsHvarGetLstr(hvar),100);

Category:

Miscellaneous Functions
