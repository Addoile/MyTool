_command typeless man(_str param='')

UNIX only. Displays man page(s) a newly created edit buffer for man help item specified. We attempt to remove special text screen formatting characters. However, this filter may need to be changed for some UNIX configurations. Lets us know if our filter does not work on your system. If ManHelpItem is not given, a dialog box is displayed which allows you to enter one.

Returns: typeless

Returns 0 if successful.

Category:

Miscellaneous Functions
