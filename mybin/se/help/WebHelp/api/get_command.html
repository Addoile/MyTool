void get_command(_str &text, int &start_sel=0, int &end_sel=0, _str &prompt="")

Be sure to look at the get_string function which prompts the user for command line input.

Returns: void

Returns the command line text, start of selection, end of selection, and prompt in the corresponding variables. If two arguments are specified, start_sel is set to the cursor position. The prompt variable is always set to the prompt before the command line and not the current text box.

Applies To:

Text Box, Combo Box

Categories:

Combo Box Methods, Text Box Methods

See Also:

command_state, command_toggle, execute, set_command
