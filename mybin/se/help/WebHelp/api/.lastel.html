typeless& _lastel()

Returns: typeless&

Returns last element in an array. Equivalent to ar[ar._length() - 1].

Example:

t[0]=1;
t[1]=2;
t[2]=3;
i = t._lastel();   // returns t[2]

Category:

Miscellaneous Functions
