void _TreeRestoreNodes(TREENODESTATE &state, int parent=-1)

Restore the specified node and all of it's children. This is used in conjunction with _TreeSaveNodes to save and restore the contents of a tree control to a Slick-C® data structure. This function will also restore the corresponding node as the current tree index.

Parameters:

state - (input) filled with state of specified tree node

index - (input) tree node to start from

Applies To:

Tree View

Category:

Tree View Methods

See Also:

_TreeCopy, _TreeSaveNodes
