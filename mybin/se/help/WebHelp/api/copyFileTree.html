int copyFileTree(_str srcDir, _str dstDir, _str options='', boolean ignoreErrors=false, _str(&FilesCopied)[]=null)

Copy all files from the specified source directory to the specified destination directory.

Parameters:

srcDir - source directory

dstDir - destination directory

options - Options: 'r' to make files read only, 'w' to make files read/write, '' to leave files untouched

ignoreErrors - If true, keep trying to copy files even if an error is encountered along the way.

Returns: int

0 OK, !0 error code

Category:

File Functions
