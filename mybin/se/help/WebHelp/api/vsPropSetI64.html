void VSAPI vsPropSetI64(int wid, int prop_id, VSINT64 value)

Sets value of a 64-bit integer property to value. For documentation on the properties, you must translate the VSP_??? constants into the Slick-C® property name. For example, VSP_BUFNAME corresponds to p_buf_name and VSP_LINE corresponds to p_line.

Parameters:

wid - Window id of object or index to resource returned by vsFindIndex, vsNameMatch, VSP_CHILD, or VSP_NEXT. 0 specifies the current object.

prop_id - One of the VSP_??? constants in "vs.h".

value - New value for property.

Example:

     // Go to line 5 in current buffer
     vsPropSetI64(0,VSP_LINE,5);

Applies To:

All Window Objects

Category:

Miscellaneous Functions

See Also:

vsPropGetB, vsPropGetZ, vsPropSetZ, vsPropSetB, vsPropGetI, vsPropGetI64, vsPropSetI, vsPropSetI64, vsPropGetL, vsPropSetL
