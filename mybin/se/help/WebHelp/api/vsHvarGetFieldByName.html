VSHVAR VSAPI vsHvarGetFieldByName(VSHVAR hvar, VSPSZ field_name)

Parameters:

hvar - handle for interpreter variable

field_name - name of non-static data member

Returns: VSHVAR

Return a handle to the given non-static data member of a class or struct. Returns CLASS_MEMBER_NOT_FOUND_RC if the field is not found. Returns INVALID_HVAR_ARGUMENT_RC if 'hvar' is not a struct or class instance.

Category:

Miscellaneous Functions

See Also:

vsHvarFieldName, vsHvarGetField
