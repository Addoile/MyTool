_str check_and_load_mode_support(_str mode_name)

If the language ID matches one of the builtin extensions supported by SlickEdit, this function will attempt to automaticaly load support for the language if it is not already loaded.

Parameters:

mode_name - Display name for language type

Returns: _str

The language ID for the language corresponding to 'mode_name'.

Category:

Miscellaneous Functions
