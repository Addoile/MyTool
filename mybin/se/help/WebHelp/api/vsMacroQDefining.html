int VSAPI vsMacroQDefining()

Use this function to determine which macro recording commands should be enabled or disabled.

Returns: int

Returns non-zero value if there is a macro is currently being recorded. This will return a non-zero value even if the pause_recording command is used to temporarily turn of recording.

Category:

Macro Programming Functions
