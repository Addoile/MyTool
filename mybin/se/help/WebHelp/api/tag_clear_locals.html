int tag_clear_locals(int local_id)

Kill all locals after 'local_id', not including 'local_id'

For synchronization, macros should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

local_id - id for the local variable to start removing at Use '0' to remove all locals.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions

See Also:

tag_get_num_of_locals, tag_insert_local2
