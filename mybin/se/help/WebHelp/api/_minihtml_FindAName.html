int _minihtml_FindAName(_str name, int flags)

Find a named anchor (for example, <A name="xxx">) within the text. Control will be positioned at that item if found.

Parameters:

name - Anchor name

flags - search flags

  • VSMHFINDANAMEFLAG_INCREASE_HEIGHT
  • VSMHFINDANAMEFLAG_CENTER_SCROLL

Returns: int

0 on success, <0 on error.

Applies To:

Mini HTML

Category:

Mini HTML Methods
