int dbg_session_update_locals(int session_id, int thread_id, int frame_id)

Update the local variables for the specified stack frame in the specified thread.

Parameters:

session_id - debugger session ID (1 .. max_session)

thread_id - index of thread in table

frame_id - index of frame in thread's stack

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

To update the local variables in the top-most stack frame: 

  int thread_id = dbg_get_cur_thread();
  int status = dbg_session_update_locals(session_id, thread_id, 1);
  if (status) {
     return status;
  }

Category:

Debugger Functions
