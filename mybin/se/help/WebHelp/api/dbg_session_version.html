int dbg_session_version(int session_id, _str &description, _str &major_version, _str &minor_version, _str &runtime_version, _str &debugger_name)

Get the version information for the debugger.

Parameters:

session_id - debugger session ID (1 .. max_session)

description - Description of debugger and version

major_version - Major version number

minor_version - Minor version number

runtime_version - Version of runtime environment

debugger_name - Name of debugger

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

To get the version information for the debugger: 

   _str description;
   _str major_version;
   _str minor_version;
   _str runtime_version;
   _str debugger_name;
   int status = dbg_session_version(session_id, description,
                                    major_version, minor_version,
                                    runtime_version, debugger_name);
   if (status) {
      return status;
   }
   _message_box("description=" :+ description :+ "\n" :+
                "major=" :+ major_version :+ "\n" :+
                "minor=" :+ minor_version :+ "\n" :+
                "runtime=" :+ runtime_version :+ "\n" :+
                "debugger_name=" :+ debugger_name);
