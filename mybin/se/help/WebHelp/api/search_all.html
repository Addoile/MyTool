int search_all(_str search_string, _str search_options, int index_list[])

Search buffer for all occurrences of search string.

Parameters:

search_string - Search string.

search_options - Search options.

index_list - Array of callbacks for each occurrence found.

Returns: int

Returns number of matching occurrences in this buffer.

Applies To:

Edit Window, Editor Control

Category:

Search Functions

See Also:

search, repeat_search, find_index
