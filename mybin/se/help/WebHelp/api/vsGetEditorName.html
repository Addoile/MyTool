char*VSAPI vsGetEditorName(int option, char *pszName, int MaxNameLen, int *pMaxNameLen=a)

Parameters:

option - One of the VSEDITORNAME_??? constants defined in "vs.h":

VSEDITORNAME_INVOCATION_NAME
Invocation name of editor. (argv[0]). This name will contain a path if the user invoked the editor with a path specification. Under Windows, this name is used when associating an extension to the editor.
VSEDITORNAME_EXE_PATH
Absolute editor executable path with trailing backslash. A few files in a single user configuration are stored in this directory (vrestore.slk, vslick.ini).
VSEDITORNAME_STATE_FILENAME
Absolute state file name.
VSEDITORNAME_AUTORESTORE_FILENAME
Absolute auto restore file name.
VSEDITORNAME_DDE_SERVER_NAME
DDE server name.
VSEDITORNAME_APPLICATION_NAME
Application name. This is currently only used by vsMessageBox as the default title. However, it is likely other dialogs will use this in the future.

pszName - Buffer to contain copy of value. This parameter is used as the return value. This can be 0.

MaxNameLen - Number of bytes allocated to pszName. Must be at least 1. We recommend VSMAXFILENAME.

pMaxNameLen - If this is not 0, this is set to the number characters you need to be allocate.

Returns: char*VSAPI

Returns string corresponding to option specified.
