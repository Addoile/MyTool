int VSAPI vsTagFindIncludeFile(const char *pszFilename, int find_first, char *pszIncludeName, int include_max)

Retrieve the name of the first file included by 'pszFileName'.

Parameters:

pszFilename - "outer" filename, for example, src.jar

find_first - Find first included file, or find next?

pszIncludeName - (reference) pointer to buffer to copy include file name into

include_max - size of buffer for 'pszIncludeName'

Returns: int

0 on sucess, <0 on error.

Category:

Tagging Functions
