int tag_normalize_classes(_str class_parents, _str cur_class_name, _str file_name, _str(&tag_files)[], boolean allow_locals, boolean case_sensitive, _str &normal_parents, _str &normal_types, _str &normal_files, ...)

For each item in 'class_parents', normalize the class and place it in 'normal_parents', along with the tag type, placed in 'normal_types'.

Parameters:

class_parents - list of class names, seperated by semicolons

cur_class_name - class context in which to normalize class name

file_name - source file where reference to class name is

tag_files - Array of tag files to search

allow_locals - allow local classes in list

case_sensitive - case sensitive tag search?

normal_parents - (output) list of normalized class names

normal_types - (output) list of tag types found for normalized class names

normal_files - (output) list of tag files parent classes are found in

visited - (optional) hash table of prior results

depth - (optional) depth of recursive search

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
