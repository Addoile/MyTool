void _macro_append(_str string)

If macro recording and output is on (_macro()!=0), the string argument specified is appended to the end of the macro being recorded.

Parameters:

string - text to append to macro definition.

Example:

   _macro_call('find','Find this', 'ri');
   // The above is identical to
   _macro_append("find('Find this', 'ri')");
   // and is also identical to
   _macro_append( "find(" _quote('Find this') ',' _quote('ri') ")" );

Category:
