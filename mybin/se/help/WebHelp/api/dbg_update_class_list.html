int dbg_update_class_list(int list_wid, _str parent_class)

Update the list of classes with the information retrieved by the debugger system. The classes are inserted with their names only.

Parameters:

list_wid - window ID of tree control

parent_class - (optional) last classes that derive from a this class

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
