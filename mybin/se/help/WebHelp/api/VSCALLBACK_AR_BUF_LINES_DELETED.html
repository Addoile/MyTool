1 VSCALLBACK_AR_BUF_LINES_DELETED

void (VSAPI *pfnDeleteLine)(int buf_id,seSeekPos FirstLinenum,seSeekPos Noflines)

The LINES_DELETED callback is called when lines are deleted in a buffer.

WARNING: It is not safe to access editor buffer data during this callback.

Parameters:

buf_id - Buffer id.

FirstLineNum - Line number (VSP_LINE and not VSP_RLINE. No save lines are counted). First line deleted.

Noflines - Number of lines deleted.

Category:

Editor Control Callback Functions
