int tag_list_globals_of_type(int tree_wid, int tree_index, _str(&tag_files)[], int type_id, int mask, int nonzero, var vnum_matches, int max_matches)

List the global symbols of the given type. See tag_find_global for more details.

For synchronization, macros should perform a tag_lock_matches(true) and tag_lock_context(false) prior to invoking this function.

Parameters:

tree_wid - Window ID of tree control to insert into, 0 indicates to insert into a match set, see tag_insert_match.

tree_index - Parent tree node index to insert item under, ignored if tree_wid is zero.

tag_files - (Input) Tag files to search

type_id - Tag type ID. See tag_get_type for a list of all standard tag types. if (type_id<0), returns tags with ID>VS_TAGTYPE_LASTID

mask - Tag flags indicating symbol attributes, see tag_insert_simple for details.

nonzero - if 1, succeed if mask & tag.flags != 0 if 0, succeed if mask & tag.flags == 0

vnum_matches - (Input, Output) number of matches

max_matches - Maximum number of matches allowed

Returns: int

Returns 0 on success, <0 on error.

Category:

Tagging Functions

tag_find_global, tag_get_type, tag_insert_match, tag_tree_insert_tag, tag_tree_prepare_expand
