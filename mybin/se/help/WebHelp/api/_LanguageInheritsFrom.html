boolean _LanguageInheritsFrom(_str parent, _str lang=null)

Does the current source language match or inherit from the given language?

If 'lang' is not specified, the current object must be an editor control.

Parameters:

parent - language ID to compare to

lang - current language ID (default=p_LangId)

Returns: boolean

'true' if the language matches, 'false' otherwise.

Since:

13.0

Applies To:

Edit Window, Editor Control

Category:

Tagging Functions

See Also:

_SetLanguageInheritsFrom, _FindLanguageCallbackIndex
