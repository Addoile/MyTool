char*VSAPI vsUTF8CharPrev(const char *pLine, const char *pCurrent)

Parameters:

pLine - Buffer contain UTF-8 text.

pCurrent - Pointer to the beginning of a UTF-8 character in pLine.

Returns: char*VSAPI

Returns pointer to the beginning of the previous composite character or pLine if pCurrent==pLine.

Category:

Unicode Functions
