int def_cd

Change directory options. Bitset of the following:

  • CDFLAG_CHANGE_DIR_IN_BUILD_WINDOW -- Also change directory in build window.
  • CDFLAG_EXPAND_ALIASES_IN_CD_FORM -- Expand aliases in the change directory form
  • CDFLAG_NO_SYS_DIR_CHOOSER -- Windows only - defeat windows "Browse for folder" dialog

Default:

3

Category:

Configuration Variables
