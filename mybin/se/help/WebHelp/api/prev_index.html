int prev_index(_str new_value="", _str option="D")

Returns: int

Returns index of previous command executed by a key or the built-in function call_key. If new_value is specified and not '', the index of the previous command is set to new_value.

When the second parameter is 'C', this command returns (and sets if new_value not '') the index of the previous command executed by any mechanism.
