void VSAPI vsHvarArrayDeleteEl(HVAR hvarArray, int start, int Nofitems=a)

Deletes elements from an array variable.

Parameters:

hvarArray - array to delete item(s) from

start - start index corresponding to item(s) to delete

Nofitems - (default 1) number of items to delete

Example:

   vsHvarMakeEmpty(VSHVAR_RC);
   hvar = vsHvarArrayEl(VSHVAR_RC, 0);
   vsHvarSetI(hvar, 1);
   hvar = vsHvarArrayEl(VSHVAR_RC, 1);
   vsHvarSetI(hvar, 2);
   hvar = vsHvarArrayEl(VSHVAR_RC, 2);
   vsHvarSetI(hvar, 3);
   vsHvarArrayDeleteEl(VSHVAR_RC, 1); // t[1] will now contain 3

Category:

Miscellaneous Functions
