int dbg_get_exception_class(int exception_id, _str &class_name)

Get the class and method scope of the specified exception.

Parameters:

exception_id - Exception to examine

class_name - Name of class that exception is set in

method_name - Name of method that exception is set in

Returns: int

1 for enabled, 0 for disabled, <0 on error.

Category:

Debugger Functions
