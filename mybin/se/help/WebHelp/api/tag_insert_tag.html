int tag_insert_tag(_str tag_name, _str type_name, _str file_name, int line_no, _str class_name, int tag_flags, _str signature)

Insert the given tag with accompanying information into the database. The only difference between this function and tag_insert_simple is that in this function, type_name is a string, allowing you to insert tags with user defined tag types.

Parameters:

tag_name - tag string

type_name - string specifying tag type. If the string is not a standard type, a new type will be created and inserted in the tag database. See tag_get_type.

file_name - File path of source file the tag is located in. See VS_TAGSEPARATOR_file for more details about constructing this value.

line_no - (optional) line number of tag within file

class_name - Name of class or package or other container that the tag belongs to. See VS_TAGSEPARATOR_class and VS_TAGSEPARATOR_package for more details about constructing this value. If the tag is not in a class scope, simple pass the empty string for this value.

tag_flags - Tag flags indicating symbol attributes, see tag_insert_simple for details.

signature - (optional) Tag signature, includes type, value, and arguments for the symbol being inserted. See VS_TAGSEPARATOR_args and VS_TAGSEPARATOR_equals for more details about constructing this value.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
