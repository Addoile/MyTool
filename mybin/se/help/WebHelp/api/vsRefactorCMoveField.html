int VSAPI vsRefactorCMoveField(int handle, const char *filename, const char *symbolName, const char *className, const char *classFileName, const char *classDefFileName, seSeekPos startSeekPos, seSeekPos endSeekPos)

Move a field. Move a static field from one class to another and fix all references.

Parameters:

handle - Handle to transaction

filename - Filename to find symbol in

symbolName - Name of symbol to move

className - Name of class to move symbol to.

classFileName - File that contains the class declaration to move symbol to.

classDefFileName - File that contains the class definition to move symbol to.

startSeekPos - Seek position of symbol or position to start search

endSeekPos - Seek position to end symbol search (0 if begin is actual symbol)

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
