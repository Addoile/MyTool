_str path_search(_str filename, _str env_var="PATH", _str options="", _str dotExtList=null)

Searches the path specified for a file or program. The path given with the filename is searched first. Then the directories specified by the environment variable env_var are searched.

Parameters:

options - is a string of zero or more of the following:

'P'
Program search. Does not effect UNIX.
'M'
Program search including search for .e and .ex Slick- C batch macros.
'S'
Don't search in current directory. Does not effect UNIX.

Non-UNIX platforms: The search order for a program or macro is:

.com
.exe
.e
(if M option specified)
.ex
(if M option specified)
.bat
(Windows and Windows 95/98 only)
.cmd
(Windows NT only)

UNIX: The search order for a program or macro is:

filename
(No changes to filename)
.e
(if M option specified)
.ex
(if M option specified)

Returns: _str

If successful, a complete file specification for a filename is returned. Otherwise '' is returned.

Example:

name=path_search('vs','PATH','P');  // Find vs.exe
name=path_search('draw','PATH','M');   // Find external draw
macro
name=path_search('vslick.ini');  // Find ini file
