int mou_limit(int x1, int y1, int x2, int y2)

(Supported under Windows, Windows 95/98, and Windows NT only)Limits the range of the mouse to pixel rectangle (x1,y1) to (x2,y2) relative to the current window. To cancel a limit, specify 0 for all parameters.

Parameters:

x1 - left x

y1 - top y

x2 - right x

y2 - bottom y

Example:

   mou_limit(0,0,p_client_x,p_client_height);
   for (;;){
        event=get_event();
        // Move controls within current window until get lbutton_up event.
        ...
   }
   // Cancel mouse range limit
   mou_limit(0,0,0,0);

Applies To:

All Window Objects

Category:

Mouse Functions
