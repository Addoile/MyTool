int eventtab_index(int root_keys_index, int mode_keys_index, int key_index, _str option="")

Returns name table index bound to the key corresponding to key_index. The fourth parameter may be set to "U", to return the index of the event table used. If the mode event table has a non zero binding for the key corresponding to key_index this index is returned. Otherwise the root key table binding is returned.

Example:

key=get_event();
root_keys_index= find_index('default-keys',EVENTTAB_TYPE);
index=eventtab_index(root_keys_index,p_mode_eventtab,event2index(key));
// Display information about the binding to this key.
message('name='name_name(index)' type='name_type(name));
