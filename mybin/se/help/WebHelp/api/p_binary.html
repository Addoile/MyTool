boolean p_binary

Determines the binary mode of the buffer attached to this window. Affects the _save_file() function. When on, all reformatting options given to the _save_file function are ignored. This property is initialized by the load_files function. The "+lb" load_files switch turns this buffer property on. This property is only available when the control is displayed.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Properties, Editor Control Properties
