void activate_window(int window_id)

Determines the current object/current window.

Parameters:

window_id - ID of window

Example:

#include "slick.sh"
defmain()
{
   //  Calling a function as a method like below
   _mdi.p_child.myproc();

   // is identical to the code below
   new_wid= _mdi.p_child;
   orig_wid=p_window_id;
   p_window_id=new_wid;
   myproc();
   p_window_id=orig_wid;
}
static myproc()
{
}

Applies To:

All Window Objects

Category:

Window Functions

p_window_id, get_window_id, load_files, _next_window, _prev_window, _delete_window, _open_temp_view, _delete_temp_view, _create_window
