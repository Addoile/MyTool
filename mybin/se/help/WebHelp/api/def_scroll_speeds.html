_str def_scroll_speeds

Controls how fast the editor or a list box is scrolled when you create a selection and drag the mouse outside of the editor control. Scrolling gets faster as you move the mouse further past edge. Contains scroll speed levels, ordered from slowest to fastest. Each pair is of the form ( timer delay in milliseconds . number of lines to scroll ).

Default:

'30.0 20.3 10.4 10.6'

Category:

Configuration Variables
