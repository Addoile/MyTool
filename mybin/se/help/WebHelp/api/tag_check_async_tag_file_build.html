int tag_check_async_tag_file_build(_str pszTagDatabase, boolean &isRunning, int &percentProgress)

Check the status of the given asynchronous tag file build or rebuild.

Parameters:

pszTagDatabase - name of tag file to check build progress for

isRunning - set to 'true' if the build is still running

percentProgress - set to the percentage 0-100 of progress for the build

Returns: int

0 on success, <0 on error

Since:

16.0

Category:

Tagging Functions
