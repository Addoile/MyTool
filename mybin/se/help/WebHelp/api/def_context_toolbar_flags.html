int def_context_toolbar_flags

Options for current context tool window. This is a bitset of the following flags:

  • CONTEXT_TOOLBAR_ADD_LOCALS -- Do not include local variables in the list and do not display them as the current context.
  • CONTEXT_TOOLBAR_SORT_BY_LINE -- sort the items in the list by line number rather than alphabetically.
  • CONTEXT_TOOLBAR_DISPLAY_LOCALS -- include local variables when displaying the current symbol under the cursor. Set to "0" to only show tags in context. Note that the drop down list will always be display locals for navigation purposes independent of this setting.

Default:

CONTEXT_TOOLBAR_ADD_LOCALS=1

Category:

Configuration Variables
