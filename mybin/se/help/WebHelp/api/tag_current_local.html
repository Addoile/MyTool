int tag_current_local()

Return the index of the current local variable.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Returns: int

Returns the index of the current local variable tag, calculated using the current buffer seek position. Returns <0 on error or 0 if no current local variable.

Category:

Tagging Functions
