_command void ispf_paste(_str arglist="")

The paste command copies lines from the clipboard to the specified position in the buffer, or the current line.

Syntax:

   PASTE [ clipboardname ] [ AFTER label ] [BEFORE label ] [KEEP]

Arguments:

AFTER label
The destination for the data being pasted. The label may be either an ISPF label or a line number. The data will be inserted after the specified line.
BEFORE label
The destination for the data being pasted. The label may be either an ISPF label or a line number. The data will be inserted after the specified line.
KEEP
Do not clear out the clipboard after doing a paste operation.

If neither AFTER nor BEFORE is specified, the default location to insert the data is after the current line (cursor position). However, if there is an A or B line command in the prefix area, the data will be inserted at that point.

Category:

ISPF Primary Commands
