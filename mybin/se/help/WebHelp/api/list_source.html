_command typeless list_source(...)

This command creates a buffer called "vusrdefs.e" (UNIX: "vunxdefs.e") and inserts Slick-C® source code representing your current configuration. All configuration changes are listed except modified or new dialog box templates and menu templates. See list_objects, list_sys_objects for information on generating source code for your dialog box and menu templates. The purpose of generating this file is usually to update to a new version of SlickEdit that can not read your old "vslick.sta" file. The vusrdefs (UNIX: vunxdefs) batch macro can be executed in the new version of SlickEdit by typing the name on the command line.

Returns: typeless

Returns 0 if successful.

Applies To:

Edit Window

Category:

Miscellaneous Functions
