int VSAPI vsCommandIndexForKey(int wid, int event)

Returns command index of command bound to the specified event (key)

Parameters:

wid - Window id of window that is the context for the key. This is important because some key bindings are "document mode" dependent

event - The key or key sequence we want to find the bound command for. Values come from vsevents.h

Returns: int

0 if no command is bound to this key, otherwise it returns the command index
