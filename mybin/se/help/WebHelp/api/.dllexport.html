int _dllexport(_str FuncProto, _str NameInfo="", int arg2=0)

Registers a DLL command or function that can be called from the Slick-C® macro. This function is identical to the vsDLLExport function except that you always need to specify the dll module the function is contained in. See vsDLLExport for information on parameters.
