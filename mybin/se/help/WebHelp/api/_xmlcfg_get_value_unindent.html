typeless _xmlcfg_get_value_unindent(int iHandle, int NodeIndex)

Gets the node value, removing any leading indent from beautification

Parameters:

iHandle - Handle to an XML tree

NodeIndex - Index to set the name for

Returns: typeless

Returns the value for the node specified. The value returned is null except for comment, attribute, CDATA, and PCDATA nodes.

Category:

XMLCFG Functions
