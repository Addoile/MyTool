_command void cbacktab()

Moves the cursor to the previous indent level or tab stop. To set the tab stops see help on gui_tabs or tabs command.

Applies To:

Text Box, Combo Box, Edit Window, Editor Control

Categories:

Combo Box Methods, CursorMovement Functions, Edit Window Methods, Editor Control Methods, Text Box Methods
