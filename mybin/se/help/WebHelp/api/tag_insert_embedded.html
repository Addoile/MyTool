int tag_insert_embedded(int start_seekpos, int end_seekpos)

Insert the start and end seek positions, and optionally the file extension information for an embedded context.

For synchronization, threads should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

start_seekpos - start seek position of embedded context

end_seekpos - end seek position of embedded context

Returns: int

0 on success, <0 on error

Category:

Tagging Functions
