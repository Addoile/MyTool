int VSAPI vsStrTranslate(char *pOutput, int OutputLen, size_t *pBytesRequired, const char *pInputStr, int InputLen, const char *pReplaceStr, int ReplaceLen, const char *pSearchStr, int SearchLen, int SearchFlags=0)

Parameters:

pOutput - (output) Character array to copy output to.

OutputLen - Number of bytes allocated to output array.

pBytesRequired - Number of bytes required for pOutput (can be NULL).

pInputStr - Input string to translate.

InputLen - Length of input string, -1 if null-terminated.

pReplaceStr - String to replace search matches with.

ReplaceLen - Length of replace string, -1 if null- terminated.

pSearchStr - String to search for.

SearchLen - Length of search string, -1 if null- terminated.

SearchFlags - One or more of the following flags:

VSSTRPOSFLAG_IGNORE_CASE
Case insensitive search.
VSSTRPOSFLAG_RE
SlickEdit regular expression search. See SlickEdit Regular Expressions.
VSSTRPOSFLAG_WORD
Reserved for future used.
VSSTRPOSFLAG_UNIXRE
UNIX regular expression search. See UNIX Regular Expressions.
VSSTRPOSFLAG_BINARY
Binary search. This allows start positions in the middle of a DBCS or UTF-8 character. This option is useful when editing binary files (in SBCS/DBCS mode) which may contain characters which look like DBCS but are not. For example, if you search for the character 'a', it will not be found as the second character of a DBCS sequence unless this option is specified.
VSSTRPOSFLAG_ACP
Specifies that pSearchFor and pBuf contain active code page data and that an SBCS/DBCS mode search should be performed. This flag is ignored if Unicode support is not active.
VSSTRPOSFLAG_BRIEFRE
Brief regular expression search. See Brief Regular Expressions.
VSSTRPOSFLAG_WILDCARDS
Wildcards expression search. See Wildcard Expressions.

Returns: int

Returns pInputStr with all occurrences of pSearchStr replaced with pReplaceStr. See vsStrPos function for information on valid search options.

Category:

String Functions
