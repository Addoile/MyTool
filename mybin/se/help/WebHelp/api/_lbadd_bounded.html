void _lbadd_bounded(_str text, int max_items=25)

Adds an item to the top of a list box. If the item is already in the list, just move it to the top. Limit the total number of items to 'max_items'. This function is useful for keeping a most-recently-used list of items, such as you would from a search drop-down.

Parameters:

text - The text of item to be added to the list box.

max_items - Maximum number of items to keep in list box.

Applies To:

List Box

Category:

List Box Methods

See Also:

_lbadd_item, _lbsearch, _lbget_item, _lbset_item
