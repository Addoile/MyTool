int def_buflist

Buffer list options.

  • SORT_BUFLIST_FLAG -- sort the buffer list
  • SEPARATE_PATH_FLAG -- separate paths
  • SHOW_SYSTEM_BUFFERS_FLAG -- show system buffers
  • SELECT_ACTIVE_BUFFER -- select the active buffer
  • FORCE_OLD_BUFLIST -- Use old list buffers dialog

Default:

SORT_BUFLIST_FLAG | SEPARATE_PATH_FLAG

Category:

Configuration Variables

See Also:

list_buffers, list_modified
