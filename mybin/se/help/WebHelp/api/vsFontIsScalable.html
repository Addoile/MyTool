int VSAPI vsFontIsScalable(int flags, const char *pszFontName, void *reserved=a)

Parameters:

flags - One or more of the VSFONTLISTFLAG_??? flags ORed together.

VSFONTLISTFLAG_SCREEN_FONTS
Test screen font.
VSFONTLISTFLAG_PRINTER_FONTS
Test printer font.

pszFontName - Name of font to list sizes for. Specify 0 to list available font names.

Returns: int

Returns non-zero value if the font is scalable. You should not list font sizes for fonts which are scalable since all sizes are supported.

Category:

Font Functions
