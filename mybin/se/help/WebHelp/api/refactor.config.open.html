int refactor_config_open(_str xmlConfigFile)

Open the master refactoring [compiler] configuration file, either for reading or editing. The configuration file is an XML file conforming to vsrefactor.dtd.

Parameters:

xmlConfigFile - Fully qualified path to XML configuration file

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions

See Also:

refactor_config_save, refactor_config_close
