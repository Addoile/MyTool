int _create_window(int object_index, int parent_wid, _str title, int x, int y, int width, int height, int cw_flags=0, int bds_style=-1)

Creates an object and returns an instance handle.

Parameters:

object_index - An object constant with prefix "OI_" from "slick.sh". Can't use OI_MDI_FORM.

parent_wid - Must be an existing window id, _mdi constant, or _app constant. _mdi is a window handle for the MDI window. _app is a window handle to the SlickEdit application window (exists but can be seen).

title - Used by OI_FORM, OI_FRAME, OI_COMMAND_BUTTON, OI_CHECK_BOX, and OI_RADIO_BUTTON objects only.

x -

y -

width -

height -

cw_flags - Create window flags. Many additional properties for objects can be set after the object is created. These flags are a combination of the flags below.

CW_HIDDEN
Create the window initially hidden.
CW_CHILD
Create the object as a child of the parent window, parent_wid. This flag is not valid for an OI_FORM object and REQUIRED for all other objects.
CW_PARENT
Keep form object on top of another window. This flag is only supported by form objects.
CW_LEFT_JUSTIFY, CW_RIGHT_JUSTIFY
Displays text on left or right for OI_CHECK_BOX or OI_RADIO_BUTTON.
CW_BSDEFAULT
Selects a OI_COMMAND_BUTTON to be the default button.
CW_EDIT
Creates the window for editing in the dialog editor.
CW_COMBO_LIST_ALWAYS
Used to create OI_COMBO_BOX in the list always style. Effects OI_COMBO_BOX object only.
CW_COMBO_NOEDIT
Used to create OI_COMBO_BOX in the no edit style. Effects OI_COMBO_BOX object only.

bds_style - Border style. Effects OI_FORM object only. May be one of the following:

BDS_NONE
BDS_FIXED_SINGLE
BDS_SIZABLE
BDS_DIALOG_BOX

Returns: int

Returns the window id of root window of the object created. On error, a negative error number is returned.

Example:

form_wid=_create_window(OI_FORM,_mdi,"Title",0,0,4000,2000,
              CW_PARENT|CW_HIDDEN,
              BDS_DIALOG_BOX);
text_wid=form_wid._create_window(OI_TEXT_BOX,form_wid,"",100,100,3500,100,
   CW_CHILD);
form_wid._center_window(_mdi)
form_wid.p_visible=1;

Category:

Window Functions
