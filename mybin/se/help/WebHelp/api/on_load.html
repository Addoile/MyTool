event on_load()

The on_load event is sent to a form (after the on_create and on_create2 events) when a dialog box template is loaded with the show or _load_template function. The on_load event is intended to be used to set the control which has the initial focus. Use the on_create event unless you need to set the initial focus.

Example:

#include 'slick.sh'
defeventtab form1;
form1.on_load()
{
    // Change the focus to the text2 control
    p_window_id=_control text2;
}

Applies To:

Form

Category:

Form Events
