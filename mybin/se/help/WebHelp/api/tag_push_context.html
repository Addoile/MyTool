int tag_push_context()

Push the current match set onto the stack of match sets and transfer the context set to the top of the match set stack.

For synchronization, macros should perform a tag_lock_matches(true) and tag_lock_context(true) prior to invoking this function.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
