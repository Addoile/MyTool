int _ntRegFindFirstValue(int root, _str _strSubKeyName, _str &hvarValName, _str &hvarValVal, int findFirst)

Retrieves a name/value pair from a subkey.

Parameters:

root - one of the HKEY_* constants

_strSubKeyName - name of subkey

hvarValName - (by-ref) name

hvarValVal - (by-ref) value

FindFirst - 1 to find the first pair, 0 to find the next pair, -1 to close the key Note that after opening a subkey to look at its data, you must call this function with FindFirst set to -1 to close the key and clean up after yourself.

Returns: int

int 0 on success, 1 on error, 2 if no more keys can be found

Category:

Miscellaneous Functions
