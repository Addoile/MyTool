int VSAPI vsNameInsert(const char *pszName, int kind, const char *pszInfo=a, int reserved=a)

This function creates a new names table item.

Parameters:

pszName - Name for new names table item. For kind==VSTYPE_PICTURE, pszName is the filename of a ".bmp", ".ico", ".png", or ".xpm" file whose path will be stripped off. Your picture should start with an "_" (ex. "c:\slickedit\_stackex.ico") to indicate that your bitmap is global. Otherwise when the configuration is written your bitmap will get deleted if it is not attatched to a Slick-C® dialog box.

kind - Type of new names table item. May be VSTYPE_MISC, VSTYPE_PICTURE, VSTYPE_EVENTTAB, or VSTYPE_MENU.

pszInfo - Optional name information for new names table item. Use vsNameInfo function to retrieve this information later.

Returns: int

If successful, the index of new names table item is returned. Otherwise, a negative error code is returned. 0 is never returned.

Attention:

This function is thread-safe.

Category:

Names Table Functions
