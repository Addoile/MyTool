int _select_char(_str markid="", _str options='EN')

Starts or extends the character selection specified. Used for processing sentences of text which do not start and end on line boundaries. The first _select_char becomes the pivot point. Subsequent select_char calls will extend the selection between the pivot point and the cursor.

Parameters:

mark_id - is a selection handle allocated by the _alloc_selection built-in. A mark_id of '' specifies the active selection or selection showing and is always allocated.

options - may be zero or more of the following letters:

'E'
Specifies that selecting an area of text requires _select_char be executed to select the end of the text area as well as the beginning. If the 'C' letter is not specified, this select style is used.
'C'
Specifies that the selection extend as the cursor moves.
'I'
Specifies an inclusive selection. Currently only character selections are affected by this option.
'N'
Specifies a non-inclusive selection. Currently only character selections are affected by this option. If the 'I' letter is not specified, the character selection will be non-inclusive selection.
'P'
A value of 'P' specifies a persistent select style and may be specified in addition to the other options above. Macros use this to help determine if a selection should be unhighlighted when the cursor moves.

Returns: int

Returns 0 if successful. Possible returns are TEXT_NOT_SELECT_RC. and TEXT_ALREADY_SELECTED_RC.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions

See Also:

_select_block, _select_line, _show_selection
