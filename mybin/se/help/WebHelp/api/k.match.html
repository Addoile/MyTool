_str k_match(_str name, boolean find_first)

Parameters:

name - macro name to match

find_first - 'true' to find first, 'false' to find next

Returns: _str

Returns the name of the user recorded macro matching the prefix name_prefix. If a match is not found, '' is returned. If find_first is non-zero, matching starts from the first user recorded macro. Otherwise matching starts after the previous match.

Categories:

Completion Functions, Macro Programming Functions
