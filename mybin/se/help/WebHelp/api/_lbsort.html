int _lbsort(_str options="", int start_linenum=1, int end_linenum=-1)

Sort items in a list box. By default, a case insensitive string sort is performed. If cmdline is specified and is not '', a case sensitive sort will be performed unless the 'I' option letter is specified.

Parameters:

cmdline - is a string in the format:

[ A|  D]   [ E | I ]    [  -N | -F]

cmdline - may contain multiple options having the following meaning:

A
(Default) Sort in ascending order.
D
Sort in descending order.
I
Case insensitive sort (Ignore case).
E
Case sensitive sort (Exact case).
-N
Sort numbers
-F
Sort filenames

start_point - and end_point are seek positions to the beginning of lines in the list box returned by the point function. If start_point is given, sorting does not affect lines above the line on this seek position. If end_point is given, sorting does not affect lines below the line on this seek position.

Example:

#include "slick.sh"
defeventtab form1;
list1.on_create()
{
    _lbadd_item("10");
    _lbadd_item("1");
    _lbadd_item("4");
    _lbsort("-n");   // Sort the number in the list box
    _lbsort("d -n"); // Now sort them in descending order
    _lbclear();
    _lbadd_item("c");
    _lbadd_item("A");
    _lbadd_item("C");
    _lbsort(); // Sort strings case insensitive in ascending order.
    _lbsort("d"); // Sort strings case sensitive in descending order.
    _lbsort("i d");  // Sort strings case insensitive in descending order.
}

Applies To:

List Box

Category:

List Box Methods
