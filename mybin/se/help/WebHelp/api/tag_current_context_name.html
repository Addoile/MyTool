int tag_current_context_name()

Return the index of the current context item whose name and name location matches the symbol under the cursor.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Returns: int

<0 on error, 0 if no current context.

Category:

Tagging Functions

See Also:

tag_current_context, tag_set_context_name_location
