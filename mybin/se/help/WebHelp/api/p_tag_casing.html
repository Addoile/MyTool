int p_tag_casing

Determines the preferred style of casing for tag elements for the current buffer. This property is available only when the control is displayed and is valid only in languages where tag elements are used (e.g. HTML, XML).

values and respective results:

  1. WORDCASE_PRESERVE - does not change
  2. WORDCASE_LOWER - lowercase
  3. WORDCASE_UPPER - uppercase
  4. WORDCASE_CAPITALIZE - capitalize the first letter only

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
