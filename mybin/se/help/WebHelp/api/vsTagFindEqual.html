int VSAPI vsTagFindEqual(const char *pszTagName, const char *pszClassName=a, int exact_match=a, int case_sensitive=a, int find_first=a, int skip_duplicates=a)

Retrieve the first tag with the given tag name or tag name prefix. Use vsTagGetTagInfo to extract the details about the tag.

Parameters:

pszTagName - Name of tag to search for.

pszClassName - (optional) class name to search for tag_name having

exact_match - (optional) exact match or prefix match of pszTagName

case_sensitive - (optional) default search is case-insensitive.

find_first - (optional) find first match, or next match

skip_duplicates - (optional) skip duplicate tags

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Category:

Tagging Functions
