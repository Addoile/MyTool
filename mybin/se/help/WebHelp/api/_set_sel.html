void _set_sel(int start_pos, int end_pos=start_pos)

Returns: void

Sets the cursor position or selection. If end_pos is not given, the new cursor position is set to start_pos and no text is selected. If end_pos is given, the selection starts from start_pos and ends at the cursor which is set to end_pos. The first text position is 1.

Applies To:

Text Box, Combo Box

Categories:

Combo Box Methods, Text Box Methods

See Also:

_get_sel, p_sel_start, p_sel_length
