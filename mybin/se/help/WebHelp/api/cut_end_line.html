_command typeless cut_end_line(boolean push=true)

Deletes text from cursor to end of line and copies it to the clipboard. For an edit window or editor control, if the cursor is past the last character of the line, the next line is joined with the current line. Invoking this command from the keyboard multiple times in succession creates one clipboard. You may retrieve the deleted text with the command paste.

Returns: typeless

Returns 0 if successful. Common return codes are BOTTOM_OF_FILE_RC and TOO_MANY_SELECTIONS_RC. On error, message is displayed.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Clipboard Functions, Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods
