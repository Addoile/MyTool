int _FindLanguageCallbackIndex(_str callback_name, _str lang=null)

This function is used to look up a language-specific callback function.

Return the names table index for the callback function for the current language, or a language we inherit behavior from. The current object should be an editor control.

Parameters:

callback_name - name of callback to look up, with a '%s' marker in place where the language ID would be normally located.

lang - current language ID (default=p_LangId)

Returns: int

Names table index for the callback. 0 if the callback is not found or not callable.

Since:

13.0

Applies To:

Edit Window, Editor Control

Category:

Tagging Functions

See Also:

_SetLanguageInheritsFrom, _LanguageInheritsFrom
