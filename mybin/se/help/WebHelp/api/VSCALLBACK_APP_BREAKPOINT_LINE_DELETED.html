34 VSCALLBACK_APP_BREAKPOINT_LINE_DELETED

int (VSAPI * vspfnBreakPointLinedDeleted)(int BreakPointIndex, int flags, void *puserdata);
This callback gets called when the line that any kind a break point is on is deleted.

Parameters:

BreakPointIndex - Index of break point being deleted.

flags - Flags for this break point.

puserdata - Pointer to user data for this break point.

Returns: 34

Return 0 to delete the break point.

Example:

static int VSAPI CallbackBreakPointLineDeleted(int BreakPointIndex, int
flags, void *puserdata)
{
     if (! (flags &
            (VSBPFLAG_BREAKPOINT|VSBPFLAG_BREAKPOINTDISABLED) ) ) {
            // Don't delete this because it is an execution or
            // stack executing break point.
            return(1);
     }
     ..
     return(0);
}
