void mou_capture(bool grabMouse=true)

Forces all mouse messages to the current window. This function is typically called after an lbutton_down event which performs a get_event loop which waits for an lbutton_up event to occur.

Parameters:

grabMouse - Set to true to grab mouse events for ALL windows (even those not belonging to the application). Defaults to true.
