int VSAPI vsGetTimeoutRemaining()

Check how many milliseconds are remaining before the timeout set using _SetTimeout() will expire. This is a software timeout. Nothing will happen when the timeout passes, it's up to you to call _CheckTimeout() yourself and handle the situtation.

Returns: int

0 if the timeout is expired, >0 is the number of milliseconds remaining, >= MAXINT if there is no timeout set.
