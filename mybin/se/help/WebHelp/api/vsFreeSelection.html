int VSAPI vsFreeSelection(int markid)

Frees the selection handle or bookmark handle corresponding to mark_id. _free_selection will not free the active selection (the one that is seen on screen). mark_id is a handle to a selection returned by vsAllocSelection or vsDuplicateSelection. A mark_id of -1 is not allowed. The default selection or selection show (vsShowSelection) can be freed.

Parameters:

mark_id - Selection id to free.

Category:

Selection Functions
