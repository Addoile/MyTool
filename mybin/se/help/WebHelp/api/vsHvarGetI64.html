VSINT64 VSAPI vsHvarGetI64(VSHVAR hVar)

Returns: VSINT64

Returns integer value of interpreter variable corresponding to hvar. If the value of the variable is not a valid 64-bit integer, an unknown value is returned.

Example:

       VSHVAR hvar;
       VSINT64 i;
       int index;

       // The rc variable is a global variable which always exists.
       index=vsFindIndex("rc",VSTYPE_VAR);
       // Get handle to interpreter variable
       hvar=vsGetVar(index);
       i=vsHvarGetI64(hvar);
