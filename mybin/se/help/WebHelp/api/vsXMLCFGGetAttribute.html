const char*VSAPI vsXMLCFGGetAttribute(int iHandle, int iIndex, VSPSZ pszRequestedAttr, int *pValueLen=a, VSPSZ pszDefaultValue="")

Deprecated. Use vsXMLCFGGetAttributeString instead whereever possible.

Retrieves attribute value for the attribute name given.

Parameters:

iHandle - Handle to an XMLCFG tree returned by vsXMLCFGOpen(), vsXMLCFGCreate(), or vsXMLCFGOpenFromBuffer().

iIndex - Tree node index.

pszRequestedAttr - Name of attribute to retrieve value for.

pValueLen - Number of bytes in returned value.

pszDefaultValue - Value to return if attributre does not exist on an error occurs.

Returns: const char*VSAPI

Returns attribute value if successful. Otherwise pszDevaultValue is returned.

Category:

XMLCFG Functions
