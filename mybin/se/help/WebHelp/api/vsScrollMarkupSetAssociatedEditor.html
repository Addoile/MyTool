void VSAPI vsScrollMarkupSetAssociatedEditor(int wid, int iAssociatedWID)

Associate a standalone scrollbar with an editor control - this is only for the purposes of markup, it will not handle the actual scrolling.

Currently only supports vertical scrollbars

Parameters:

wid - Window ID of scrollbar

iAssociatedWID - Window ID of the editor control to associate.

Returns: void

void

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
