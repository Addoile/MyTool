int dbg_get_breakpoint_scope(int breakpoint_id, _str &class_name, _str &method_name)

Get the class and method scope of the specified breakpoint.

Parameters:

breakpoint_id - Breakpoint to examine

class_name - Name of class that breakpoint is set in

method_name - Name of method that breakpoint is set in

Returns: int

1 for enabled, 0 for disabled, <0 on error.

Category:

Debugger Functions
