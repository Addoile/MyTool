_command void cut_sentence()

Used in EMACS emulation. Deletes the text from cursor to end of sentence and copies it to the clipboard. End of sentence is defined to be one of the character ., !, or ? followed by any number of the characters ., !, ?, ", ', ), or ] followed by end of line or two spaces. Invoking this command multiple times in succession creates one clipboard.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
