_str getDocCommentFlags(_str langID, _str defaultValue=null)

Gets the doc comment flags for this language. This value is controlled by the options available on the Comments form for each applicable language.

Parameters:

langID - language ID (see p_LangId)

defaultValue - if the value is not available, then we just return this value. If this parameter is not specified, then the default value for the overall def-var is used.

Returns: _str

doc comment flags for this language

Category:

LanguageSettings API
