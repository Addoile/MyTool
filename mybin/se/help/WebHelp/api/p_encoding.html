int p_encoding

Determines the default encoding that the buffer will be saved in. After opening a file, this property indicates the encoding of the original file.

Use the p_UTF8 property to check whether a buffer contains UTF-8.

VSCP_ACTIVE_CODEPAGE indicates that the buffer contains SBCS/DBCS active code page data. All other encodings indicate that the buffer contains UTF-8 data.
