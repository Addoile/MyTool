_command void gui_draw()

The gui_draw command is used to draw lines in the current buffer using the PCDOS graphics characters. Line drawing characters are not available under most UNIX systems. The Draw Lines dialog box is displayed which prompts you for a box character set. You will want to use the OEM Fixed Font (default) in order to view the graphic characters. Under UNIX, we don't know which font if any supports the line drawing characters.
