_command void move_text_tab()

If Indent With Tabs is on and the cursor is on or before the first non- blank character, one more indent level is added to the line.

If indent with tabs is on and the cursor is after the first non-blank character, a tab character is inserted.

If indent with tabs is off, enough spaces are inserted to move the text, from the cursor to the end of the line, to the next indent level.

The Indent with Tabs option ("Document", "Indent with Tabs") toggles indenting with tabs for the current buffer.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box
