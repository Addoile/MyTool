int VSAPI vsRefactorGetError(int handle, const char *filename, int errorIndex, char *errorString, int maxLength)

Get the specified error string from the refactoring transaction.

Parameters:

handle - Handle to transaction

filename - Filename of file that failed to parse

errorIndex - Index of error

errorString - Error string

maxLength - Maximum error string length

Returns: int

Length of error string on success, <0 on error

Category:

Refactoring Functions
