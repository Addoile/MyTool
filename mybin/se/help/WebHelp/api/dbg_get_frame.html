int dbg_get_frame(int thread_id, int frame_id, _str &method_name, _str &signature, _str &return_type, _str &class_name, _str &file_name, int &line_number, _str &address)

Get the details about the given stack frame

Parameters:

thread_id - index of thread to get stack frame from

frame_id - index of frame in stack frame list

method_name - (reference) name of method on stack frame

signature - (reference) method signature

return_type - (reference) return type of method

class_name - (reference) name of class on stack frame

file_name - (reference) name of file that stack frame is positioned at

line_number - (reference) line number that stack frame is positioned at

address - (reference) address of current instruction in stack frame

Returns: int

0 on success, <0 on error

Category:

Debugger Functions
