int _ini_set_value(_str filename, _str section_name, _str field_name, _str new_value, _str case_option=null)

Sets value of field_name defined within section of filename to value specified.

Returns: int

0 if successful 

     ".ini" File Syntax:
         [section1]
         field_name1=value1
         field_name2=value2
         ...
         [section2]
         field_name1=value1
         field_name2=value2
         ...
