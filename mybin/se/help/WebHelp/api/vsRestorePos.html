void VSAPI vsRestorePos(int wid, const VSSAVEPOS *pbuf)

Restores the editor cursor position and scroll position information. pbuf must have been initialized by vsSavePos. NO EDITS should have been made to this buffer inbetween the vsSavePos and vsRestorePos calls. Use vsSavePos2 and vsRestorePos2 to save and restore where edits are made.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

pbuf - Structure initialized by vsSavePos.

Category:

CursorMovement Functions
