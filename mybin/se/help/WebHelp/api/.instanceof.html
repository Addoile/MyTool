boolean _instanceof(_str typeName)

Parameters:

typeName - String containing name of type to test.

Returns: boolean

Return true if the given class is an instance of the named class. Note that it is preferable to use the "instanceof" operator instead of this builtin wherever possible.

Example:

class AAAA { int x; int y; };
class BBBB : AAAA { int z; };
class CCCC { _str ch; };
void testInstanceOf() {
   AAAA a;
   BBBB b;
   CCCC c;
   true1  := b._instanceof("AAAA");  // BBBB inherits from AAAA
   true2  := b._instanceof("BBBB");  // b._typename() = BBBB
   false1 := b._instanceof("CCCC");  // no relation to CCCC
   false2 := b._instanceof("DDDD");  // no such class DDDD
}
