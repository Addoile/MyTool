int VSAPI vsMessageBox(const char *pszMessage, const char *pszTitle=a, int vsmb_flags=VSMB_OK, int default_button=a)

Creates a modal dialog box and displays pszMessage and various buttons. pszMessage may contain carriage return characters to break a string message into multiple lines. By default, pszMessage is broken into multiple lines on word boundaries if necessary.

Parameters:

pszMessage - Message to display.

pszTitle - Title for message box. If 0, the default application title is used.

vsVSMB_flags - is a coVSMBination of zero or more VSID button flags and VSMB_* flags:

Note: The VSMB_OK, VSMB_OKCANCEL, VSMB_ABORTRETRYIGNORE, VSMB_YESNOCANCEL, VSMB_RETRYCANCEL are coVSMBinations of the VSID button flags (ex. VSMB_OKCANCEL is VSIDOK|VSIDCANCEL)

VSIDOK
Display an OK button
VSIDSAVE
Display a Save button
VSIDSAVEALL
Display a Save All button
VSIDOPEN
Display an Open button
VSIDYES
Display a Yes button
VSIDYESTOALL
Display a Yes to All button
VSIDNO
Display a No button
VSIDNOTOALL
Display a No to All button
VSIDABORT
Display an Abort button
VSIDRETRY
Display a Retry button
VSIDIGNORE
Display an Ignore button
VSIDCLOSE
Display a Close button
VSIDCANCEL
Display a Cancel button
VSIDDISCARD
Display a Discard button
VSIDHELP
Display a Help button
VSIDAPPLY
Display an Apply button
VSIDRESET
Display a Reset button
VSIDRESTOREDEFAULTS
Display Restore Defaults button
VSMB_OK
Display an OK button.
VSMB_OKCANCEL
Display an OK and Cancel button.
VSMB_ABORTRETRYIGNORE
Display Abort, Retry, Ignore buttons. Not supported on UNIX.
VSMB_YESNOCANCEL
Display Yes, No, and Cancel buttons.
VSMB_YESNO
Display Yes, and No buttons.
VSMB_RETRYCANCEL
Display a Retry and Cancel button. Not supported on UNIX.
VSMB_ICONHAND
Display a stop sign picture to the left of the message.
VSMB_ICONQUESTION
Display a question picture to the left of the message.
VSMB_ICONEXCLAMATION
Display an exclamation point picture to the left of the message.
VSMB_ICONINFORMATION
Display an 'i' picture to the left of the message.
VSMB_ICONSTOP
Display a stop sign picture to the left of the message.

default_button - If specified, determines which of the buttons is the default button. One of the following button constants:

  • VSIDOK
  • VSIDSAVE
  • VSIDSAVEALL
  • VSIDOPEN
  • VSIDYES
  • VSIDYESTOALL
  • VSIDNO
  • VSIDNOTOALL
  • VSIDABORT
  • VSIDRETRY
  • VSIDIGNORE
  • VSIDCLOSE
  • VSIDCANCEL
  • VSIDDISCARD
  • VSIDHELP
  • VSIDAPPLY
  • VSIDRESET
  • VSIDRESTOREDEFAULTS

Returns: int

Returns the button id of the button pressed which closed the dialog box. One of the following constants is returned:

  • VSIDOK
  • VSIDSAVE
  • VSIDSAVEALL
  • VSIDOPEN
  • VSIDYES
  • VSIDYESTOALL
  • VSIDNO
  • VSIDNOTOALL
  • VSIDABORT
  • VSIDRETRY
  • VSIDIGNORE
  • VSIDCLOSE
  • VSIDCANCEL
  • VSIDDISCARD
  • VSIDHELP
  • VSIDAPPLY
  • VSIDRESET
  • VSIDRESTOREDEFAULTS

Example:

result=vsMessageBox(
              "Save changes?",
              "",
              VSMB_YESNO|VSMB_ICONQUESTION);
if (result==VSIDYES) {
    // save the file
} else {
    // Don't save it
}
// Same as above but using VSID button flags instead of MB_OK.
// Also change the default button
result=vsMessageBox(
              "Save changes?",
              "",
              VSIDYES|VSIDNO|VSMB_ICONQUESTION,
              VSIDNO);
if (result==VSIDYES) {
    // save the file
} else {
    // Don't save it
}
