int VSAPI vsRefactorConfigSave(const char *xmlConfigFile=NULL)

Save the master refactoring [compiler] configuration file. An alternate configuration file path may be specified.

Parameters:

xmlConfigFile - Fully qualified path to XML configuration file

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions

See Also:
