_command typeless gui_save_as()

Saves the current buffer under a name you specify. The Explorer Standard Open dialog box or Standard Open dialog box is displayed to prompt you for the output file name. The name of the current buffer is changed unless the Keep old name check box is checked. You may use the "Save File As Type" combo box to change file format to DOS, UNIX, or Macintosh ASCII format.

Returns: typeless

Returns 0 if successful.

Applies To:

Edit Window

Category:

File Functions

See Also:

save, name, save_as
