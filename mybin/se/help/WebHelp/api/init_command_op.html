void init_command_op()

A command that operates on a single line in a buffer may be extended to operate on all text boxes including the command line by calling init_command_op before, and retrieve_command_results after the command. Calls to this function may not be nested.

Example:

_command upcase_line()
      name_info(','VSARG2_REQUIRES_EDITORCTL |VSARG2_TEXT_BOX)
{
     init_command_op();
     get_line(line);replace_line(upcase(line));
     retrieve_command_results();
}
