_command int get(_str src_filename='', _str obsolete='', _str insert_style='', int start_line=0, int end_line=0, _str load_options='')

Inserts file before or after current line

Parameters:

src_filename - Name of file to insert

obsolete - This argument is not used.

insert_style - Selects whether to insert after or before the current line. 'A', 'B', or '' which corresponds to 'After', 'Before', or def_line_insert setting.

start_line -

end_line -

load_options - may be "" or only may contain the following

  • +bi <buf_id>
  • +d
  • +b

Returns: int

Returns 0 if successful. Common return codes are FILE_NOT_FOUND_RC, PATH_NOT_FOUND_RC, TOO_MANY_SELECTIONS_RC, and TOO_MANY_FILES_RC. On error, message is displayed.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, File Functions
