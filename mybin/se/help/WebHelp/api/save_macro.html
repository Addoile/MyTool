_command typeless save_macro(_str macro_info='')

Saves the last macro recorded under the command name you specify. The Slick-C® source for the macro is appended to the "vusrmacs.e" file. The name and flag_names strings are used to create the following command definition: 

   _command name() name_info(','flag_names)

Parameters:

cmdline - a string in the format: name flag_names

Returns: typeless

Returns 0 if successful.

Category:

Macro Programming Functions
