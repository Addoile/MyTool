int _TreeGetParentIndex(int ItemIndex)

Parameters:

ItemIndex - Specifies a tree item.

Returns: int

Returns the tree item index of the parent of the node specified by ItemIndex. -1 is returned if the node specified has no parent (this only happens when ItemIndex is 0).

Applies To:

Tree View

Category:

Tree View Methods

See Also:

_TreeGetNextSiblingIndex, _TreeGetPrevSiblingIndex, _TreeGetFirstChildIndex
