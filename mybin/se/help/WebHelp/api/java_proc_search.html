int java_proc_search(_str &proc_name, int find_first)

If proc_name is '', then this procedure searches for the first or next occurrence of a Java language function definition. Search always starts from cursor position. The find_first parameter indicates whether this is the first or next call. proc_name is set to the name of the function found.

If proc_name is not '', then a search for a Java language function definition with name proc_name is performed. Searching begins at the cursor position. find_first must be non-zero if proc_name is not ''. The cursor is placed on the definition found, if one is found.

Returns: int

Returns 0 if successful. Otherwise a non-zero value is returned.

Applies To:

Editor Control, Edit Window

Categories:

Edit Window Methods, Editor Control Methods, Search Functions
