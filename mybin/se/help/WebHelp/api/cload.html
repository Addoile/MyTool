_command _str cload(...)

Loads color coding lexer definition file specified. If filename is not specified and the current buffer has a ".vlx" extension, it is loaded. Otherwise a Standard Open File dialog box is displayed which allows you to select a lexer definition file to load. See Color Coding for information on syntax of lexer definitions.

Returns: _str

Returns 0 if successful.

Category:

File Functions

See Also:

color_toggle, color_modified_toggle
