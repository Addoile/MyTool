_command int quick_highlight()

Highlight all occurrences of the word at cursor or selection. Use clear_highlights command to clear all highlights or you can also use Undo to remove the highlights.

Returns: int

Returns 0 if the search string is found. Otherwise, STRING_NOT_FOUND_RC is returned.. On error, message is displayed.

Applies To:

Edit Window, Editor Control

Category:

Search Functions

See Also:

find, gui_find, quick_search, clear_highlights
