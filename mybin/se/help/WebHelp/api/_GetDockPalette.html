int _GetDockPalette(vsDockingArea area)

Returns window id of dock palette for a particular MDI frame docking area. If no palette exists for the that side 0 is returned.

Parameters:

area - One of the following:

  • DOCKINGAREA_LEFT
  • DOCKINGAREA_TOP
  • DOCKINGAREA_RIGHT
  • DOCKINGAREA_BOTTOM

Applies To:

MDI Window

Category:

MDI Window Methods
