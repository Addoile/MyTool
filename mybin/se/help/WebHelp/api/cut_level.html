_command typeless cut_level()

Used in EMACS emulation. Deletes text from cursor to character before end of next parenthesis level and copies it to the clipboard. See NEXT-LEVEL command for information on how the end of the next parenthesis level is found. If the end of the next parenthesis level is not found, text from cursor to end of buffer is deleted and copied to the clipboard. Invoking this command multiple times in succession creates one clipboard.

Returns: typeless

Returns 0 if successful. Common return value is TOO_MANY_MARKS_RC. On error, message displayed.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

next_level, cut_prev_level, prev_level
