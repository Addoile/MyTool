_str name2event(_str name)

Returns: _str

Returns the binary string representation of the key named string_constant. The binary key string returned is the same as the binary key string returned by the GET_EVENT and TEST_EVENT built-in functions.

Example:

         key=get_event();
         if (key:==name2event('C-S')) {
             ...
         }

Category:

Keyboard Functions
