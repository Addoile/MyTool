int _menu_destroy(int menu_handle)

Destroy menu loaded by the _menu_load function. This function is typically used to destroy the old menu bar when replacing a menu bar or to destroy a pop-up menu after it has been displayed. menu_handle is typically returned by p_menu_handle or _menu_load.

Returns: int

Returns 0 if successful.
