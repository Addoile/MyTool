25 VSCALLBACK_APP_TAG_FILE_ADD_REMOVE

This callback is called after tag files are removed or added. This happens when the Tag Files dialog box is used or when the current project changes. This callback does get called when the vsTagSetTagFiles function is called.

Parameters:

pszTagFilename - Name of tag file being added or removed. Most of the time this if "" which indicates that the tag file(s) being added or removed are unknown so you must assume anything could have happened to the list of tag files.

pszOption - Specifies whether the tag file is being added or removed. "A" when adding a tag file and "R" when removing a tag file. When pszTagFileName is "", the value of pszOption is undefined.

Category:

Editor Control Callback Functions
