int dbg_session_reload(int session_id, _str file_names, int options)

Reload the given module into the debugger and continue debugging.

Parameters:

session_id - debugger session ID (1 .. max_session)

file_names - Modules (class files) to reload

options - System-specific options

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

   int status = dbg_session_reload(session_id, "mylibrary.so", 0);
   if (status) {
      return status;
   }

Category:

Debugger Functions
