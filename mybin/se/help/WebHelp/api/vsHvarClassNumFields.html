int VSAPI vsHvarClassNumFields(VSPSZ class_name)

Parameters:

class_name - Slick-C® class name

Returns: int

Return the number of non-static data members in a class or struct. Returns CLASS_NAME_NOT_FOUND_RC if the class is not found.

Category:

Miscellaneous Functions
