_command typeless gui_save_macro()

Saves the last macro recorded under the command name you specify. The Slick-C® source for the macro is appended to the "vusrmacs.e" file. The Save Macro dialog box is displayed to prompt for a name for the macro and options.

Returns: typeless

Returns 0 if successful.

Category:

Macro Programming Functions
