void VSAPI vsSetDefaultExtensionReferTo(const char *pszExtension, const char *pszLangId)

Allows you to specify that a file extension is associated with the specified language. This information is used to determine what language mode to choose when a file with the given extension is opened.

Parameters:

pszExtension - File extension. For list of file extension types, use our Extension Manager dialog ("Tools", "Options...", "File Extension Manager").

pszLangId - Language ID (see p_LangId For list of language types, Language Options dialog ("Tools", "Options", "Language Setup...").

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions

See Also:

vsGetDefaultExtensionReferTo, vsSetDefaultLanguageOptions, vsDeleteLanguageOptions, vsUpdateExistingBufferLanguageOptions
