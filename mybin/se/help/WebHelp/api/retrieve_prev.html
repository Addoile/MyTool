_command void retrieve_prev()

Moves the line position of the retrieve buffer ".command", one line up and places the contents of the line on the command line. If the current line is the first line of the buffer, the cursor is placed on the last line and the contents of the last line are placed on the command line. When a command is executed, the current line of the retrieve buffer becomes the last line. However, the first execution of retrieve_prev after the command is executed will place the contents of the last line on the command line and not move the line position.
