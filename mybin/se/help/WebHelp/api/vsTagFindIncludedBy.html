int VSAPI vsTagFindIncludedBy(const char *pszFileName, int find_first, char *pszIncludedBy, int included_max)

Retrieve the name of a file that includes 'pszFileName'.

Parameters:

pszFileName - name of included file

find_first - find first item, or next

pszIncludedBy - (reference) set to name of included file

included_max - size of buffer for 'pszIncluded'

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
