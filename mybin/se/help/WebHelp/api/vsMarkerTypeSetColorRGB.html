void VSAPI vsMarkerTypeSetColorRGB(int type, int ColorRGB)

Sets the marker type color RGB for the marker type specified. Currently only supported by stream markers with VSMARKERTYPEFLAG_DRAW_SCROLL_BAR_MARKER flag.

Parameters:

type - Index of marker type.

ColorIndex - SlickEdit RGB Color for type.

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
