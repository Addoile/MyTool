int _menu_delete(int menu_handle, int menu_pos)

Deletes a menu or menu item in a loaded menu or a menu resource.

Parameters:

menu_handle - Handle of loaded menu returned by p_menu_handle, _menu_load, _menu_find, or _menu_get_state. This handle may also be a handle to menu resource returned by find_index or name_match. menu_pos Position within menu menu_handle. The first menu item position is 0.

Returns: int

Returns 0 if successful.

Example:

#include "slick.sh"
defmain()
{
     status=_menu_find(_mdi.p_menu_handle,"M","help -using",mh,mpos);
     if (!status) _menu_delete(mh,mpos);
}
