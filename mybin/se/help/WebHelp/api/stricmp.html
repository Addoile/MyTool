int stricmp(_str s1, _str s2)

Perform a case insensitive Unicode (or DBCS) string compare. Compare is typically Unicode except for OEM's which may start the editor in non-Unicode mode.

Parameters:

s1 - String 1

s2 - String 2

Returns: int

1
s1>s2
0
s1==s2
-1
s1
