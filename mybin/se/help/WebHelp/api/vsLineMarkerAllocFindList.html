int*VSAPI vsLineMarkerAllocFindList(int wid, seSeekPos LineNum, seSeekPos LineOffset=(seSeekPos) VSNULLSEEK, int CheckNofLines=0, int *pNofItems=0)

Lists line markers

Parameters:

wid - Window id of editor control. 0 specifies the current object.

LineNum - Line number returned by vsPropGetI64(wid,VSP_LINE)

LineOffset - Optional specifies offset to search for markers which use MarkIds.

CheckNofLines - When on, only line markers with NofLines which include this line are returned.

pNofItems - May be null. Set to number of items in array returned.

Returns: int*VSAPI

Returns list of line markers indexes which match the line (or lines if CheckRange is non-zero) specified. -1 is appended to the end of the list to indicate the end of the list. 0 is returned if no matches are found. Call vsFree to free the list.

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
