void VSAPI vsMarkerTypeSetFlags(int type, int MarkerTypeFlags)

Sets the marker type flags for the marker type specified.

Parameters:

type - Index of marker type.

MarkerTypeFlags - Marker flags may be one or more of the VSMARKERTYPEFLAG_* flags. See flags for details.

VSMARKERTYPEFLAG_AUTO_REMOVE
Specifies that the marker be removed when all marker text is deleted.
VSMARKERTYPEFLAG_DRAW_BOX
Specifies that a box be drawn around the text. For line markers, the number of lines contained in the box is determined by the NofLines property of the line marker.
VSMARKERTYPEFLAG_DRAW_FOCUS_RECT
Specifies that a focus rectangle be drawn around the text. For line markers, the number of lines contained in the box is determined by the NofLines property of the line marker.
VSMARKERTYPEFLAG_DRAW_SQUIGGLY
Specifies that a squiggly underline be drawn under the text. For line markers, the number of lines contained in the box is determined by the NofLines property of the line marker.
VSMARKERTYPEFLAG_DRAW_SCROLL_BAR_MARKER

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
