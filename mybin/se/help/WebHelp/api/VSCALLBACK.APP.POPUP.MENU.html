9 VSCALLBACK_APP_POPUP_MENU

void (VSAPI *pfnPopupMenu)(const char *pszMenuName,int menu_handle);
The POPUP_MENU callback is called when a right click menu is displayed. Later we may expand this hook to support when any pop- up menu is displayed.

Parameters:

pszMenuName - Name of Slick-C® menu about to be displayed.

menu_handle - Handle to loaded menu. Use vsMenu??? API's to make any kind of modification to the menu before it gets display. For example, you might disable menu items, remove menu items, or even add menu items before the menu is displayed.

Category:

Editor Control Callback Functions
