int _StreamMarkerGet(int StreamMarkerIndex, VSSTREAMMARKERINFO &info)

Retrieve the details about the given stream marker.

Parameters:

StreamMarkerIndex - Index of stream marker returned by a vsStreamMarkerAdd function.

info - (reference) initialized with stream marker information.

Returns: int

0 if stream marker index is valid, <0 on error.

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
