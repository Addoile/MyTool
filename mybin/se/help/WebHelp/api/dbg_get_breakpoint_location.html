int dbg_get_breakpoint_location(int breakpoint_id, _str &file_name, int &line_number)

Get the file and line number for the specified breakpoint.

Parameters:

breakpoint_id - Breakpoint to examine

file_name - Name of file breakpoint is in

line_number - Line that breakpoint is on

Returns: int

1 for enabled, 0 for disabled, <0 on error.

Category:

Debugger Functions
