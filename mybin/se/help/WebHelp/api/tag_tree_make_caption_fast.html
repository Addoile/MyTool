_str tag_tree_make_caption_fast(int match_type, int local_or_context_id, boolean include_class=true, boolean include_args=true, boolean include_tab=false)

For the current tag, local, context, or match item, encode class name, member name, signature, etc. in order to make caption for tree item very quickly. Returns the resulting tree caption as a NULL terminated string. See tag_tree_make_caption() for further details about the output

For synchronization, macros should perform a tag_lock_context(false) prior to invoking this function.

Parameters:

match_type - VS_TAGMATCH_

local_or_context_id - 0 if inserting current tag from database, otherwise, specifies the unique integer ID of the item from the context, locals, or match set, as specified by match_type.

include_class - If 0, does not include class name

include_args - if 0, does not include function signature

include_tab - append class name after signature if 1, prepend class name with :: if 0

Returns: _str

Caption formatted in standard form as normally presented in symbol browser.

Category:

Tagging Functions
