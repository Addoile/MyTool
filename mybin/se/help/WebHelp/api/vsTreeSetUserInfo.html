int VSAPI vsTreeSetUserInfo(int wid, int iHandle, VSHVAR hvar)

Sets the user info value for the specified tree item.

Parameters:

wid - Window id of tree control. 0 specifies the current object.

ItemIndex - Tree item index.

hvarinfo - New value for user info of this tree item.

Applies To:

Tree View

Category:

Tree View Methods

See Also:

_TreeSetCaption, _TreeGetCaption, _TreeSetInfo, _TreeGetInfo, _TreeGetUserInfo
