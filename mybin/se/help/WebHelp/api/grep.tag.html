_command typeless grep_tag(_str search_str='', _str options="")

Search for tags matching the given search pattern. Note: 'gt' is shorthand for the grep-tag command. 

  Usage:  gt/tag_name_regex/search_options

Parameters:

search_str - Regex and search options. The following search options are accepted:

E
Case-sensitive match
I
Case-insensitive match
R
use SlickEdit regular expressions
U
use Unix regular expressions
B
use Brief regular expressions
L
use Perl regular expressions
&
use SlickEdit wildcard (filename) expressions
A
Search all tag files, not just the workspace tag file.
P
Limit search to the current project only

Returns: typeless

Returns 0 if successful. Otherwise, a non-zero value is returned.

Category:

Tagging Functions
