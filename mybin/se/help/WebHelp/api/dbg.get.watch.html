int dbg_get_watch(int thread_id, int frame_id, int watch_id, int &tab_number, _str &expr, _str &context_name, _str &value, int &expandable, int &base, _str &raw_value, _str &type)

Get the details about a watched variable.

Parameters:

thread_id - (optional, use 0) index of thread

frame_id - (optional, use 0) index of stack frame

watch_id - index of variable in list of watches

tab_number - (reference) [1-4], which watch tree to insert into

expr - (reference) expression to watch

context_name - (reference) the context to evaluate watch in

value - (reference) display value of variable

expandable - (reference) set to 1 if this node is expandable

base - (reference) base to display this variable's value in

raw_value - (reference) raw value of variable

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
