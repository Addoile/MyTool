boolean p_auto_select

Gets or sets the auto selection when tabbing between controls. When p_auto_select is non-zero, and you tab from a text box or combo box, the selection is removed. In addition, when you tab to a text box or combo box, the text is selected. If you do not want a text selection to change, set the p_auto_select property to zero.

Applies To:

Text Box, Check Box

Categories:

Text Box Properties, Check Box Properties
