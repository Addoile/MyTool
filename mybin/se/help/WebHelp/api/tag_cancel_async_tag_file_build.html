int tag_cancel_async_tag_file_build(_str pszTagDatabase, int waitForMS=0)

Cancel a tag file build or rebuild that is currently running in the background.

Parameters:

pszTagDatabase - name of tag file to cancel build/rebuild for

waitForMS - time in MS to wait for the thread to stop

Returns: int

0 on success, <0 on error.

Since:

16.0

Category:

Tagging Functions
