boolean batch_call_list(_str callback_name, typeless batch_arg=null, boolean forceCallbackOnTimer=false)

In a frequently called event handler, sometimes you will want to group events together and handle them all in one shot using a more general refresh function instead of a piecemeal handler for the specific event.

Call this function to schedule a one-shot timer to invoke a general refresh function. It only scedules the timer to call the refresh function after four singular events have passed. This way the simple, faster, piecemeal handler can be used for isolated events and you only go to the general, more expensive refresh handler for larger updates.

Parameters:

callback_name - name of general refresh callback to call_event

batch_arg - argument to pass to refresh callback

forceCallbackOnTimer - force callback, don't wait for threshold

Returns: boolean

The function returns 'true' if the threshold has been reached and the rest of the event handler callback should be ignored. Otherwise, it returns false, and the caller is responsible for handling the event handler callback itself.
