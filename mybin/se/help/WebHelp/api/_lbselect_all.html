void _lbselect_all()

Selects all lines in a list box. The p_Nofselected property is set to the number of lines in the list box. Do not call this function if the p_multi_select property is MS_NONE.

Applies To:

List Box

Category:

List Box Methods

See Also:

_lbdeselect_all, _lbselect_line, _lbdeselect_line, _lbinvert, _lbisline_selected, _lbmulti_select_result, _lbfind_selected
