int mou_mode(int mode=-1)

Gets and optionally sets the mouse event mode. The current new mouse event mode is returned.

Parameters:

mode - One of the following: 

-1 Do not set mode. Just return current mode.
 0 Turn mode off.
 1 Turn mode on. When on, mouse button-up, and move events are returned by get_event() and test_event().

Returns: int

Current mouse event mode.

Applies To:

All Window Objects

Category:

Mouse Functions

See Also:

get_event, test_event
