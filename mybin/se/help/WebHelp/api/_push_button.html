boolean _push_button(typeless Nofstates='', typeless new_value='')

Called after an lbutton_down event to give the effect of pushing a command button. The p_value property is temporarily set to the next state. You can specify the new_value to select the temporary button state. This function is typically called for 2 state pictures (p_Nofstates==2). You can specify a different number of states than the p_Nofstates property by specifying the Nofstates parameter.

Returns: boolean

Returns 1 if the button is pressed and released while the mouse pointer was within the window. Otherwise, 0 is returned.

Applies To:

Picture Box, Image

Categories:

Image Methods, Picture Box Methods
