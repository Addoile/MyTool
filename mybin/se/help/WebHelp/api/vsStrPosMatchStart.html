int VSAPI vsStrPosMatchStart(int MatchGroup)

Parameters:

TaggedExpression - Tagged expression number. For SlickEdit and Brief regular expressions the first tagged expression is 0 and the last is 9. For UNIX, the first tagged expression is 1 and the last is 0.

Returns: int

Returns start of tagged expression found for last search performed by vsStrPos, or vsStrLastPos. Result is not defined if the tagged expression was not found.

For SlickEdit and Brief regular expressions the first tagged expression is 0 and the last is 9. For UNIX and Perl, the first tagged expression is 1 and the last is 0.

Example:

See vsStrPos for an example.
