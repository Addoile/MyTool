int _TreeEndUpdate(int ItemIndex)

Finish effecient update of tree contents. This function will delete all tree nodes that were not recycled during the tree update.

Do not set column flags (_TreeSetColButtonInfo) between calls to _TreeBeginUpdate and _TreeEndUpdate, they will be reset.

Parameters:

ItemIndex - Specifies a tree item (parent node to update children of)

Applies To:

Tree View

Category:

Tree View Methods

See Also:

_TreeBeginUpdate, _TreeDelete
