_command void alt_bookmark()

Sets a bookmark identified by a letter or number corresponding to the key pressed which invoked this command.

This command should only be bound to the following groups of key combinations:

  • Alt+0-9
  • Alt+A-Z
  • Alt+F1-F12
  • Ctrl+0-9
  • Ctrl+A-Z
  • Ctrl+F1-F12
  • Ctrl+Alt+0-9
  • Ctrl+Alt+A-Z
  • Ctrl+Alt+F1-F12
  • Shift+F1-F12

Applies To:

Edit Window

Category:

Bookmark Functions
