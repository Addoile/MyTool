int _TreeSetCheckState(int index, int checkedState, ...)

Set the check state of a tree node

Parameters:

index - Handle to tree node to set check state for

checkedState - State of check box, one the following members of the TreeCheckVal enum: * TCB_UNCHECKED * TCB_CHECKED * TCB_PARTIALLYCHECKED

col - Column of tree node to set checkable (defaults to 0)

Returns: int

int 0 if successful

Applies To:

Tree View

Category:

Tree View Methods
