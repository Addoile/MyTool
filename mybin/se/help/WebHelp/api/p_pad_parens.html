boolean p_pad_parens

Determines whether to insert a space between parenthesis during syntax expansion for the current buffer. This property is available only when the control is displayed.

Example:

    // p_pad_parens = false
    if (x == 5)
    // p_pad_parens = true
    if ( x == 5 )

Applies To:

Editor_Control, Edit_Window

Categories:

Editor Control Properties, Edit Window Properties
