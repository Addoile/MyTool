void _TreeGetColButtonInfo(int ButtonIndex, int &ButtonWidth, int &ButtonFlags, int &StateNOTUSED, _str &Caption)

Retrieve the column button info for a multi-column tree control.

Parameters:

ButtonIndex - column index, 0 is the first column

ButtonWidth - column width, in twips

ButtonFlags - button flags, bitset of TREE_BUTTON_

State - this is no longer used, it is always 0

Caption - Button caption

Applies To:

Tree View

Category:

Tree View Methods
