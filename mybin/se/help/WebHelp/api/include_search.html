_str include_search(_str filename, _str multi_path='', _str multi_path2='')

Searches for filename in include directories specified by the include_dirs parameter. If the include_dirs parameter contains an '=' character, it is assumed to be an environment variable. include_dirs defaults to INCLUDE=' if not specified.

Returns: _str

If successful, a complete file specification for filename is returned. Otherwise '' is returned.

Category:

File Functions
