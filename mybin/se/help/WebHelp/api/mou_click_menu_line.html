_command typeless mou_click_menu_line()

Starts a line selection or displays an extension menu. Use the Extension Menu dialog box to set the extension specific menus. A selection can only be created if the "Select First" option is on (default is on).

Applies To:

Edit Window, Editor Control

Category:

Mouse Functions

See Also:

mou_click, mou_click_block, mou_click_line, mou_select_line, mou_extend_selection, mou_select_word, mou_click_menu, mou_click_menu_block
