int tag_is_parent_class(_str parent_class, _str child_class, _str(&tag_files)[], boolean case_sensitive, boolean normalize, ...)

Determine if parent_class is a parent of child_class, that is, does child_class derive either directly or transitively from parent_class.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Parameters:

parent_class - class to check if it is a parent (base class)

child_class - class to check if it is derived from parent

tag_files - (reference to _str[]) tag files to search

case_sensitive - case sensitive (1) or case insensitive (0)

normalize - attempt to normalize class name or take as-is?

parent_file - (optional) path to file containing the parent class

visited - (optional) hash table of prior results

depth - (optional) depth of recursive search

Returns: int

1 if 'child_class' derives from 'search_class', otherwise 0.

Category:

Tagging Functions

See Also:

tag_find_class, tag_get_inheritance
