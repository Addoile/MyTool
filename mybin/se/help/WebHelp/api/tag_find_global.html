int tag_find_global(int type_id, int mask, int nzero)

Retrieve the first tag included in this tag database with global scope that is one of the given type (type_id) and that matches the given tag flag mask (mask & tag.mask != 0). Tag names are ordered lexicographically, case insensitive. Use tag_get_info to extract the details about the tag.

Parameters:

type_id - Tag type ID. See tag_get_type for a list of all standard tag types. if (type_id<0), returns tags with ID>VS_TAGTYPE_LASTID

mask - Tag flags indicating symbol attributes, see tag_insert_simple for details.

nzero - if 1, succeed if mask & tag.flags != 0 if 0, succeed if mask & tag.flags == 0

Returns: int

0 on success, <0 on error.

Example:

The following would be used to find the first inline global function. 

  tag_find_global(VS_TAGTYPE_function, VS_TAGFLAG_inline, true);
