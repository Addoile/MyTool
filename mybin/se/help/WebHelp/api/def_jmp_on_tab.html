boolean def_jmp_on_tab

By default, moving the cursor over a tab character with the Left or Right arrow key causes the cursor to jump across the virtual space. To allow the Left and Right arrow keys to cursor into virtual space of tab characters, disable this option.

Also, when positioning the cursor with the mouse, this option also controls whether or not the cursor position is snapped to the nearest real character instead of allowing it to be placed in virtual space.

Default:

1

Category:

Configuration Variables
