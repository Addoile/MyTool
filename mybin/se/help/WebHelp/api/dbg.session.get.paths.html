int dbg_session_get_paths(int session_id, _str &cwd, _str &user, _str &sys)

Parameters:

session_id - debugger session ID (1 .. max_session) Get the user and system class paths for this debugging session.

cwd - Set to the base directory or working directory

user - Set to user path, delimited by newlines

sys - Set to system path, delimited by newlines

Returns: int

0 on success, <0 on error

Since:

10.0

Example:

   _str cwd, user, sys;
   int status = dbg_session_get_paths(session_id, cwd, user, sys);
   if (status) {
      return status;
   }
   _message_box("cwd=" :+ cwd :+ "\n" :+
                "user=" :+ user :+ "\n" :+
                "sys=" :+ sys :+ "\n");
