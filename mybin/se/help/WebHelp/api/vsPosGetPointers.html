void VSAPI vsPosGetPointers(unsigned char **pp, unsigned char **ppEndLine, unsigned char **ppEndBuf=a)

You must call vsPosInit before calling this function. Once you use a non vsPosXXX function which accesses buffer data you MUST call vsPosInit again to avoid crashing the editor. Only use this function if you want your searching to be as fast as possible.

This function is used to quickly scan part of the current Pos line (Not the same as the cursor position).

Parameters:

pp - Set to point to current byte. You must check that pp is less than *ppEndBuf.

ppEndLine - Set to point past the last valid byte of the current Pos line segment.

ppEndBuf - Set to point past the last valid byte of the current Pos buffer segment.

Applies To:

Edit Window, Editor Control

Category:

Search Functions
