typeless _SetBufferInfoHt(_str key, typeless value)

Store data to the current buffer indexed by the given key.

Use _GetBufferInfoHt() or _GetBufferInfoHtPtr() to retrieve data from a dialog by hash table key.

This function differs from _SetDialogInfoHt() because it stores data in a hash table per open file (buffer) rather than per-dialog.

IMPORTANT:
The old value stored by hash table key is returned. null is returned if there is no old value stored. You should initialize your key/value pairs if you want to safely take advantage of the return value.

Parameters:

key - A unique hash table key that represents a given item.

value - value to store in hash table

Returns: typeless

Old value stored by key. null is returned if there is no old value stored for that key.

Category:

Buffer Functions
