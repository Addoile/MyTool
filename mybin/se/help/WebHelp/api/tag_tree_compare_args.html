int tag_tree_compare_args(_str arg_list1, _str arg_list2, boolean unqualify)

Compare the two argument lists, this method works for both Delphi/Pascal/Ada style arguments and C/C++/Java style arguments.

Parameters:

arg_list1 - Argument list number 1

arg_list2 - Argument list number 2

unqualify - Loose comparison, peel off class qualifications

Returns: int

0 if they match, nonzero otherwise.

Category:

Tagging Functions

See Also:

tag_get_detail, tag_tree_format_args
