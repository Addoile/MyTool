int refactor_config_get_java_source(_str name, _str &src)

Retrieve the root directory of the given Java configuration.

Parameters:

configIndex - index of configuration to query name of [0..n-1]

src - [reference] set to root directory of Java configuration

Returns: int

0 on success, <0 on error
