int refactor_c_static_to_instance_method(int nHandle, _str szMethodName, _str szClassName, _str szFileName, int nSeekPosition, int nFlags)

Replace a static method with an instance method.

Parameters:

nHandle - Handle to transaction

szMethodName - Name of static method to be converted to an instance method.

szClassName - Name of class the method is a member of.

szFilename - Filename to find the method in

nSeekPosition - Seek position of the method or position to start search

nFlags - Flags for access modifiers

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
