int tag_find_type(_str &type_name, _str search_for=null)

Retrieve the name of the first tag type included in this tag database, or optionally, the name of a specific tag type, either to check if it is in the database or to position the iterator at a specific point.

Parameters:

type_name - (Output) full path of type containing tags

search_for - Specific type to search for (prefix search)

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions

See Also:

tag_get_type, tag_next_type, tag_reset_find_type
