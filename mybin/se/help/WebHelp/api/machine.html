_str machine()

Returns: _str

Returns 'WINDOWS' if running under Windows NT or newer on an Intel compatible machine. Returns 'NT386' if running under Windows NT on 386 compatible machine. Returns 'NTMIPS' if running under Windows NT on MIPS/compatible machine. Returns NTALPHA if running under Windows NT on a DEC ALPHA AXP machine. Returns 'SPARC' if running under SUN SPARC station. Returns 'RS6000' if running under AIX RS600. Returns 'NTPPC' if running under Windows NT on a PowerPC. Returns 'LINUX' if running under LINUX on Intel machine. Returns 'HP9000' if running under UNIX on HP9000. Returns 'SGMIPS' if running under UNIX on Silicon Graphics machine. Returns 'MACOSX11' if running under Mac OS X on a PowerPC.

  • "WINDOWS" -- Windows NT or newer on Intel compatible machine.
  • "PCDOS" -- Windows 3.1
  • "PCDOSP" -- Windows 3.1 protected mode?
  • "NTPPC" -- Windows NT on a PowerPC.
  • "NTMIPS" -- Windows NT on MIPS/compatible machine.
  • "NTALPHA" -- Windows NT DEC ALPHA AXP machine.
  • "NT386" -- Windows NT on 386 compatible machine.
  • "OS2386" -- OS/2 on 386 compatible machine
  • "OS2PPC" -- OS/2 on PowerPC.
  • "LINUX" -- LINUX on Intel machine.
  • "SPARCSOLARIS" -- Sun Solaris on Sun SPARC station.
  • "SPARC" -- SUN SPARC station.
  • "RS6000" -- AIX RS600.
  • "M88K" -- Unix on Motorola 88K
  • "HP9000" -- UNIX/HPUX on HP9000.
  • "SGMIPS" -- UNIX/Irix on Silicon Graphics machine.
  • "EPMIPS" --
  • "UNIXWARE" -- Unixware on Intel compatible machine.
  • "ALPHAOSF" -- OSF/1 on DEC ALPHA AXP machine.
  • "INTELSOLARIS" -- Sun Solaris on Intel compatible machine.
  • "DECULTRIX" -- DEC Ultrix on MIPS
  • "SCO" -- SCO on Intel compatible machine.
  • "FREEBSD" -- FreeBSD on Intel compatible machine.
  • "UNIXWARE" -- UnixWare on Intel compatible machine.
  • "S390" -- Operating System/390
  • "LINUXON390" -- Linux on System/390
  • "MACOSX" -- Native Mac OS X
  • "MACOSX11" -- Mac OS X under X11
