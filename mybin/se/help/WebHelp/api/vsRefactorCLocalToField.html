int VSAPI vsRefactorCLocalToField(int handle, VSPSZ filename, VSPSZ symbolName, seSeekPos startSeekPos, seSeekPos endSeekPos, VSPSZ szFieldName, int flags)

Convert local variable to a field

Parameters:

handle - Handle to transaction

filename - Filename to find symbol in

symbolName - Name of symbol to rename

startSeekPos - Seek position of symbol or position to start search

endSeekPos - Seek position to end symbol search (0 if begin is actual symbol)

szFieldName - Name of field

flags - Flags for access modifiers and make field static

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
