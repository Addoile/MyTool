int VSAPI vsMatchLength(int TaggedExpression=-)

Parameters:

TaggedExpression - Tagged expression number. For SlickEdit and Brief regular expressions the first tagged expression is 0 and the last is 9. For UNIX, the first tagged expression is 1 and the last is 0.

Returns: int

Returns the length of the last match or tagged expression found by the last vsSearch or vsRepeatSearch.

Example:

int status;
status=vsSearch(0,"this|that",-1,"r");
if (!status ){
    char temps[100];
    // Get the word matched
    vsGetText(0,vsMatchLength(),vsMatchStart(),temps);
    vsMessage(temps);
}

Applies To:

Edit Window, Editor Control

Category:

Search Functions
