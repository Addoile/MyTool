void VSAPI vsMarkerTypeSetPriority(int type, int DrawPriority)

Sets the drawing priority for the marker type specified. 0 is the highest priority, 255 is the lowest priority.

Parameters:

type - Index of marker type.

DrawPriority - unsigned integer between 0 and 255. 0 is the default draw priority.

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
