int signal_handler(_str option, int index=-1)

Returns: int

Returns index of current signal handler for specified signal. Currently this function only supports the 'B'=Ctrl+Break signal. Specifying the index parameter will set the signal handler. Valid values for index must be less than or equal to 0. If index is 0, the default signal handler is used which halts Slick-C® batch/macros that are running. If index is less than 0, the Ctrl+break signal is ignored.
