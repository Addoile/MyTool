_str _pad(_str text, int totalLength, _str padChar, _str options='R')

Pads a string with extra characters to get it up to a specified length. Can pad on either the right or the left.

Parameters:

text - the string to be padded

totalLength - the desired total length

padChar - the padding character to use

options - behavior options 

L - pad on the left side of the string
R (default) - pad on the right side of the string

Returns: _str

the padded string

Category:

String Functions
