_command typeless cut_word()

Deletes from the cursor to the end of the word at the cursor or the next word and copies it to the clipboard. Invoking this command from the keyboard multiple times in succession creates one clipboard. The deleted text may be retrieved by the command paste.

Returns: typeless

Returns 0 if successful. Common return codes are TOO_MANY_SELECTIONS_RC, and STRING_NOT_FOUND_RC. On error, message is displayed.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Clipboard Functions, Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods
