void _macro_delete_show(_str form_name)

If macro recording and output is on (_macro()!=0), and the last source line is of the form: 

    execute('show -modal <form_name>');
the last source line of recorded macro is deleted.

Parameters:

form_name - name of form to expect to see

Category:

Macro Programming Functions
