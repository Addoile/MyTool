void _dump_var(var v, _str caption='', int dumpToScreen=1, int dumpToFile=0)

Dump the contents of the given Slick-C variable to the debug window, or the log file.

Parameters:

v - variable to display contents of

caption - caption to print out for variable

dumpToScreen - dump the stack to the debug window

dumpToFile - dump the stack the log file

Category:

Miscellaneous Functions
