int tag_tree_type_is_package(_str type_name)

Speed-demon way to determine if the given type is a package, library, or program (package, lib, prog).

Parameters:

type_name - String specifying tag's type name (from tag_get_info)

Returns: int

1 if the item is a package, 0 otherwise.

Tagging Functions
