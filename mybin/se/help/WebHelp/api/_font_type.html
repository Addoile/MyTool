int _font_type(_str font_name, _str options="S")

This function is used by font dialog boxes. The following font type flags are returned:

RASTER_FONTTYPE
Font is a raster font.
DEVICE_FONTTYPE
Font is a printer font.
TRUETYPE_FONTTYPE
Font is a true type font
FIXED_FONTTYPE
Font is a fixed size

Parameters:

options - a string of zero or more of the following option letters:

F
This letter is ignored and is only here because it is used by other font functions.
P
Get font type for print font font_name.
S
(Default) Display screen fonts.

Category:

Miscellaneous Functions
