int _copy_to_clipboard(_str markid="", boolean isClipboard=true, boolean addClipboard=false)

Copies text selection to the operating system clipboard.

Parameters:

markid - Handle to a selection returned by one of the built-ins _alloc_selection or _duplicate_selection. A mark_id of '' or no mark_id parameter identifies the active selection.

isClipboard - Indicates whether operation is performed for the clipboard or selection. Effects Unix only.

addClipboard - When true, a text clipboard is added to the set of clipboards which will be added later when _clipboard_close() is called. _clipboard_open must be called before clipboard close.

Returns: int

Returns 0 if successful.

Applies To:

Edit Window, Editor Control

Categories:

Clipboard Functions, Edit Window Methods, Editor Control Methods
