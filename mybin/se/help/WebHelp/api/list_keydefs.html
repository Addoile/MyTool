_command typeless list_keydefs()

Runs the external batch macro 'listkeys.e' with no arguments. This command creates a new buffer and inserts Slick-C® source code representing all the key definitions including key definitions for all modes. You may want to print this file and use it as a reference of your key definitions.

Category:

Keyboard Functions
