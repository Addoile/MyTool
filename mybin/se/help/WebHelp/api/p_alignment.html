int p_alignment

Gets or sets the alignment. Radio buttons and check boxes do not support this property under UNIX.

Label - Determines where p_caption text gets displayed. The p_alignment property has no effect if p_auto_size is non-zero. p_alignment is one of the following constants defined in "slick.sh":

AL_LEFT
Display caption left justified.
AL_RIGHT
Display caption right justified.
AL_CENTER
Display caption horizontally centered at top.
AL_VCENTER
Display caption vertically centered at left.
AL_VCENTERRIGHT
Display caption vertically centered at right.
AL_CENTERBOTH
Display caption centered vertically and horizontally.

Radio Button, Check Box - Determines whether p_caption text gets displayed on left or right. We use the same conventions as Visual Basic (which are backwards). p_alignment is one of the following constants defined in "slick.sh":

AL_LEFT
Display caption right justified.
AL_RIGHT
Display caption left justified.

Applies To:

Label, Radio Button, Check Box

Categories:

Label Properties, Radio Button Properties, Check Box Properties
