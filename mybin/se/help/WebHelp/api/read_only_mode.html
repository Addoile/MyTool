_command void read_only_mode()

Switches to read only mode. While in read only mode, you will not be able to modify the current buffer by a command which modifies text. If the off parameter is specified and not "", read only mode is turned off.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
