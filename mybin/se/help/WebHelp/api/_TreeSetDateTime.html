void _TreeSetDateTime(int index, int column, int year, int month, int day, ...)

Set the type of a treenode of a tree to a date so that the dates will be sorted properly and displayed according to the OS's settings. Do not set the caption if you use this.

Parameters:

index - tree node index

col - tree column

year -

month -

day -

hour - (optional) 0-23

minute - (optional) 0-59

second - (optional) 0-59

milisecond - (optional) 0-999

Applies To:

Tree View

Category:

Tree View Methods
