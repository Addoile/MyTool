_str _clipboard_format(_str format, boolean isClipboard=true)

Determines if a particular operating system clipboard format is available. Use the clipboard_itype function to determine the format of an internal TEXT clipboard.

Parameters:

format - may be one of the following constants defined in "slick.sh": 

   CBF_TEXT There is a TEXT format clipboard.
   CBF_VSTEXT  There is a SlickEdit TEXT format clipboard.
   CBF_VSCONTROLS
   There is a SlickEdit CONTROLS format clipboard.

isClipboard - Indicates whether operation is performed for the clipboard or selection. Effects Unix only.

Returns: _str

Returns 0 if the clipboard format is not available. Otherwise, a non-zero value is returned. If CBF_VSTEXT is given and the format exists, a string in the format "type process_id more_info" is returned.

Category:

Clipboard Functions
