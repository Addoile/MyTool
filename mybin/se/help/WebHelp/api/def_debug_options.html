int def_debug_options

Debugging support options flags Bitset of VSDEBUG_OPTION_

  • VSDEBUG_OPTION_SUPPORT_HOTSWAP -- support hot swap (Java)
  • VSDEBUG_OPTION_CONFIRM_DIRECTORY -- confirm directory when locating source files
  • VSDEBUG_OPTION_ALLOW_EDITING -- allow source files to be edited during debugging
  • VSDEBUG_OPTION_MOUSE_OVER_INFO -- show values of variables when mouse floats over them
  • VSDEBUG_OPTION_AUTO_CORRECT_BP -- automatically correct breakpoint locations

Default:

0xffffffff

Category:

Configuration Variables
