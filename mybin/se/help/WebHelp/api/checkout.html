_command int checkout(_str filename="")

The checkout command checks out the files specified from the version control system specified by the current project. If cmdline is not given or '', the Check Out dialog box is displayed.

Parameters:

options - is a string of one or more of the following switches: 

   + or -r  Checkout the file read only.  Ignored by Delta.
   + or -e  Edit the file after checking it out.

   You may specify '.' to check out the current buffer ("checkout .").

Returns: int

Returns 0 if successful.

Exmple:

cmdline is a string in the format: [options] file1 file2 file3 ...
