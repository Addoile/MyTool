int VSAPI SETagInsertInfoInDatabase(int dbHandle, slickedit::SEHashSet*oldTags, const slickedit::SETagInformation &tagInfo)

API function for inserting a tag entry with supporting info

Parameters:

dbHandle - database handle

oldTags - old tag information (from SETagFileInsertStart())

tagInfo - tag information to insert into database

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
