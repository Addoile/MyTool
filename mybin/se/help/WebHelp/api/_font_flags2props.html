void _font_flags2props(int font_flags)

This function sets the font properties of the current object based on the font flags. font_flags is a combination of the following flag constants defined in "slick.sh":

    F_BOLD
    F_ITALIC
    F_STRIKE_THRU
    F_UNDERLINE
    F_PRINTER
