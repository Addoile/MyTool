int VSAPI vsTagGetDate(const char *pszFilename, char *pszDate=a, const char *pszIncludedBy=a)

Retrieve the date of tagging for the given file. The string returned by this function is structured such that consecutive dates are ordered lexicographically, and is reported in local time coordinates (YYYYMMDDHHMMSSmmm). This function has the side effect of positioning the file iterator on the given file name, and may be used to test for the existence of a file in the database.

Parameters:

pszFilename - File path of source file to retrieve tagging date for.

pszDate - (20 bytes, optional) Modification date of file when tagged. Format is YYYYMMDDHHMMSSmmm (year, month, day, hour, minute, second, ms)

pszIncludedBy - File path of source file which includes pszFilename

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if the file is not found.

Category:

Tagging Functions
