void _TreeGetComboDataNodeCol(int index, int col, STRARRAY comboBoxLines, ...)

Get combo box data for a given node and column

Parameters:

index - index of item to set data for

col - column of item to set data for (0 is the leftmost)

comboBoxLines - Filled in with times from combo box

comboBoxStyle - One of the PSCBO_* styles.

Applies To:

Tree View

Category:

Tree View Methods
