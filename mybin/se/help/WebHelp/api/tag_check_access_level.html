int tag_check_access_level(int context_flags, int tag_flags)

Determines access to a tag

Parameters:

context_flags - VS_TAGCONTEXT_

tag_flags - Tag flags indicating symbol attributes, see tag_insert_simple for details.

Returns: int

Returns true if at the current access level (represented by context_flags), we have access to the member with the given access restrictions (represented by tag_flags), false if not.

Category:

Tagging Functions

See Also:

tag_check_context_flags, tag_insert_simple
