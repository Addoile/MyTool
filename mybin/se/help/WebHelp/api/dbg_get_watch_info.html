int dbg_get_watch_info(int thread_id, int frame_id, _str watch_path, _str &name, _str &class_name, _str &signature, _str &value, int &flags, _str &raw_value)

Get the details about a variable or expanded watch variable

Parameters:

thread_id - index of thread

frame_id - index of stack frame

watch_path - path to auto-watch variable in tree

name - (reference) name of variable

class_name - (reference) name of class the member is in

signature - (reference) type of variable

value - (reference) display value of variable

flags - (reference) set to bitset of DEBUG_MODEL_FLAGS_

raw_value - (reference) raw value of variable

Returns: int

0 on success, <0 on error

Category:

Debugger Functions
