_command void toolbars(_str formName='1')

Displays the Toolbar Customization dialog box. This dialog box is used to add and remove buttons from a toolbar. You can also create new toolbars and choose which toolbars you want visible.

The following toolbars are available:

Project
Default toolbar display at left of MDI frame. Includes tabs for symbol browser, project files, buffers, and disk files.

Output
Default toolbar display at bottom of MDI frame. Includes tabs for build output, multi-file search output, and tag symbols for word at th cursor.

Standard
Default toolbar displayed at top of MDI frame. Includes buttons for file open, file save, clipboard operations, searching, help, version control, and displaying the current file in a web browser.

Tools
Includes buttons for the source code beautifier, difference editor, 3-way merge, find file, calculator, command prompt, spell checker, and hex mode toggle.

Edit
Includes buttons for uppercase, lowercase, shift selection left, shift selection right, indent, unindent, tabify selection, untabify selection, and match parenthesis.

Selective Display
Includes selective display buttons for function headings, hide code block, selective display dialog box, hide selected lines, and show all.
