int tag_set_local_template_signature(int local_id, _str template_args)

Set the template signature for the given local tag.

For synchronization, macros should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

local_id - Local ID [ from 1 to tag_get_num_of_locals() ] of local variable tag representing class or struct to set inheritance relationships for.

template_signature - Template signature for this local tag, in language specific format, normally delimited by commas.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions

See Also:

tag_get_local2, tag_get_detail2, tag_insert_local, tag_set_inheritance
