int VSAPI vsRefactorConfigAdd(const char *configName, const char *configHeader, const char *includePath)

Add a new C/C++ compiler configuration to the master refactoring configuration file.

Parameters:

configName - name for this configuration

configHeader - header file to configure the compiler

includePath - PATHSEP deliminated include file path

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
