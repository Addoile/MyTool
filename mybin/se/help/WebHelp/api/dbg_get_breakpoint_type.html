int dbg_get_breakpoint_type(int breakpoint_id, int &breakpoint_flags)

Get the type of a breakpoint (line, method, or watch)

  • VSDEBUG_BREAKPOINT_LINE -- file/line oriented breakpoint
  • VSDEBUG_BREAKPOINT_METHOD -- class/method breakpoint
  • VSDEBUG_BREAKPOINT_ADDRESS -- address within a function
  • VSDEBUG_WATCHPOINT_READ -- read access on breakpoint condition
  • VSDEBUG_WATCHPOINT_WRITE -- write access on breakpoint condition
  • VSDEBUG_WATCHPOINT_ANY -- any access on breakpoint condition
  • VSDEBUG_WATCHPOINT_INSTANCE_READ -- read access on breakpoint condition
  • VSDEBUG_WATCHPOINT_INSTANCE_WRITE -- write access on breakpoint condition
  • VSDEBUG_WATCHPOINT_INSTANCE_ANY -- any access on breakpoint condition

Parameters:

breakpoint_id - index of variable in list of breakpoints

breakpoint_flags - (reference) set to bitset of VSDEBUG_WATCHPOINT_FLAG_

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
