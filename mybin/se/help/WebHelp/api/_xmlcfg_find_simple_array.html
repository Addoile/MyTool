int _xmlcfg_find_simple_array(int iHandle, _str QueryStr, _str(&Array)[], int NodeIndex=TREE_ROOT_INDEX, int FindFlags=0)

Searches for nodes based on an subset XPath expression.

Parameters:

iHandle - Handle to an XMLCFG tree returned by _xmlcfg_open() ,_xmlcfg_create(), or _xmlcfg_open_from_buffer().

QueryStr - This is a small subset of an XPath expression. See _xmlcfg_find_simple() for information on this parameter.

Array - Set to node indexes or values found

NodeIndex - Tree node index to start search from.

FindFlags - Zero or more of the following flags:

VSXMLCFG_FIND_APPEND
When specified, items are append to the end of Array.
VSXMLCFG_FIND_VALUES
When specified, values are returned instead of node indexes.

Returns: int

Returns 0 if successful. Check the length (Array._length()) of the the array to see if any nodes were found. Otherwise a negative error code is returned.

Example:

// Find all file element nodes under /project
_xmlcfg_find_simple_array(handle,"/project/file");

// Find the "file" element nodes that are anywhere
// beneath the node specified.
_xmlcfg_find_simple_array(handle,"//file");

Category:
