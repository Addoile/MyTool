int _GetDefaultLanguageOptions(_str lang, _str origExtension, VS_LANGUAGE_OPTIONS &langOptions)

Gets language specific options for a specific language type.

Parameters:

lang - Language ID (see p_LangId). For list of language types, use our Language Manager dialog ("Tools", "Options...", "Language Manager")

origExtension - File extension referred to this language. This is necessary for retrieving options which are genuinely per extension, such as encoding and associated open application.

langOptions - Initialized to language specific options for pszLangId.

Returns: int

Returns 0 if successful.

Since:

13.0

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions

See Also:

_SetDefaultLanguageOptions, _SetExtensionReferTo, _DeleteLanguageOptions, _SetDefaultLanguageOptions
