void _clex_xmlAddKeywordAttrs(_str tag_name, _str attr_list, ...)

Adds the tag and attributes specified to the XML color coding for the current buffer.

Parameters:

tag_name - XML tag name.

attr_list - Space delimited list of attributes which belong to the tag_name specified.

Applies To:

Edit Window, Editor Control

Categories:

Editor Control Methods, Miscellaneous Functions
