int _TreeGetNextIndex(int ItemIndex)

Parameters:

ItemIndex - Specifies a tree item.

Returns: int

Returns the tree item index of the next tree node as though the tree is a list box. By default, will not return hidden nodes. Specify 'H' to return hidden nodes. -1 is returned if the node specified has no next node.

Applies To:

Tree View

Category:

Tree View Methods

See Also:

_TreeGetParentIndex, _TreeGetNextIndex, _TreeGetPrevIndex, _TreeGetNextSiblingIndex, _TreeGetPrevSiblingIndex, _TreeGetFirstChildIndex
