_command int cut(boolean push=true, boolean doCopy=false, _str name='')

Deletes selection and copies it to the clipboard. For an edit window or editor control, if no text is selected, the current line is deleted and copied to the clipboard. All clipboards are stored in the ".clipboards" buffer. The last clipboard may be inserted by the paste command (Ctrl+V). Previous clipboards may be inserted with the list_clipboards command (Ctrl+Shift+V).

In ISPF emulation, this command is not called when invoked from the command line. Instead ispf_cut is called. Use ("Edit", "Cut") to explicitly invoke the cut command.

Parameters:

push -

doCopy -

name -

Returns: int

Returns 0 if successful. Common return codes are 1 (no line at cursor) and TOO_MANY_SELECTIONS_RC. On error, message is displayed.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box
