int _TreeAdjustColumnWidths(int col=-1, int *pNumWrappedRows=null)

Adjust the widths of one or all tree columns depending on the text in the tree.

Parameters:

col - column to adjust, -1 to adjust all columns

Returns: int

if given a specific column, return the width of that column. If adjusting all columns, return the total width of all columns.

Applies To:

Tree View

Category:

Tree View Methods
