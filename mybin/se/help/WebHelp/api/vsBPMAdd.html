int VSAPI vsBPMAdd(int wid, seSeekPos RealLineNum, int flags, void *puserdata=a)

Deprecated. Use vsLineMarker* functions

Adds a break point.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

RealLineNum - Real line number (VSP_RLINE).

flags - One of the flags below:

  • VSBPFLAG_BREAKPOINT
  • VSBPFLAG_EXEC
  • VSBPFLAG_STACKEXEC
  • VSBPFLAG_BREAKPOINTDISABLED

puserdata - Optional pointer to user data. Use this to store debugger specific information.

Returns: int

Returns an index to a break point which is valid input to vsBPMGet().

Category:

Deprecated Breakpoint Functions
