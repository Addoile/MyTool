int tag_tree_add_members_in_category(int type_id, int mask, int nzero, _str category_name, int &in_count)

Add globals with the given type the given category, as set up using tag_tree_prepare_expand. Refer to tag_find_global for details about the parameters type_id, mask, and non_zero.

Parameters:

type_id - Type id (see tag_get_type) If (type_id<0), inserts global tags with any user defined tag types.

mask - Tag flags indicating symbol attributes, see tag_insert_simple for details.

nzero - add member if (mask & tag_flags) is zero or nonzero?

category_name - tag category to add members from

in_count - (reference, input, output), number of items inserted

Returns: int

0 on error, <0 on error.

Category:

Tagging Functions
