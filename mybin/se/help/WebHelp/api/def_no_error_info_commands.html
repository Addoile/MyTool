_str def_no_error_info_commands

This list is examined when certain commands, such as project commands, are automatically generated and when ENTER is pressed in the build window.

This is a list of commands for which we do not need to insert the current directory.

Default:

Windows: 'cl javac sj sgrep vst msdev'; Unix: 'cc CC gcc g++ vst javc sgrep c89 c++ vscomp.rexx'

Category:

Configuration Variables
