boolean tag_return_type_equal(struct VS_TAG_RETURN_TYPE &rt1, struct VS_TAG_RETURN_TYPE &rt2, boolean case_sensitive=true)

Compare two return type info sets for equality. This function does not support templates and ignore the match_tag.

Parameters:

rt1 - first return type to compare

rt2 - second return type to compare

case_sensitive - case sensitive comparison?

Returns: boolean

true if equal, false otherwise

Category:

Tagging Functions
