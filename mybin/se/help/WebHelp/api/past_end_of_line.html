boolean past_end_of_line(boolean moveToEndIfPast=false)

Deteremine if the cursor is past the end of the current line

Parameters:

moveToEndIfPast - If true and cursor is past end of line, move cursor to the end of line

Returns: boolean

true if the cursor is past the end of the current line

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
