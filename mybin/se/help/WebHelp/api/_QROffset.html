long _QROffset()

Returns: long

Returns the number of bytes before the cursor. Non-savable lines (lines with the NOSAVE_LF flag set) are not included. This method is intended for dealing with disk seek position. However, if you have changed the load options to translate files when they are openned, these offsets will not match what is on disk.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Search Functions

See Also:
