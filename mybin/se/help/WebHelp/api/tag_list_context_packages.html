int tag_list_context_packages(int tree_wid, int tree_index, _str prefix, _str(&tag_files)[], var vnum_matches, int max_matches, boolean exact_match, boolean case_sensitive)

List the packages matching the given prefix expression.

For synchronization, macros should perform a tag_lock_matches(true) and tag_lock_context(false) prior to invoking this function.

Parameters:

tree_wid - window id of tree control to insert into, 0 indicates to insert into a match set

tree_index - tree index to insert matches under

prefix - symbol prefix to match

tag_files - (reference to _str[]) tag files to search

vnum_matches - (Output) number of matches

max_matches - maximum number of matches allowed

exact_match - exact match or prefix match (0)

case_sensitive - case sensitive (1) or case insensitive (0)

Returns: int

Returns 0 on success, <0 on error.

Category:

Tagging Functions
