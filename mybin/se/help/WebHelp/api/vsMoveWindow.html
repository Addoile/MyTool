void VSAPI vsMoveWindow(int wid, int x, int y, int width, int height, char state=a, int reserved=a)

Moves and sizes the current window to the position and size specified. The input position and size parameters are specified in the parent scale mode (VSP_XYSCALEMODE). For an editor control created with vsCreateEditorCtl, the parent scale mode is always in pixels.

After calling this function, you need to call vsRefresh with the 'A' option so that the scroll bars are updated.

Parameters:

wid - Window id of object. 0 specifies the current object.

x - X position of window relative to parent.

y - Y position of window relative to parent.

width - Width of window.

height - Height of window.

state - One of the following:

0 or 'C'
Change position and size of window for its current state.
'N'
Change normalized position and size of window.

Applies To:

All Window Objects
