int tag_find_tag(_str tag_name, _str type_name, _str class_name, _str arguments=null)

Retrieve first tag with the given tag name, type, and class (all are necessary to uniquely identify a tag). If class_name is unknown, simply use NULL. Use tag_get_info to extract the details about the tag.

Parameters:

tag_name - name of tag to search for

type_name - Tag type name. See tag_get_type for a list of all standard tag types.

class_name - name of class that tag is present in, use concatenation (as defined by language rules) to specify names of inner classes.

arguments - function arguments to attempt to match. Ignored if they result in no matches.

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Category:

Tagging Functions

See Also:

tag_find_equal, tag_find_prefix, tag_get_detail, tag_get_info, tag_next_tag, tag_reset_find_tag
