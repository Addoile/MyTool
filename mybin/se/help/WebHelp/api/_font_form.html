_font_form

Displays the Font dialog box. The 'P' option specifies that printer fonts be listed. font specifies the initial font displayed in this dialog box.

Syntax _str show('-modal _font_form', options , font)

Parameters:

options - a string of zero or more of the following option letters:

F
Display fixed fonts only.
P
Display printer fonts. If not specified, screen fonts are displayed.

font - a string in the format: font_name, font_size, font_flags [,]

font_size - a font point size.

font_flags - is a combination of the following flags defined in "slick.sh":

F_BOLD
F_ITALIC
F_STRIKE_THRU
F_UNDERLINE

Returns:

Returns '' if the user cancels the dialog box. Otherwise, a string in the same format as the font argument parameter is returned.

Example:

   font='Courier,10,0';
    // You can use the _font_param function to build the above string like this
    //     font=_font_param("Courier", 10, 0);
   result = show('-modal _font_form',
                 'F',      // Display fixed fonts only
                  font
                );
   if (result == '') {
      return(COMMAND_CANCELLED_RC);
   }
   parse result with font_name','font_size','font_flags','

Applies To:

All Window Objects

Category:

Forms
