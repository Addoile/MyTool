void _lxy2dxy(int scale_mode, int &x, int &y)

Converts the x and y variables which are in the scale, scale_mode, to pixels. scale_mode may be SM_PIXEL or SM_TWIP.

Parameters:

scale_mode - may be SM_PIXEL or SM_TWIP.

x - x coordinate

y - y coordinate

Category:

Miscellaneous Functions
