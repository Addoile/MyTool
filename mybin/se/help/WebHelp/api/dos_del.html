_command typeless dos_del(_str cmdline='')

Passes text to the operating system del (UNIX: rm) command. See operating system manual for syntax of arguments. Use the delete_file built-in for reliable and portable error codes.

In ISPF emulation, this command is not called when invoked from the command line. Instead ispf_delete is called. Use dos_del to explicitly invoke the del command.
