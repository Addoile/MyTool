void _insertel(typeless value, typeless index, int Nofitems=1)

Inserts an element into an array or hash table. This method is not necessary for hash tables, but very useful for inserting into an array and shifting the the array items correctly. It also allows you to specify a number of items, so you can use this method to effeciently initialize an array. Nofitems defaults to 1.

Parameters:

value - value to insert into the array

index - index / key corresponding to item to insert

Nofitems - number of items to insert (arrays only)

Example:

t[0]=1;
t[1]=2;
t[2]=3;
t._insertel(4,1,2);  // t = {1,4,4,2,3}

Category:

Miscellaneous Functions
