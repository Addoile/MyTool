void VSAPI vsKeyin(int wid, const char *pBuf, int BufLen)

Inserts or overwrites string specified into the command line or text area depending on the insert state and the cursor position. If BufLen is -1, then pBuf must point to a null terminated string. This function is slow because it performs word wrap. Use vsInsertLine or vsInsertText function for better performance.

Parameters:

wid - Window id of object. 0 specifies the current object.

pBuf - Text to insert. May contain line separator characters.

BufLen - Number of characters in pBuf. -1 specifies that pBuf is null terminate and length can be determined from it.

Applies To:

Text Box, Combo Box, Edit Window, Editor Control

Categories:

Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods

See Also:

vsQInsertState, vsSetInsertState
