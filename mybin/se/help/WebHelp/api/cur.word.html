_str cur_word(int &start_col, _str from_cursor="", boolean end_prev_word=false, boolean multi_line=false)

Returns the current word. If from_cursor=="1", word is extracted starting at the cursor position instead of the beginning of the word. "" is returned if there is no word at the cursor. start_col is set to the physical position within the current line of the word returned.

Parameters:

start_col -

from_cursor -

end_prev_word -

multi_line -

Returns: _str

Returns the current word.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
