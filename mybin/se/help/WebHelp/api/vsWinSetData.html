void VSAPI vsWinSetData(int wid, void *pdata)

This function should only be used by OEM's to store some data per editor control and not by users writing macros.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

pdata - Pointer to window specific data.

Returns: void

Sets pointer to OEM window data.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
