_str p_message

Picture Box, Image - Determines the message that gets displayed when the mouse is hovered within the window.

Menu, Menu Item - Determines the message displayed when this menu item is selected. Currently this property is only supported by the MDI menu bar. In the future, we will add an event (maybe called on_menu_select) so that forms can display menu messages any where they want. When menus are displayed, there is no instance handle. Menu and menu item properties can only be accessed in menu resources. The _menu_get_state and _menu_set_state function may be used to get and set menu properties of displayed menus.

Applies To:

Picture Box, Image, Menu, Menu Item

Categories:

Picture Box Properties, Image Properties, Menu Properties, Menu Item Properties
