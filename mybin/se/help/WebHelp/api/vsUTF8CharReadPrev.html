char*VSAPI vsUTF8CharReadPrev(const char *pLine, const char *pCurrent, unsigned &uch)

Reads the previous UTF-8 character. This function does not read composite characters.

Parameters:

pLine - Buffer contain UTF-8 text.

pCurrent - Pointer to current character in pLine.

uch - Recieves UTF-32 character.

Returns: char*VSAPI

Returns pointer to the previous character.

Category:

Unicode Functions
