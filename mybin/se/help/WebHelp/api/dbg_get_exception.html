int dbg_get_exception(int exception_id, int &stop_when, int &count, _str &condition, _str &class_name, _str &thread_name, boolean &enabled)

Get the details about an exception.

Parameters:

exception_id - index of variable in list of exceptions

stop_when - (reference) THROWN, CAUGHT, UNCAUGHT

count - (reference) number of times to skip breakpoint -1 indicates that it is a one-time breakpoint

condition - (reference) expression for conditional exception

class_name - (reference) name of exception to catch

thread_name - (reference) name of thread breakpoint is restricted to

enabled - (reference) is the exception enabled or disabled

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
