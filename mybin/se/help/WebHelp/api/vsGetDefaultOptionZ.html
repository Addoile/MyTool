char*VSAPI vsGetDefaultOptionZ(int option, char *pszResult, int MaxResult, int *pMaxResultLen=a)

Parameters:

option - One of the following options:

VSOPTIONZ_PAST_EOF
String displayed for lines in an editor control past the end of the file. Currently this string can't be longer than 1 character.
VSOPTIONZ_SPECIAL_CHAR_XLAT_TAB
Editor control display translation table.

MaxResult - should be set to VSSPECIALCHAR_MAX. Each character corresponds to one of the VSSPECIALCHAR_??? constants.

pszResult - Ouput buffer for option. This can be 0.

MaxResult - Number of characters allocated to pszResult.

pMaxResult - If this is not 0, this is set to the number of characters you need to allocate to pszResult.

Returns: char*VSAPI

Returns pszResult.

Edit Window Functions, Editor Control Functions
