int VSAPI vsHvarClassFieldIndex(VSPSZ class_name, VSPSZ field_name)

Parameters:

class_name - Slick-C® class name

field_name - name of struct or class data field (non-static)

Returns: int

Return the index >= 0 for the given non-static member of a class or struct. Returns CLASS_NAME_NOT_FOUND_RC if the class is not found. Returns CLASS_MEMBER_NOT_FOUND_RC if the field is not found.

Category:

Miscellaneous Functions
