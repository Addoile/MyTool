int VSAPI vsColWidthSet(int wid, int i, int width)

Sets column width of column i. Column text is separated with a tab character.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

i - Number between 0 and 199.

width - New column width in the scale mode (p_xyscale_mode) of the parent.

Returns: int

Returns 0 if successful.

Applies To:

List Box, Tree View

Categories:

List Box Methods, Tree View Methods
