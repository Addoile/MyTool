int VSAPI vsMenuSetState(int menu_handle, const char *pszFindCommand, int flags, char by_category, const char *pszItemText=a, const char *pszCommand=a, const char *pszCategory=a, const char *pszHelpCommand=a, const char *pszHelpString=a)

Sets menu item values for the menu items which contain the command or category specified.

Parameters:

menu_handle - Handle of loaded menu returned by vsMenuLoad.

pszFindCommand - Command or category of menu item to find. If a command is specified, the command must match exactly. See by_category parameter.

by_category - May be 'M', 'C', 'P'. If the 'M' option is given, psaFindCommand is the command of the menu item to be found. If the 'C' option is given, pszFindCommand is the category of the menu item to be found. If the 'P' option is given, pszFindCommand is an integer (you must cast pszFindCommand to int) position (0..vsMenuQInfo(menu_handle )-1) of the menu item with in menu_handle. Defaults to 'C'.

mf_flags - may be zero or more of the following flags defined in "vs.h":

  • VSMF_CHECKED
  • VSMF_UNCHECKED
  • VSMF_GRAYED
  • VSMF_ENABLED
  • VSMF_SUBMENU

pszItemText - Menu item title.

pszCategory - Menu item categories. May be one or more categories separated with a '|' character. See vsMenuSetState function for more information about categories.

pszHelpCommand - Menu item help command. Executed when F1 is pressed while on the menu item.

pszHelpMessage - Menu item help message. Displayed when menu item is selected.

Returns: int

Returns 0 if successful.

Example:

void VSAPI dllproc ()
{
    // Gray out all menu items which are not allowed when there
    // no child windows.
    vsMenuSetState(
           vsPropGetI(VSWID_MDI,VSP_MENUHANDLE),
           "new",VSMF_GRAYED,'M');
}
