int mouse_drag(int(*move_callback)(int wid), _str move_picture='')

This is a generic function for dragging the mouse within an editor control. It can be used (for example) to move icons in the gutter.

The starting point for this function is typically in response to a lbutton_down event(). The mouse drag ends when we get a lbutton_up() event or the user hits escape.

Parameters:

move_callback - callback for move animation events

move_picture - picture to show in gutter as mouse moves

Returns: int

0 on success, COMMAND_CANCELLED_RC on cancel.

Applies To:

Edit Window, Editor Control

Category:

Mouse Functions
