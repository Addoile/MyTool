int dbg_find_session(_str session_name)

Return the session ID of the first session with the specified name

Parameters:

session_name - debugger session name

Returns: int

session ID (1 .. max_session) pointer, <0 on failure

Since:

10.0

Example:

To find a debugging session named "MySession" 

   int session_id = dbg_find_session("MySession");
   if (session_id < 0) {
      printf("session not found\n");
   }

Category:

Debugger Functions
