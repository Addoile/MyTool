int _ly2ly(int in_scale, int out_scale, int y)

Returns: int

Returns the y coordinate which is in the scale, in_scale_mode, converted to scale, out_scale_mode. The scale mode parameters may be SM_PIXEL or SM_TWIP.

Category:

Miscellaneous Functions
