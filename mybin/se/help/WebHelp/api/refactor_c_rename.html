int refactor_c_rename(int handle, _str filename, _str symbolName, int beginSeekPos, int endSeekPos, _str newName, int flags)

Rename a symbol

Parameters:

handle - Handle to transaction

filename - Filename to find symbol in

symbolName - Name of symbol to rename

beginSeekPos - Seek position of symbol or position to start search

endSeekPos - Seek position to end symbol search (0 if begin is actual symbol)

newName - New name for symbol

flags - Optional flags, bitset of VSREFACTOR_RENAME_* (above)

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
