int p_parent

Returns the parent window id. When a form has a parent, the form is not clipped within the parent window except in the case of an edit window form which is clipped with the MDI window. Controls are always clipped to the parent window. Often the parent window id is used to determine what object a dialog box should act on. Due to our implementation of the MDI window, p_parent will be 0 if you show a dialog box with the -mdi option.

Applies To:

All Window Objects
