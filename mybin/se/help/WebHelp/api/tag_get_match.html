int tag_get_match(int match_id, _str &tag_file, _str &tag_name, _str &type_name, _str &file_name, int &line_no, _str &class_name, int &tag_flags, _str &arguments, _str &return_type)

Retrieve information about the given tag match. Though this function is highly optimized, it is often more efficient to use tag_get_detail2 to retrieve only specific information about a match rather than everything.

For synchronization, macros should perform a tag_lock_matches(false) prior to invoking this function.

Parameters:

match_id - Match ID [ from 1 to tag_get_num_of_matches() ] of tag match to retrieve information about.

tag_file - (Output) tag file that match came from

tag_name - (Output) Tag string (native case)

type_name - (Output) Tag type name, see tag_get_type.

file_name - (Output) Full path of file the tag is located in

line_no - (Output) Start line number of tag within file

class_name - (Output) Name of class or package or other container that the tag belongs to. See VS_TAGSEPARATOR_class and VS_TAGSEPARATOR_package for more details about constructing this value. If the tag is not in a class scope, simple pass the empty string for this value.

tag_flags - (Output) Tag flags indicating symbol attributes, see tag_insert_simple for details.

arguments - (Output) arguments or formal parameters

return_type - (Output) constant value or return type

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
