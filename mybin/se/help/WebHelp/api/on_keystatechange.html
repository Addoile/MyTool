event on_keystatechange()

The on_keystatechange event is returned by the get_event function to indicate that a shift key has been pressed or released. You must call the mou_mode function with a one for this event to be returned by the get_event function.

Example:

defmain()
{
     mou_mode(1);
OuterLoop:
     for (;;) {
         event=get_event();
         switch( event ) {
         case ESC:
              break OuterLoop;
          case ON_KEYSTATECHANGE:
              if (_IsKeyDown(CTRL) ){
                   message("CTRL is down");
               } else {
                    message("CTRL is not down");
               }
           }
      }
      mou_mode(0);
           }
