char*VSAPI vsEvent2Name(int event, char *pszName, int MaxNameLen, char option='L', int *pMaxNameLen=a, int version=a)

Converts a SlickEdit event constant into its corresponding name.

Parameters:

event - SlickEdit event constant.

pszName - Output buffer for name of event. This can be 0.

MaxNameLen - Number of characters allocated to pszName. We recommend VSMAXEVENTNAME.

option - One of the following option letters:

'S'
Return Slick-C® source code event name. These will never change so that our Slick-C® source code will be backward compatible.
'L'
Return long event names intended for display on a menu. like "Alt+F4" instead for "A+F4".
'C'
Return short event names intended for display on a menu.

pMaxNameLen - If this is not 0, this is set to the number of characters you need to allocate.

Returns: char*VSAPI

Returns pszName.

Category:

Keyboard Functions
