int tag_tree_type_is_func(_str type_name)

Speed-demon way to determine if the given type is a function or procedure type. (proc, proto, func, constr, or destr).

Parameters:

type_name - string specifying tag's type name (from tag_get_info)

Returns: int

1 if the item is a func, 0, otherwise.

Category:

Tagging Functions
