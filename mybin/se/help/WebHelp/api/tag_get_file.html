int tag_get_file(int file_id, _str &file_name)

Retreive the name of the file with the given file ID.

Parameters:

file_id - ID of file, from tag_get_detail()

file_name - (Output) Full path of file containing tags

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Category:

Tagging Functions

See Also:

tag_get_detail, tag_find_file, tag_next_file, tag_set_date
