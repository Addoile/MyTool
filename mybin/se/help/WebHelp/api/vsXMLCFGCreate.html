int VSAPI vsXMLCFGCreate(const char *pszFilename, int iEncoding, int iCreateOption=VSXMLCFG_CREATE_IF_EXISTS_CREATE)

Creates or opens an existing XMLCFG tree.

Parameters:

pszFilename - Name of XML file to create. No disk I/O occurs.

iEncoding - For standard XML files, this parameter should be VSENCODING_UTF8. For Visual Studio 2002/2003 project files, this parameter should be VSCP_ACTIVE_CODEPAGE. For Visual Studio 2005, use VSENCODING_UTF8. Do not use VSENCODING_AUTOXML for creating new documents. But VSENCODING_AUTOXML is valid when using VSXMLCFG_CREATE_IF_EXISTS_OPEN.

encoding may be one of the following: 

VSCP_ACTIVE_CODEPAGE
VSCP_EBCDIC
VSCP_CYRILLIC_KOI8_R
VSCP_ISO_8859_1
VSCP_ISO_8859_2
VSCP_ISO_8859_3
VSCP_ISO_8859_4
VSCP_ISO_8859_5
VSCP_ISO_8859_6
VSCP_ISO_8859_7
VSCP_ISO_8859_8
VSCP_ISO_8859_9
VSCP_ISO_8859_10
Any valid Windows code page
VSENCODING_AUTOUNICODE
VSENCODING_AUTOXML
VSENCODING_UTF8
VSENCODING_UTF8_WITH_SIGNATURE
VSENCODING_UTF16LE
VSENCODING_UTF16LE_WITH_SIGNATURE
VSENCODING_UTF16BE
VSENCODING_UTF16BE_WITH_SIGNATURE
VSENCODING_UTF32LE

VSENCODING_UTF32LE_WITH_SIGNATURE
VSENCODING_UTF32BE
VSENCODING_UTF32BE_WITH_SIGNATURE

iCreateOption - One of the following options.

VSXMLCFG_CREATE_IF_EXISTS_CLEAR
If this XMLCFG tree already exists, clear the contents of the tree.
VSXMLCFG_CREATE_IF_EXISTS_OPEN
If this XMLCFG tree already exists, just return the existing handle.
VSXMLCFG_CREATE_IF_EXISTS_ERROR
If this XMLCFG tree already exists, return an error.
VSXMLCFG_CREATE_IF_EXISTS_CREATE
If this XMLCFG tree already exists, create a new tree with the same name.

Returns: int

Returns XMLCFG tree handle if successful. Otherwise a negative error code is returned.

Category:

XMLCFG Functions
