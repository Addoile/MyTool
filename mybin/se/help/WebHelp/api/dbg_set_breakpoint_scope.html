int dbg_set_breakpoint_scope(int breakpoint_id, _str class_name, _str method_name)

Modify the class name / method scope for a breakpoint.

Parameters:

breakpoint_id - Breakpoint to enable

class_name - class that breakpoint is set in

method_name - name of function breakpoint is set in

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
