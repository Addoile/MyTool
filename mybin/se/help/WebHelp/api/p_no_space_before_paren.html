boolean p_no_space_before_paren

Determines whether to insert a space before the opening parenthesis during syntax expansion for the current buffer. This property is available only when the control is displayed.

Example:

   // p_no_space_before_paren = false
      if (x == 5)
   // p_no_space_before_paren = true
      if(x == 5)

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
