int tag_current_local_name()

Return the index of the current local item whose name and name location matches the symbol under the cursor.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Returns: int

<0 on error, 0 if no current local.

Category:

Tagging Functions

See Also:

tag_current_local, tag_set_local_name_location
