int name_match(_str name_prefix, int find_first, int name_type_flags=~0 &~IGNORECASE_TYPE)

Parameters:

name_prefix - Symbol name prefix to search for

find_first - Find first match (true) or next match?

name_type_flags - The name type flags are listed in the file "slick.sh".

PROC_TYPE
Matches global function
VAR_TYPE
Matches global variable
EVENTTAB_TYPE
Matches event table
COMMAND_TYPE
Matches command
CLASS_TYPE
Matches class name
INTERFACE_TYPE
Matches interface name
STRUCT_TYPE
Matches struct name
CONST_TYPE
Matches const name
ENUM_TYPE
Matches enumerated type
MODULE_TYPE
Matches module
PICTURE_TYPE
Matches picture.
BUFFER_TYPE
Matches buffer scope variable
OBJECT_TYPE
Matches any type of dialog box template. Use the oi2type to build the correct object flags. See example below.
MISC_TYPE
Matches miscellaneous.
IGNORECASE_TYPE
Perform case insensitive search.

Returns: int

Returns name table index of the name with prefix matching name_prefix and where (name_t & name_type(index)) is true. A non-zero value for find_first, begins a new search. If find_first is zero, the next matching index is returned. 0 is returned if no match is found. Search is case sensitive unless one of the flags MODULE_TYPE, PICTURE_TYPE, or IGNORECASE_TYPE is given. When one of the flags MODULE_TYPE or PICTURE_TYPE is given, search is case sensitive for file systems like UNIX which is case sensitive. The search is always case insensitive when the IGNORECASE_TYPE flag is given. Underscores in name_prefix are translated to dashes before search takes place.

Example:

// COMMAND_TYPE is defined in "slick.sh"
name_t= COMMAND_TYPE;
name_prefix='p';           // Find names that start with p
index= name_match(name_prefix,1,name_t);   // Find first
// Press Ctrl+Break to break a macro during a messageNwait
for (;;) {
    if (!index ) break;
    messageNwait(''name found='name_name(index));
    index= name_match(name_prefix,0,name_t) // Find next
}
