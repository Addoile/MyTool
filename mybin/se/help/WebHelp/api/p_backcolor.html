int p_backcolor

Determines the background color of a control.

Form, Label, Spin, Text Box, Editor Control, Frame, Radio Button, Check Box, List Box, File List Box, Directory List Box, Gauge, Tree View - Determines the background color of the window.

Combo Box - Determines the background color of the text box and list box.

Drive List - Determines the background color of the text box and list box.

Picture Box, Image - Determines the background color of the window. No background of the window is seen if the p_auto_size property is non-zero and the p_picture property has a valid picture.

Applies To:

All Window Objects

Category:

All Windows Properties
