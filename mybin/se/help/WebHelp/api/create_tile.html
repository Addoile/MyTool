_command typeless create_tile()

Allows you to create a new tiled window by pointing to edge for new window with cursor keys.

Returns: typeless

Returns 0 if successful. Common return codes are 1 (window too small to split), COMMAND_CANCELLED_RC, TOO_MANY_WINDOWS_RC, and TOO_MANY_SELECTIONS_RC. On error, message is displayed.

Applies To:

Edit Window

Category:

Window Functions

See Also:

move_edge, next_window, _prev_window, change_window, delete_tile, window_left, window_right, window_above, window_below
