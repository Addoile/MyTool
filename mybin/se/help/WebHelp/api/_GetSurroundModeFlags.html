int _GetSurroundModeFlags(_str lang='')

Get the language-specific surround options.

The options are stored per language. If the options are not yet defined for a language, then use def_surround_mode_options as the default.

Parameters:

lang - language ID (p_LangId)

Returns: int

bitset of VS_SURROUND_MODE_* options.

Applies To:

Edit Window, Editor Control

Category:

Tagging Functions
