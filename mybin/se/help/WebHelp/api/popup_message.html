_command void popup_message(_str string="", typeless flags="", _str title="")

Displays message_text in message box with an exclamation point displayed to the left of the message. Use the new and more powerful _message_box function instead of this function. This function is typically used with the _post_call function to display an error message box during on_got_focus or on_lost_focus events.

Parameters:

string - message to display

flags - message box flags (MB_*)

title - message box title

Category:

Miscellaneous Functions
