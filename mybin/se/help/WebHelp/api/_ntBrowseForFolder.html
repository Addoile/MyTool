_str _ntBrowseForFolder(_str dir_from_dialog, _str title=null, int flags=0)

Show the standard Windows style directory browser dialog.

Parameters:

dir_from_dialog - directory to initialize dialog to

title - dialog caption (default "Select a directory")

flags - bitset

  • 0x1 -- show make new directory button
  • 0x2 -- validate that directory exists

Returns: _str

Returns the selected directory, or '' if cancelled.

Since:

13.0

Category:

File Functions
