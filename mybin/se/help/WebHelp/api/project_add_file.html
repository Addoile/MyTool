_command int project_add_file(_str newfilename="", boolean quiet=false, _str ProjectName=_project_name, _str &msg=null)

Adds a file or buffer to the current project.

Parameters:

newfilename - Name of file or buffer to add to the project. If "" is given, the current buffer is added to the project.

newfilename -

quiet - When false, messages are displayed.

ProjectName -

msg - Set to status message on error. Useful when quiet=true.

Returns: int

Returns 0 if successful.
