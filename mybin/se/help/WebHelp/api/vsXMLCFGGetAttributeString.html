VSDLLEXPORT slickedit::SEString vsXMLCFGGetAttributeString(int iHandle, int NodeIndex, VSPSZ pszAttrName, const char *pszDefaultValue=0)

Retrieves attribute value for the attribute name given. This function is thread-safe.

Parameters:

iHandle - Handle to an XMLCFG tree returned by vsXMLCFGOpen(), vsXMLCFGCreate(), or vsXMLCFGOpenFromBuffer().

NodeIndex - Tree node index.

pszAttrName - Name of attribute to return value of

pszDefaultValue - default value to return if attribute not present

Returns: VSDLLEXPORT slickedit::SEString

Returns attribute value if successful. Otherwise pszDevaultValue is returned.

Category:

XMLCFG Functions
