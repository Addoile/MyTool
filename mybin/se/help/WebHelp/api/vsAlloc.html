void*VSAPI vsAlloc(size_t len)

Allocates memory. The memory is not initialized with zeros. Use the vsFree function to free memory allocated by this function.

Parameters:

len - Number of bytes to allocated.

Returns: void*VSAPI

If successful, returns a pointer to len allocated bytes of memory. Otherwise 0 is returned.

Category:

Miscellaneous Functions

See Also:

vsFree, vsRealloc
