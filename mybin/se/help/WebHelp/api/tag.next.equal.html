int tag_next_equal(boolean case_sensitive=false, _str class_name=null, boolean skip_duplicates=false)

Retrieve the next tag with the same tag name as the last one retrieved. Should be called only after calling tag_find_equal or tag_find_tag.

Parameters:

case_sensitive - Case sensitive tag name comparison

class_name - Class name to search for tag in

skip_duplicates - Skip tags with same name, type and class.

Returns: int

0 on success, <0 on error, or if no such tag.

Category:

Tagging Functions
