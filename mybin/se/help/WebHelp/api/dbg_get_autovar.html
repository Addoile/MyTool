int dbg_get_autovar(int thread_id, int frame_id, int watch_id, _str &expr, _str &value, _str &raw_value)

Get the details about a auto-watch variable.

Parameters:

thread_id - index of thread

frame_id - index of stack frame

watch_id - index of variable in list of auto-watches

expr - (reference) expression to evaluate

value - (reference) value of variable

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
