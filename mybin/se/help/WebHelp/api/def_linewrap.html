boolean def_linewrap

The line wrap on text option affects whether line wrapping occurs when the left margin is reached or column one is reached for the Left and Backspace key configurations. By default, when the word wrap option is enabled, wrapping occurs when the left margin is reached regardless of the Left or Backspace key configurations. Mark this check box if you want wrapping to occur when column one is reached.

Default:

1

Configuration Variables
