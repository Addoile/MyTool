_command void ispf_cut(_str arglist="")

Cut lines out of the current buffer and place them in the specified clipboard.

Sytnax:

   CUT [ line_range ] [DEFAULT | clipboardname ] [REPLACE] [DISPLAY]

line_range
Two numbers or labels, specifying the starting and ending lines to cut to the clipboard.
clipboardname
Place the result on the named clipboard.
DEFAULT
Place the result on the default clipboard.
REPLACE
Replace the contents of the given clipboard or append to the end of the clipboard.
DISPLAY
Display list of clipboards.

Category:

ISPF Primary Commands

See Also:

ispf_paste, list_clipboards, cut
