int tag_set_language(_str file_name, _str lang)

API function for setting the language type for the given filename. This corresponds to the p_LangId property of 'file_name', not necessarily the literal file extension.

Parameters:

file_name - File path of source file to set language property for. See VS_TAGSEPARATOR_file for more details about constructing this value.

lang - p_LangId property for file_name.

Returns: int

0 on success, <0 on error.

Since:

13.0

Category:

Tagging Functions
