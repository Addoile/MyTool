int VSAPI vsRefactorConfigGetName(int configIndex, char *configName, int maxLength)

Retrieve the name of the given configuration.

Parameters:

configIndex - index of configuration to query name of [0..n-1]

configName - [reference] set to name of configuration

maxLength - length of the configName buffer

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions

See Also:

vsRefactorConfigAdd, vsRefactorConfigCount, vsRefactorConfigSetName
