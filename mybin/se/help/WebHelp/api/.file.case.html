_str _file_case(_str a)

Parameters:

filename - File path to convert.

Returns: _str

Returns filename in operating system specific case. For DOS and Windows NT, file names are case-insensitive, so they are converted to lower case. This is useful for storing filenames in a case-agnostic manner.

_file_case() should be used under rare cirumstances. It is preferrable to use file_eq() to compare files without modifying their case.

Category:

File Functions
