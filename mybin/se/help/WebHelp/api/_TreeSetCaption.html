void _TreeSetCaption(int ItemIndex, _str caption, int ColIndex=-1)

Sets the caption for the specified tree item.

Parameters:

ItemIndex - Specifies a tree item.

caption - Caption to set tree item to

ColIndex - Column index to set. First column is 0. The default is to set all columns, separated by tabs.

Applies To:

Tree View

Category:

Tree View Methods

See Also:

_TreeGetCaption, _TreeSetInfo, _TreeGetInfo, _TreeSetUserInfo, _TreeGetUserInfo
