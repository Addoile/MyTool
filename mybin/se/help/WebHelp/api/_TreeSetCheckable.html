int _TreeSetCheckable(int index, int checkable, int isTriState, ...)

Set whether or not a tree node is checkable

Parameters:

index - Handle to tree node to set checkable

checkable - 1 for checkable, 0 for non-checkable

isTriState - 1 for tri-state, 0 for 2-state. NOTE: for tri-state checkbox in the treeview, the third (TCB_PARTIALLYCHECKED )state can only be set programatically.

checkedState - Initial state of check box, one the following members of the TreeCheckVal enum: * TCB_UNCHECKED * TCB_CHECKED * TCB_PARTIALLYCHECKED

col - Column of tree node to set checkable (defaults to 0)

Returns: int

int 0 if successful

Applies To:

Tree View

Category:

Tree View Methods
