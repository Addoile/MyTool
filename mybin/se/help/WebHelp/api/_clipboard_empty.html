int _clipboard_empty(boolean isClipboard=true)

Returns non-zero value if the operating system clipboard is empty. After an auto restore of your previous clipboards, the operating system clipboard may be empty, but there will be some internal clipboards. Use the clipboard_itype function to determine if there are any internal clipboards.

Parameters:

isClipboard - Indicates whether operation is performed for the clipboard or selection. Effects Unix only.

Returns: int

Returns non-zero value if the operating system clipboard is empty

Category:

Clipboard Functions
