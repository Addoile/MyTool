int VSAPI vsSortSelection(int wid, int markid=-, int SortFlags=a)

Sorts the selection specified in ascending or descending order in case sensitivity specified. Sorting defaults to ascending and case sensitive. If the buffer containing the specified selection is the active buffer of wid, the resulting lines are inserted after the end of the selection. Otherwise the resulting lines are inserted after the cursor. If a character selection is used, it is converted to a line type selection.

Parameters:

wid - Ouput window id. Window id of editor control. 0 specifies the current object.

markid - mark_id is a selection handle allocated by the vsAllocSelection or vsDuplicateSelection function.

SortFlags - Zero or more of the following flags Ored together:

VSSORT_DESCENDING
Sort in descending order.
VSSORT_IGNORECASE
Case insensitive sort according to locale.
VSSORT_IGNORECASE_NONLOCALE
Case insensitive sort according to U.S. locale.
VSSORT_NUMBER
Data being sorted contains floating point numbers.
VSSORT_FILENAME
Data being sorted containts file names.

Returns: int

Returns 0 if successful. Common return codes are LINE_OR_BLOCK_SELECTION_REQUIRED_RC, NOT_ENOUGH_MEMORY_RC, TOO_MANY_SELECTIONS_RC, and INVALID_SELECTION_HANDLE_RC.

Applies To:

Edit Window, Editor Control
