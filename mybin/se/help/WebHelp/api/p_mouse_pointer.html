int p_mouse_pointer

Determines the mouse pointer displayed when the mouse is within the window. This property may be one of the following constants defined in "slick.sh":

MP_DEFAULT
Shape of mouse pointer depends on control.
MP_ARROW
MP_CROSS
MP_IBEAM
MP_SIZE
MP_SIZENESW
MP_SIZENS
MP_SIZENWSE
MP_SIZEWE
MP_UP_ARROW
MP_HOUR_GLASS
Try the above constants to see what they look like.

Example:

   #include "slick.sh"
   defmain()
   {
      p_mouse_pointer=MP_HOUR_GLASS;
   }

Applies To:

All Window Objects

Category:

All Windows Properties
