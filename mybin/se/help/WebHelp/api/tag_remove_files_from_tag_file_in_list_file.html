int tag_remove_files_from_tag_file_in_list_file(_str pszTagDatabase, int rebuildFlags, _str pszFilename)

Remove all the files listed in the given list file from the tag file.

Parameters:

pszTagDatabase - name of tag file to update

rebuildFlags - bitset of VS_TAG_REBUILD_

  • VS_TAG_REBUILD_SYNCHRONOUS -- Remove the files in the foreground instead of finishing the work using a background thread.

pszFilename - name of file on disk containing list of source files to remove.

Returns: int

0 on success, <0 on error.

Since:

16.0

Category:

Tagging Functions
