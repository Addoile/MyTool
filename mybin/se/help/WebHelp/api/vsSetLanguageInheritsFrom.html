void VSAPI vsSetLanguageInheritsFrom(const char *pszLangId, const char *pszParentLangId)

Allows you to create a language which has it's own set of options, but can inherit support callbacks from another language.

Parameters:

pszLangId - Language ID (see p_LangId For list of language types, Language Options dialog ("Tools", "Options", "Language Setup...").

pszParentLangId - pszLangId is set to inherit language specific callbacks from the language specified by pszParentLangId. If NULL, remove language inheritance for pszLangId.

Since:

13.0

Applies To:

Edit Window, Editor Control

Category:

Tagging Functions

See Also:

vsSetDefaultLanguageOptions, vsSetDefaultExtensionReferTo, vsDeleteLanguageOptions, vsUpdateExistingBufferLanguageOptions, vsLanguageInheritsFrom, vsFindLanguageCallbackIndex
