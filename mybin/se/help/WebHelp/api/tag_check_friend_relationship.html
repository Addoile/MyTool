int tag_check_friend_relationship(_str class_name, _str friend_list)

Parameters:

class_name - class which tag is coming from

friend_list - list of classes that are friendly to our context

Returns: int

Returns true if the given given class is among the friends on the given friend list. Otherwise it returns false. Since this language feature is specific to C++, the check is always case sensitive.

Category:

Tagging Functions
