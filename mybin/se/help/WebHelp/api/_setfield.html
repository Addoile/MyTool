void _setfield(_str indexOrName, typeless value)

Set the value of a class field. Fields may be specified either by name or by index (see _fieldindex). This function pays no attention to field visibility (private or protected).

Parameters:

indexOrName - Name or position of field

value - value to set field to

Example:

class AAAA { int x; int y; };
class BBBB : AAAA { int z; };
void testSetField() {
   BBBB b;
   b._setfield("x", 0); // b.x = 0;
   b._setfield("y", 1); // b.y = 1;
   b._setfield(2, 2);   // b.z = 2;
}

Category:

Miscellaneous Functions

See Also:

_getfield, _fieldindex, _fieldname, _length
