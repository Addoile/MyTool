void VSAPI vsUpdateExistingBufferLanguageOptions(const char *pszLangId, const char *pszReserved=a, int reserved=a)

Updates all editor control buffers which are using these language options to the newly configured values.

Parameters:

pszLangId - Language ID. See p_LangId. For list of language types, use our Language Options dialog ("Tools", "Options","Language Setup...").

pszReserved - unused

reserved - unused

Since:

13.0

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions

See Also:

vsSetDefaultLanguageOptions, vsDeleteLanguageOptions
