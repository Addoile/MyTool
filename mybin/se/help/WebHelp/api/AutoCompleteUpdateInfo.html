Update the completion suggestions. This function is called from the auto-save timer, and synchronously from AutoCompleteDoKey() and the autocomplete command.

Parameters:

alwaysUpdate - Update now, do not wait for idle timer

forceUpdate - Force update, independent of auto complete options

Returns: int

0 on success, <0 on error.

Edit Window, Editor Control, Text Box, Combo Box
