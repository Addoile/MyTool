int tag_find_include_file(_str file_name, _str &include_name)

Retrieve the name of the first file included by file_name

Parameters:

file_name - Full path of "main" source file

include_name - (Output) Full path of included file

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Category:

Tagging Functions
