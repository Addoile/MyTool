int VSAPI vsSearchReplace(int wid, const char *pReplaceString, int ReplaceStringLen=-, int RepeatSearch=a)

Replaces the match found by the last vsSearch or vsRepeatSearch with the replace string specified and optionally searches for the next occurrence.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

pReplaceString - Last search string found is replaced with this string.

ReplaceStringLen - Number of characters in pReplaceString. -1 means pReplaceString is null terminated.

RepeatSearch - If this is non-zero, a search for the next occurrence of the last search string is performed with the same options and direction specified by the last vsSearch or vsRepeatSearch call

Returns: int

Returns 0 if successful. Common return codes are BREAK_KEY_PRESSED_RC and STRING_NOT_FOUND_RC.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Search Functions

vsMatchLength, vsMatchStart, vsMatchCursorLength, vsSelectMatch, vsSearch, vsRepeatSearch, vsSaveSearch, vsRestoreSearch
