int dbg_session_expand_member(int session_id, int thread_id, int frame_id, _str member_path)

Expand the member variable at the given member variable path for the specified stack frame in the specified thread.

Parameters:

session_id - debugger session ID (1 .. max_session)

thread_id - index of thread in table

frame_id - index of frame in thread's stack

member_path - path to member variable to expand

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

To expand the structured member at ID 5 under the member with ID 8 in the current thread and the top-most stack frame: 

  int thread_id = dbg_get_cur_thread();
  int status = dbg_session_expand_member(session_id, thread_id, 1, "8 5");
  if (status) {
     return status;
  }
