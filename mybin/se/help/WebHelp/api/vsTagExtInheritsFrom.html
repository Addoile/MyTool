int VSAPI vsTagExtInheritsFrom(const char *parent, const char *lang=a)

Deprecated. use vsLanguageInheritsFrom()

Does the current source language match or inherit from the given language?

If 'lang' is not specified, the current object must be an editor control.

Parameters:

parent - extension to compare to

lang - current source language (default=p_LangId)

Returns: int

'true' if the extension matches, 'false' otherwise.

Category:

Tagging Functions
