void _TreeSetInfo(int ItemIndex, int ShowChildren, int NonCurrentBMIndex=-1, int CurrentBMIndex=-1, int moreFlags=0, int setCurrentNodeIfHidden=1, int flagMask=-1)

Modifies some tree item properties.

Parameters:

ItemIndex - Specifies a tree item. The root of the tree can't be deleted and always has an item index of 0.

ShowChildren - Indicates whether to show the children of this node. TREE_NODE_COLLAPSED specifies not to show the children. Specify TREE_NODE_EXPANDED to initially show children of this node. Set this to TREE_NODE_LEAF for leaf items.

NonCurrentBMIndex - Names table index to a picture which gets displayed to the left of the caption text when this node is not the current node. We recommend you specify the same values for NonCurrentBMIndex and CurrentBMIndex for leaf items.

CurrentBMIndex - Names table index to a picture which gets displayed to the left of the caption text when this node is the current node.

moreFlags - Combination of TREENODE_* flags.

setCurrentNodeIfHidden - Set this to 0 for performance if you are hiding a lot of nodes and will set the current node yourself when finished.

flagMask - Mask indicating which TREENODE_* flags to set

Applies To:

Tree View

Category:

Tree View Methods

See Also:

_TreeSetCaption, _TreeGetCaption, _TreeGetInfo, _TreeSetUserInfo, _TreeGetUserInfo
