_command int set_next_error(_str beQuiet='', _str gotoEndOfLine='')

Sets the next_error commands start search position to the cursor. This command is no longer very useful since the cursor_error command also sets the start search position.

Parameters:

beQuiet - do not display message box on error

gotoEndOfLine - jump to end of line containing message

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

reset_next_error, cursor_error, next_error
