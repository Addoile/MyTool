_str env_match(_str name_prefix, int find_first)

Returns: _str

Returns the name of the environment variable matching the prefix name_prefix. If a match is not found, the global rc variable is set to STRING_NOT_FOUND_RC and '' is returned. If find_first is non-zero, matching starts from the first environment variable. Otherwise matching starts after the previous match.
