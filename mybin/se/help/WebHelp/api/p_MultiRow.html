int p_MultiRow

Deprecated. No longer supported

Determines how tabs are displayed. p_MultiRow may be one of the following:

    SSTAB_MULTIROW_NONE
    If tabs don't fit, display a scroll bar.
    SSTAB_MULTIROW_MULTIROW
    If tabs don't fit, make another row. Use the p_TabsPerRow property to set the number of tabs displayed on each row.
    SSTAB_MULTIROW_BESTFIT
    If tabs don't fit, try to reduce space taken by each tab. If tabs still don't fit, display a scroll bar.

    Applies To:

    SSTab

    Category:

    SSTab Properties
