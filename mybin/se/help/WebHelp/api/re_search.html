_command typeless re_search(_str commandName='')

Sets the default search pattern sensitivity to regular or non-regular expression for command line search commands. Does not affect the i_search or reverse_i_search commands. If no argument is given the current setting is displayed on the command line for editing. Used in BRIEF emulation.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions, Search Functions

See Also:

replace, find, i_search, reverse_i_search, search_case, re_toggle, case_toggle, search_forward, search_backward, translate_forward, translate_backward, translate_again, search_again.
