int VSAPI vsGetDefaultFileEncoding(const char *pszLangId, const char *pszExtension=a)

Looks up the default file encoding to be used when attempting to load files of the given langauge. It looks first in the language specific file encodings and then for an extension specific file encodings, and finally it will default to the default file encoding.

Parameters:

pszLangId - Language ID (see p_LangId For list of language types, Language Options dialog ("Tools", "Options", "Language Setup...").

pszExtension - File extension. For list of file extension types, use our Extension Manager dialog ("Tools", "Options...", "File Extension Manager").

Attention:

This function is thread-safe.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions
