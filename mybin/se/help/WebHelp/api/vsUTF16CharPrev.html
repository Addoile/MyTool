unsigned short*VSAPI vsUTF16CharPrev(const unsigned short *pLine, const unsigned short *pCurrent)

Parameters:

pLine - Buffer contain UTF-16 text.

pCurrent - Pointer to the beginning of a UTF-16 character in pLine.

Returns: unsigned short*VSAPI

Returns pointer to the beginning of the previous composite or surrogate character or pLine if pCurrent==pLine.

Category:

Unicode Functions
