int tag_insert_match2(_str tag_file, _str tag_name, _str type_name, _str file_name, int line_no, int seekpos, int scope_line_no, int scope_seekpos, int end_line_no, int end_seekpos, _str class_name, int tag_flags, _str signature)

Add a tag to the match set. The tag database allows you to maintain a list of tags very efficiently through the match set. This is useful when searching for specific tags. Many of the Context Tagging® functions are designed to insert directly into the match set.

For synchronization, macros should perform a tag_lock_matches(true) prior to invoking this function.

Parameters:

tag_file - (optional) full path of tag file match came from

tag_name - tag string

type_name - Tag type name. See tag_get_type for a list of all standard tag types.

file_name - full path of file the tag is located in (p_buf_name)

start_linenum - start line number of tag within file

start_seekpos - start seek position of tag within file

scope_linenum - Line number where the scope of the tag begins in file_name. For example, this is the line on which opening brace is found in a C function. For tags where scope is not applicable, simply the same value as start_linenum.

scope_seekpos - Seek position where the scope of the tag begins in file_name. See explanation above.

end_linenum - (optional) ending line number of tag within file

end_seekpos - (optional) Line number where the tag ends in file_name. Technically, this is the seek position of the last character in the tag. Using a C function for example, this is the seek position of the close brace ending the function body.

class_name - (optional) Name of class or package or other container that the tag belongs to. See VS_TAGSEPARATOR_class and VS_TAGSEPARATOR_package for more details about constructing this value. If the tag is not in a class scope, simple pass the empty string for this value.

tag_flags - (optional) Tag flags indicating symbol attributes, see tag_insert_simple for details.

signature - (optional) Tag signature, includes type, value, and arguments for the symbol being inserted. See VS_TAGSEPARATOR_args and VS_TAGSEPARATOR_equals for more details about constructing this value.

Returns: int

sequence number (match_id) of matching tag on success, or <0 on error.
