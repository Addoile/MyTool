void _insert_toggle()

Toggles insert mode on/off. Normally the cursor shape is smaller when in insert mode. When in insert mode, characters are inserted at the cursor position. When in over-write mode, the characters at the cursor position are replaced.

Example:

         // set insert mode on.
         if (!_insert_state()) _insert_toggle();
         keyin('insert these characters into the command line');

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods
