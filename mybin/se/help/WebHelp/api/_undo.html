void _undo(_str option="U")

Undo the last operation performed by the editor control.

Parameters:

option - letter defaults to 'U' and may be one of the following:

'U'
If the current buffer allows more than 0 undoable steps, the last operation is undone. Otherwise, the current line is restored to its original value before the cursor moved onto it.
'C'
Same 'U' option except when the current buffer has more than 0 undoable steps. Consecutive cursor motion is undone as one step.
'R'
If the current buffer has more than 0 undoable steps, the last undo operation is redone. Otherwise, this function has no affect.
'S'
Starts a new level of undo.

Returns: void

On successful, completion, a number greater than or equal to zero is returned which is composed of the flags below (defined in "slick.sh"):

  • LINE_DELETES_UNDONE
  • CURSOR_MOVEMENT_UNDONE
  • MARK_CHANGE_UNDONE
  • TEXT_CHANGE_UNDONE
  • LINE_INSERTS_UNDONE
  • MODIFY_FLAG_UNDONE
If not successful, NOTHING_TO_UNDO_RC or NOTHING_TO_REDO_RC is returned.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
