int _xmlcfg_get_xpath_from_node(int handle, int elementNode, _str &queryStr)

Construct an XPath query string for the element specified.

Parameters:

handle - Handle to an XMLCFG tree returned by _xmlcfg_open(), _xmlcfg_create(), or _xmlcfg_open_from_buffer().

elementNode - Node index of element to construct XPath query string for.

queryStr - (output). Query string for element node.

Returns: int

0 on success, <0 on error.

Example:

Element node:

  <file name="main.c" option="debug">

Resulting XPath expression:

  /file[@name='main.c'][@option='debug']

Category:

XMLCFG Functions
