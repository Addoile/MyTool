_str refer_ext(_str ext, _str file_name='', boolean currentObjectIsBuffer=false, boolean setAutoSelectLanguage=false)

Deprecated. Use _Filename2LangId()

Parameters:

ext - source file name extension

file_name - source file name with path

currentObjectIsBuffer - pass 'true' if if the current object is an editor control containing the file in question

setAutoSelectLanguage - if 'true' set p_AutoSelectLanguage

Returns: _str

Returns the language mode associated with the given file name extension. The Language Options dialog box allows you to map an extension that to a language mode. This function performs that translation, and will also use the [optional] buffer name to determine the language mode.
