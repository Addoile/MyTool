_command int plusminus(_str CheckCurLineOnly='')

Expands or collapses selective display blocks.

  • If the cursor is on a line that contains a "+" bitmap for selective display, expand the collapsed block.
  • If the cursor is on a line that contains a "-" bitmap for selective display, collapse the block.
  • otherwise, if the cursor is on a line that has no "+" or "-" bitmap, but the line is the first line of a block collapse the block, creating a new selective display region. The definition of the current block depends on the language.

Applies To:

Edit Window, Editor Control

Categories:
