int VSAPI vsHvarFindInterfaceMethod(VSHVAR hvar, VSPSZ class_name, VSPSZ method_name)

Check if the given variable is an instance of the given class and if it implements the method named.

Parameters:

hvar - handle for interpreter variable

class_name - name of class or interface to test

method_name - name of method to find

Returns: int

Return the name index of the given (virtual) method for the a Slick-C® class. For virtual methods, this will find the nearest virtual method, starting with the actual type of the given class. Returns CLASS_NAME_NOT_FOUND_RC if the class is not found. Returns CLASS_MEMBER_NOT_FOUND_RC if the method is not found.

Category:

Miscellaneous Functions
