_command int select_proc(typeless count=0, int markid=-1, _str notQuiet='', SELECT_PROC_FLAGS options=def_select_proc_flags)

Selects current function contents and (optionally) function heading.

Parameters:

count - reserved

markid - Can be '' for active selection

notQuiet - show message box if not supported

options - select_proc options - specify SELECT_PROC_NO_COMMENTS to not include header comments in selection

Returns: int

Returns 0 if successful.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions
