void _TreeAdjustColumnWidthsByColumnCaptions(int extraGoesTo=-1)

Adjust all the column widths so that the column names all fit.

If there is extra width in the tree, it can be added to the specified column. If no column is specified, the tree automatically adds that extra to the last column.

Parameters:

extraGoesTo -

Applies To:

Tree View

Category:

Tree View Methods
