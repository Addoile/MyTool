int _isloaded(int form_index, _str options="")

Parameters:

option - may be 'N' or 'E'. Specify the 'N' option if you only want to find a non-edited instance of an object. Specify the 'E' option if you only want to find an edited instance of an object.

Returns: int

Returns the window id of a displayed form whose p_template property matches form_index. Returns 0 if a match is not found. form_name is an index into the names table of a form. When a form is loaded with the _load_template function (show calls this function), the p_template property is set to the form_index given to the _load_template function. This allows a macro to determine the correct name of a form when you press Shift-SPACE to edit a dialog box. Using the form name would be incorrect since this can be changed after the form is loaded. The p_template property is read only and is set only by the _load_template and _update_template functions. Beware, by default, this function will find edited or non-edited instances of an object.
