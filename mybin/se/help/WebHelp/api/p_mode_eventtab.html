int p_mode_eventtab

Determines the language-specific event table for the current file. The binding for a key, mouse, and a few other events is determined by checking event tables for bindings in the following order.

  1. p_eventtab (window property)
  2. p_mode_eventtab (buffer property)
  3. Root event table (global stored in _default_keys variable)

This property is available only when the control is displayed.

Applies To:

Editor Control, Edit Window

Categories:

Editor Control Properties, Edit Window Properties
