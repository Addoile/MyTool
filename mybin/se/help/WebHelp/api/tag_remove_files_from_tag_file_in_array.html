int tag_remove_files_from_tag_file_in_array(_str pszTagDatabase, int rebuildFlags, _str(&sourceFileArray)[])

Remove all the files in the given array from the tag file.

Parameters:

pszTagDatabase - name of tag file to update

rebuildFlags - bitset of VS_TAG_REBUILD_

  • VS_TAG_REBUILD_SYNCHRONOUS -- Remove the files in the foreground instead of finishing the work using a background thread.

sourceFileWid - Editor control containing a list of source files, one per line

Returns: int

0 on success, <0 on error.

Since:

16.0

Category:

Tagging Functions
