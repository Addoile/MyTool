_command void maybe_split_insert_line()

If the cursor is on the command line, the command is executed.

Otherwise if insert state is on, the current line is split at the cursor. Enough blanks are appended to the beginning of the new line to align it with the first non blank character of the original line.

If the insert state is off, the cursor is moved to column one of the next line.

Applies To:

Edit Window, Editor Control, Command Line

Categories:

Edit Window Methods, Editor Control Methods
