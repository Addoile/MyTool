int dbg_session_probe_breakpoint(int session_id, _str class_name, _str method_name, _str file_name, int line_number)

Probe if a breakpoint can be set on the given line and return the corrected line, file, class, and method that the breakpoint would be placed at.

Parameters:

session_id - debugger session ID (1 .. max_session)

class_name - name of class breakpoint is in

method_name - name of function breakpoint is in

file_name - name of file breakpoint is in

line_number - line number in file breakpoint is in

Returns: int

"fixed" line number >0 on success, <0 on error.

Since:

10.0

Example:

To probe setting a breakpoint in a function name "xmain" in a class named "Application": 

   int status = dbg_session_probe_breakpoint(session_id,
                                             "Application", "xmain",
                                             "cfront.c", 0);
   if (status) {
      return status;
   }
