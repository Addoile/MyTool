_command int ispf_find(_str arglist="")

This command finds occurrences of the given search string or ISPF regular expression in the current buffer. If an occurrence is found in an excluded line, the line will be unexcluded. Find takes the following common searching arguments (also used in the change, exclude, and locate commands):

Syntax:

   string [ range ] [NEXT | ALL | FIRST | LAST | PREV | CHARS | SUFFIX | WORD | X | NX | QUIET] [col1 [col2 ]]

string
The string or regular expression to search for.
line
range Two numbers or labels, specifying the starting and ending lines to search within, inclusive.
NEXT
Find the next occurrence starting from the cursor.
ALL
Find all matches in the specified range.
FIRST
Find only the first occurrence in the specified range.
LAST
Find only the last occurrence in the specified range.
PREV
Find the previous occurrence starting from the cursor.
CHARS
(default) Locate match anywhere.
PREFIX
Only find matches at the beginning of a word.
SUFFIX
Only find matches at the end of a word.
WORD
Only find matches for the entire word.
X
Look for matches only in excluded lines.
NX
Do not report matches in excluded lines.
col1
Starting column boundary to search within.
col2
Ending column boundary to search within.
QUIET
Do not display messages.

The ISPF find command supports ISPF regular expressions (known as picture strings). The following table shows the specific picture string characters supported, as well as how they may be expressed as SlickEdit Regular Expressions

=  ?          any character.
^  [^ \t]     non-blank character.
.  [0-9]      non-ASCII character (0x20..0x7E)
#  [0-9]      any single decimal digit.
-  [^0-9]     non-numeric character.
@  [a-zA-Z]   alphabetic character (case-insensitive).
<  [a-z]      lowercase alphabetic character.
>  [A-Z]      uppercase alphabetic character.
$  [~ \ta-zA-Z0-9]  punctuation (not alphabetic, digit, or blank).

Returns: int

Returns 0 if the search string specified is found. Common return codes are STRING_NOT_FOUND_RC, INVALID_OPTION_RC and INVALID_REGULAR_EXPRESSION_RC. On error, a message is displayed.

Category:

ISPF Primary Commands
