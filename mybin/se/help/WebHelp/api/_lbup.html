_str _lbup()

Places the cursor on the previous item in the list box. If the p_multi_select property is MS_NONE or MS_EXTENDED, selected lines are deselected. The first item in a list box is on line 1 (p_line==1). If there are non items in the list box, the current line will be line 0 (p_line==0). If you call the _lbup function when on line 1, the cursor will be placed on line 0. You might want to check if the list is empty (p_Noflines==0) or if you are on line 1 (p_line==1) before calling this function.

Returns: _str

Returns non-zero value call when on line 0 (p_line==0).

Applies To:

List Box

Category:

List Box Methods
