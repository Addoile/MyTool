void VSAPI vsGoToOldLineNumber(int wid, seSeekPos OldLineNum, int Reserved=a)

Searches for the OldLineNumber specified and places the cursor on the line with that old line number or the closest old line number. If no lines have an old line number set, then the cursor is placed on the real line (p_RLine) number given. The "real line number" does not count non-savable lines (that is lines with the VSLF_ NOSAVE flag). Use the vsSetAllOldLineNumbers function to set the old line numbers. Note that the Slick-C® save command takes an option to set the old line numbers on save.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

OldLineNumber - Indicates the old line number to find.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

vsSetAllOldLineNumbers
