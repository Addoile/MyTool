void _cbset_text(_str text, int pic_index=0)

Set the text in a combo box to the given text. If the given string is not in the combo box's list, add it to the list.

Parameters:

text - string to set caption to

pic_index - bitmap index for item if added

Applies To:

List Box Combo Box

Categories:

List Box Methods, Combo Box Methods

See Also:

p_text, _lbfind_item, _lbadd_item, _lbsearch
