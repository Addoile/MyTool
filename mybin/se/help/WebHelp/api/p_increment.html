int p_increment

Determines the amount the spin control should increment the text box. If p_increment is non-zero, there must be a text box with a tab index one less than the spin control. This allows the spin control to spin the previous window which can be located with the p_prev property. The value in the text box may be a floating pointer number (sorry, p_increment is only an int). If p_increment is 0, the spin control will call the on_spin_up and on_spin_down events.
