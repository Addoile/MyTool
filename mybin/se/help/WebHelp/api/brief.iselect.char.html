_command typeless brief_iselect_char()

Starts or deselects an inclusive character selection. Used for processing sentences of text which do not start and end on line boundaries. If no selection is active an inclusive character selection is started. if an inclusive character selection is already active, it is unmarked. If a different select style than inclusive character selection is active, the select style is switched to an inclusive character selection.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions

See Also:

brief_select_char, brief_select_line, brief_select_block
