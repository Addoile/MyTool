int VSAPI vsDeleteText(int wid, int DelLen, char option=a)

Parameters:

wid - Window id of editor control. 0 specifies the current object.

DelLen - Number of characters to delete. Specify -1 to delete to end of line. Specify -2 to delete to end of file. When DelLen is greater than the number of characters left in the file, the text up to the end of file is deleted.

option - Maybe 0 or 'C'. Specify 'C' to indicate that DelLen is a positive imaginary count (as if tabs were expanded) of characters to delete in the current line.

If option is zero, number of characters specified by DelLen starting from the cursor location are deleted. Each end of line character is treated as 1 character (as is).

If option is 'C', the number of columns (as if tabs were expanded) specified by DelLen are deleted from the current line. The end of line characters are not deleted.

Returns: int

Returns 0 if successful.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
