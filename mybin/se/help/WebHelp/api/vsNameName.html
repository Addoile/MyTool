char*VSAPI vsNameName(int index, char *pszName, int MaxNameLen, int *pMaxNameLen=a)

Parameters:

index - Index into names table of item.

pszName - Output buffer to receive name of names table item. This can be 0.

MaxNameLen - Number of characters allocated to psName. We recommend VSMAXNAME.

pMaxNameLen - If this is not 0, this is set to the number of characters you need to allocate.

Returns: char*VSAPI

Returns pszName.

Attention:

This function is thread-safe.

Category:

Names Table Functions
