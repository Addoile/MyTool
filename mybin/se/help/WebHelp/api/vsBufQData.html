void*VSAPI vsBufQData(int buf_id)

This function should only be used by OEM's to store some data per buffer and not by users writing macros.

Parameters:

buf_id - Buffer id returned by vsPropGetI(wid,VSP_BUF_ ID), vsBufEdit, or vsBufMatch.

Returns: void*VSAPI

Returns pointer to OEM buffer data set by the vsBufSetData function.

Category:

Buffer Functions
