int tag_get_async_tagging_result(_str &fileName, int &taggingFlags, int &bufferId, int &lastModified, int &updateFinished, _str &tagDatabase, int waitTimeInMS=0, int deferSlowerJobs=0)

Check the result of an asynchronous tagging operation, in order to report our progress as we create a tag file in the background, or update the current context or locals in the background, or update the tagging for other buffers in the background.

Parameters:

fileName - (output) set to name of file tagged

taggingFlags - (output) set to tagging flags (VSLTF_*) for operation

bufferId - (output) set to buffer Id of file which was tagged

lastModified - (output) set to the generation count of the file which was tagged

updateFinished - (output) set to true if the context, locals, statements, or tag database was already updated on the thread.

tagDatabase - (output) tag database being updated (can be empty string)

waitTimeInMS - (input) time to wait for result to appear on queue

deferSlowerJobs - (input) if 'true', defer slower jobs that that require foreground processing until later.

Returns: int

0 on success, <0 on error

Category:

Tagging Functions
