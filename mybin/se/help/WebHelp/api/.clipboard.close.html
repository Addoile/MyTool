void _clipboard_close(boolean isClipboard)

Places the newly created clipboard onto the system clipboard.

Note: _clipboard_close() must be called to free memory.

Parameters:

isClipboard - Indicates whether operation is performed for the clipboard or selection. Effects Unix only.

Category:

Clipboard Functions
