void VSAPI vsEBCDICToASCII(const char *pSrcBuf, char *pDestBuf, int SrcBufLen=-1)

Converts EBCDIC text to ASCII. pSrcBuf maybe the same as pDestBuf (OK if pSrcBuf==pDestBuf).

Parameters:

pSrcBuf - Buffer contain EBCDIC data to be converted.

pDestBuf - Destination buffer for ASCII.

SrcBufLen - Number of bytes in pSrcBuf.

Category:

Unicode Functions
