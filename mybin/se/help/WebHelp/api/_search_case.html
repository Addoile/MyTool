_str _search_case(_str option=null)

Gets and/or sets the default search case sensitivity to exact or ignore case and returns current case sensitivity. 'E' specifies case sensitive searching by default. 'I' specifies case insensitive searching by default. The built-in search command will default to this case unless the 'E' or 'I' is specified to override the default.
