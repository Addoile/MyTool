int dbg_session_disable_exception(int session_id, int exception_id)

Disable the given exception

Parameters:

session_id - debugger session ID (1 .. max_session)

exception_id - Exception to disable

Returns: int

0 on success, <0 on error.

Since:

10.0

Example:

   int status = dbg_session_disable_exception(session_id, 1);
   if (status) {
      return status;
   }

Category:

Debugger Functions
