int VSAPI vsMaybeExecuteKey(int wid, int event, int checkCtrlAlt)

Execute the command bound to the specified event (key) if one exists.

Parameters:

wid - Window id of window that should perform the command for the specified key

event - Key or key combination to use to find the command bound to that key. The values come from vsevents.h

checkCtrlAlt - Flag: 1 to also check for CONTROL and/or ALT modifier keys being pressed before the key is executed, 0 to ignore the modifier keys

Returns: int

0 if no command for the specified key or key sequence. -1 if command found

Category:

Keyboard Functions
