int _length()

Returns: int

Returns the number of elements in the given variable.

  • array -- the number of elements in the array
  • hash table -- the number of items in the hash table
  • struct or class instance -- the number of fields
  • string -- the length of the string

Example:

int a[];
a[0]=1;
a[1]=2;
messageNwait("Number of elements="a._length());
