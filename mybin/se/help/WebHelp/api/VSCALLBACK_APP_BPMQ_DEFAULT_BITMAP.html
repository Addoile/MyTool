19 VSCALLBACK_APP_BPMQ_DEFAULT_BITMAP

int (VSAPI *pfnQDefaultBitmap)(int wid,seSeekPos LineNum,seSeekPos RealLineNum,seSeekPos OldLineNum,int reserved);
The break point manager calls this function when there are no debug flags (break point bitmaps) on the current line. This callback is typically used to display a bitmap indicating whether a break point can be set on this line.

Parameters:

wid - Window id of editor control.

RealLineNum - Real line number (VSP_RLINE). This line number does not include lines with the VSLF_NOSAVE (non-savable line) flag set. The difference editor adds non-savable lines. OldLineNum Old line number (VSP_OLDLINENUMBER).

Returns: 19

Returns index into the names table of the picture or 0 to indicate no picture.

Category:

Editor Control Callback Functions
