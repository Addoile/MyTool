int VSAPI vsRefactorCPushDown(int handle, const char *filename, const char *className, seSeekPos startSeekPos, seSeekPos endSeekPos, const char *derivedClass, const char *membersToMove, char *memberWorkingOn, const char *classDefInfoList, const char *origClassDefFiles)

Push Down class member(s) to a derived class

Parameters:

handle - Handle to transaction

className - Name of class

filename - Name of file containing class

startSeekPos - Seek position of beginning of class reference

endSeekPos - Seek position of end of class reference

derivedClass - Name of derivedclass to move symbol to

membersToMove - Members of class to move to superclass

memberWorkingOn - (out)This is used for errors. This is the member being worked when error occurs

classDefInfoList - Classesand Files to move members to. Format: NumOfClasses @ className0 @ classFile0 @ classNameN @ classFileN @

origClassDefFiles - Format: NumOfFiles @ defFile0 @ defFileN @ membersToMove is in the form: number of members [@] member [@] member [@] ...

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
