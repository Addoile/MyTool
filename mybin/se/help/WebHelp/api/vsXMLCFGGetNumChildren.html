int VSAPI vsXMLCFGGetNumChildren(int iHandle, int NodeIndex, int NodeTypeFlags=~VSXMLCFG_NODE_ATTRIBUTE)

Parameters:

iHandle - Handle to an XMLCFG tree returned by vsXMLCFGOpen(), vsXMLCFGCreate(), or vsXMLCFGOpenFromBuffer().

NodeIndex - Tree node index.

NodeTypeFlags - Indicates node type of child to find. 1 or more VSXMLCFG_NODE_* flags. See _xmlcfg_add() for more information on this parameter.

Returns: int

Return the number of nodes under the specified index.

Category:

XMLCFG Functions
