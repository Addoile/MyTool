int tag_check_cached_locals(int start_seekpos, int end_seekpos, int context_id, int list_all_option)

Check if the list of symbols for the current file and position are already up-to-date. Switch to one of the cached local variable lists if necessary.

For synchronization, macros should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

start_seekpos - start seek position of function to search

end_seekpos - end seek position of function to search

context_id - context item to search for locals in

list_all_option - list all locals, or only in scope?

Returns: int

1 if the locals are up-to-date, 0 otherwise.

Category:

Tagging Functions

See Also:

tag_check_cached_context, tag_get_local
