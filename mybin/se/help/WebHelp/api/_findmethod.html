int _findmethod(_str methodName)

Parameters:

methodName - name of the method to look up

Returns: int

Return the names table index of the given class method.

Example:

class AAAA {
   void saySomething() {
      message("something");
   }
};
int testFindMethod(_str name) {
   AAAA a;
   return a._findmethod(name);
}

Category:

Miscellaneous Functions
