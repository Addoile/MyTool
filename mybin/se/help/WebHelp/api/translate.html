_str translate(_str string, _str output_table, _str input_table, _str pad="", int compliment=0, boolean binary=false)

Parameters:

string - Input string

output_table - table of characters to replace matches with

input_table - table of characters to look for

pad - pad character if length(output_table) < length(input_table)

compliment - Replace char with 'pad' character if it matches input table

binary - When true, input string is treated as an array of bytes (dbcs or utf8 characters are not special).

Returns: _str

Returns string with characters translated according to tables. Characters that are in the input_table are translated to the corresponding character in the output table. If the length of input_table is greater than the length of output_table, output_table is padded with the pad character given. Pad character defaults to ''. If output_table is not given, string is converted to upper case.

If 'compliment' is given and not zero, characters matching the input table will be preserved, and all *other* characters will be replaced with the pad character.

Example:

   translate('bottom_of_window','-','_')  =='bottom-of-window'
   translate('bottom_of_window','','_','*')  =='bottom*of*window'
   translate('bottom_of_window') =='BOTTOM_OF_WINDOW'
