int p_init_style

Determines the initialization style. The p_init_style property is one or more of the following flags:

IS_SAVE_XY
Indicates that the form x and y values should be saved in the ".command" buffer when the form is closed. The show function maybe given the -xy option to display the form in its previous position.
IS_REINIT
Indicates that when _delete_window is called to delete a form window, the form window should be made invisible instead of destroyed. The destroy events and all other side effects of the _delete_window function still take place.
IS_HIDEONDEL
Same as IS_REINIT.

Applies To:

Form

Category:

Form Properties
