void process_make_rc(_str error_code, _str module, boolean quiet=false)

Processes error codes during rebuild of the state file. error_code is either a negative number corresponding to one of the error constants in "rc.sh" or 1 which indicates error is from the Slick-C® translator trying to compile the file module_name. module_name is the name of the file during which the error occurred.

When a compilation error occurs and the st command has not been loaded, a message is displayed and the editor is exited. Otherwise, a message is displayed and the editor remains resident.
