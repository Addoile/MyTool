int VSAPI vsGetDefaultFont(int field, char *pszFontName, int MaxFontName, int *pFontSizex10, int *pFontFlags, int *pCharSet=a)

Returns information about the font configuration.

Parameters:

field - Field to change font for. One of the following constants defined in "vs.h":

VSCFG_CMDLINE
Font used by the command line.
VSCFG_WINDOW_TEXT
Font used by the editor control window text.
VSCFG_MESSAGE
Font used by the message line.
VSCFG_STATUS
Font used by the status line.

pszFontName - Output buffer for font name.

MaxFontName - Number of characters for font name.

pFontSizex10 - Point size of font x 10.

pFontFlags - One or more of the following flags:

  • VSFONTFLAG_BOLD
  • VSFONTFLAG_ITALIC
  • VSFONTFLAG_STRIKE_THRU
  • VSFONTFLAG_UNDERLINE

pCharSet - Indicates the character set of the font. These character sets map directly onto the Windows character sets. See VSCHARSET_??? constants defined in "vs.h". We do not expect this parameter to be used under UNIX.

Returns: int

Returns 0 if parameters are valid.

Categories:

Edit Window Functions, Editor Control Functions, Miscellaneous Functions

See Also:

vsSetDefaultFont, vsGetDefaultColor, vsSetDefaultColor
