int p_multi_select

Determines whether the list box allows multiple items to be selected and the user interface style. In addition, when the p_multi_select property is set to MS_EDIT_WINDOW, the list box becomes an editor control. The p_multi_select property may be one of the following constants defined in "slick.sh":

MS_NONE
Allow only one item to be selected.
MS_SIMPLE_LIST
Allow multiple items to be selected.
MS_EXTENDED
(List Box only)Allow multiple items to be selected.

Applies To:

List Box, Tree View

Categories:

List Box Properties, Tree View Properties
