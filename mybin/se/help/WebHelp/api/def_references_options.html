int def_references_options

Options for the symbol referencing system. This is a bitset of the following flags.

  • VSREF_FIND_INCREMENTAL -- Find references incrementally instead of searching all matching files right away.
  • VSREF_DO_NOT_GO_TO_FIRST -- Just find the references, do not navigate to the first reference when the references search is invoked.
  • VSREF_NO_WORKSPACE_REFS -- Do not build workspace tag files with cross-referencing indexes by default.
  • VSREF_HIGHLIGHT_MATCHES -- Highlight references matches in the editor.
  • VSREF_SEARCH_WORDS_ANYWAY -- Search for word matches if a symbol can not be found by the tagging system.
  • VSREF_ALLOW_MIXED_LANGUAGES -- Allow a references search to include files that are in different language modes.

Default:

0

Category:

Configuration Variables
