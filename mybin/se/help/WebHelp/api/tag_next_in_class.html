int tag_next_in_class()

Find the next tag in the given class. Tags are returned unsorted. Should be called only after calling tag_find_in_class. Use tag_get_info to extract the details about the tag.

Returns: int

0 on success, < 0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Category:

Tagging Functions
