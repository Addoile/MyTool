int tag_find_friends_to_tag(_str tag_name, _str class_name, _str(&tag_files)[], _str &friend_list)

Find all the classes which are friendly towards the given tag name or it's class or outer classes.

Parameters:

tag_name - tag to find friends to

class_name - class to find friends to

tag_files - List of tag files to search

friend_list - (reference) set to list of friends, separated by VS_TAGSEPARATOR_parents

Returns: int

0 on success, <0 on error. Having no friends is considered as a success.

Category:

Tagging Functions
