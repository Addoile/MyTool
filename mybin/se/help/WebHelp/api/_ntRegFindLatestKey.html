int _ntRegFindLatestKey(int RootKey, _str Path, _str &subkey, int requiredMajor=0)

Deprecated. Use _ntRegFindLatestVersion()

Finds the latest version among a set of subkeys in the registry.

Parameters:

RootKey - one of the HKEY_* constants

Path - path to key to check for version subkeys

subkey - name of latest version subkey

requiredMajor - required major version (must find a major version greater than this)

Returns: int

int 0 on success, non-zero on error

Category:

Miscellaneous Functions
