void VSAPI vsSetDefaultExtensionOptions(const char *pszLangId, const VS_LANGUAGE_OPTIONS *pLangOptions, int reserved=a, int version=VS_LANGUAGE_OPTIONS_VERSION)

Deprecated. Use vsSetDefaultLanguageOptions

Sets language specific options for a specific file extension.

Parameters:

pszExtension - File extension. For list of file extension types, use our File Extension Manager dialog ("Tools", "Options...", "File Extension Manager")

pLangOptions - New language specific options for pszExtension. Since all options are must be set, use the vsGetDefaultExtensionOptions first to query the existing value before setting new values.

reserved - unused

version - normally VS_LANGUAGE_OPTIONS_VERSION

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Functions, Editor Control Functions

See Also:

vsGetDefaultExtensionOptions, vsSetDefaultLanguageOptions, vsSetDefaultExtensionReferTo, vsDeleteExtensionOptions, vsDeleteLanguageOptions, vsUpdateExistingBufferExtensionOptions, vsUpdateExistingBufferLanguageOptions
