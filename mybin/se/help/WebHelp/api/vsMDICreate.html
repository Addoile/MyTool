int VSAPI vsMDICreate(VSSYSHWND hwndframe, VSSYSHWND hwnd, VSSYSHWND hwndmdiclient)

This function registers some MDI window handles. Currently, only one MDI window is supported. SlickEdit will automically disable the hwndframe window when a modal dialog box is displayed.

Parameters:

hwndframe - System window handle to outer most MDI frame window. For Windows, this is always the same as hwnd.

hwnd - System window handle MDI frame client window. The hwndmdiclient window is inside this window.

hwndmdiclient - System window handle of MDI client window. In a Windows MDI, this is the parent window for all MDI child windows.

Returns: int

Returns MDI window instance handle.

See our MDI sample program for an example.
