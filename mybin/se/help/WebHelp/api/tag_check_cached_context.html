int tag_check_cached_context(int list_tags_flags)

Check if the list of symbols for the current file are already up-to-date. Switch to one of the cached file lists if necessary.

For synchronization, macros should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

list_tags_flags - (optional) statement level tagging option

Returns: int

1 if the context is up-to-date, 0 otherwise.

Category:

Tagging Functions
