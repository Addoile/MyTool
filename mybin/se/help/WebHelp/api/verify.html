_str verify(_str string, _str reference, _str option="", int start=1)

If option =='', this function returns the first character position in string which is not one of the characters in reference. If string is only composed of characters from reference, then 0 is returned.

If option == 'M' , this function returns the first character position in string which matches one of the characters in reference. If no characters in reference match a character in string, 0 is returned.

If start is specified, the searching begins at start.

Example:

verify("test.*","*?",'M')  ==6
verify("?*st.*","*?",'M',3)   ==6
verify("?*st.*","*?")   ==3
