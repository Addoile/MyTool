int _TreeCompare(int parent1Index, int parent2Index, _str reservedOptions="")

Compares children of and . These must be in the same tree control. Does not recurse

Parameters:

int - parent1Index index of first sub-tree to compare

int - parent2Index index of second sub-tree to compare

_str - reservedOptions

Returns: int

int 0 if sub-trees match. Otherwise returns the strcmp output from comparing the first item that didn't match

Applies To:

Tree View

Category:

Tree View Methods
