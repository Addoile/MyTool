_command typeless cut_prev_word(boolean doDelete=false, int subwordOption=0)

Deletes from the cursor to the beginning of the previous word and copies it to the clipboard. Invoking this command from the keyboard multiple times in succession creates one clipboard. The deleted text may be retrieved by the command paste.

Applies To:

Edit Window, Editor Control, Text Box, Combo Box

Categories:

Clipboard Functions, Combo Box Methods, Edit Window Methods, Editor Control Methods, Text Box Methods
