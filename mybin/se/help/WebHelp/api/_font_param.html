_str _font_param(_str font_name, _str font_size, _str font_flags, _str rgb_color="")

This function is used to assist in creating a font string argument used as input to several Slick-C® functions. The rgb_color argument is ignored by some Slick-C® functions that don't support it.

Returns: _str

Returns a string in the format: font_name, font_size, font_flags,
