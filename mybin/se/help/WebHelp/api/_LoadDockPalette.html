int _LoadDockPalette(int index, vsDockingArea area=DOCKINGAREA_TOP)

Loads a form as dock palette onto an MDI frame docking area. index corresponds to a form in the names table returned by find_index.

Parameters:

index - Index of form to load as dock palette.

area - One of the following:

  • DOCKINGAREA_LEFT
  • DOCKINGAREA_TOP
  • DOCKINGAREA_RIGHT
  • DOCKINGAREA_BOTTOM

Applies To:

MDI Window

Category:

MDI Window Methods

_GetDockPalette
