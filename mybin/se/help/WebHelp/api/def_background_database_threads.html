int def_background_database_threads

Specifies whether an additional tagging thread is dedicated to writing to the tag database. This improves performance by pipelining parsing and writing to the tag database. In addition, it eliminaates contention for the tag database. This option has no effect if def_background_tagging_threads == 0. This setting requires you to restart the editor for a change to take effect.

Default:

1

Category:

Configuration Variables
