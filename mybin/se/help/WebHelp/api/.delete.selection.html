int _delete_selection(_str mark_id="")

Deletes the marked text specified. No clipboard is created.

Parameters:

mark_id - is a handle to a selection returned by one of the built-ins _alloc_selection or _duplicate_selection. A mark_id of '' or no mark_id parameter identifies the active selection. This function performs a "binary" delete when in hex mode (p_hex_mode==true). A binary delete allows bisecting of end of line pairs like CR,LF.

Returns: int

Returns 0 if successful. Otherwise TEXT_NOT_SELECTED_RC is returned. On error, message is displayed.

Category:

Selection Functions
