int tag_find_refer_to(int inst_id, int &ref_type, _str &file_name, int &line_no)

Find the first tag instance referenced by the given tag instance. The tag is identified by its unique ID, see tag_match_instance(). This is typically used with functions (caller/callee relationship) or structures (container/item relationships).

Parameters:

inst_id - Unique ID of tag instance.

ref_type - (Output) reference type (see VS_REFTYPE_*)

file_name - (Output) Full path of file the tag is located in

line_no - (Output) Line number of tag within file

Returns: int

Returns instance ID (> 0) of tag referenced in context represented by inst_id or < 0 on error, BT_RECORD_NOT_FOUND_RC if not found.

Category:

Tagging Functions

See Also:

tag_get_instance_info, tag_match_instance, tag_next_refer_to
