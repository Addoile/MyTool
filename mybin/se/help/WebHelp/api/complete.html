void complete(...)

Completes the current command argument being entered on the command line according to the command's completion info or completion_info specified. completion_info must be zero or more completion constants delimited with space characters. For example, complete(FILE_ARG' 'FILE_ARG) would specify that the command takes two file name arguments. Completion constants have the suffix "_ARG" and are listed in "slick.sh".
