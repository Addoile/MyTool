int dbg_get_frame_path(int thread_id, int frame_id, _str &file_name, int &line_number)

Get the details about file and line number for the the given stack frame.

Parameters:

thread_id - index of thread to get stack frame from

frame_id - index of frame in stack frame list

file_name - (reference) name of file that stack frame is positioned at

line_number - (reference) line number that stack frame is positioned at

Returns: int

0 on success, <0 on error

Category:

Debugger Functions
