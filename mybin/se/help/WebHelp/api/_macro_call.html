void _macro_call(_str proc_name, ...)

void _macro_call( _str proc_name [,_str arg1 [,_str arg2 .. ]] )

If macro recording and output is on (_macro()!=0), the call to function proc_name with arguments arg1, arg2.. is appended to the end of the macro being recorded.

Parameters:

proc_name - Slick-C® procedure to call

arg1 - (optional) first argument

arg2 - (optional) second argument

... - subsequent arguments

Example:

   _macro_call('find','Find this', 'ri');
   // The above is identical to
   _macro_append("find('Find this', 'ri')");
   // and is also identical to
   _macro_append( "find(" _quote('Find this') ',' _quote('ri') ")" );

Category:

Macro Programming Functions
