int _end_select_compare(_str markid="")

Compares current line position to last line of mark specified.

Parameters:

markid - is a handle to a selection returned by one of the built-ins _alloc_selection or _duplicate_selection. A mark_id of '' or no mark_id parameter identifies the active selection.

Returns: int

0
Current line is on last line of mark
> 0
Current line is after last line of mark
< 0
Current line is before last line of mark
-1
Text is not marked or mark is not in current buffer

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions

See Also:

_begin_select_compare
