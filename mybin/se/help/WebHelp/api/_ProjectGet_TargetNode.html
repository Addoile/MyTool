int _ProjectGet_TargetNode(int handle, _str Target, _str config=GetCurrentConfigName())

Locates the target node with the given name. A target is a project-specific build tool.

Parameters:

handle - handle to project file

Target - target name

config - configuration to search. To use the current configuration, do not send this argument. To find a target in any configuration, send an empty string.

Returns: int

int target node, -1 if not found

Category:

Project Functions
