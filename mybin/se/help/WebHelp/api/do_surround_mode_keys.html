boolean do_surround_mode_keys(boolean force=false, NotificationFeature callingFeature=-1)

Process keys handled by surround mode. Immediately exit from surround mode if any unrecognized key or mouse event happens.

While in surround mode, cursor-down will move the line after the last line of the block down one line and indent it within the block. Likewise, cursor-up will move the last line in the block outside and unindent it to it's original state.

Parameters:

force - (default=false) force surround mode even if disabled

Category:

Editor Control Methods
