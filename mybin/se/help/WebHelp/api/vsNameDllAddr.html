void*VSAPI vsNameDllAddr(int index)

Parameters:

index - Index into names table of global function.

Returns: void*VSAPI

Returns address of function. 0 is returned if index is not valid or can not be resolved.

Attention:

This function is thread-safe.

Category:

Names Table Functions
