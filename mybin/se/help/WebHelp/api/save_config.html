_command int save_config(_str save_immediate='')

Updates source code representing your configuration changes. Source code representing your configuration changes is saved in the files "vusrdefs.e" (UNIX: "vunxdefs.e"), "vusrobjs.e" (UNIX: "vunxobjs.e"), and "vusrsNNN.e" (UNIX: "vunxsNNN.e"). The state file will be saved to "vslick.sta" (UNIX: "vslick.stu") if you loaded new macro modules. When the editor is invoked, the state file is loaded and then the source code representing your configuration changes are applied.

Returns: int

Returns 0 if successful. Common return codes are: COMMAND_CANCELLED_RC, ACCESS_DENIED_RC, ERROR_OPENING_FILE_RC, INSUFFICIENT_DISK_SPACE_RC, ERROR_READING_FILE_RC, ERROR_WRITING_FILE_RC, DRIVE_NOT_READY_RC, and PATH_NOT_FOUND_RC. On error, message box displayed.

Category:

Miscellaneous Functions
