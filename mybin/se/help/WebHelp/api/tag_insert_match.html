int tag_insert_match(_str tag_file, _str tag_name, _str type_name, _str file_name, int line_no, _str class_name, int tag_flags, _str signature)

Add a search match tag and its information to the matches list.

For synchronization, macros should perform a tag_lock_matches(true) prior to invoking this function.

Parameters:

tag_file - (optional) full path of tag file match came from

tag_name - tag string

type_name - Tag type name. See tag_get_type for a list of all standard tag types.

file_name - full path of file the tag is located in

line_no - start line number of tag within file

class_name - (optional) name of class that tag is present in, use concatenation (as defined by language rules) to specify names of inner classes.

tag_flags - (Optional) Tag flags indicating symbol attributes, see tag_insert_simple for details.

signature - (optional) tag signature (return type, arguments, etc)

Returns: int

sequence number (match_id) of matching tag on success, or <0 on error.

Category:

Tagging Functions
