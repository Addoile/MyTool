int VSAPI vsCRefactorCExtractMethodFinish(int handle, const char *filename, const char *newName, int createMethodCall, const char *paramInfo, const char *commentInfo, int flags, int syntaxIndent)

Finish the extract method operation

Parameters:

handle - Handle to transaction

filename - Filename to do refactoring within

newName - Name for new function

createMethodCall - should the refactoring replace the selected code with a method call?

paramInfo - List of parameters for new function

commentInfo - Comments to insert for the new function

flags - Optional flags, bitset of VSREFACTOR_FORMAT_* (above)

syntaxIndent - Syntax indent amount

Returns: int

0 on success, <0 on error

Category:

Refactoring Functions
