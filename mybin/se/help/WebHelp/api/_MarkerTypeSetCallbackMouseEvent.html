int _MarkerTypeSetCallbackMouseEvent(int type, _str event, typeless pfnCallback)

Parameters:

type - Index of marker type.

event - Mouse event constant. Only the constants LBUTTON_DOWN, LBUTTON_DOUBLE_CLICK, RBUTTON_DOWN are supported.

pfnCallback - Callback. Either index into names table of function or pointer to Slick-C® function. int (* pfnCallback)(int wid,int LineNum, int LineMarkerIndex,int BMIndex,int Type, const char *pszMessage, void *pUserData);

Returns: int

Returns 0 if successful. Sets the callback for the specified mouse event.

Applies To:

Edit Window, Editor Control

Category:

Marker Functions
