_command void fileman_mode()

The fileman mode keys are activated after executing the fileman_mode, dir, list, or fileman commands. The keys are redefined as indicated below

Alt+Shift+A
Select all files.
Alt+Shift+B
Backup selected files.
Alt+Shift+C
Copy selected files.
Alt+Shift+D
Delete selected files.
Alt+Shift+E
Edit selected files.
Alt+Shift+G
Search and Replace on selected files.
Alt+Shift+M
Move select files.
Alt+Shift+N
Keyin current line filename.
Alt+Shift+O
File Sort dialog box.
Alt+Shift+R
Repeat Command on Selected dialog box.
Alt+Shift+T
Fileman Set Attributes dialog box.
ENTER
Edit current file or insert directory files.
Space bar
Select current file toggle.
'8'
Select and cursor up.
'2'
Select and cursor down.
'9'
Deselect and cursor up.
'3'
Deselect and cursor down.
RButtonDown
Displays file manager pop-up menu.
Selected directories are skipped by Backup, Move, Delete, Edit, and Set file attribute commands.
