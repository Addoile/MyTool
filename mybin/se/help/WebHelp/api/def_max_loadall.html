_str def_max_loadall

This option controls whether a file is loaded entirely into memory or if it should be partially loaded by block-by-block. The format of this variable is: 

   [on/off]  [ksize]
where [on/off] is 0 or 1, and [ksize] is the number of kilobytes for the load partial threshold.

Default:

'1 8000'

Category:

Configuration Variables
