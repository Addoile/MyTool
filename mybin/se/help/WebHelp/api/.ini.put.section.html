int _ini_put_section(_str filename, _str section_name, int temp_view_id)

Replaces all field_name=value lines in the section specified with the buffer text contained in temp_view_id.

Returns: int

zero if successful 

     ".ini" File Syntax:
         [section1]
         field_name1=value1
         field_name2=value2
         ...
         [section2]
         field_name1=value1
         field_name2=value2
         ...
