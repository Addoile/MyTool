_str tags_filename(_str lang="", boolean includeSlickC=true)

Return a list of tag files associated with the given (optional) extension. If the extension given is the empty string, and the current object is an editor control, get the extension from the p_LangId property. If the current object is not an editor control, retrieve all tag files and project tag files, reguarless of extension relationship.

Note: The new version of this function tags_filenamea is used more often, and is slightly more efficient because it returns its results as an array.

Parameters:

ext - If not '', the workspace tag file and all tag files for the extension specified by ext. If this is '', all extension specific tag files are returned except optionally the tag files for Slick-C®.

includeSlickC - Specifies whether the tag files for Slick-C® should be returned. This option has no effect if ext!=''.

Returns: _str

string containing FILESEP separted tag file paths.

Category:

Tagging Functions
