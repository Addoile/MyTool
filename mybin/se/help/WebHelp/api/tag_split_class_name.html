void tag_split_class_name(_str class_name, var inner_name, var outer_name, boolean outer_first_only=false)

Decompose a class name into its outer component and inner class name only. This is strictly a string function, no file I/O or searching is involved.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Parameters:

class_name - Class name to split. See VS_TAGSEPARATOR_class and VS_TAGSEPARATOR_package for more details.

inner_name - (Output) 'inner' class name (class name only)

outer_name - (Output) 'outer' class name (class_name - inner_name)

outer_first_only - When true, set outer_name to just the name of the outer class rather than the qualified class name.

Category:

Tagging Functions
