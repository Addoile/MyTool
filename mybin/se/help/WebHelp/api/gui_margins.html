_command typeless gui_margins()

This command sets the left, right, and new paragraph margins for the current buffer. For convenience, we decided to turn word wrap on when the margins are set. Each buffer has one set of margins. The Margins dialog box is displayed to prompt you for the new margins. To set up different initial margins for specific file extensions, use the Extension Options dialog box ("Tools", Configuration...", "File Extension Setup...", select the Word Wrap tab).

Applies To:

Edit Window

Category:

Miscellaneous Functions
