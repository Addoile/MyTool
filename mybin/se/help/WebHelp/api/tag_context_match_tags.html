int tag_context_match_tags(var errorArgs, _str &tag_name, boolean exact_match, boolean case_sensitive, boolean find_parents, int &num_matches, int max_matches, ...)

The current object must be an editor control positioned on the tag that you wish to find matches for, with the edit mode selected.

For synchronization, macros should perform a tag_lock_matches(true) prior to invoking this function.

Parameters:

errorArgs - Array of strings for return code

tag_name - Name of occurrence to match against

exact_match - Exact match or prefix match?

case_sensitive - Case sensitive tag search?

find_parents - Find instances of this tag in parent classes

num_matches - (output) number of matches found

max_matches - maximum number of matches to find

visited - (optional) hash table of prior results

depth - (optional) depth of recursive search

Returns: int

0 on success, <0 on error, >0 on Context Tagging® error.

Category:

Tagging Functions
