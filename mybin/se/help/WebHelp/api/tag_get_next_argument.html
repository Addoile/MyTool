int tag_get_next_argument(_str params, int &arg_pos, _str &argument, _str ext=null)

Disect the next argument (starting at 'arg_pos') from the given parameter list (params). Place the result in 'argument'. This is designed to handle both C/C++ style, comma separated argument lists, and Pascal/Ada semicolon separated argument lists.

If you do not know the number of bytes to allocate to 'argument', call this with 'argument' as NULL, and calculate ('arg_pos' - 'orig_pos' + 1).

Parameters:

params - complete parameter list to parse

arg_pos - current position in argument list use 0 to find first argument

argument - [output] set to the next argument in the argument list

ext - current file extension

Returns: int

Returns the position where the argument begins in 'params'. Note, this may differ from the original value of 'arg_pos' if there are leading spaces. Returns STRING_NOT_FOUND_RC and sets 'argument' to the empty string if there are no more arguments in the parameter list.

Category:

Tagging Functions
