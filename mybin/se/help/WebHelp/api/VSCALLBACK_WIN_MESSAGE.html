4 VSCALLBACK_WIN_MESSAGE

void (VSAPI *pfnMessage)(int wid,const char *pszMsg,int Immediate);
The MESSAGE callback function is called to display unobtrusive informational messages like popup menu messages, incremental search messages, and for many of the editor commands.

Parameters:

wid - Window id of editor control.

pszMsg - String to display.

Immediate - Indicates whether the screen should be updated immediately or if it is OK if the screen gets update later in a paint message.

Category:

Editor Control Callback Functions
