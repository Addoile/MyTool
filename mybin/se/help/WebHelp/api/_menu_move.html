int _menu_move(int src_menu_index, int src_menu_pos, int dest_menu_index, int dest_menu_pos)

Moves menu or menu item from source position to destination position. src_menu_index and dest_menu_index do not have to be the same menu. src_menu_index and dest_menu_index are returned by find_index, name_match, p_child, p_next, or p_child. This function currently only supports menu resources and not loaded menus.
