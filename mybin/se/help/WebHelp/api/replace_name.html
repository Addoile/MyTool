int replace_name(int index, _str name, _str info="", int copy_index=0)

Replaces information for a name in the names table. Valid types are listed in "slick.sh" and have the suffix "_TYPE". Internal name types may not be inserted except for PICTURE_TYPE and EVENTTAB_TYPE. If the type given is PICTURE_TYPE, name should be a filename.

Returns: int

If successful, the index of the name table entry is returned. Otherwise a negative error code is returned.

Category:

Names Table Functions
