int dbg_set_breakpoint_location(int breakpoint_id, _str file_name, int line_number, _str address)

Modify the file, line, and address location information for a breakpoint.

Parameters:

breakpoint_id - Breakpoint to enable

file_name - file breakpoint is set in

line_number - line number to set breakpoint at

address - breakpoint address

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
