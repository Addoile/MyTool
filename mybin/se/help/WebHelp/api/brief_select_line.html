_command typeless brief_select_line()

Starts or deselects a line selection. Used for processing complete lines of text. If no selection is active a line selection is started. if a line selection is already active, it is unmarked. If a different select style than a line selection is active, the select style is switched to a line selection.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods, Selection Functions

See Also:

brief_select_char, brief_select_line, brief_select_block
