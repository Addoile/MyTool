_str absolute(_str filename, _str toDir=null)

Returns filename with exact path specification as if the current directory was toDir.

Parameters:

filename - The filename to build the path for.

toDir - The directory to use as the current working directory. If null, the current directory is used.

Returns: _str

A string containing the absolute path to the filename.

Example:

/* Assuming current working directory is 'c:\vslick' */

absolute('..\autoexec.bat')      :== 'c:\autoexec.bat'
absolute('vs.exe')               :== 'c:\vslick\vs.exe'
absolute('vs.exe','d:\vslick\')  :== 'd:\vslick\vs.exe'
