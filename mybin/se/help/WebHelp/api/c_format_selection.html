_command int c_format_selection(boolean use_form_options=false, boolean quiet=false)

Beautifies the current selection using the current options. If there is no current selection the entire buffer is beautified. Use the C/C++/Java/JavaScript/C# Beautifier dialog box set beautifier options used by this command.

Applies To:

Edit Window, Editor Control

Category:

Edit Window Methods

See Also:

gui_beautify, c_beautify
