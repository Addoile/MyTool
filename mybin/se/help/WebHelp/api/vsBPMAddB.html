int VSAPI vsBPMAddB(const char *pszBufName, seSeekPos LineNum, int flags, void *puserdata=a)

Deprecated. Use vsLineMarker* functions Adds a break point. You can use this function to set a break point in a file that has not yet been loaded (deferred break point). When the file is loaded, the break point will be displayed.

Parameters:

pszBufName - Name of file or buffer.

RealLineNum - Real line number (VSP_RLINE).

flags - One of the flags below:

  • VSBPFLAG_BREAKPOINT
  • VSBPFLAG_EXEC
  • VSBPFLAG_STACKEXEC
  • VSBPFLAG_BREAKPOINTDISABLED

puserdata - Optional pointer to user data. Use this to store debugger specific information.

Returns: int

Returns an index to a break point which is valid input to vsBPMGet().

Category:

Deprecated Breakpoint Functions
