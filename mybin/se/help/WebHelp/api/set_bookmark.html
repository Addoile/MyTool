_command typeless set_bookmark(_str arg1='', boolean quiet=false)

This function is used to set a bookmark on the current line.

Parameters:

arg1 - command line options string of the form: 

option bookmark_name
where option is:
  • -n -- Use new smaller dialog to prompt for just bookmark name
  • -q -- do not prompt for bookmark name
  • -r -- set bookmark without prompting
  • -m -- do not set duplicate mark
and bookmark_name is the name of the bookmark to create.

Applies To:

Edit Window

Category:

Bookmark Functions

See Also:

push_bookmark, pop_bookmark, toggle_bookmark, goto_bookmark
