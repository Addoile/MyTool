_str letter_prompt(_str msg, _str letters='')

Remark Displays msg given on message line and waits for a character key contained in the letters argument, or the cancel key to be pressed. The letters defaults to 'YN' if not specified.

Returns: _str

Returns key pressed which caused return. If it is one of the letter keys, it is returned in upper case.

Category:

Keyboard Functions
