int VSAPI vsHvarCallMethodByName(int wid, VSHVAR hvar, VSPSZ func_name, int Nofargs, VSARGTYPE *pArgList)

Call a class method with the given argument list.

Parameters:

wid - window ID

hvar - handle for interpreter variable

func_name - name of class function to call

Nofargs - number of arguments

pArgList - array of arguments

Returns: int

0 on success. Returns INVALID_ARGUMENT_RC if the method index is invalid. Returns INVALID_HVAR_ARGUMENT_RC if 'hvar' is not a class instance.

Category:

Miscellaneous Functions

See Also:

vsHvarFindMethod, vsHvarCallMethod
