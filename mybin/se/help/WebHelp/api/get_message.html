_str get_message(int rc=MAXINT, ...)

Returns message corresponding to error number error_code. If error_code does not exists, "Message not available" is returned. If the error_code argument is not given or is MAXINT, the message currently displayed on the SlickEdit message line is returned. The error codes are listed in the file "rc.sh".

Parameters:

rc - error status to look up Also can pass in the following special arguments

  • -age -- return the age of the current message on the message line
  • -sticky -- return 'true' if the current message on the message line is sticky
  • -sticky_age -- return the age of the current message, even if it is a sticky message

s0 - string to replace %s0 with

s1 - string to replace %s1 with

sN - string to replace %sN with

Returns: _str

Message string for the given error code with arguments collated.
