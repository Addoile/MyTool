_str tag_join_class_name(_str class_name, _str outer_class, _str(&tag_files)[], boolean case_sensitive, boolean allow_anonymous=false)

Determine whether the class name and outer class name should be joined using a package separator or class separator and return the resulting string. This involves searching for the outer class name as a package, if found, then use the package separator VS_TAGSEPARATOR_package, otherwise use the class separator VS_TAGSEPARATOR_class. The current object must be an editor control or current buffer.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Parameters:

class_name - name of class to join (inner_class)

outer_class - name of outer class or package to join

tag_files - (reference, read-only array of strings) list of tag files to search

case_sensitive - Specifies search case sensitivity.

allow_anonymous - Allow class_name to be an anonymous class.

Returns: _str

Returns static string if outer_class :: class_name was found in the context or tag database, otherwise returns '';
