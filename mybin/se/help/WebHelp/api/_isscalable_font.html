int _isscalable_font(_str font_name, _str options="S")

Parameters:

options - is a string of zero or more of the following option letters:

F
Display fixed fonts only.
P
Display printer fonts.
S
(Default) Display screen fonts.

The S and P options may not both be specified.

Returns: int

Returns non-zero value if the font_name given is a scalable font.

Miscellaneous Functions
