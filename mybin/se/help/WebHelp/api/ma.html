_command void ma(_str cmdline="")

Parameters:

cmdline - is a string in the format left [right [new-paragraph]]

The ma or margins command sets the left, right, and new paragraph margins. Each buffer has one set of margins. To set up different initial margins for specific file extensions, use the Extension Options dialog box ("Tools", Configuration...", "File Extension Setup...", select the Word Wrap tab).

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
