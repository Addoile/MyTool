_command typeless upcase_word()

Translates the current word to upper case and places the cursor after the current word.

Returns: typeless

Returns 0 if successful. Returns 1 if no word exists at cursor. On error, message is displayed.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods

See Also:

lowcase_word, cap_word, upcase, upcase_selection
