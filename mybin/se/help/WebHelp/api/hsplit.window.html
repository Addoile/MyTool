_command typeless hsplit_window(_str load_options="")

Splits the current window horizontally in half. Optionally load buffer into split window.

Parameters:

load_options - Option to load buffer into split window. See load_files.

Returns: typeless

Returns 0 if successful. Common return codes are TOO_MANY_WINDOWS_RC and TOO_MANY_SELECTIONS_RC. On error, message is displayed.

Applies To:

Edit Window

Category:

Window Functions
