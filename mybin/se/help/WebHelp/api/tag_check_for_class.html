int tag_check_for_class(_str tag_name, _str outer_class, boolean case_sensitive, _str(&tag_files)[])

Look up tag_name and see if it is a class or struct name. If so, return the signature of the template in template_sig. The current object must be an editor control or current buffer.

For synchronization, threads should perform a tag_lock_context(false) prior to invoking this function.

Parameters:

tag_name - Tag name to check if it is a template class.

outer_class - Name of class that tag belongs to.

case_sensitive - Speciifies case_sensitive comparison

tag_files - (Input) Tag files to search

Returns: int

1 if a match was found, 0 of none found.

Category:

Tagging Functions
