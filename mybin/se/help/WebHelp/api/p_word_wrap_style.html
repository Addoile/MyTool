int p_word_wrap_style

Determines the word wrap style for the current buffer. The p_word_wrap_style is a combination of the following flags defined in "slick.sh":

STRIP_SPACES_WWS
Determines whether the paragraph reformatting functions such as reflow_paragraph reformat spaces between words. This flag should not be used with the JUSTIFY_WWS flag.

WORD_WRAP_WWS
Determines whether word wrap is on. When word wrap is on text is word wrapped as it moves past the right margin.

JUSTIFY_WWS
Determines whether the paragraph reformatting functions such as reflow_paragraph left and right justify the text. This flag should not be used with the STRIP_SPACES_WWS flag.

ONE_SPACE_WWS
Determines whether one or two spaces are placed after punctuation characters.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Properties, Editor Control Properties
