int dbg_translate_capability(_str capability)

Convert the named debugger capability to an int in the capability enumeration type.

Parameters:

capability - name of capability to look up

  • initialize -- initialize debugger connection
  • finalize -- clean up debugger connection
  • terminate -- stop debugging
  • detach -- detach from debugging session
  • restart -- restart from start of program
  • do_command -- perform debugger command
  • error_message -- get last error message
  • print_output -- print output to vsdebugio
  • print_info -- send info to vsdebugio
  • print_error -- send error message to vsdebugio
  • send_direct_input -- send input to direct to debugee
  • version -- get version of debugger
  • test_version -- test version of debugger
  • reload -- reload module and continue
  • get_paths -- get debugger paths
  • resolve_path -- resolve source file path
  • suspend -- suspend execution
  • resume -- resume execution
  • interrupt -- send interrupt to thread
  • step_into -- step into function
  • step_over -- step over statement
  • step_out -- step out of function
  • step_into_runtimes -- step into runtime function
  • step_instr -- step by instruction
  • set_instruction_pointer -- move instruction pointer
  • disable_breakpoint -- disable breakpoint
  • enable_breakpoint -- enable breakpoint
  • probe_breakpoint -- test breakpoint
  • disable_watchpoint -- disable watchpoint
  • enable_watchpoint -- enable watchpoint
  • disable_exception -- disable exception
  • enable_exception -- enable exception
  • handle_event -- poll for asynchronous event
  • update_threads -- update thread list
  • update_thread_groups -- update thread group tree
  • update_thread_names -- update thread names
  • update_thread_states -- update thread states
  • update_stack -- update stack
  • update_locals -- update local variables
  • expand_local -- expand local variable
  • modify_local -- modify local variable
  • update_members -- update class members variables
  • expand_member -- expand class member variable
  • modify_member -- modify class member variable
  • update_watches -- update user defined watches
  • expand_watch -- expand watch variable
  • modify_watch -- modify watch variable
  • update_auto_watches -- update auto watch variables
  • expand_auto_watch -- expand auto watch variable
  • modify_auto_watch -- modify auto watch variable
  • eval_condition -- evaluate condition
  • eval_expression -- evaluate expression
  • update_classes -- update classes
  • expand_class -- expand class members
  • expand_methods -- expand class methods
  • expand_fields -- expand class fields
  • expand_nested -- expand inner classes
  • expand_parents -- expand class parents
  • modify_field -- modify static class field
  • update_registers -- update registers
  • update_memory -- update memory contents

Returns: int

DEBUG_CAPABILITY_NOT_FOUND if no such capability exists, otherwise, it returns the debugger capability index.

Since:

10.0

Example:

To look up the capability ID for initialize 

   int cap = dbg_translate_capability("initialize");
   return cap;

Category:

Debugger Functions
