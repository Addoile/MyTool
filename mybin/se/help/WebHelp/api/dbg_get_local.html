int dbg_get_local(int thread_id, int frame_id, _str local_path, _str &name, _str &class_name, _str &signature, int &flags, _str &value, int &line_number, int &is_in_scope, _str &raw_value)

Get the details about a local variable.

Parameters:

thread_id - index of thread

frame_id - index of stack frame

local_path - path to local in list of local variables

name - (reference) name of local

class_name - (reference) name of class that we are in

signature - (reference) type of local variable

flags - (reference) set to bitset of DEBUG_MODEL_FLAGS_

value - (reference) value of variable

line_number - (reference) line number that it is declared at

is_in_scope - (reference) is the variable in scope?

Returns: int

0 on success, <0 on error.

Category:

Debugger Functions
