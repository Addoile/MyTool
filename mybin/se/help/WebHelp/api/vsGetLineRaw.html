int VSAPI vsGetLineRaw(int wid, char *pszBuf, int BufLen)

This function is identical to the vsGetLine function, except that the output string is in the same format as the internal buffer data which can be SBCS/DBCS or UTF-8. See "Unicode and SBCS/DBCS C API Programming".

Applies To:

Edit Window, Editor Control

Edit Window Methods, Editor Control Methods
