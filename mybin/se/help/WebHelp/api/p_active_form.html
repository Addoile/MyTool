int p_active_form

Returns the window id of the active form. You can not use the name (p_name) of the form to access properties of the active form. You must use the p_active_form property. This property is not available when the form is not displayed.

Example:

   defeventtab form1;
   command1.lbutton_up()
   {
      // Close the active dialog box.
      p_active_form._delete_window();
   }

Applies To:

All Window Objects

Category:

All Windows Properties
