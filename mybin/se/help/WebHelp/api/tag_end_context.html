int tag_end_context(int context_id, int end_line_no, int end_seekpos)

Set the end positions of the context with the given context ID.

For synchronization, macros should perform a tag_lock_context(true) prior to invoking this function.

Parameters:

context_id - Context ID [ from 1 to tag_get_num_of_context() ] of context tag to set end position of.

end_line_no - ending line number of tag within file

end_seekpos - Line number where the tag ends. Technically, this is the seek position of the last character in the tag. Using a C function for example, this is the seek position of the close brace ending the function body.

Returns: int

0 on success, <0 on error.

Category:

Tagging Functions
