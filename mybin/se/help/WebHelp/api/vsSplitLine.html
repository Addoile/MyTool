int VSAPI vsSplitLine(int wid, int col=a)

Splits the current line at the cursor position. More specifically inserts p_newline character string at cursor. If the cursor is past the end of the line, the p_newline character string is inserted after the last character of the line but before the line termination characters.

Parameters:

wid - Window id of editor control. 0 specifies the current object.

Applies To:

Edit Window, Editor Control

Categories:

Edit Window Methods, Editor Control Methods
