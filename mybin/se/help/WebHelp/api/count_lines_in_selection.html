int count_lines_in_selection(_str markid='', boolean count_nosave_lines=false)

Counts the number of lines or sub-lines within the marked area identified by mark_id. mark_id is a handle to a selection returned by one of the built-ins _alloc_selection or _duplicate_selection. A mark_id of '' or no mark_id parameter identifies the active selection.

Returns: int

If successful, returns number of lines or sub-lines within the selection. Otherwise, 0 is returned.

Applies To:

Edit Window, Editor Control

Category:

Selection Functions
