int getCodehelpFlags(_str langID, int defaultValue=null)

Gets the codehelp flags for this language. The options for this value are available on the GUI on the Context Tagging form for each applicable language.

Parameters:

langID - language ID (see p_LangId)

defaultValue - if the value is not available, then we just return this value. If this parameter is not specified, then the default value for the overall def-var is used.

Returns: int

codehelp flags for this language

Category:

LanguageSettings API
