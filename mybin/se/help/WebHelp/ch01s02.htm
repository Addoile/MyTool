New Features and Enhancements

This section describes the new features and enhancements in SlickEdit®.

New MDI Architecture

The MDI child window layout has been completely rewritten to provide many new features and more modern look. MDI windows can be arranged in multiple vertical and horizontal tab groups. Individual files can be zoomed to encompass the entire MDI area, then unzoomed to restore the view of all file tabs and groups.

New Beautifiers

Java and C# are now supported in the new beautifier framework. New options have been added to support beautification for JavaDoc, Doxygen, and XMLDoc comments. See C/C++ Beautifier for more information.

Android Project Support

Added support for NDK projects, including NDK debugging. Context Tagging support has been added for symbols located in XML resource files. See Working with Android Projects.

Large File Support

Reading and saving files greater than 2GB is now supported. In theory the maximum limit is 1TB, but available system memory and hard drive speed are now the limiting factors. Most of our testing has been with files in the 2GB to 8GB range.

Language Support

Support for CoffeeScript, Markdown, and Google Go has been added. TCL and CSS language support has been greatly improved with new parsers. Initial 'color coding only' support has been added for Matlab and TTCN-3.

Version Control Repository Browser

A new repository history log browser is now available for Subversion and Mercurial. When you have a project or file open that is using SVN or Hg, invoke the svc-gui-browse-repository command, or use the Tools → Version Control → Browse repository... menu command. This will display history for the entire repository, initially displaying all log entries from the past week.

Enhanced special character display

Unicode files now have a separate configuration section to specify how special character sequences are displayed. Multi-character substitutions are supported.

Reload with Encoding

This new menu item on the File → Reload with Encoding... menu enables you to quickly reload an open file with a new encoding.
